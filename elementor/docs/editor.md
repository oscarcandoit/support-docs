# [\#](https://developers.elementor.com/docs/editor/\#elementor-editor) Elementor Editor

Elementor CoreBasic

The editor is the environment in which users create pages and control site settings. Editor is a general term that includes two main areas - the preview and the panel. Each of these areas contains inner areas which can be extended by 3rd party developers.

## [\#](https://developers.elementor.com/docs/editor/\#structure) Structure

The editor refers to the entire editing screen and consists of two main areas:

- [Elementor Preview](https://developers.elementor.com/docs/editor/elementor-preview/)
- [Elementor Panel](https://developers.elementor.com/docs/editor/elementor-panel/)

## [\#](https://developers.elementor.com/docs/editor/\#editor-panels) Editor Panels

The panel area displays various controls and setting. The controls and functionality displayed depend on what specific panel is being displayed:

- [Menu Panel](https://developers.elementor.com/docs/editor/menu-panel/)
- [Site Settings Panel](https://developers.elementor.com/docs/editor/site-settings-panel/)
- [User Preferences Panel](https://developers.elementor.com/docs/editor/user-preferences-panel/)
- [Page Settings Panel](https://developers.elementor.com/docs/editor/page-settings-panel/)
- [History Panel](https://developers.elementor.com/docs/editor/history-panel/)
- [Widget Panel](https://developers.elementor.com/docs/editor/widgets-panel/)
