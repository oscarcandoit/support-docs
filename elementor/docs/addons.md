# [\#](https://developers.elementor.com/docs/addons/\#building-addons) Building Addons

Elementor CoreBasic

Elementor addons are basically WordPress plugins that add functionality to Elementor. Before you start creating Elementor addons, you need a background in creating WordPress plugins. Afterwards, you need to learn the basics of how Elementor addons are built.

## [\#](https://developers.elementor.com/docs/addons/\#wordpress-plugins) WordPress Plugins

[WordPress(opens new window)](https://wordpress.org/) has a very extendable code base. It lets anyone extend core functionality by using plugins. For in-depth information check out the [Plugin Handbook(opens new window)](https://developer.wordpress.org/plugins/), which is part of the [WordPress Developer Resources(opens new window)](https://developer.wordpress.org/).

## [\#](https://developers.elementor.com/docs/addons/\#elementor-addon) Elementor Addon

When creating Elementor addons, you should implement all WordPress coding standards and best practices. In addition, you should follow the Elementor standards and best practices as outlined on this site.

To start creating an Elementor addon, you first need to build a simple WordPress plugin focused on Elementor, using the following information:

- [File & Folder Structure](https://developers.elementor.com/docs/addons/file-folder-structure/)
- [Header Comments](https://developers.elementor.com/docs/addons/plugin-header/)
- [Loading Process](https://developers.elementor.com/docs/addons/load/)
- [Main Class](https://developers.elementor.com/docs/addons/main-class/)
- [Compatibility Checks](https://developers.elementor.com/docs/addons/compatibility/)
- [Initialization Process](https://developers.elementor.com/docs/addons/initialization/)
- [Using Namespaces](https://developers.elementor.com/docs/addons/namespaces/)
- [Addon Example](https://developers.elementor.com/docs/addons/addon-example/)

## [\#](https://developers.elementor.com/docs/addons/\#what-s-next) What's Next?

When you finish building a wrapper, you can start extending Elementor components. Check out the following tutorials for details and examples of how you can extend individual components:

- [Create new widgets](https://developers.elementor.com/docs/widgets/)
- [Create new controls](https://developers.elementor.com/docs/controls/)
- [Create new dynamic tags](https://developers.elementor.com/docs/dynamic-tags/)
- [Create new form actions](https://developers.elementor.com/docs/form-actions/)
- [Create new form fields](https://developers.elementor.com/docs/form-fields/)
- [Create new finder categories](https://developers.elementor.com/docs/finder/)
- [Extend the context menu](https://developers.elementor.com/docs/context-menu/)

Of course these are just some of the ways you can extend Elementor.
