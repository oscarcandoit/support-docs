# [\#](https://developers.elementor.com/docs/cli/system-info/\#system-info) System Info

Elementor CoreAdvanced

System info shows details about the website configuration. This data can be used to debug issues, helping developers and support engineers identify problems. The Elementor CLI `system info` command will print the site's system info in a JSON format.

## [\#](https://developers.elementor.com/docs/cli/system-info/\#command) Command

```
wp elementor system-info

```

1

## [\#](https://developers.elementor.com/docs/cli/system-info/\#arguments) Arguments

None.

## [\#](https://developers.elementor.com/docs/cli/system-info/\#usage-examples) Usage Examples

```
wp elementor system-info

```

1

This command will display the entire system info.
