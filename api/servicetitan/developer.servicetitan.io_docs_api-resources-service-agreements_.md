# Service Agreements

A service agreement is the way to create and manage customized maintenance and service agreements for commercial customers.

Service Agreement — A service agreement includes the customer, total agreement price, estimated labor and material costs, estimated profit margins, start date and duration, planned visit schedule, covered locations and equipment, billing schedule, and other terms and conditions that have been agreed upon with the customer.

Service Agreement Template — A service agreement template includes accounting instructions for how revenue from the agreement is handled, and optional default values that are applied to any agreements created from the template.

High level use cases for these APIs:

- Get a list of all active agreements

- Get a list of all canceled agreements

- Get the details of a service agreement
