# Getting access

If you need to [Login to the My Apps](https://developer.servicetitan.io/signin) section of the developer portal to add new and manage apps, or generate an application key you need to be a user of your organization’s integration or production environment.

**ServiceTitan Customers**

ServiceTitan customers have access to both Integration and Production environments. Production environment access is provided by default.

**Requesting Integration Environment Access:**

To request Integration environment access, the ServiceTitan admin must email [integrations@servicetitan.com](https://developer.servicetitan.io/docs/get-going-getting-access/#). ServiceTitan will perform authentication steps and provide the Integration environment details.

**Integration Environment Features**

- The Integration environment contains the latest copy of your Production data. You can request periodic data refreshes for the Integration environment by emailing [integrations@servicetitan.com](https://developer.servicetitan.io/docs/get-going-getting-access/#).

- Each organization is provided with one instance of the Integration environment. Within this instance, you can create an unlimited number of users.


**Third-Party Developers**

Developers can access **only the Integration environment** provided by ServiceTitan.

- **Requesting Access:**

  - Visit the "Request Access" tab and fill out the form.

  - The ServiceTitan team will:

    - Verify your information.

    - Contact you if necessary.

    - Respond with your Integration environment details and credentials.
- **Integration Environment Features:**

  - Comes with standard sample demo data to help you get started.

  - Each organization is provided one Integration environment instance.

  - You can create an unlimited number of users within this instance.
- **Third-Party Developer Guidelines:**

  - Do not ask customers to create users in their Integration or Production environments to develop or create an app.

  - As a third-party developer, you must use only the Integration environment instance provided to you, per the API Terms of Service.
