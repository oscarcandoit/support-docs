# FB page messaging setup:

1\. Leadconnector should have access to send message and access message for the page ( [https://www.facebook.com/settings?tab=business\_tools&ref=settings](https://www.facebook.com/settings?tab=business_tools&ref=settings))

2\. User account should have full access to page and should have message access. ( Link: [https://business.facebook.com/latest/settings](https://business.facebook.com/latest/settings))

# Instagram Messaging setup:

1\. Leadconnector should have access to send message and access message for the page ( [https://www.facebook.com/settings?tab=business\_tools&ref=settings](https://www.facebook.com/settings?tab=business_tools&ref=settings))

# ![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155046611201/original/560dNU3lKrXB3ADHgcxA9uUfY8O2B6oMvw.png?1747220054)

2\. Verify that you have access to send and receive message on instagram under instagram accounts ( [https://business.facebook.com/latest/settings](https://business.facebook.com/latest/settings))

![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155046820921/original/tLINkNzirrxoXfTSJdGxULjH4cx8k9f3Ug.png?1747628845)

3\. Verify instagram account is present in connected assets list in the page connected.

![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155046820913/original/rThbgLELyFW3QsFfS1WTuVteuNKAyl2-ag.png?1747628832)

4\. Instagram messaging is enabled in Settings >> Integration >> Facebook card >> Settings

![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155046820968/original/rYafkZPj6wQIXvvuNGOOqY4P2wQWSO_S0Q.png?1747629000)

5\. Also, confirm if connected tools have access over instagram messages. The below setting needs to be checked over instagram app:

- Login to Instagram App
- Go to settings and activity.
- Under Messages and Story reply, go to messages request.
- Allow access to connected tools
