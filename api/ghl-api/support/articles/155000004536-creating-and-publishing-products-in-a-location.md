## Overview

This guide walks you through the process of creating a product in your location. By following these steps, you can add and manage products seamlessly.

## Steps to Create a Product

### **Step 1: Navigate to the Product Section**

1. Navigate to **Payments** \> **Products**.
2. Click on **Add Product** to create a new product.



![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155042397131/original/_Pr2tNILV7Sc-nzlyS-86hBRMvN-D5yFJg.png?1740718460)

### **Step 2: Enter Product Information**  ![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155042397639/original/jAfuLtZjro7kzuRiJhe89bycJ5b_medO7w.png?1740719506)

1. **"Include in Online Store" toggle**\- Enable the toggle if you want to publish the product in your Ecommerce store.
2. **Product Name** – Enter a unique and descriptive name for your product.
3. **Description** – Provide a detailed product description. You can format it using rich text options.
4. **Media** \- Upload images in the order you want it to be displayed
5. **Enable Product Label** _**(for Ecommerce Stores)**_ – Product label can be added on the products in stores. Store owners can set the time period visibility as well for the labels, post which labels will get removed.
6. **Collections** – Assign the product to one or more collections for better organization in stores. Users can search and assign the product to multiple collections. If no collection exists, then new collections can be created and products can be added by clicking "+" button alongside the search bar.




[![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155042694834/original/Bu5rVe0jb686xVPUegyiou8S6BxA6ox-AQ.png?1741176462)](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155042694834/original/Bu5rVe0jb686xVPUegyiou8S6BxA6ox-AQ.png?1741176462)

7. **Product Taxes -** Users can select tax code for automatic tax calculation by choosing from the dropdown.
1. For Include tax in prices:
      1. As per global settings - If Global settings chosen, then the global default tax settings will be applied for whether to include or exclude the taxes in product price.
      2. Yes - Tax will be included in the purchase price which gets displayed.
      3. No - Tax will not be included in product prices before checkout. At the final step taxes will be added separately.
2. Attach tax rates - manual tax rates can be attached with the products. This will be added if enabled along with automatic taxes.





      ![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155042695045/original/ASM4B_3PKCpjYGrFBaYcOadKJUj_hMsNAg.png?1741176598)

### **Step 3: Set Pricing**

### [![Price creation image](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155042696029/original/Aoi5teSDR1hVmAjsLjbWR54-0ASQaoKI0A.png?1741177180)](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155042696029/original/Aoi5teSDR1hVmAjsLjbWR54-0ASQaoKI0A.png?1741177180)

1. **Pricing Name – Enter the price name that you want.**
2. **Pricing Type** – It can be either Onetime or Recurring. On choosing Onetime, a fix one time amount will be set for the product. For Recurring price, Billing period (time duration can be selected as per needed), Trial period, Number of payments and Setup Fee needs to be defined.

![Recurring price type image](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155042696615/original/hcDuF0uWVpK-ZPaKxiV3JJw82U5NWtAfxQ.png?1741177558)

3. **Compare-at Price** (Optional) – Set a higher price to show discounts.
4. **Track Inventory**– Enable if you want to track sock levels automatically.
1. Define the Available Quantity when track inventory is enabled
5. **Continue Selling when out of stock**\- If enabled, product will be available for purchase by customers irrespective of available quantity.
6. **Additional Option** \- Price description can be defined and a membership offer can be enabled and attached.

### **Step 4: Variant configuration**![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155042703826/original/a_9eM5hKkZj3M_UKflaZUJKwH96rHlPArg.png?1741181325)

1. **Option name**\- Enable Variants if the product has multiple options (e.g., size, color).
2. **Option values**\-  Define Variant values (e.g., "Color" with values like "Red, Blue, Green").
3. **Price** – Set a higher price that will be shown at checkout.
4. **Compare-at Price** (Optional) – Set a higher price to show discounts.
