## LC WordPress Plugin \| Fixing Cron Jobs/Crashing Issue  [Print](https://help.leadconnectorhq.com/support/solutions/articles/155000004673-lc-wordpress-plugin-fixing-cron-jobs-crashing-issue\# "Print this Article")

Modified on: Mon, 24 Feb, 2025 at 2:40 AM

* * *

For Resolving the LC Plugin creating too many cron jobs/crashing your website please follow the steps provided below.

**Step 1 -**

Deactivate the LC plugin by going to Plugins -> Installed Plugins. _Skip if already in Inactive state._

![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155042057299/original/3rjzw4zQMymtRZTB65J4-lVWM-5LXdCqbA.png?1740207689)

**Step 2 -**

Download the leadconnector-cron-fix plugin by clicking [**here**](https://2ly.link/24UnG)

( MD5 Checksum - 5a80c8666c97f834392a0465726c219a )

**Step 3 -**

Install the new plugin by going to Plugins -> Add new plugin -> Upload plugin.

![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155042056236/original/EBXLYvhn0ZLkqQZPubZXqVulZkrMSsjs4Q.png?1740203382)

**Step 4 -**

Next Click on Activate Plugin.

![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155042056328/original/ORRvhofyWnfFf_IhBC_BTFBwOVSKHqhGRQ.png?1740203505)

**Step 5 -**

After activating the plugin Go to Tools -> LC Cron Fixer

If you see this screen you are good to activate the main LC Plugin now

![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155042057167/original/ISdT5rL5KKX3WdzXT3QmQ0Px3vxT7lWxcg.png?1740207101)

or if you see this screen you can click on the "fix cron jobs" button.

![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155042057184/original/bMfQb3760iADRVCzJxEeBOMz4MK2bl6ZlQ.png?1740207198)

**if incase you do not see the first screen after multiple attempts please reach out to LeadConnector Support we will help you resolve the issue further.**

**Step 6 -**

Go Back to Plugins -> List Plugins and click on Activate against LC Plugin. ( If incase your plugin is not on the latest version please do update the same aswell.

![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155042057277/original/6q8NM7WEeN_WC1f6pkj01BVxGMeHJ_bqFQ.png?1740207504)

**Step 7 -**

Lastly Deactivate and Delete the Lead Connector Cron Fix Plugin and you are all set!
