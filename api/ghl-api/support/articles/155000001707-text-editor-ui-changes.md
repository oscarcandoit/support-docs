## Text Editor UI Changes  [Print](https://help.leadconnectorhq.com/support/solutions/articles/155000001707-text-editor-ui-changes\# "Print this Article")

Modified on: Thu, 18 Jan, 2024 at 7:44 AM

* * *

We are thrilled to announce some exciting UI changes in for the Text Editor. This document will take you through all the functionalities of the new editor.

- When "Custom Values" icon is clicked it will populate all the options from where you can select the custom values for the template.

![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155018081860/original/_-ion_5Tqy9p2wpcJaK4iory5em6PKjQCw.png?1705584105)

- By clicking on "Trigger Links" you can select from the links available in the dropdown to add in the template.


![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155018081974/original/GbGDsrSej2mOFgIcbTZIJT1MCplwsux0ZA.png?1705584177)

- Below mentioned functions are highlighted in the next screenshot



- Undo
- Redo
- Bold
- Italic
- Underline
- Font Type

![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155018082296/original/7xfq25nWDBo4ZYeAXCxI-ZdlN4bJWacWwg.png?1705584256)

- Below mentioned functions are highlighted in the next screenshot
  - Font Size

  - Paragraph Format

  - Line Height

  - Text Color

  - Background Colour

  - Disc List

  - Decimal List

  - Text Align

![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155018082461/original/BqwBAPM_k-B545TW7yct2F5Fq17ASoYRXg.png?1705584327)

- On clicking the more options icon (3 dots) you will be able to select from the below mentioned actions



- Strike
- Super Script
- Sub Script
- Code
- Code Block
- Block Quote
- Format Selected
- Format all document

![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155018082564/original/RyHkiUY3vjtOyjl5x6KQ6R9mjBPw4mUrug.png?1705584369)

- When the "Add Link" icon is clicked a new window will open. In this pop up window you can paste the URL or select a trigger link by clicking on the icon in the right. Select the "Action: from Self or New Window and click on Save.

![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155018082674/original/TVOX2qbKxRXO1gX4Vcw2erGgHrKz7AKjwA.png?1705584405)

![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155018082725/original/c1WX5AZHi1KKy1dwP6oLcqJJdRRsmnU2Dg.png?1705584415)

- When the "Image" icon is clicked a new window will open. In this pop up window you can
  - Add URL of your Image
  - Upload the Image by clicking on the "Upload" button
  - Set the Width and Height
  - Enter an Alternative Text
  - Enter a link you want to open when user clicks on the Image
  - Open Link In - This field is only editable if "Image Action Link" is selected. You can select the link should open in same or different window

![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155018082803/original/1Gd2FRMFmr5kwxl4CUJWb0xuNzg88lpZJQ.png?1705584469)

- When the "Source Code" icon is clicked a new window will open. In this pop up window you can see the Source Code for the template and edit the same.

![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155018082868/original/zEt6wKVurPgxPVmS41vPZw2R4WhVRKuYHQ.png?1705584510)

![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155018082882/original/xcy2NXmc-HPjwbzUqEQfs7_SYOUD-O6Big.png?1705584519)
