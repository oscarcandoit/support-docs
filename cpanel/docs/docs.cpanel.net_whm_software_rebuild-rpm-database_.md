## Overview

Use WHM’s _Rebuild RPM Database_ interface ( _WHM » Home » Software » Rebuild RPM Database_) to rebuild the package manager database if it becomes corrupt.

Note:

You will not see this interface if your cPanel & WHM server is built on an Ubuntu® OS.

## Rebuild RPM database

To begin the rebuild process, click _Proceed_.
