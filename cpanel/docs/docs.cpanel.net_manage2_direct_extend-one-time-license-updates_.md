## Extend One Time License Updates

Last modified: _2020 April 13_

* * *

## Overview

This feature allows partners to extend the software updates for their one time licenses.

Note:

We do **not** currently offer new one time licenses.

## Extend One Time License Updates

This feature allows users with one time licenses to order one-year extensions for software updates.

To order a one-year extension, perform the following steps:

1. In the _Server IP Address box_, enter the IP address of the server for which you wish to extend updates.
2. Click _Go_.
3. Review the update extension’s details.






Note:





When you extend updates for a server, Manage2 bills your account a non-refundable fee. Make **certain** that you review the amount before you continue.

4. Click _Extend Updates_.
