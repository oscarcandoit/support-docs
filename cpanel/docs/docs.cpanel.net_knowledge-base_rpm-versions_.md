## The rpm.versions System

* * *

#### [Introduction to the rpm.versions System](https://docs.cpanel.net/knowledge-base/rpm-versions/introduction-to-the-rpm-versions-system/)

The rpm.versions system manages many of the packages on your cPanel & WHM server.

[Read more](https://docs.cpanel.net/knowledge-base/rpm-versions/introduction-to-the-rpm-versions-system/)

* * *

#### [How to Override the rpm.versions System](https://docs.cpanel.net/knowledge-base/rpm-versions/how-to-override-the-rpm-versions-system/)

System administrators and third-party developers can modify cPanel-provided package management.

[Read more](https://docs.cpanel.net/knowledge-base/rpm-versions/how-to-override-the-rpm-versions-system/)

* * *

#### [Target Settings](https://docs.cpanel.net/knowledge-base/rpm-versions/target-settings/)

Target settings represent the group of packages that must exist on the server for a specific service to function properly.

[Read more](https://docs.cpanel.net/knowledge-base/rpm-versions/target-settings/)

* * *

#### [The check\_cpanel\_pkgs Script](https://docs.cpanel.net/knowledge-base/rpm-versions/the-check_cpanel_pkgs-script/)

This script scans every cPanel-managed package on your server for problems.

[Read more](https://docs.cpanel.net/knowledge-base/rpm-versions/the-check_cpanel_pkgs-script/)

* * *

#### [The rpm.versions File](https://docs.cpanel.net/knowledge-base/rpm-versions/the-rpm-versions-file/)

This file contains all of the cPanel-provided settings for the rpm.versions system.

[Read more](https://docs.cpanel.net/knowledge-base/rpm-versions/the-rpm-versions-file/)

* * *

#### [The update\_local\_rpm\_versions Script](https://docs.cpanel.net/knowledge-base/rpm-versions/the-update_local_rpm_versions-script/)

Use this script to update the /var/cpanel/rpm.versions.d/local.versions file.

[Read more](https://docs.cpanel.net/knowledge-base/rpm-versions/the-update_local_rpm_versions-script/)
