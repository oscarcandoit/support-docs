# transfers

* * *

#### [Transfer Tool](https://docs.cpanel.net/whm/transfers/transfer-tool/)

This interface copies one or more cPanel accounts, packages, and configurations from a source server to your destination cPanel & WHM server.

[Read more](https://docs.cpanel.net/whm/transfers/transfer-tool/)

* * *

#### [Transfer Tool — Prepare The Source Server](https://docs.cpanel.net/whm/transfers/transfer-tool/prepare-a-source-server/)

To use WHM's Transfer Tool feature, you must prepare the source server to transfer the correct data.

[Read more](https://docs.cpanel.net/whm/transfers/transfer-tool/prepare-a-source-server/)

* * *

#### [Transfer Tool — Select What to Transfer](https://docs.cpanel.net/whm/transfers/transfer-tool/select-what-to-transfer/)

Once you have prepared a source server in WHM's \_Transfer Tool\_ interface (\_WHM >> Home >> Transfers >> Transfer Tool\_), this section will appear to allow you to select which configurations, packages, and accounts you want to transfer to the destination server.

[Read more](https://docs.cpanel.net/whm/transfers/transfer-tool/select-what-to-transfer/)

* * *

#### [Transfer Tool — The Transfer Process](https://docs.cpanel.net/whm/transfers/transfer-tool/the-transfer-process/)

When you click \_Copy\_ in the Transfer Tool feature, the following transfer process begins.

[Read more](https://docs.cpanel.net/whm/transfers/transfer-tool/the-transfer-process/)

* * *

#### [Transfer Tool — After The Transfer](https://docs.cpanel.net/whm/transfers/transfer-tool/after-the-transfer/)

After you transfer a cPanel account with the Transfer Tool, you must restore some configurations manually.

[Read more](https://docs.cpanel.net/whm/transfers/transfer-tool/after-the-transfer/)

* * *

#### [Transfer Tool — Only Transfer IP Address](https://docs.cpanel.net/whm/transfers/transfer-tool/only-transfer-ip-address/)

If you only want to transfer an IP address from one server to another, use the \_IP Migration Wizard\_ interface.

[Read more](https://docs.cpanel.net/whm/transfers/transfer-tool/only-transfer-ip-address/)

* * *

#### [Account Restore](https://docs.cpanel.net/whm/transfers/account-restore/)

The Account Restore interface displays the status and progress of your account restoration, and any warnings.

[Read more](https://docs.cpanel.net/whm/transfers/account-restore/)

* * *

#### [Account Transfer](https://docs.cpanel.net/whm/transfers/account-transfer/)

The Account Transfer interface displays the status and progress of your account restoration, and any warnings.

[Read more](https://docs.cpanel.net/whm/transfers/account-transfer/)

This document describes how to use SSH keys to securely copy an account from one server (source server) to another server (destination server).

[Read more](https://docs.cpanel.net/knowledge-base/transfers-and-restores/how-to-copy-an-account-with-ssh-keys/)

* * *

#### [How to Manually Transfer an Account Between Servers](https://docs.cpanel.net/knowledge-base/transfers-and-restores/how-to-manually-transfer-an-account-between-servers/)

When accounts are too large to move with the WHM Transfers features, you can move the accounts manually.

[Read more](https://docs.cpanel.net/knowledge-base/transfers-and-restores/how-to-manually-transfer-an-account-between-servers/)

#### [How to Migrate a WordPress® Installation to a cPanel & WHM Server](https://docs.cpanel.net/knowledge-base/transfers-and-restores/how-to-migrate-a-wordpress-installation-to-a-cpanel-server/)
