## Service Configuration

* * *

#### [Apache Configuration](https://docs.cpanel.net/whm/service-configuration/apache-configuration/)

This collection of features allows you to configure Apache.

[Read more](https://docs.cpanel.net/whm/service-configuration/apache-configuration/)

* * *

#### [cPanel Log Rotation Configuration](https://docs.cpanel.net/whm/service-configuration/cpanel-log-rotation-configuration/)

This interface allows you to select cPanel & WHM log files for the cpanellogd daemon to compress and archive.

[Read more](https://docs.cpanel.net/whm/service-configuration/cpanel-log-rotation-configuration/)

* * *

#### [cPanel Web Disk Configuration](https://docs.cpanel.net/whm/service-configuration/cpanel-web-disk-configuration/)

This interface allows you to edit the TLS/SSL cipher and protocol lists for Web Disk.

[Read more](https://docs.cpanel.net/whm/service-configuration/cpanel-web-disk-configuration/)

* * *

#### [cPanel Web Services Configuration](https://docs.cpanel.net/whm/service-configuration/cpanel-web-services-configuration/)

This interface allows you to edit the TLS/SSL Cipher List and TLS/SSL Protocol list for cPanel, WHM, and Webmail.

[Read more](https://docs.cpanel.net/whm/service-configuration/cpanel-web-services-configuration/)

* * *

#### [DirectoryIndex Priority](https://docs.cpanel.net/whm/service-configuration/directoryindex-priority/)

The DirectoryIndex Priority interface allows you to specify the filenames that Apache recognizes and displays as index pages.

[Read more](https://docs.cpanel.net/whm/service-configuration/directoryindex-priority/)

* * *

#### [Exim Configuration Manager](https://docs.cpanel.net/whm/service-configuration/exim-configuration-manager/)

Use the features in the Exim Configuration Manager interface to modify your Exim configuration.

[Read more](https://docs.cpanel.net/whm/service-configuration/exim-configuration-manager/)

* * *

#### [Exim Configuration Manager Basic Editor](https://docs.cpanel.net/whm/service-configuration/exim-configuration-manager-basic-editor/)

Select the Basic Editor tab in the Exim Configuration Manager interface to modify your server's Exim configuration settings.

[Read more](https://docs.cpanel.net/whm/service-configuration/exim-configuration-manager-basic-editor/)

* * *

#### [FTP Server Configuration](https://docs.cpanel.net/whm/service-configuration/ftp-server-configuration/)

The FTP Server Configuration interface allows you to customize your chosen FTP server's behavior.

[Read more](https://docs.cpanel.net/whm/service-configuration/ftp-server-configuration/)

* * *

#### [FTP Server Selection](https://docs.cpanel.net/whm/service-configuration/ftp-server-selection/)

This interface allows you to select an FTP server daemon, or to disable FTP services.

[Read more](https://docs.cpanel.net/whm/service-configuration/ftp-server-selection/)

* * *

#### [Global Configuration](https://docs.cpanel.net/whm/service-configuration/global-configuration/)

This feature allows you to adjust several of the Apache® web server's advanced features.

[Read more](https://docs.cpanel.net/whm/service-configuration/global-configuration/)

* * *

#### [Include Editor](https://docs.cpanel.net/whm/service-configuration/include-editor/)

This interface allows you to add other configuration files to your Apache configuration file (httpd.conf) through Apache's Include directive.

[Read more](https://docs.cpanel.net/whm/service-configuration/include-editor/)

* * *

#### [Log Rotation](https://docs.cpanel.net/whm/service-configuration/log-rotation/)

This interface allows you to select Apache log files for the cpanellogd daemon to compress and archive.

[Read more](https://docs.cpanel.net/whm/service-configuration/log-rotation/)

* * *

#### [Mailserver Configuration](https://docs.cpanel.net/whm/service-configuration/mailserver-configuration/)

This interface allows you to configure the POP3 and IMAP protocols that the Dovecot mail servers use.

[Read more](https://docs.cpanel.net/whm/service-configuration/mailserver-configuration/)

* * *

#### [Manage Service SSL Certificates](https://docs.cpanel.net/whm/service-configuration/manage-service-ssl-certificates/)

This interface allows you to manage certificates for your server's services.

[Read more](https://docs.cpanel.net/whm/service-configuration/manage-service-ssl-certificates/)

* * *

#### [Memory Usage Restrictions](https://docs.cpanel.net/whm/service-configuration/memory-usage-restrictions/)

This interface calculates and sets a new memory limit for Apache processes.

[Read more](https://docs.cpanel.net/whm/service-configuration/memory-usage-restrictions/)

* * *

#### [Nameserver Selection](https://docs.cpanel.net/whm/service-configuration/nameserver-selection/)

This interface allows you to select your server's nameserver software or to disable DNS services.

[Read more](https://docs.cpanel.net/whm/service-configuration/nameserver-selection/)

* * *

#### [Piped Log Configuration](https://docs.cpanel.net/whm/service-configuration/piped-log-configuration/)

This interface configures Apache to use a single target for all virtual host access and bandwidth logs.

[Read more](https://docs.cpanel.net/whm/service-configuration/piped-log-configuration/)

* * *

#### [Reserved IP Address Editor](https://docs.cpanel.net/whm/service-configuration/reserved-ip-address-editor/)

The Reserved IP Address Editor interface allows you to configure Apache to ignore HTTP requests on certain IP addresses.

[Read more](https://docs.cpanel.net/whm/service-configuration/reserved-ip-address-editor/)

* * *

#### [Service Manager](https://docs.cpanel.net/whm/service-configuration/service-manager/)

This interface lists your server's services and it allows you to enable and monitor them.

[Read more](https://docs.cpanel.net/whm/service-configuration/service-manager/)
