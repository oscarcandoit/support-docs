## Overview

phpPgAdmin is a third-party tool that you can use to manipulate PostgreSQL® databases. For example, you can use phpPgAdmin to add or delete records in a PostgreSQL database. For more information about how to use phpPgAdmin, visit the [phpPgAdmin website](http://phppgadmin.sourceforge.net/).
