## Locales

* * *

#### [Copy a Locale](https://docs.cpanel.net/whm/locales/copy-a-locale/)

This interface allows you to duplicate a locale in to a standard locale or a non-standard locale.

[Read more](https://docs.cpanel.net/whm/locales/copy-a-locale/)

* * *

#### [Delete a Locale](https://docs.cpanel.net/whm/locales/delete-a-locale/)

This interface allows you to delete a locale.

[Read more](https://docs.cpanel.net/whm/locales/delete-a-locale/)

* * *

#### [Edit a Locale](https://docs.cpanel.net/whm/locales/edit-a-locale/)

Use this interface to edit the locales on your server.

[Read more](https://docs.cpanel.net/whm/locales/edit-a-locale/)

* * *

#### [Locale XML Download](https://docs.cpanel.net/whm/locales/locale-xml-download/)

This interface allows you download an existing locale (language) in XML format.

[Read more](https://docs.cpanel.net/whm/locales/locale-xml-download/)

* * *

#### [Locale XML Upload](https://docs.cpanel.net/whm/locales/locale-xml-upload/)

This feature allows you to upload a locale in XML format to your server.

[Read more](https://docs.cpanel.net/whm/locales/locale-xml-upload/)

* * *

#### [View Available Locales](https://docs.cpanel.net/whm/locales/view-available-locales/)

This interface allows you to view available locales, cross-referenced against the themes for which they are available.

[Read more](https://docs.cpanel.net/whm/locales/view-available-locales/)
