## Important Notices

* * *

#### [CVE 2017 1000369 Exim Stack Clash](https://docs.cpanel.net/knowledge-base/important-notices/cve-2017-1000369-exim-stack-clash/)

On Monday, June 19, 2017, Qualsys announced memory handling vulnerabilities in a number of software distributions, including a vulnerability that could leverage a bug in the Exim software to achieve a local privilege escalation to root.

[Read more](https://docs.cpanel.net/knowledge-base/important-notices/cve-2017-1000369-exim-stack-clash/)

* * *

#### [CVE 2017 16943 and CVE 2017 16944 Exim](https://docs.cpanel.net/knowledge-base/important-notices/cve-2017-16943-and-cve-2017-16944-exim/)

On Friday, November 24 2017, Exim announced two vulnerabilities in versions 4.88 and later.

[Read more](https://docs.cpanel.net/knowledge-base/important-notices/cve-2017-16943-and-cve-2017-16944-exim/)

* * *

#### [CVE 2019 7524 Buffer overflow when reading extension header from Dovecot index files](https://docs.cpanel.net/knowledge-base/important-notices/cve-2019-7524-buffer-overflow/)

We were made aware of a CVE in Dovecot Versions 2.0.14 - 2.3.5 that involves using Solr on Thursday, March 28th 2019.

[Read more](https://docs.cpanel.net/knowledge-base/important-notices/cve-2019-7524-buffer-overflow/)

* * *

#### [CVE-2015-2035-GHOST](https://docs.cpanel.net/knowledge-base/important-notices/cve-2015-2035-ghost/)

On 27 January 2015, a vulnerability in all versions of the GNU C library (glibc) was announced by Qualys.

[Read more](https://docs.cpanel.net/knowledge-base/important-notices/cve-2015-2035-ghost/)

* * *

#### [CVE-2016-1238 Perl](https://docs.cpanel.net/knowledge-base/important-notices/cve-2016-1238-perl/)

On July 25 2016, Perl announced a vulnerability in all versions of the Perl 5 software.

[Read more](https://docs.cpanel.net/knowledge-base/important-notices/cve-2016-1238-perl/)

* * *

#### [CVE-2016-1531 Exim](https://docs.cpanel.net/knowledge-base/important-notices/cve-2016-1531-exim/)

On Wednesday, March 2, 2016, Exim announced a vulnerability in all versions of the Exim software.

[Read more](https://docs.cpanel.net/knowledge-base/important-notices/cve-2016-1531-exim/)

* * *

#### [CVE-2016-3714 ImageMagick](https://docs.cpanel.net/knowledge-base/important-notices/cve-2016-3714-imagemagick/)

On Tuesday, May 3 2016, ImageMagick announced a vulnerability in all versions of the ImageMagick software. ImageMagick is a software package commonly used by web services to process images.

[Read more](https://docs.cpanel.net/knowledge-base/important-notices/cve-2016-3714-imagemagick/)

* * *

#### [CVE-2016-5387 HTTPOXY](https://docs.cpanel.net/knowledge-base/important-notices/cve-2016-5387-httpoxy/)

This document describes the Apache-disclosed vulnerability that affects application code which runs in CGI, or CGI-like environments.

[Read more](https://docs.cpanel.net/knowledge-base/important-notices/cve-2016-5387-httpoxy/)

* * *

#### [CVE-2016-6662 MySQL](https://docs.cpanel.net/knowledge-base/important-notices/cve-2016-6662-mysql/)

This document describes multiple MySQL® vulnerabilities reported by an independent researcher on 12 September, 2016.

[Read more](https://docs.cpanel.net/knowledge-base/important-notices/cve-2016-6662-mysql/)

* * *

#### [CVE-2016-9963 Exim](https://docs.cpanel.net/knowledge-base/important-notices/cve-2016-9963-exim/)

On Sunday, December 25, 2016, Exim announced a vulnerability in versions 4.69 to 4.87 of the Exim software.

[Read more](https://docs.cpanel.net/knowledge-base/important-notices/cve-2016-9963-exim/)

* * *

#### [CVE-2017-5753, CVE-2017-5715, CVE-2017-5754 Meltdown](https://docs.cpanel.net/knowledge-base/important-notices/cve-2017-5753-cve-2017-5715-cve-2017-5754-meltdown/)

Red Hat has been made aware of multiple microarchitectural (hardware) implementation issues affecting many modern microprocessors, requiring updates to the Linux kernel, virtualization-related components, and/or in combination with a microcode update.

[Read more](https://docs.cpanel.net/knowledge-base/important-notices/cve-2017-5753-cve-2017-5715-cve-2017-5754-meltdown/)

* * *

#### [CVE-2019-10149 Exim](https://docs.cpanel.net/knowledge-base/important-notices/cve-2019-10149-exim/)

Exim maintainers announced that they received a report of a potential remote exploit in Exim from version 4.87 to version 4.91.

[Read more](https://docs.cpanel.net/knowledge-base/important-notices/cve-2019-10149-exim/)
