# email

* * *

#### [The convert\_mdbox\_to\_maildir Script](https://docs.cpanel.net/whm/scripts/the-convert_mdbox_to_maildir-script/)

This script converts mailboxes from mdbox to maildir format.

[Read more](https://docs.cpanel.net/whm/scripts/the-convert_mdbox_to_maildir-script/)

* * *

#### [The copy\_user\_mail\_as\_root Script](https://docs.cpanel.net/whm/scripts/the-copy_user_mail_as_root-script/)

This script copies a cPanel account's mail to or from a remote server as the root user.

[Read more](https://docs.cpanel.net/whm/scripts/the-copy_user_mail_as_root-script/)

* * *

#### [The copy\_user\_mail\_as\_user Script](https://docs.cpanel.net/whm/scripts/the-copy_user_mail_as_user-script/)

This script copies a cPanel account's mail to or from a remote server as the account.

[Read more](https://docs.cpanel.net/whm/scripts/the-copy_user_mail_as_user-script/)

* * *

#### [The cpanel\_dovecot\_solr\_rebuild\_index Script](https://docs.cpanel.net/whm/scripts/the-cpanel-dovecot-solr-rebuild-index-script/)

This script rebuilds your Apache Solr™ index files if they become corrupt.

[Read more](https://docs.cpanel.net/whm/scripts/the-cpanel-dovecot-solr-rebuild-index-script/)

* * *

#### [The fix\_dns\_zone\_ttls Script](https://docs.cpanel.net/whm/scripts/the-fix_dns_zone_ttls-script/)

This script fixes problems with TTL values in DNS zone files.

[Read more](https://docs.cpanel.net/whm/scripts/the-fix_dns_zone_ttls-script/)

* * *

#### [The remove\_dovecot\_index\_files Script](https://docs.cpanel.net/whm/scripts/the-remove_dovecot_index_files-script/)

Use the /usr/local/cpanel/scripts/remove\_dovecot\_index\_files script to remove the Dovecot® index logs and cache files from cPanel mail accounts.

[Read more](https://docs.cpanel.net/whm/scripts/the-remove_dovecot_index_files-script/)

* * *

#### [The rescan\_user\_dovecot\_fts Script](https://docs.cpanel.net/whm/scripts/the-rescan_user_dovecot_fts-script/)

Use the /usr/local/cpanel/scripts/rescan\_user\_dovecot\_fts script to perform a full text search (FTS) scan on an account's mailboxes.

[Read more](https://docs.cpanel.net/whm/scripts/the-rescan_user_dovecot_fts-script/)

* * *

#### [The setupmailserver Script](https://docs.cpanel.net/whm/scripts/the-setupmailserver-script/)

This script lets you select a mail server.

[Read more](https://docs.cpanel.net/whm/scripts/the-setupmailserver-script/)

* * *

#### [Track Delivery](https://docs.cpanel.net/cpanel/email/track-delivery/)

This interface displays reports about email message delivery attempts from your account.

[Read more](https://docs.cpanel.net/cpanel/email/track-delivery/)

* * *

#### [View Mail Statistics Summary](https://docs.cpanel.net/whm/email/view-mail-statistics-summary/)

This interface displays statistics about the mail that your server sends and receives over a period of time.

[Read more](https://docs.cpanel.net/whm/email/view-mail-statistics-summary/)

* * *

#### [View Relayers](https://docs.cpanel.net/whm/email/view-relayers/)

This interface lists users who have relayed mail. Relayed email is email that a user on your server forwards to a remote account.

[Read more](https://docs.cpanel.net/whm/email/view-relayers/)

* * *

#### [View Sent Summary](https://docs.cpanel.net/whm/email/view-sent-summary/)

This interface displays the total number of message delivery attempts that each domain on your server made within a specified time and date range.

[Read more](https://docs.cpanel.net/whm/email/view-sent-summary/)
