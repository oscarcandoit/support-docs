## Experimental Repository Change Log

Last modified: _2024 August 21_

* * *

EasyApache 4 does **not** use versioning and organizes changes by date only.

Warning:

cPanel & WHM does **not** guarantee the functionality of software in the [Experimental Repository](https://docs.cpanel.net/ea4/git-repositories/the-experimental-repository), and we provide it on an experimental basis **only**. You assume all risk for any software that you install from the Experimental Repository. Installation of this software could cause significant functionality failures, even for experienced administrators.
