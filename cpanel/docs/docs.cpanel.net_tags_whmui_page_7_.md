# whmui

* * *

#### [Manage cPAddons Site Software](https://docs.cpanel.net/whm/cpanel/manage-cpaddons-site-software/)

The Manage cPAddons Site Software interface allows you to upgrade or uninstall individual installations of cPAddon software.

[Read more](https://docs.cpanel.net/whm/cpanel/manage-cpaddons-site-software/)

* * *

#### [Manage Database Access Hosts](https://docs.cpanel.net/whm/database-services/manage-database-access-hosts/)

This interface allows you to permit remote hosts (servers) to access MySQL® and MariaDB® databases on your server. It also allows you to access remote database servers that you configured via WHM's Manage Database Profiles interface.

[Read more](https://docs.cpanel.net/whm/database-services/manage-database-access-hosts/)

* * *

#### [Manage Database Profiles](https://docs.cpanel.net/whm/database-services/manage-database-profiles/)

This interface allows you to manage multiple MySQL® or MariaDB® profiles.

[Read more](https://docs.cpanel.net/whm/database-services/manage-database-profiles/)

* * *

#### [Manage Database Users](https://docs.cpanel.net/whm/database-services/manage-database-users/)

This interface allows server administrators to retrieve a list of MySQL® and MariaDB® database users, which account owns them, and their databases.

[Read more](https://docs.cpanel.net/whm/database-services/manage-database-users/)

* * *

#### [Manage Database Users](https://docs.cpanel.net/whm/sql-services/manage-database-users/)

This interface allows server administrators to retrieve a list of database users, each user's database, and which account owns each database user.

[Read more](https://docs.cpanel.net/whm/sql-services/manage-database-users/)

* * *

#### [Manage Databases](https://docs.cpanel.net/whm/database-services/manage-databases/)

This interface allows you to list cPanel-managed databases on your server, each database's engine, and which user owns each database.

[Read more](https://docs.cpanel.net/whm/database-services/manage-databases/)

* * *

#### [Manage Databases](https://docs.cpanel.net/whm/sql-services/manage-databases/)

This interface allows you to list cPanel-managed databases on your server, each database's engine, and which user owns each database.

[Read more](https://docs.cpanel.net/whm/sql-services/manage-databases/)

* * *

#### [Manage Demo Mode](https://docs.cpanel.net/whm/account-functions/manage-demo-mode/)

This feature allows you to convert any cPanel account on your server to a demo account.

[Read more](https://docs.cpanel.net/whm/account-functions/manage-demo-mode/)

* * *

#### [Manage External Authentications](https://docs.cpanel.net/whm/security-center/manage-external-authentications/)

The Manage External Authentications interface allows you to manage your server's OpenID® Connect™-compliant identity providers.

[Read more](https://docs.cpanel.net/whm/security-center/manage-external-authentications/)

* * *

#### [Manage Hooks](https://docs.cpanel.net/whm/development/manage-hooks/)

This interface allows you to perform a variety of actions on the hooks that exist on your server.

[Read more](https://docs.cpanel.net/whm/development/manage-hooks/)

* * *

#### [Manage MySQL® Profiles](https://docs.cpanel.net/whm/sql-services/manage-mysql-profiles/)

This interface allows you to manage multiple MySQL® profiles.

[Read more](https://docs.cpanel.net/whm/sql-services/manage-mysql-profiles/)

* * *

#### [Manage Plugins](https://docs.cpanel.net/whm/cpanel/manage-plugins/)

The Manage Plugins interface allows you to add and remove plugins.

[Read more](https://docs.cpanel.net/whm/cpanel/manage-plugins/)

* * *

#### [Manage Reseller's IP Delegation](https://docs.cpanel.net/whm/resellers/manage-resellers-ip-delegation/)

This interface allows you to control a reseller's access to the server's IP addresses.

[Read more](https://docs.cpanel.net/whm/resellers/manage-resellers-ip-delegation/)

* * *

#### [Manage Reseller's Shared IP](https://docs.cpanel.net/whm/resellers/manage-resellers-shared-ip/)

This interface allows you to specify a shared IP address for a reseller.

[Read more](https://docs.cpanel.net/whm/resellers/manage-resellers-shared-ip/)

* * *

#### [Manage Service SSL Certificates](https://docs.cpanel.net/whm/service-configuration/manage-service-ssl-certificates/)

This interface allows you to manage certificates for your server's services.

[Read more](https://docs.cpanel.net/whm/service-configuration/manage-service-ssl-certificates/)

* * *

#### [Manage Sudo Group Users](https://docs.cpanel.net/whm/security-center/manage-sudo-group-users/)

This interface allows you to manage the wheel group's membership.

[Read more](https://docs.cpanel.net/whm/security-center/manage-sudo-group-users/)

* * *

#### [Manage Wheel Group Users](https://docs.cpanel.net/whm/security-center/manage-wheel-group-users/)

This interface allows you to manage the wheel group's membership.

[Read more](https://docs.cpanel.net/whm/security-center/manage-wheel-group-users/)

* * *

#### [Memory Usage Restrictions](https://docs.cpanel.net/whm/service-configuration/memory-usage-restrictions/)

This interface calculates and sets a new memory limit for Apache processes.

[Read more](https://docs.cpanel.net/whm/service-configuration/memory-usage-restrictions/)

* * *

#### [Modify cPanel & WHM News](https://docs.cpanel.net/whm/cpanel/modify-cpanel-whm-news/)

This feature allows you to display custom messages in your users' and resellers' cPanel & WHM interfaces.

[Read more](https://docs.cpanel.net/whm/cpanel/modify-cpanel-whm-news/)

* * *

#### [Modify/Upgrade Multiple Accounts](https://docs.cpanel.net/whm/multi-account-functions/modify-upgrade-multiple-accounts/)

This interface allows you to simultaneously change the attributes of multiple accounts to use the same settings.

[Read more](https://docs.cpanel.net/whm/multi-account-functions/modify-upgrade-multiple-accounts/)
