## Upload Forms

Last modified: _2020 April 7_

* * *

## Overview

The _Upload Forms_ interface allows you to upload digital copies of authorization forms.

Note:

The accepted formats are the `.pdf`, `.jpeg`, `.png`, or `.gif` file types.


## Upload a file

To upload a file, perform the following steps:

1. Click _Browse_.
2. Select the appropriate file and then click _Open_.
3. Click _Upload_.

You will receive a confirmation message when file upload completes.
