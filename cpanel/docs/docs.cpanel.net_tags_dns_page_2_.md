# dns

* * *

#### [Nameserver Record Report](https://docs.cpanel.net/whm/dns-functions/nameserver-record-report/)

The Nameserver Record Report interface lists the server's nameserver information.

[Read more](https://docs.cpanel.net/whm/dns-functions/nameserver-record-report/)

* * *

#### [Nameserver Selection](https://docs.cpanel.net/whm/service-configuration/nameserver-selection/)

This interface allows you to select your server's nameserver software or to disable DNS services.

[Read more](https://docs.cpanel.net/whm/service-configuration/nameserver-selection/)

* * *

#### [Set Zone Time to Live (TTL)](https://docs.cpanel.net/whm/dns-functions/set-zone-time-to-live/)

This interface allows you to set the value for time to live (TTL) in your server's DNS zones.

[Read more](https://docs.cpanel.net/whm/dns-functions/set-zone-time-to-live/)

* * *

#### [Setup/Edit Domain Forwarding](https://docs.cpanel.net/whm/dns-functions/setup-edit-domain-forwarding/)

This feature allows you to redirect one domain to another domain.

[Read more](https://docs.cpanel.net/whm/dns-functions/setup-edit-domain-forwarding/)

* * *

#### [Synchronize DNS Records](https://docs.cpanel.net/whm/dns-functions/synchronize-dns-records/)

This feature compares zone files among servers, recognizes which are the most up-to-date, and updates the servers in the DNS cluster.

[Read more](https://docs.cpanel.net/whm/dns-functions/synchronize-dns-records/)

* * *

#### [The cleandns Script](https://docs.cpanel.net/whm/scripts/the-cleandns-script/)

This script removes orphaned and duplicate DNS zone definitions.

[Read more](https://docs.cpanel.net/whm/scripts/the-cleandns-script/)

* * *

#### [The dnssec-cluster-keys Script](https://docs.cpanel.net/whm/scripts/the-dnssec-cluster-keys-script/)

This script allows you to sync and revoke currently active DNSSEC keys in a DNS cluster.

[Read more](https://docs.cpanel.net/whm/scripts/the-dnssec-cluster-keys-script/)

* * *

#### [The enable\_spf\_dkim\_globally Script](https://docs.cpanel.net/whm/scripts/the-enable_spf_dkim_globally-script/)

This script enables SPF and DKIM for the specified users, and creates the appropriate DNS records for their domains.

[Read more](https://docs.cpanel.net/whm/scripts/the-enable_spf_dkim_globally-script/)

* * *

#### [The servicedomains Script](https://docs.cpanel.net/whm/scripts/the-servicedomains-script/)

This script lets you to create DNS records for service subdomains.

[Read more](https://docs.cpanel.net/whm/scripts/the-servicedomains-script/)

* * *

#### [Track DNS](https://docs.cpanel.net/cpanel/advanced/track-dns/)

This interface contains tools to help you retrieve network information.

[Read more](https://docs.cpanel.net/cpanel/advanced/track-dns/)

* * *

#### [Zone Editor](https://docs.cpanel.net/cpanel/domains/zone-editor/)

This feature allows you to create, edit, and delete Domain Name System (DNS) zone records.

[Read more](https://docs.cpanel.net/cpanel/domains/zone-editor/)
