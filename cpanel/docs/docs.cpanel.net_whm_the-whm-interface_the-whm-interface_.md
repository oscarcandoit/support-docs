## Overview

Note:

- If you have not already installed cPanel & WHM on your server, use our [Installation Guide](https://docs.cpanel.net/installation-guide/install) to complete the installation process.
- If you are logging in to cPanel & WHM for the first time, read our [Getting Started](https://docs.cpanel.net/whm/the-whm-interface/getting-started) documentation.

The WHM interface provides the server’s `root` user and resellers with `root`-level privileges to all of WHM’s server administration features.

![WHM Home interface with Jupiter theme](https://docs.cpanel.net/img/whm-jupiter-interface-130.png)

The WHM interface consists of the following sections:

1. [Top navigation bar](https://docs.cpanel.net/whm/the-whm-interface/the-whm-interface/#top-navigation-bar)
2. [Side navigation menu](https://docs.cpanel.net/whm/the-whm-interface/the-whm-interface/#side-navigation-menu)
3. [Important next steps](https://docs.cpanel.net/whm/the-whm-interface/the-whm-interface/#important-next-steps)
4. [Favorites](https://docs.cpanel.net/whm/the-whm-interface/the-whm-interface/#favorites)
5. [Statistics panel](https://docs.cpanel.net/whm/the-whm-interface/the-whm-interface/#statistics-panel)
6. [Server Monitoring](https://docs.cpanel.net/whm/the-whm-interface/the-whm-interface/#server-monitoring)
7. [Tools](https://docs.cpanel.net/whm/the-whm-interface/the-whm-interface/#tools)
8. [Footer navigation bar](https://docs.cpanel.net/whm/the-whm-interface/the-whm-interface/#footer-navigation-bar)

## Top navigation bar

WHM’s top navigation bar remains visible as you navigate through the WHM interface. The top navigation bar contains the following features:

| Feature | Description |
| --- | --- |

Use the arrow keys or mouse to select the search result that you want to view. <br>- Use the forward slash (`/`) keyboard shortcut to quickly access the _Search_ text box.<br>- This feature shows ranked results, prioritizing exact matches.<br>- This feature displays cPanel accounts based on the text matching a username or a primary domain owned by that cPanel account. |
| _Notifications_ | WHM will show you notifications of recent activity, problems with the server, and other information of importance. If your _Notifications_ icon is red (![red notification icon](https://docs.cpanel.net/img/jupiter-whm-notification.png)), you have a message, warning, and/or an error on your WHM account. Click the _Notifications_ icon to display them. |
| _User Menu_ | To change the `root` password (if you are logged in as the `root` user) or log out from the account, click the account icon (![account icon](https://docs.cpanel.net/img/whm-account-icon-jupiter.png)). |

| _Hostname_ | This feature displays the server’s hostname. |

Click the alert to navigate to the _Server Monitoring_ interface ( _WHM » Home » Plugins » Server Monitoring_). |

## Side navigation menu

WHM’s side navigation menu provides access to all of WHM’s interfaces, and allows you to quickly locate specific features. For more information about WHM’s features and interfaces, read our [WHM](https://docs.cpanel.net/whm) documentation.

We minimize the feature categories by default. You can expand a feature category to see the grouped features by clicking the feature name. You can expand and collapse all feature categories by clicking _Expand_ or _Collapse_ at the top of the side navigation menu.

The _Favorites_ category will contain the tools that you select with the [_Favorites_](https://docs.cpanel.net/whm/the-whm-interface/the-whm-interface/#favorites) feature.

To search for a tool, enter one or more keywords in the search text box. Search results will update the options on the side navigation menu to show only those features that match the entered search criteria.

## Favorites

The _Favorites_ section of the WHM _Home_ interface provides immediate access to the WHM features that you select. You can choose which features you prefer and the order in which they appear. You can also choose whether to display the feature descriptions. The tools that you select will appear in the _Favorites_ category in WHM’s [side navigation menu](https://docs.cpanel.net/whm/the-whm-interface/the-whm-interface/#side-navigation-menu).

To add features to the _Favorites_ section, perform the following steps:

1. Click _Edit Favorites_. A star icon (![star icon](https://docs.cpanel.net/img/favorite-star-1.jpg)) will appear next to each feature.
2. Click the star icon (![star icon](https://docs.cpanel.net/img/favorite-star-1.jpg)) for each feature that you wish to add.
3. Click _Done Editing_ to stop adding features.

To reorder features in the _Favorites_ section, perform the following steps:

1. Click _Edit Favorites_.
2. Click and drag the features to the position you prefer.
3. Click _Done Editing_.

To remove features from the _Favorites_ section, perform the following steps:

1. Click _Edit Favorites_.
2. Click the x icon (![x icon](https://docs.cpanel.net/img/favorite-x-1.jpg)) for each feature that you wish to remove.
3. Click _Done Editing_ to stop removing features.

To enable or disable feature descriptions, perform the following steps:

1. Click _Edit Favorites_.
2. Click the _Compact_ or _Detailed_ toggle.
3. Click _Done Editing_.

## Statistics panel

In the WHM _Home_ interface _Statistics_ panel, you will find useful information about your server. The statistics panel contains the following features:

| Feature | Description |
| --- | --- |
| _Hostname_ | The _Hostname_ section displays the hostname of your server. |
| _Server Monitoring_ | The _Server Monitoring_ section displays a quick overview of your server’s [resource use](https://docs.cpanel.net/whm/the-whm-interface/the-whm-interface/#server-monitoring-icons). |
| _Load Averages_ | The _Load Averages_ section displays the load average for your server for the past minute, five minutes, and fifteen minutes. It will also tell you if your load is trending down, trending up, or remaining steady. |
| _Operating System_ | The _Operating System_ section displays the operating system and version that your cPanel & WHM server runs. |
| _Product_ | The _Product_ section displays the version and the license type of your cPanel & WHM server. |
| _User Analytics ID_ | The _User Analytics ID_ displays your Universally Unique Identifier ( [UUID](https://en.wikipedia.org/wiki/Universally_unique_identifier)). For more information, read our [_cPanel Analytics_](https://docs.cpanel.net/knowledge-base/cpanel-product/cpanel-analytics/) documentation. <br>Note:<br>This section **only** appears if the following conditions are both true:<br>- The user logged in as a non-`root` reseller account.<br>- You [allow users to manage Interface Analytics](https://docs.cpanel.net/knowledge-base/general-systems-administration/how-to-allow-users-to-manage-interface-analytics/). |

### Server Monitoring icons

The color of each icon indicates the current usage level of your server’s resources as follows:

| Icon Color | Description |
| --- | --- |
| _Green_ | - CPU and Disk usage are below 50%.<br>- Memory usage is below 70%. |
| _Yellow_ | - CPU and Disk usage exceed 50%.<br>- Memory usage exceeds 70%. |
| _Red_ | - CPU and Disk usage exceed 90%.<br>- Memory usage exceeds 95%. |

Click these icons to navigate to the [_Server Monitoring_](https://docs.cpanel.net/whm/plugins/server-monitoring/) interface ( _WHM » Home » Plugins » Server Monitoring_).

## Server Monitoring

Note:

WebPros International, LLC currently provides this feature in a limited release, with plans to expand its availability in future updates.

The _Server Monitoring_ section displays statistics and metrics for your server’s performance at a glance. Here you can see details such as _CPU_ usage, _CPU load average_, and _Top processes_. For a more detailed view, click _View Reports_ to be taken to the [_Server Monitoring (Powered by 360 Monitoring)_](https://docs.cpanel.net/whm/plugins/server-monitoring) interface.

## Tools

The WHM _Home_ interface provides quick access to the features that manage cPanel & WHM servers and accounts. These features are grouped for quick access to functions of a similar type.

Note:

By default, we hide _Feature Showcase_ items on new installations until new items are available for the next server update. For more information and steps to view these items, read our [Guide to the Feature Showcase](https://api.docs.cpanel.net/guides/guide-to-the-feature-showcase/) documentation.
