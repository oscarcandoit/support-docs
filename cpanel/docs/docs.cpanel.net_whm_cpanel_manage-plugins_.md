## Overview

The _Manage Plugins_ interface allows you to add and remove plugins. Plugins provide additional functionality for your server. The plugins that this interface lists vary from server to server.

Important:

- This interface does **not** appear on cPanel DNSOnly® servers.
- For more information about WHM plugin development, read our [Guide to WHM Plugins](https://api.docs.cpanel.net/guides/guide-to-whm-plugins/) documentation.

## Install a plugin

To install a plugin, click _Install_ for the plugin that you wish to install.

A progress window will appear in the interface, and it will display a success message when installation finishes.

Note:

To access an installed plugin, use WHM’s _Home_ interface or left-side navigation menu to access the Plugins section and its interfaces.


## Uninstall a plugin

To uninstall a plugin, click _Uninstall_ for the plugin that you wish to uninstall.
