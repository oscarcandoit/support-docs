## Server Monitoring (Powered by 360 Monitoring) Change Log

Last modified: _2025 November 13_

* * *

### 2.0.2-1

* * *

##### **2025-11-13**

- CPCLOUD-5927: Release server monitoring to safe list partners
  - Add new companies Ids from safe list excluding Milesweb
  - Enabled mini-app also

### 2.0.1-1

* * *

##### **2025-07-25**

- DUCKS-3453: Improved interface messaging for server monitoring home page widgets.

### 2.0.0-3

* * *

##### **2025-06-24**

- IDUCKS-2648:
  - Added a new top bar widget for server monitoring alerts.
  - Added a new whm homepage stats widget for server monitoring.
  - Added localization for the new cpanel components.
  - Added local server enable/disable for server monitoring overriding rules provided by WebPros in the Feature Showcase for Server Monitoring.
  - UI improvements for better awareness of what plan the user is on.
  - Pop-up with number of alerts missed. “You missed 5 alerts”
  - Show the upgrade button as soon as the account is created (instead of waiting 15 days until the acc is no longer in “trial” mode)
  - Change title to “Server Monitoring” in all views for consistency.
  - Add tracking for mixpanel when users toggle the “Mini” visibility.

### 1.6.0-1

* * *

###### **2025-05-28**

- DUCKS-3034: Enable Server Monitoring feature for selective cPanel tiers versions 128 & above.

### 1.5.0-1

* * *

##### **2025-04-03**

- DUCKS-1910: Enable Server Monitoring feature by default for cPanel versions 128 and above

### 1.4.0-1

* * *

##### **2025-03-13**

- DUCKS-2421: Re-add button to enable upgrades for registered users

### 1.3.0-1

* * *

##### **2025-03-10**

- DUCKS-1824: Updates include:
  - Add tooling to reinitialize the feature on configuration changes, such as server license updates, that affect feature access.
  - Prevent users with bookmarked internal links to the Social Media
    Management page from accessing the page when it is disabled.

### 1.2.2-1

* * *

##### **2024-12-20**

- DUCKS-1295: Updates include:
Auto enable the whm home page server monitoring mini-app.
Enable upgrades for server monitoring.
Other minor fixes to server monitoring.

### 1.1.0-1

* * *

##### **2024-11-25**

- DUCKS-1170: Enable server monitoring for cPanel Direct (265) and some internal testing servers.

### 1.0.0

* * *

##### **2024-10-24**

- DUCKS-930: Enable the cpanel-monitoring-plugin for WHM. (soft release)
