## Server Status

* * *

#### [Apache Status](https://docs.cpanel.net/whm/server-status/apache-status/)

This interface displays the status of your Apache server.

[Read more](https://docs.cpanel.net/whm/server-status/apache-status/)

* * *

#### [Daily Process Log](https://docs.cpanel.net/whm/server-status/daily-process-log/)

This interface displays information about your server's consumption of processing power and memory.

[Read more](https://docs.cpanel.net/whm/server-status/daily-process-log/)

* * *

#### [Server Information](https://docs.cpanel.net/whm/server-status/server-information/)

This interface displays information about your hardware configuration, such as the speed of your processor.

[Read more](https://docs.cpanel.net/whm/server-status/server-information/)

* * *

#### [Service Status](https://docs.cpanel.net/whm/server-status/service-status/)

This interface displays a list of the server's monitored services.

[Read more](https://docs.cpanel.net/whm/server-status/service-status/)

* * *

#### [Task Queue Monitor](https://docs.cpanel.net/whm/server-status/task-queue-monitor/)

This interface displays information about cPanel & WHM’s task queue in real time.

[Read more](https://docs.cpanel.net/whm/server-status/task-queue-monitor/)
