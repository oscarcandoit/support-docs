# sitequality

* * *

#### [Site Quality Monitoring](https://docs.cpanel.net/cpanel/metrics/site-quality-monitoring/)

The Site Quality Monitoring feature allows you to monitor your website for common issues. The Site Quality Monitoring interface contains tabs that manage monitoring in different ways.

[Read more](https://docs.cpanel.net/cpanel/metrics/site-quality-monitoring/)

* * *

#### [Site Quality Monitoring — Crawling](https://docs.cpanel.net/cpanel/metrics/site-quality-monitoring/site-quality-monitoring-crawling/)

Site Quality Monitoring's Crawling tab allows you to run checks for dead links, missing images and files, and oversize files.

[Read more](https://docs.cpanel.net/cpanel/metrics/site-quality-monitoring/site-quality-monitoring-crawling/)

* * *

#### [Site Quality Monitoring — Extras](https://docs.cpanel.net/cpanel/metrics/site-quality-monitoring/site-quality-monitoring-extras/)

Site Quality Monitoring's Extras tab contains configurable monitoring extensions for users with paid Site Quality Monitoring subscriptions.

[Read more](https://docs.cpanel.net/cpanel/metrics/site-quality-monitoring/site-quality-monitoring-extras/)

* * *

#### [Site Quality Monitoring — Monitoring](https://docs.cpanel.net/cpanel/metrics/site-quality-monitoring/site-quality-monitoring-monitoring/)

Site Quality Monitoring's Monitoring tab contains monitoring check categories and their results.

[Read more](https://docs.cpanel.net/cpanel/metrics/site-quality-monitoring/site-quality-monitoring-monitoring/)

* * *

#### [Site Quality Monitoring — Settings](https://docs.cpanel.net/cpanel/metrics/site-quality-monitoring/site-quality-monitoring-settings/)

Site Quality Monitoring's Settings tab allows you to manage your monitored URLs and websites. It also allows you to set the frequency of error alert emails and who receives them.

[Read more](https://docs.cpanel.net/cpanel/metrics/site-quality-monitoring/site-quality-monitoring-settings/)

* * *

#### [Site Quality Monitoring — Website Summary](https://docs.cpanel.net/cpanel/metrics/site-quality-monitoring/site-quality-monitoring-website-summary/)

Site Quality Monitoring's Website Summary tab summarizes the result of recent monitoring checks.

[Read more](https://docs.cpanel.net/cpanel/metrics/site-quality-monitoring/site-quality-monitoring-website-summary/)

* * *

#### [How to Activate a Site Quality Monitoring Subscription](https://docs.cpanel.net/knowledge-base/general-systems-administration/how-to-activate-a-site-quality-monitoring-subscription/)

To use Site Quality Monitoring, you must activate a subscription.

[Read more](https://docs.cpanel.net/knowledge-base/general-systems-administration/how-to-activate-a-site-quality-monitoring-subscription/)

#### [How to Manage a Site Quality Monitoring Subscription](https://docs.cpanel.net/knowledge-base/general-systems-administration/how-to-manage-a-site-quality-monitoring-subscription/)
