## Scripts

* * *

#### [The unsuspendacct Script](https://docs.cpanel.net/whm/scripts/the-unsuspendacct-script/)

[Read more](https://docs.cpanel.net/whm/scripts/the-unsuspendacct-script/)

* * *

#### [The upcp Script](https://docs.cpanel.net/whm/scripts/the-upcp-script/)

This script lets you update cPanel & WHM on the command line.

[Read more](https://docs.cpanel.net/whm/scripts/the-upcp-script/)

* * *

#### [The upcp-running Script](https://docs.cpanel.net/whm/scripts/the-upcp-running-script/)

This script returns whether the upcp script is running.

[Read more](https://docs.cpanel.net/whm/scripts/the-upcp-running-script/)

#### [The update\_existing\_mail\_quotas\_for\_account Script](https://docs.cpanel.net/whm/scripts/the-update_existing_mail_quotas_for_account-script/)
