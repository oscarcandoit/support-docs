## Scripts

* * *

#### [The convert\_mdbox\_to\_maildir Script](https://docs.cpanel.net/whm/scripts/the-convert_mdbox_to_maildir-script/)

This script converts mailboxes from mdbox to maildir format.

[Read more](https://docs.cpanel.net/whm/scripts/the-convert_mdbox_to_maildir-script/)

* * *

#### [The copy\_user\_mail\_as\_root Script](https://docs.cpanel.net/whm/scripts/the-copy_user_mail_as_root-script/)

This script copies a cPanel account's mail to or from a remote server as the root user.

[Read more](https://docs.cpanel.net/whm/scripts/the-copy_user_mail_as_root-script/)

* * *

#### [The copy\_user\_mail\_as\_user Script](https://docs.cpanel.net/whm/scripts/the-copy_user_mail_as_user-script/)

This script copies a cPanel account's mail to or from a remote server as the account.

[Read more](https://docs.cpanel.net/whm/scripts/the-copy_user_mail_as_user-script/)

* * *

#### [The cpanel\_dovecot\_solr\_rebuild\_index Script](https://docs.cpanel.net/whm/scripts/the-cpanel-dovecot-solr-rebuild-index-script/)

This script rebuilds your Apache Solr™ index files if they become corrupt.

[Read more](https://docs.cpanel.net/whm/scripts/the-cpanel-dovecot-solr-rebuild-index-script/)

* * *

#### [The cpconftool Script](https://docs.cpanel.net/whm/scripts/the-cpconftool-script/)

This script backs up, restores, and transfers server configurations.

[Read more](https://docs.cpanel.net/whm/scripts/the-cpconftool-script/)

* * *

#### [The cpuser\_port\_authority Script](https://docs.cpanel.net/whm/scripts/the-cpuser_port_authority-script/)

The cpuser\_port\_authority script assigns one or more 5-digit port numbers for a user's exclusive use.

[Read more](https://docs.cpanel.net/whm/scripts/the-cpuser_port_authority-script/)

* * *

#### [The dbmaptool Script](https://docs.cpanel.net/whm/scripts/the-dbmaptool-script/)

This script grants ownership of databases and database users to a cPanel account.

[Read more](https://docs.cpanel.net/whm/scripts/the-dbmaptool-script/)

* * *

#### [The dbstoregrants Script](https://docs.cpanel.net/whm/scripts/the-dbstoregrants-script/)

Use this script to syncronize changes to MySQL® users' database grants

[Read more](https://docs.cpanel.net/whm/scripts/the-dbstoregrants-script/)

* * *

#### [The dnssec-cluster-keys Script](https://docs.cpanel.net/whm/scripts/the-dnssec-cluster-keys-script/)

This script allows you to sync and revoke currently active DNSSEC keys in a DNS cluster.

[Read more](https://docs.cpanel.net/whm/scripts/the-dnssec-cluster-keys-script/)

* * *

#### [The enable\_spf\_dkim\_globally Script](https://docs.cpanel.net/whm/scripts/the-enable_spf_dkim_globally-script/)

This script enables SPF and DKIM for the specified users, and creates the appropriate DNS records for their domains.

[Read more](https://docs.cpanel.net/whm/scripts/the-enable_spf_dkim_globally-script/)

* * *

#### [The export\_horde\_calendars\_to\_ics Script](https://docs.cpanel.net/whm/scripts/the-export_horde_calendars_to_ics-script/)

Use this script to export Horde calendars into files you can import into Roundcube or other email clients.

[Read more](https://docs.cpanel.net/whm/scripts/the-export_horde_calendars_to_ics-script/)

* * *

#### [The export\_horde\_contacts\_to\_vcf Script](https://docs.cpanel.net/whm/scripts/the-export_horde_contacts_to_vcf-script/)

Use this script to export Horde contacts into files you can import into Roundcube or other email clients.

[Read more](https://docs.cpanel.net/whm/scripts/the-export_horde_contacts_to_vcf-script/)

* * *

#### [The find\_and\_fix\_rpm\_issues Script](https://docs.cpanel.net/whm/scripts/the-find_and_fix_rpm_issues-script/)

The /usr/local/cpanel/scripts/find\_and\_fix\_rpm\_issues script detects issues with your server's RPM database and rebuilds the database if problems exist.

[Read more](https://docs.cpanel.net/whm/scripts/the-find_and_fix_rpm_issues-script/)

* * *

#### [The find\_outdated\_services Script](https://docs.cpanel.net/whm/scripts/the-find_outdated_services-script/)

This script checks all services to determine whether they require a restart.

[Read more](https://docs.cpanel.net/whm/scripts/the-find_outdated_services-script/)

* * *

#### [The fix\_dns\_zone\_ttls Script](https://docs.cpanel.net/whm/scripts/the-fix_dns_zone_ttls-script/)

This script fixes problems with TTL values in DNS zone files.

[Read more](https://docs.cpanel.net/whm/scripts/the-fix_dns_zone_ttls-script/)

* * *

#### [The fix-web-vhost-configuration Script](https://docs.cpanel.net/whm/scripts/the-fix-web-vhost-configuration-script/)

You can use the /usr/local/cpanel/scripts/fix-web-vhost-configuration script to iterate through each user’s web virtual host configuration files and remove excess alias names.

[Read more](https://docs.cpanel.net/whm/scripts/the-fix-web-vhost-configuration-script/)

* * *

#### [The gather\_update\_log\_stats Script](https://docs.cpanel.net/whm/scripts/the-gather_update_log_stats-script/)

The /usr/local/cpanel/scripts/gather\_update\_log\_stats script gathers your server's update logs and sends them to WebPros International, LLC for analysis.

[Read more](https://docs.cpanel.net/whm/scripts/the-gather_update_log_stats-script/)

* * *

#### [The gather\_update\_logs\_setupcrontab Script](https://docs.cpanel.net/whm/scripts/the-gather_update_logs_setupcrontab-script/)

The /usr/local/cpanel/scripts/gather\_update\_logs\_setupcrontab script schedules when to send your server's update logs to WebPros International, LLC for analysis.

[Read more](https://docs.cpanel.net/whm/scripts/the-gather_update_logs_setupcrontab-script/)

* * *

#### [The hook Script](https://docs.cpanel.net/whm/scripts/the-hook-script/)

This script executes standardized hooks.

[Read more](https://docs.cpanel.net/whm/scripts/the-hook-script/)

* * *

#### [The install\_dovecot\_fts Script](https://docs.cpanel.net/whm/scripts/the-install_dovecot_fts-script/)

This script enables Internet Message Access Protocol (IMAP) Full-Text Search (FTS) Indexing (powered by Apache Solr™), which provides fast search capabilities for IMAP mailboxes.

[Read more](https://docs.cpanel.net/whm/scripts/the-install_dovecot_fts-script/)
