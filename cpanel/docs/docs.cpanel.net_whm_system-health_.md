## System Health

* * *

#### [Background Process Killer](https://docs.cpanel.net/whm/system-health/background-process-killer/)

This interface allows you to select processes that the system will terminate when the upcp script calls the system maintenance script every night.

[Read more](https://docs.cpanel.net/whm/system-health/background-process-killer/)

* * *

#### [Process Manager](https://docs.cpanel.net/whm/system-health/process-manager/)

This interface displays your server's active processes, and allows you to trace and kill any processes.

[Read more](https://docs.cpanel.net/whm/system-health/process-manager/)

* * *

#### [Show Current Disk Usage](https://docs.cpanel.net/whm/system-health/show-current-disk-usage/)

This feature displays your server's current partition and hard disk usage.

[Read more](https://docs.cpanel.net/whm/system-health/show-current-disk-usage/)

* * *

#### [Show Current Running Processes](https://docs.cpanel.net/whm/system-health/show-current-running-processes/)

This interface displays a table of all of the active processes on your server.

[Read more](https://docs.cpanel.net/whm/system-health/show-current-running-processes/)
