# greylisting

* * *

#### [Common Mail Service IP Addresses](https://docs.cpanel.net/knowledge-base/email/common-mail-service-ip-addresses/)

Add these IP addresses and ranges to the Trusted Hosts in WHM's Greylisting to prevent them from being identified as spam.

[Read more](https://docs.cpanel.net/knowledge-base/email/common-mail-service-ip-addresses/)

* * *

#### [Configure Greylisting](https://docs.cpanel.net/cpanel/email/configure-greylisting/)

This interface allows you to enable or disable greylisting for each domain on your account. Greylisting defends email users against spam.

[Read more](https://docs.cpanel.net/cpanel/email/configure-greylisting/)

* * *

#### [Greylisting](https://docs.cpanel.net/whm/email/greylisting/)

This interface allows you to configure Greylisting, a service that protects your server against unwanted email or spam.

[Read more](https://docs.cpanel.net/whm/email/greylisting/)

* * *

#### [The manage\_greylisting Script](https://docs.cpanel.net/whm/scripts/the-manage-greylisting-script/)

The script manages the cPanel Greylisting service.

[Read more](https://docs.cpanel.net/whm/scripts/the-manage-greylisting-script/)

* * *

#### [The setup\_greylist\_db Script](https://docs.cpanel.net/whm/scripts/the-setup_greylist_db-script/)

The setup\_grelist\_db script sets up the SQLite database for the cPanel Greylisting service.

[Read more](https://docs.cpanel.net/whm/scripts/the-setup_greylist_db-script/)
