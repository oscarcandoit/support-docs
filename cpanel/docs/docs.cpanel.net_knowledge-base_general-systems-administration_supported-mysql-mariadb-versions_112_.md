## Overview

This document provides the supported MySQL® and MariaDB® versions for each [operating system](https://docs.cpanel.net/installation-guide/system-requirements) that cPanel & WHM supports.

If you do not [customize your installation](https://docs.cpanel.net/installation-guide/customize-your-installation/), cPanel & WHM installs MySQL 8.0 by default.

## MySQL supported versions

Warning:

While your existing systems may use MySQL 5.7, you **cannot** select this version for a new installation. You also **cannot** modify your MySQL version to use 5.7. For more information, read our [cPanel Deprecation Plan](https://docs.cpanel.net/knowledge-base/cpanel-product/cpanel-deprecation-plan).

The following table shows the MySQL database versions available for your operating system:

| Operating system | OS version number | Available relational database version number |
| --- | --- | --- |
| AlmaLinux OS | 8 | `8.0` |
| AlmaLinux | 9 | `8.0` |
| CloudLinux™ | 8 | `8.0` |
| Rocky Linux™ | 8 | `8.0` |
| Rocky Linux | 9 | `8.0` |
| Ubuntu® | 20 | `8.0` |

## MariaDB supported versions

Warning:

While your existing systems may use MariaDB 10.0, 10.1, or 10.2, you **cannot** select any of these versions for a new installation. For more information, read our [cPanel Deprecation Plan](https://docs.cpanel.net/knowledge-base/cpanel-product/cpanel-deprecation-plan).

The following table shows the MariaDB database versions available for your operating system:

| Operating system | OS version number | Available relational database version number |
| --- | --- | --- |
| AlmaLinux | 8 | `10.3`, `10.5`, `10.6` |
| AlmaLinux | 9 | `10.5`, `10.6` |
| CloudLinux | 8 | `10.3`, `10.5`, `10.6` |
| Rocky Linux | 8 | `10.3`, `10.5`, `10.6` |
| Rocky Linux | 9 | `10.5`, `10.6` |
| Ubuntu | 20 | `10.5`, `10.6` |
