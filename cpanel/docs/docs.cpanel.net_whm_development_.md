## Development

* * *

#### [API Shell for WHM](https://docs.cpanel.net/whm/development/api-shell-for-whm/)

Allows you to call WHM API 1 functions from the UI.

[Read more](https://docs.cpanel.net/whm/development/api-shell-for-whm/)

* * *

#### [Apps Managed by AppConfig](https://docs.cpanel.net/whm/development/apps-managed-by-appconfig/)

This interface displays applications that you installed and registered with AppConfig.

[Read more](https://docs.cpanel.net/whm/development/apps-managed-by-appconfig/)

* * *

#### [cPanel Plugin File Generator](https://docs.cpanel.net/whm/development/cpanel-plugin-file-generator/)

This interface generates a plugin installation file for the cPanel interface's theme.

[Read more](https://docs.cpanel.net/whm/development/cpanel-plugin-file-generator/)

* * *

#### [Manage API Tokens in WHM](https://docs.cpanel.net/whm/development/manage-api-tokens-in-whm/)

This interface lets you to create, list, update, and revoke API tokens.

[Read more](https://docs.cpanel.net/whm/development/manage-api-tokens-in-whm/)

* * *

#### [Manage Hooks](https://docs.cpanel.net/whm/development/manage-hooks/)

This interface allows you to perform a variety of actions on the hooks that exist on your server.

[Read more](https://docs.cpanel.net/whm/development/manage-hooks/)
