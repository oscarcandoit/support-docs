# cpanelui

* * *

#### [Email Routing](https://docs.cpanel.net/cpanel/email/email-routing/)

This interface allows you to configure how the system routes a domain's incoming mail.

[Read more](https://docs.cpanel.net/cpanel/email/email-routing/)

* * *

#### [Encryption](https://docs.cpanel.net/cpanel/email/encryption/)

This interface allows you create, store, and manage GnuPG keys.

[Read more](https://docs.cpanel.net/cpanel/email/encryption/)

* * *

#### [Error Pages](https://docs.cpanel.net/cpanel/advanced/error-pages/)

Error pages inform visitors about problems when they attempt to access your site.

[Read more](https://docs.cpanel.net/cpanel/advanced/error-pages/)

* * *

#### [Errors](https://docs.cpanel.net/cpanel/metrics/errors/)

The interface displays up to 300 of the most recent entries in your web server's error logs, in reverse chronological order.

[Read more](https://docs.cpanel.net/cpanel/metrics/errors/)

* * *

#### [File and Directory Restoration for cPanel](https://docs.cpanel.net/cpanel/files/file-and-directory-restoration-for-cpanel/)

The File and Directory Restoration interface allows you to restore items from local backup sources.

[Read more](https://docs.cpanel.net/cpanel/files/file-and-directory-restoration-for-cpanel/)

* * *

#### [File Manager](https://docs.cpanel.net/cpanel/files/file-manager/)

Use the File Manager interface to manage and edit your files.

[Read more](https://docs.cpanel.net/cpanel/files/file-manager/)

* * *

#### [Forwarders](https://docs.cpanel.net/cpanel/email/forwarders/)

This interface allows you to configure an email address to forward copies of incoming emails to another address.

[Read more](https://docs.cpanel.net/cpanel/email/forwarders/)

* * *

#### [FTP Accounts](https://docs.cpanel.net/cpanel/files/ftp-accounts/)

You can use this interface to create and manage your website's File Transfer Protocol (FTP) accounts.

[Read more](https://docs.cpanel.net/cpanel/files/ftp-accounts/)

* * *

#### [FTP Connections](https://docs.cpanel.net/cpanel/files/ftp-connections/)

This interface displays information about current connections to your FTP server and allows you to terminate those connections.

[Read more](https://docs.cpanel.net/cpanel/files/ftp-connections/)

* * *

#### [Git™ Version Control](https://docs.cpanel.net/cpanel/files/git-version-control/)

This feature allows you to easily host Git repositories on your cPanel account.

[Read more](https://docs.cpanel.net/cpanel/files/git-version-control/)

* * *

#### [Gitweb](https://docs.cpanel.net/cpanel/files/gitweb/)

Gitweb allows you to browse, view, and review your repositories from within a simple interface.

[Read more](https://docs.cpanel.net/cpanel/files/gitweb/)

* * *

#### [Global Email Filters](https://docs.cpanel.net/cpanel/email/global-email-filters/)

This interface allows you to configure how your server filters all of the email that your domains' email accounts receive.

[Read more](https://docs.cpanel.net/cpanel/email/global-email-filters/)

* * *

#### [Hotlink Protection](https://docs.cpanel.net/cpanel/security/hotlink-protection/)

A hotlink occurs when someone embeds content from your site in another site and uses your bandwidth to serve the files. You can use this interface to prevent this issue.

[Read more](https://docs.cpanel.net/cpanel/security/hotlink-protection/)

* * *

#### [How to Set Up a cPanel Email Account on iOS® and Android™](https://docs.cpanel.net/knowledge-base/email/how-to-set-up-a-cpanel-email-account-on-ios-and-android/)

This article provides instructions to set up a cPanel email account on an iOS® or Android™ device.

[Read more](https://docs.cpanel.net/knowledge-base/email/how-to-set-up-a-cpanel-email-account-on-ios-and-android/)

* * *

#### [HTML Editor (Beta)](https://docs.cpanel.net/cpanel/files/html-editor/)

The HTML Editor (Beta) interface allows you to easily edit your HTML documents from the File Manager interface.

[Read more](https://docs.cpanel.net/cpanel/files/html-editor/)

* * *

#### [Images](https://docs.cpanel.net/cpanel/files/images/)

The Images interface allows you to modify and manage images that you saved to your account.

[Read more](https://docs.cpanel.net/cpanel/files/images/)

* * *

#### [Indexes](https://docs.cpanel.net/cpanel/advanced/indexes/)

Use the Indexes interface to define how the server displays a specific directory's index to a visitor, or to disable index display for a directory.

[Read more](https://docs.cpanel.net/cpanel/advanced/indexes/)

* * *

#### [IP Blocker](https://docs.cpanel.net/cpanel/security/ip-blocker/)

The IP Blocker interface allows you to block access to your website for one or more IP addresses or fully qualified domain names (FQDNs).

[Read more](https://docs.cpanel.net/cpanel/security/ip-blocker/)

* * *

#### [Leech Protection](https://docs.cpanel.net/cpanel/security/leech-protection/)

The Leech Protection interface allows you to detect unusual levels of activity in password-restricted directories.

[Read more](https://docs.cpanel.net/cpanel/security/leech-protection/)

* * *

#### [Mailing Lists](https://docs.cpanel.net/cpanel/email/mailing-lists/)

Mailing lists allow you to use a single address to send mail to multiple email addresses.

[Read more](https://docs.cpanel.net/cpanel/email/mailing-lists/)
