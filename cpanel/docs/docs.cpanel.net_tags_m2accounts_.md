# m2accounts

* * *

#### [Account History](https://docs.cpanel.net/manage2/billing/account-history/)

The Account History interface displays your account's billing history.

[Read more](https://docs.cpanel.net/manage2/billing/account-history/)

* * *

#### [Add User](https://docs.cpanel.net/manage2/users/add-user/)

This interface allows you to create additional Manage2 user accounts, each with their own Manage2 login information and permissions.

[Read more](https://docs.cpanel.net/manage2/users/add-user/)

* * *

#### [API Authorization Keys](https://docs.cpanel.net/manage2/admin/api-authorization-keys/)

This interface allows you to assign privileges for an IP address.

[Read more](https://docs.cpanel.net/manage2/admin/api-authorization-keys/)

* * *

#### [API Pickup Passphrases](https://docs.cpanel.net/manage2/admin/api-pickup-passphrases/)

The API Pickup Passphrases interface allows you to create and manage the pickup phrases to use when you authenticate with your Manage2 account.

[Read more](https://docs.cpanel.net/manage2/admin/api-pickup-passphrases/)

* * *

#### [Change Password](https://docs.cpanel.net/manage2/security/change-password/)

This feature allows you to change the password for your Manage2 account.

[Read more](https://docs.cpanel.net/manage2/security/change-password/)

* * *

#### [Edit Users](https://docs.cpanel.net/manage2/users/edit-users/)

This interface allows you to modify the information for a specific user, and the account's privileges

[Read more](https://docs.cpanel.net/manage2/users/edit-users/)

* * *

#### [Manage Users](https://docs.cpanel.net/manage2/users/manage-users/)

This interface allows you to manage users and their privileges.

[Read more](https://docs.cpanel.net/manage2/users/manage-users/)

* * *

#### [Manage2 API Function - List License Information](https://docs.cpanel.net/manage2/api/manage2-api-list-license-information/)

This function returns a list of the Manage2 account's licenses.

[Read more](https://docs.cpanel.net/manage2/api/manage2-api-list-license-information/)

* * *

#### [Manage2 API Function - Reactivate Expired Licenses](https://docs.cpanel.net/manage2/api/manage2-api-reactivate-expired-licenses/)

This function reactivates an account's expired cPanel license.

[Read more](https://docs.cpanel.net/manage2/api/manage2-api-reactivate-expired-licenses/)

* * *

#### [Search for a Group or Direct Account](https://docs.cpanel.net/manage2/direct/search-for-a-group-or-direct-account/)

This interface allows you to search for an individual Group or Direct Account.

[Read more](https://docs.cpanel.net/manage2/direct/search-for-a-group-or-direct-account/)

* * *

#### [Support Usage Report](https://docs.cpanel.net/manage2/support/support-usage-report/)

This interface provides a detailed overview for support incident tickets usage.

[Read more](https://docs.cpanel.net/manage2/support/support-usage-report/)

* * *

#### [Transfer a license](https://docs.cpanel.net/manage2/licenses/transfer-a-license/)

This interface allows you to transfer an active license from the current IP address to a new IP address.

[Read more](https://docs.cpanel.net/manage2/licenses/transfer-a-license/)

* * *

#### [Update Company Information](https://docs.cpanel.net/manage2/company/update-company-information/)

This interface allows you to view or update your company information.

[Read more](https://docs.cpanel.net/manage2/company/update-company-information/)

* * *

#### [Update My Information](https://docs.cpanel.net/manage2/users/update-my-information/)

This interface allows you to update the information for your own user account.

[Read more](https://docs.cpanel.net/manage2/users/update-my-information/)
