## How to Purchase a WP Guardian (cPanel Addon) License

Last modified: _2024 May 28_

* * *

## Overview

cPanel Partners can purchase a WP Guardian (cPanel Addon) license that includes both WP Toolkit Deluxe and WP Guardian’s advanced security measures.

- [WP Toolkit](https://docs.cpanel.net/knowledge-base/cpanel-developed-plugins/wp-toolkit) provides a feature-rich management solution for WordPress® installations. It includes AI-powered automated updates, easy site migrations, plugin management, theme management, and other features.
- [WP Guardian](https://wpguardian.io/) provides effective security for WordPress websites. It includes server-level scanning, flexible site management, virtual patching, and complete infrastructure analysis.

## Purchase a WP Guardian (cPanel Addon) license from Manage2

Important:

To buy a WP Guardian (cPanel Addon) license through Manage2, you **must** be a cPanel Partner.

To purchase a WP Guardian (cPanel Addon) license, perform the following steps:

1. Log in to the [Manage2](https://manage2.cpanel.net/) interface.
2. In the _Licenses_ section, click _Add Licenses_.
3. Select the group to which you wish to add the WP Guardian (cPanel Addon) license.
4. Select the desired WP Guardian (cPanel Addon) package.
5. Enter the IPv4 address to associate with the license.
6. Click _Add Licenses_.

A success message containing the activation code will appear at the top of the Manage2 interface. You will also receive an email message with more details about your license.

For more information, read our [Add License](https://docs.cpanel.net/manage2/licenses/add-license) documentation.
