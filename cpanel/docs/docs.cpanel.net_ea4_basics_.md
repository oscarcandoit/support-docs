## Basics

* * *

#### [About EasyApache 4](https://docs.cpanel.net/ea4/basics/about-easyapache-4/)

EasyApache 4 installs, configures, updates, and validates your web server and its components, including PHP.

[Read more](https://docs.cpanel.net/ea4/basics/about-easyapache-4/)

* * *

#### [EasyApache 4 Create a Profile](https://docs.cpanel.net/ea4/basics/easyapache-4-create-a-profile/)

This document discusses how to create an EasyApache 4 profile.

[Read more](https://docs.cpanel.net/ea4/basics/easyapache-4-create-a-profile/)

* * *

#### [EasyApache 4 File System Layout](https://docs.cpanel.net/ea4/basics/easyapache-4-file-system-layout/)

This document provides an abridged list of the main EasyApache 4 file locations.

[Read more](https://docs.cpanel.net/ea4/basics/easyapache-4-file-system-layout/)

* * *

#### [EasyApache 4 Recommendations](https://docs.cpanel.net/ea4/basics/easyapache-4-recommendations/)

The EasyApache 4 interface gives you a variety of information about the packages that you can install. This could include security advice or notes about your configuration.

[Read more](https://docs.cpanel.net/ea4/basics/easyapache-4-recommendations/)

#### [Package Manager Basics](https://docs.cpanel.net/ea4/basics/package-manager-basics/)

Package managers allow you to easily update, install, and remove packages on your system.

[Read more](https://docs.cpanel.net/ea4/basics/package-manager-basics/)
