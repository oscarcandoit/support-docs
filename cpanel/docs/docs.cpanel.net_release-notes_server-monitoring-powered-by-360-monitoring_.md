## Server Monitoring (Powered by 360 Monitoring)

Last modified: _2025 January 9_

* * *

Note:

WebPros International, LLC currently provides this feature in a limited release, with plans to expand its availability in future updates.

## Server Monitoring 1.2.2

_Released January 12, 2025_

#### Added Server Monitoring to the Feature Showcase

[Server Monitoring (Powered by 360 Monitoring)](https://docs.cpanel.net/whm/plugins/server-monitoring/) now appears in the _Feature Showcase_ interface.

#### New WHM API 1 functions

- [`EcosystemFeatures-local_enable`](https://api.docs.cpanel.net/openapi/whm/operation/local_enable/) — This function allows you to locally enable Server Monitoring, even if your hosting provider has disabled it by default.
- [`EcosystemFeatures-local_disable`](https://api.docs.cpanel.net/openapi/whm/operation/local_disable/) — This function allows you to locally disable Server Monitoring, even if your hosting provider has enabled it by default.

## Server Monitoring version 1.0

_Released October 2024_

#### Added the Server Monitoring panel to the WHM interface

In cPanel & WHM version 118, we added the _Server Monitoring_ section to the [WHM interface](https://docs.cpanel.net/whm/the-whm-interface). The _Server Monitoring_ section displays statistics and metrics for your server’s performance at a glance. Here you can see details such as _CPU Usage_, _Load Averages_, _Top Processes_ and more. For a more detailed view, click _View Reports_ to be taken to the [_Server Monitoring (Powered by 360 Monitoring_](https://docs.cpanel.net/whm/plugins/server-monitoring) interface ( _WHM » Plugins » Server Monitoring_).

#### Added the Server Monitoring (Powered by 360 Monitoring) interface

We added the [_Server Monitoring_](https://docs.cpanel.net/whm/plugins/server-monitoring) interface ( _WHM » Plugins » Server Monitoring_) to cPanel & WHM. The _Server Monitoring (powered by 360 Monitoring)_ interface provides real-time insights into the performance, availability, and health of your websites, servers, and applications. It tracks key metrics like uptime, response times, and server resources to spot issues before they affect users. This interface helps businesses maintain optimal performance by providing clear reports and alerts, allowing them to fix problems quickly.
