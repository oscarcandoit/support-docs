Looking for this interface?

Note:

Your [hosting provider](https://docs.cpanel.net/knowledge-base/cpanel-product/how-to-identify-your-hosting-provider/) can enable or disable this interface for resellers in WHM's [_Edit Reseller Nameservers and Privileges_](https://docs.cpanel.net/whm/resellers/edit-reseller-nameservers-and-privileges) interface ( _WHM >> Home >> Resellers >> Edit Reseller Nameservers and Privileges_).

## Overview

This interface allows you to list and manage accounts that exceed their disk quota.

## List accounts

The interface will display a table with accounts that exceed their disk quota. The table contains the following columns:

- _User_ — The account’s domain name and user name.
- _Space Used_ — The amount of disk space that the account uses.
- _Quota_ — The account’s quota.

Note:

Click _Show only this reseller’s accounts_ to display only accounts which belong to that reseller.


### Change quota

To change an account’s quota, perform the following steps:

1. Enter the new quota for the account in the appropriate text box, in megabytes (MB), or select _Unlimited_ to grant the user unlimited disk space.
2. Click _Save_. The interface will display a success message.
