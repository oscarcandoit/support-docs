## Software

* * *

#### [Application Manager](https://docs.cpanel.net/cpanel/software/application-manager/)

This interface allows you to deploy applications with the Phusion Passenger® application server.

[Read more](https://docs.cpanel.net/cpanel/software/application-manager/)

* * *

#### [MultiPHP INI Editor for cPanel](https://docs.cpanel.net/cpanel/software/multiphp-ini-editor-for-cpanel/)

This interface allows you to configure your PHP settings.

[Read more](https://docs.cpanel.net/cpanel/software/multiphp-ini-editor-for-cpanel/)

* * *

#### [MultiPHP Manager for cPanel](https://docs.cpanel.net/cpanel/software/multiphp-manager-for-cpanel/)

cPanel's MultiPHP Manager interface allows you to easily manage your account's PHP configuration.

[Read more](https://docs.cpanel.net/cpanel/software/multiphp-manager-for-cpanel/)

* * *

#### [Optimize Website](https://docs.cpanel.net/cpanel/software/optimize-website/)

This interface allows you to configure your server to automatically compress specified types of content when visitors access that content.

[Read more](https://docs.cpanel.net/cpanel/software/optimize-website/)

* * *

#### [Perl Modules](https://docs.cpanel.net/cpanel/software/perl-modules/)

This interface allows you to search for and install Perl modules from the CPAN repository to your website.

[Read more](https://docs.cpanel.net/cpanel/software/perl-modules/)

* * *

#### [PHP PEAR Packages](https://docs.cpanel.net/cpanel/software/php-pear-packages/)

The PHP PEAR Packages interface allows you to search for and add PEAR packages to your website, or view all of your website's available PHP packages.

[Read more](https://docs.cpanel.net/cpanel/software/php-pear-packages/)

* * *

#### [Ruby on Rails](https://docs.cpanel.net/cpanel/software/ruby-on-rails/)

This interface allows you to create and deploy a Ruby on Rails® application.

[Read more](https://docs.cpanel.net/cpanel/software/ruby-on-rails/)

* * *

#### [RubyGems](https://docs.cpanel.net/cpanel/software/rubygems/)

RubyGems is a package manager for the Ruby programming language. It provides a self-contained format in which developers can distribute Ruby programs and libraries.

[Read more](https://docs.cpanel.net/cpanel/software/rubygems/)

* * *

#### [Site Software](https://docs.cpanel.net/cpanel/software/site-software/)

This interface allows you to add, manage, upgrade, and remove cPanel Addons (cPAddons).

[Read more](https://docs.cpanel.net/cpanel/software/site-software/)
