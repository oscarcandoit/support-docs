## The Manage2 Dashboard

Last modified: _2025 June 9_

* * *

## Overview

Manage2’s _Dashboard_ interface provides you with access to features that allow you to manage your cPanel & WHM licenses.

![the Manage2 Dashboard interface](https://docs.cpanel.net/img/manage2dashboard.png)

The Manage2 _Dashboard_ consists of the following sections:

1. [The application toolbar](https://docs.cpanel.net/manage2/admin/the-manage2-dashboard/#the-application-toolbar)
2. [The module toolbar](https://docs.cpanel.net/manage2/admin/the-manage2-dashboard/#the-module-toolbar)
3. [Features](https://docs.cpanel.net/manage2/admin/the-manage2-dashboard/#features)
4. [Server information](https://docs.cpanel.net/manage2/admin/the-manage2-dashboard/#server-information)

Note:

The features that appear in your Manage2 _Dashboard_ interface correspond with your account’s administrator level:

- Higher numbers correlate with greater access privileges and more available features. You can locate your administrator level in the Manage2 _Dashboard_ interface’s [_Server information_](https://docs.cpanel.net/manage2/admin/the-manage2-dashboard/#server-information) section.
- This value defaults to `0` for your primary account and defaults to `-10` for all additional accounts.

For information about Manage2 administrator levels, read our [Manage Users](https://docs.cpanel.net/manage2/users/manage-users) documentation.

## The application toolbar

The application toolbar is at the top of the Manage2 _Dashboard_ interface. This toolbar contains the following menu selections:

- The _cPanel_ logo (![the cPanel logo](https://docs.cpanel.net/img/cplogo.png)) — Returns you to the Manage2 _Dashboard_ interface.
- _Manage2_ — Displays the navigation menu for Manage2.
- _Account Manager_ — Displays the account manager’s name and contact information.
- _Documentation_ — Displays a menu of cPanel & WHM documentation resources.
- _Automation_ — Displays the _cPanel License Management API SDK_ and the [API Guide](https://docs.cpanel.net/manage2/knowledge-base/guide-to-the-manage2-api) document links.
- _Profile_ — Displays a menu of user profile options.
- _Help_ — Directs you to the WebPros International, LLC contact details and information page.
- _User menu_ — Displays the _Open Tickets_, _Change Password_, and _Logout_ links.

## The module toolbar

In the module toolbar, the _Dashboard_ interface displays the available [Manage2 feature modules](https://docs.cpanel.net/manage2/). Selecting a module opens a menu of the module’s available [Manage2 features](https://docs.cpanel.net/manage2/admin/the-manage2-dashboard/#features).

The _My Shortcuts_ feature provides a menu of user-defined links to Manage2 interfaces. You can customize the features that appear in this menu in Manage2’s [_Change Preferences_](https://docs.cpanel.net/manage2/users/change-preferences#my-shortcuts) interface ( _Manage2 » Dashboard » Users » Change Preferences_).

## Features

Manage2’s _Dashboard_ interface displays your account’s available features in groups, by functionality. For information about a specific feature, read our [Manage2](https://docs.cpanel.net/manage2/) documentation.

Note:

- The _Announcement_ and _Notices_ modules appear when your account has an update, such as server notices, past-due balances, and account suspensions.
- If you see any account notifications, make **certain** to check your [_Account History_](https://docs.cpanel.net/manage2/billing/account-history/) interface ( _Manage2 » Dashboard » Billing » Account History_) for more information and account balance details.

## Server information

The server information is located at the bottom of the Manage2 _Dashboard_ interface. This section displays the following information:

- Your host information.
- Your database information.
- Your account’s [administrator level](https://docs.cpanel.net/manage2/users/manage-users).
