## How to Set Up a Signature in Webmail

Last modified: _2023 August 2_

* * *

## Overview

An email signature is a personalized block of text at the bottom of an email.

The _[Webmail](https://docs.cpanel.net/webmail/the-webmail-interface)_ interface ( _cPanel_ » _Home_ » _Email Accounts_ » _Check Email_ ) does not have a setting for assigning a signature for your email to use. You will need to configure signatures within the [Webmail](https://docs.cpanel.net/webmail/webmail-clients/) client that you use.

## Configure a signature in Roundcube

To create an email signature in the [Roundcube Webmail](https://docs.cpanel.net/webmail/webmail-clients/#roundcube) client, follow these steps:

1. Log in to Roundcube.
2. In the side menu bar, click _Settings_.
3. In the first panel, click _Identities_.
4. Select the existing identity, or click _Create_ to create a new identity.
5. In the _Signature_ text box, enter the desired signature.
6. Click _Save_.

### Add an image to your signature in Roundcube

If you wish to add an image to your signature in the [Roundcube Webmail](https://docs.cpanel.net/webmail/webmail-clients/#roundcube) client, follow these steps:

01. Follow steps 1-4 in the procedure referenced above.
02. In the upper lefthand corner of the _Signature_ box, click the HTML editor icon (![Switch to HTML editor](https://docs.cpanel.net/img/switch-to-html.png)).
03. In the toolbar, click the insert/edit image icon (![Roundcube insert/edit image icon](https://docs.cpanel.net/img/insert-roundcube-sig-image.png)).
04. In the _Insert/edit image_ window, click the folder icon (![Roundcube image folder icon](https://docs.cpanel.net/img/roundcube-open-image.png)).
05. Click _Add image_.
06. Select the image file to upload.






    Note:





- The image file size must be smaller than the maximum allowed file size displayed in the interface.
- The image file must be in JPEG or GIF format.

07. In the bottom righthand corner, click _Close_.
08. In the _Select filename_ window, click the image name.
09. Enter the desired dimensions for the image.
10. Click _OK_.
11. Underneath the _Signature_ box, click _Save_.

Important:

The image only appears in the signature if you use the HTML editor to compose messages. To compose all messages with HTML in Roundcube, follow these steps:

1. Go to _Settings » Preferences » Composing Messages_.
2. Under the _Main Options_ setting, in the _Compose HTML messages_ menu, select _always_.
3. Click _Save_.
