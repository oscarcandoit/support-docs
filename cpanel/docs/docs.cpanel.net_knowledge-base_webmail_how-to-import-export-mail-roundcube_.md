## How to Import and Export Mail in Roundcube

Last modified: _2024 November 7_

* * *

## Overview

This document explains how to import and export mail using Roundcube.

## Access the Roundcube interface

To access the Roundcube interface, perform the following steps:

1. [Log in](https://docs.cpanel.net/knowledge-base/accounts/how-to-log-in-to-your-server-or-account/#how-to-access-webmail) to your Webmail account.
2. Ensure that you are in the Roundcube interface.
   - If this is the first login, the _Webmail_ interface will appear. If Roundcube is your default webmail client, click the Roundcube logo under the _Open your inbox_ section. Otherwise, select the Roundcube logo from the _Change your webmail client_ section. The Roundcube interface will appear.

## Import mail

Before you begin, make sure you have a file that contains the mail to import. You can use files in [MIME](https://en.wikipedia.org/wiki/Email#Message_format) or [Mbox](https://en.wikipedia.org/wiki/Mbox) format.

Note:

Files over 50 megabytes cause import errors. You may need to split large files.

To import mail into Roundcube, perform the following steps:

1. [Access the Roundcube interface](https://docs.cpanel.net/knowledge-base/webmail/how-to-import-export-mail-roundcube/#access-the-roundcube-interface).
2. In the Roundcube interface, select _More_, then select _Import_.
3. To upload your file, click _Browse_ and select your file.
4. Click _Import_. When the import is complete, you will see a success message.

## Export mail

To export mail from Roundcube, perform the following steps:

1. [Access the Roundcube interface](https://docs.cpanel.net/knowledge-base/webmail/how-to-import-export-mail-roundcube/#access-the-roundcube-interface).
2. In the Roundcube interface, select the message to export.
3. Select _More_, then select _Export_. Your browser downloads the message to your computer as an `.eml` file.






Note:





You can **only** download one message at a time.
