## API Authorization Keys

Last modified: _2020 May 15_

* * *

## Overview

The _API Authorization Keys_ interface allows you to assign privileges for an IP address.

## Authorization keys table

If you have no authorization keys, the interface will display the _There are no authorization keys_ message.

If you have authorization keys, the authorization keys table will appear. The authorization keys table will display the following information:

- _IP_ — The IP address associated with the account.
- _Service_ — The name of the service that uses the authentication scripts.
- _Privileges_ — The privileges available to assign to the IP address. You can assign any of the following privileges:

  - _Add a New License_ — Allows the user to add a new license.
  - _cPanel Security Verification Program_ — Allows the user to access the _Security Verification System_ interface to look up an IP address or license ID.
  - _Expire a License_ — Allows the user to expire a license.
  - _Extend One Time License Updates_ — Allows the user to extend one time license updates.
  - _List active licenses_ — Allows the user to view a list of active licenses.
  - _Manage Authkeys_ — Allows the user to manage authorization keys.
  - _Modify/Transfer a License_ — Allows the user to modify or transfer a license.
  - _Reactivate a License_ — Allows the user to reactivate a license.
  - _Search for a License_ — Allows the user to search for a license.
- _Remove?_ — Select _Remove_ to remove all privileges for that IP address.

## Assign and remove privileges

To assign privileges, perform the following steps:

1. Find the IP address to which you wish to assign privileges.
2. Select the desired privileges.
3. Click _Submit authkey modifications_.

To remove privileges, perform the following steps:

1. Find the IP address to which you wish to remove privileges.
2. Deselect the privileges you wish to remove. If you wish to remove all privileges, select _Remove_.
3. Click _Submit authkey modifications_.
