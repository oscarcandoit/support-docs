## Licenses Activated in the Past 48 Hours

Last modified: _2023 February 3_

* * *

## Overview

This interface displays licenses that you added to your account in the past 48 hours.

## License information

You can view the following information about each new license:

- _Group_ — The group to which the license is assigned.






Note:





Click on a group name to view that group’s _View Group Licenses_ interface _(Dashboard » Groups » View Group Licenses)_.

- _Package_ — The package to which the license is assigned.
- _$$_ — The monthly charge for the license.
- _Hostname_ — The hostname associated with the license.
- _Ip_ — The IP address associated with the license.
- _Contact_ — The contact listed for the license.
- _Start Date_ — The date on which the license will start.
