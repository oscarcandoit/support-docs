# cpanelui

* * *

#### [Awstats](https://docs.cpanel.net/cpanel/metrics/awstats/)

The Awstats interface displays traffic statistics from the Advanced Web Statistics (AWStats) software, which compiles information about how users access your website.

[Read more](https://docs.cpanel.net/cpanel/metrics/awstats/)

* * *

#### [Backup for cPanel](https://docs.cpanel.net/cpanel/files/backup-for-cpanel/)

This interface allows you to download and store your site in a backup file.

[Read more](https://docs.cpanel.net/cpanel/files/backup-for-cpanel/)

* * *

#### [Backup Wizard](https://docs.cpanel.net/cpanel/files/backup-wizard/)

This interface allows you to back up all or part of your website, or to restore it from the most recent backup file.

[Read more](https://docs.cpanel.net/cpanel/files/backup-wizard/)

* * *

#### [Bandwidth](https://docs.cpanel.net/cpanel/metrics/bandwidth/)

The Bandwidth interface displays bandwidth usage information in several sets of graphs.

[Read more](https://docs.cpanel.net/cpanel/metrics/bandwidth/)

* * *

#### [BoxTrapper](https://docs.cpanel.net/cpanel/email/boxtrapper/)

The BoxTrapper feature filters spam from your inbox through challenge-response verification.

[Read more](https://docs.cpanel.net/cpanel/email/boxtrapper/)

* * *

#### [Change Language](https://docs.cpanel.net/cpanel/preferences/change-language/)

You can choose to view the cPanel interface in many different languages.

[Read more](https://docs.cpanel.net/cpanel/preferences/change-language/)

* * *

#### [Configure Greylisting](https://docs.cpanel.net/cpanel/email/configure-greylisting/)

This interface allows you to enable or disable greylisting for each domain on your account. Greylisting defends email users against spam.

[Read more](https://docs.cpanel.net/cpanel/email/configure-greylisting/)

* * *

#### [Contact Information](https://docs.cpanel.net/cpanel/preferences/contact-information/)

Use this interface to store contact information for your cPanel account and to set your contact preferences.

[Read more](https://docs.cpanel.net/cpanel/preferences/contact-information/)

* * *

#### [cPanel Glossary](https://docs.cpanel.net/knowledge-base/cpanel-product/cpanel-glossary/)

A list of common terms, web hosting terms, and terms in cPanel & WHM.

[Read more](https://docs.cpanel.net/knowledge-base/cpanel-product/cpanel-glossary/)

* * *

#### [CPU and Concurrent Connection Usage](https://docs.cpanel.net/cpanel/metrics/cpu-and-concurrent-connection-usage/)

Use this interface to view and monitor resource usage.

[Read more](https://docs.cpanel.net/cpanel/metrics/cpu-and-concurrent-connection-usage/)

* * *

#### [Create a New Domain](https://docs.cpanel.net/cpanel/domains/domains/create-a-new-domain/)

Use this interface to create a domain on your cPanel account.

[Read more](https://docs.cpanel.net/cpanel/domains/domains/create-a-new-domain/)

* * *

#### [Create an Email Account](https://docs.cpanel.net/cpanel/email/create-an-email-account/)

This interface lets you create a new email account.

[Read more](https://docs.cpanel.net/cpanel/email/create-an-email-account/)

* * *

#### [Cron Jobs](https://docs.cpanel.net/cpanel/advanced/cron-jobs/)

Cron jobs are scheduled tasks that the system runs at predefined times or intervals.

[Read more](https://docs.cpanel.net/cpanel/advanced/cron-jobs/)

* * *

#### [Database Wizard](https://docs.cpanel.net/cpanel/databases/database-wizard/)

This wizard guides you through the setup of a MySQL® or MariaDB® database, user accounts, and user privileges.

[Read more](https://docs.cpanel.net/cpanel/databases/database-wizard/)

* * *

#### [Default Address](https://docs.cpanel.net/cpanel/email/default-address/)

This interface allows you to set up a domain default address (catch-all address) that receives any mail for an invalid email address for the domain.

[Read more](https://docs.cpanel.net/cpanel/email/default-address/)

* * *

#### [Directory Privacy](https://docs.cpanel.net/cpanel/files/directory-privacy/)

This interface allows you to protect specific directories in your cPanel account's files.

[Read more](https://docs.cpanel.net/cpanel/files/directory-privacy/)

* * *

#### [Disk Usage](https://docs.cpanel.net/cpanel/files/disk-usage/)

This interface displays information about your files, your directories, and how you use your account's disk space.

[Read more](https://docs.cpanel.net/cpanel/files/disk-usage/)

* * *

#### [Dynamic DNS](https://docs.cpanel.net/cpanel/domains/dynamic-dns/)

Use this interface to manage your dynamic DNS (DDNS) domains.

[Read more](https://docs.cpanel.net/cpanel/domains/dynamic-dns/)

* * *

#### [Email Disk Usage](https://docs.cpanel.net/cpanel/email/email-disk-usage/)

This interface allows you to view all of your mailboxes and remove old or large messages.

[Read more](https://docs.cpanel.net/cpanel/email/email-disk-usage/)

* * *

#### [Email Filters](https://docs.cpanel.net/cpanel/email/email-filters/)

This interface allows you to filter the email for each email account in your domain.

[Read more](https://docs.cpanel.net/cpanel/email/email-filters/)
