## Overview

The _WHM Marketplace_ interface allows you to renew, install, uninstall and purchase third-party software licenses and products for your server.

## Third Party Software

This interface displays a table with the following information:

- _Product_ — The available product’s name.

- _Status_ — The status of your product’s license:
  - _Active_ — The license is active.
  - _Inactive_ — The license is expired.
- _Expiration Date_ — The date the product’s license expired.

- _Manage_ — Access the product’s license in the [Manage2](https://docs.cpanel.net/manage2/admin/the-manage2-dashboard/) interface.







Note:





If you are not a cPanel Partner, the system will redirect you to the cPanel Store.

- _Install_ — Click _Install_ to install a third-party application. A new installation interface will appear.


Note:

Click the more icon (![More](https://docs.cpanel.net/img/columnmore.png)) to display more information about a product and license. This will display the following information:

- _Product Description_ — A brief explanation of the product.
- _Activation Date_ — The product license’s activation date.

## Adding Extensions

Click _Add Extensions_ to add extensions to the _WHM Marketplace_.
The _Add Extensions_ interface displays a table with the following information:

- _Featured_ — The featured product.
- _Extension_ — The available extension’s name.
- _Paid_ — Indicates if the product is a paid or free version.

  - _Purchase_ — Click _Purchase_ to purchase the extension via the _Manage 2_ interface.
  - _Install_ — Click _Install_ to install a free or purchased extension.
