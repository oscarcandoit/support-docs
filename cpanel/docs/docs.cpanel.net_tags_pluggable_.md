#### [cPanelID](https://docs.cpanel.net/knowledge-base/accounts/cpanelid/)

cPanelID is an integrated set of credentials that customers can use to log in to various cPanel websites and services.

[Read more](https://docs.cpanel.net/knowledge-base/accounts/cpanelid/)

* * *

#### [Manage External Authentications](https://docs.cpanel.net/whm/security-center/manage-external-authentications/)

The Manage External Authentications interface allows you to manage your server's OpenID® Connect™-compliant identity providers.

[Read more](https://docs.cpanel.net/whm/security-center/manage-external-authentications/)

* * *

#### [WebPros Account](https://docs.cpanel.net/knowledge-base/accounts/webpros-account/)

WebPros Account is an integrated set of credentials that server administrators can use to log in to cPanel & WHM servers.

[Read more](https://docs.cpanel.net/knowledge-base/accounts/webpros-account/)
