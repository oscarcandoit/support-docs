## Reseller Center

_Valid for versions 82 through the latest version_

#### Version:

#### [82](https://docs.cpanel.net/whm/resellers/reseller-center/)

Last modified: _2024 June 13_

* * *

Looking for this interface?

Note:

Your [hosting provider](https://docs.cpanel.net/knowledge-base/cpanel-product/how-to-identify-your-hosting-provider/) can enable or disable this interface for resellers in WHM's [_Edit Reseller Nameservers and Privileges_](https://docs.cpanel.net/whm/resellers/edit-reseller-nameservers-and-privileges) interface ( _WHM >> Home >> Resellers >> Edit Reseller Nameservers and Privileges_).

## Overview

This interface adds or removes reseller status for existing accounts. You can also use this interface to access WHM’s other reseller-related interfaces. For more information about resellers, read our [Guide to Reseller Accounts](https://docs.cpanel.net/knowledge-base/accounts/guide-to-reseller-accounts/) documentation.

Note:

To create a new account, use WHM’s [_Create a New Account_](https://docs.cpanel.net/whm/account-functions/create-a-new-account) interface ( _WHM » Home » Account Functions » Create a New Account_).

## Add or Remove Reseller Account Privileges

This section of the interface lets you edit an account’s reseller status. You can perform the following actions:

- _Add Reseller Account Privileges_ — Select the account to which to grant reseller status and click _Add_. If you want the new reseller to own their own account, select the _Make the new reseller own their own account_ checkbox.

- _Remove Reseller Account Privileges_ — Select the reseller account from which to remove reseller status and click _Remove_.


## Change Account Ownership or Email Resellers

In this section, you can choose from the following actions:

- _Change Ownership of an Account_ — Redirects you to WHM’s [_Modify an Account_](https://docs.cpanel.net/whm/account-functions/modify-an-account) interface ( _WHM » Home » Account Functions » Modify an Account_).

- _Email All Resellers_ — Redirects you to WHM’s [_Email All Resellers_](https://docs.cpanel.net/whm/resellers/email-all-resellers) interface ( _WHM » Home » Resellers » Email All Resellers_).

- _Reset Resellers’ Permissions_ — Redirects you to WHM’s [_Reset Resellers_](https://docs.cpanel.net/whm/resellers/reset-resellers) interface ( _WHM » Home » Resellers » Reset Resellers_).


## Reseller Actions

In this section, select a reseller account that you want to edit from the _Reseller Actions for_ menu. Then, select from one of the following actions:

- _Edit Reseller Privileges and Nameservers_ — Redirects you to WHM’s [_Edit Reseller Nameservers and Privileges_](https://docs.cpanel.net/whm/resellers/edit-reseller-nameservers-and-privileges) interface ( _WHM » Home » Resellers » Edit Reseller Nameservers and Privileges_).

- _Change Ownership of Multiple Accounts_ — Redirects you to WHM’s [_Change Ownership of Multiple Accounts_](https://docs.cpanel.net/whm/resellers/change-ownership-of-multiple-accounts) interface ( _WHM » Home » Resellers » Change Ownership of Multiple Accounts_).

- _View Reseller Usage and Manage Account Status_ — Redirects you to WHM’s [_View Reseller Usage and Manage Account Status_](https://docs.cpanel.net/whm/resellers/view-reseller-usage-and-manage-account-status) interface ( _WHM » Home » Resellers » View Reseller Usage and Manage Account Status_).

- _Manage IP Address Delegation_ — Redirects you to WHM’s [_Manage Reseller’s IP Delegation_](https://docs.cpanel.net/whm/resellers/manage-resellers-ip-delegation) interface ( _WHM » Home » Resellers » Manage Reseller’s IP Delegation_).

- _Manage Reseller’s Shared IP Address_ — Redirects you to WHM’s [_Manage Reseller’s Shared IP_](https://docs.cpanel.net/whm/resellers/manage-resellers-shared-ip) interface ( _WHM » Home » Resellers » Manage Reseller’s Shared IP_).

- _Log in to WHM_ — Log in to the WHM interface for that account with the `root` or a reseller with `root`-level privileges’s password.

- _Log in to cPanel_ — Log in to the cPanel interface for that account with the `root` or a reseller with `root`-level privileges’s password.
