## HTML Editor (Beta)

_Valid for versions 126 through the latest version_

#### Version:

#### [126](https://docs.cpanel.net/cpanel/files/html-editor/)

Last modified: _2025 September 26_

* * *

Looking for this interface?

Note:

Your [hosting provider](https://docs.cpanel.net/knowledge-base/cpanel-product/how-to-identify-your-hosting-provider/) can enable or disable this interface in WHM's [_Feature Manager_](https://docs.cpanel.net/whm/packages/feature-manager) interface ( _WHM >> Home >> Packages >> Feature Manager_).

## Overview

Experimental:

This interface is experimental and **not** fully functional. We plan to provide a full version in a future release.

The _HTML Editor (Beta)_ interface allows you to easily edit your HTML documents from cPanel’s [_File Manager_](https://docs.cpanel.net/cpanel/files/file-manager) interface ( _cPanel » Home » Files » File Manager_). This visual, [WYSIWYG](https://docs.cpanel.net/knowledge-base/cpanel-product/cpanel-glossary/#wysiwyg) editor includes powerful editing functions of familiar desktop editors (for example, Microsoft® Word).

The _HTML Editor (Beta)_ interface uses [Jodit](https://xdsoft.net/jodit/) functionality. For more information, read [Jodit’s documentation](https://xdsoft.net/jodit/docs/index.html).

Important:

This interface does **not** support JavaScript. Features that rely on JavaScript may not appear or function the same in the HTML Editor as on the live site.
