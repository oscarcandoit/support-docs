## DNS Functions

* * *

#### [Add a DNS Zone](https://docs.cpanel.net/whm/dns-functions/add-a-dns-zone/)

This feature allows you to set up a DNS zone on your server.

[Read more](https://docs.cpanel.net/whm/dns-functions/add-a-dns-zone/)

* * *

#### [Add an A Entry for Your Hostname](https://docs.cpanel.net/whm/dns-functions/add-an-a-entry-for-your-hostname/)

An A entry points a hostname or domain to an IP address, which allows other computers to find your server on the internet.

[Read more](https://docs.cpanel.net/whm/dns-functions/add-an-a-entry-for-your-hostname/)

* * *

#### [Delete a DNS Zone](https://docs.cpanel.net/whm/dns-functions/delete-a-dns-zone/)

This feature allows you to delete DNS zones from your server.

[Read more](https://docs.cpanel.net/whm/dns-functions/delete-a-dns-zone/)

* * *

#### [DNS Zone Manager](https://docs.cpanel.net/whm/dns-functions/dns-zone-manager/)

This feature allows you to edit the records in a domain's DNS zone file.

[Read more](https://docs.cpanel.net/whm/dns-functions/dns-zone-manager/)

* * *

#### [Edit Zone Templates](https://docs.cpanel.net/whm/dns-functions/edit-zone-templates/)

This interface allows you to edit the templates that cPanel & WHM uses to create DNS zone files for new domains.

[Read more](https://docs.cpanel.net/whm/dns-functions/edit-zone-templates/)

* * *

#### [Email DNS Record Manager](https://docs.cpanel.net/whm/dns-functions/email-dns-record-manager/)

Use this interface to add a default DMARC policy to your server's domains.

[Read more](https://docs.cpanel.net/whm/dns-functions/email-dns-record-manager/)

* * *

#### [Email Routing Configuration](https://docs.cpanel.net/whm/dns-functions/email-routing-configuration/)

This interface lets you configure the entry for each destination mail server.

[Read more](https://docs.cpanel.net/whm/dns-functions/email-routing-configuration/)

* * *

#### [Enable DKIM/SPF Globally](https://docs.cpanel.net/whm/dns-functions/enable-dkim-spf-globally/)

This feature creates DKIM and SPF records for all accounts on the server.

[Read more](https://docs.cpanel.net/whm/dns-functions/enable-dkim-spf-globally/)

* * *

#### [Nameserver Record Report](https://docs.cpanel.net/whm/dns-functions/nameserver-record-report/)

The Nameserver Record Report interface lists the server's nameserver information.

[Read more](https://docs.cpanel.net/whm/dns-functions/nameserver-record-report/)

* * *

#### [Park a Domain](https://docs.cpanel.net/whm/dns-functions/park-a-domain/)

A parked domain (or alias) is a secondary domain that points to the user’s primary domain.

[Read more](https://docs.cpanel.net/whm/dns-functions/park-a-domain/)

* * *

#### [Perform a DNS Cleanup](https://docs.cpanel.net/whm/dns-functions/perform-a-dns-cleanup/)

This interface cleans your named configuration file and removes duplicate entries.

[Read more](https://docs.cpanel.net/whm/dns-functions/perform-a-dns-cleanup/)

* * *

#### [Set Zone Time to Live (TTL)](https://docs.cpanel.net/whm/dns-functions/set-zone-time-to-live/)

This interface allows you to set the value for time to live (TTL) in your server's DNS zones.

[Read more](https://docs.cpanel.net/whm/dns-functions/set-zone-time-to-live/)

* * *

#### [Setup/Edit Domain Forwarding](https://docs.cpanel.net/whm/dns-functions/setup-edit-domain-forwarding/)

This feature allows you to redirect one domain to another domain.

[Read more](https://docs.cpanel.net/whm/dns-functions/setup-edit-domain-forwarding/)

* * *

#### [Synchronize DNS Records](https://docs.cpanel.net/whm/dns-functions/synchronize-dns-records/)

This feature compares zone files among servers, recognizes which are the most up-to-date, and updates the servers in the DNS cluster.

[Read more](https://docs.cpanel.net/whm/dns-functions/synchronize-dns-records/)
