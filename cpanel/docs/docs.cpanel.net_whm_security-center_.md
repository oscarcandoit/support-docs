## Security Center

* * *

#### [Apache mod\_userdir Tweak](https://docs.cpanel.net/whm/security-center/apache-mod_userdir-tweak/)

This interface allows you to disable the Apache \`mod\_userdir\` module's functionality for your users.

[Read more](https://docs.cpanel.net/whm/security-center/apache-mod_userdir-tweak/)

* * *

#### [Compiler Access](https://docs.cpanel.net/whm/security-center/compiler-access/)

This interface lets you manage your server's user access to the C and C++ compilers.

[Read more](https://docs.cpanel.net/whm/security-center/compiler-access/)

* * *

#### [Configure Security Policies](https://docs.cpanel.net/whm/security-center/configure-security-policies/)

The Configure Security Policies interface allows you to configure your security policy settings and security policy extensions.

[Read more](https://docs.cpanel.net/whm/security-center/configure-security-policies/)

* * *

#### [cPHulk Brute Force Protection](https://docs.cpanel.net/whm/security-center/cphulk-brute-force-protection/)

This interface allows you to configure cPHulk, a service that provides protection for your server against brute force attacks.

[Read more](https://docs.cpanel.net/whm/security-center/cphulk-brute-force-protection/)

* * *

#### [Host Access Control](https://docs.cpanel.net/whm/security-center/host-access-control/)

Use this interface to allow, reject, or drop access to services for specific IP addresses.

[Read more](https://docs.cpanel.net/whm/security-center/host-access-control/)

* * *

#### [Manage External Authentications](https://docs.cpanel.net/whm/security-center/manage-external-authentications/)

The Manage External Authentications interface allows you to manage your server's OpenID® Connect™-compliant identity providers.

[Read more](https://docs.cpanel.net/whm/security-center/manage-external-authentications/)

* * *

#### [Manage root's SSH Keys](https://docs.cpanel.net/whm/security-center/manage-root-ssh-keys/)

This interface allows you to add, import, and manage the SSH keys on your server.

[Read more](https://docs.cpanel.net/whm/security-center/manage-root-ssh-keys/)

* * *

#### [Manage Sudo Group Users](https://docs.cpanel.net/whm/security-center/manage-sudo-group-users/)

This interface allows you to manage the wheel group's membership.

[Read more](https://docs.cpanel.net/whm/security-center/manage-sudo-group-users/)

* * *

#### [Manage Wheel Group Users](https://docs.cpanel.net/whm/security-center/manage-wheel-group-users/)

This interface allows you to manage the wheel group's membership.

[Read more](https://docs.cpanel.net/whm/security-center/manage-wheel-group-users/)

* * *

#### [ModSecurity® Configuration](https://docs.cpanel.net/whm/security-center/modsecurity-configuration/)

This interface allows you to configure ModSecurity's global settings.

[Read more](https://docs.cpanel.net/whm/security-center/modsecurity-configuration/)

* * *

#### [ModSecurity® Tools](https://docs.cpanel.net/whm/security-center/modsecurity-tools/)

The ModSecurity® Tools interface allows you to install and manage ModSecurity rules.

[Read more](https://docs.cpanel.net/whm/security-center/modsecurity-tools/)

* * *

#### [ModSecurity® Vendors](https://docs.cpanel.net/whm/security-center/modsecurity-vendors/)

The ModSecurity® Vendors interface allows you to install and manage your ModSecurity vendors.

[Read more](https://docs.cpanel.net/whm/security-center/modsecurity-vendors/)

* * *

#### [Password Strength Configuration](https://docs.cpanel.net/whm/security-center/password-strength-configuration/)

This interface allows you to define minimum strengths for passwords for all of cPanel & WHM's features that require password authentication.

[Read more](https://docs.cpanel.net/whm/security-center/password-strength-configuration/)

* * *

#### [Security Advisor](https://docs.cpanel.net/whm/security-center/security-advisor/)

This interface runs a security scan on your server, and it advises you about how to resolve any security issues that it finds.

[Read more](https://docs.cpanel.net/whm/security-center/security-advisor/)

* * *

#### [Security Questions](https://docs.cpanel.net/whm/security-center/security-questions/)

This interface allows you to update and manage your WHM account's security questions.

[Read more](https://docs.cpanel.net/whm/security-center/security-questions/)

* * *

#### [Shell Fork Bomb Protection](https://docs.cpanel.net/whm/security-center/shell-fork-bomb-protection/)

This feature helps you protect your server from users with terminal access (SSH or Telnet) who may inadvertently or intentionally cause a fork bomb.

[Read more](https://docs.cpanel.net/whm/security-center/shell-fork-bomb-protection/)

* * *

#### [SMTP Restrictions](https://docs.cpanel.net/whm/security-center/smtp-restrictions/)

This interface allows you to configure your server so that only the mail transport agent (MTA), Mailman mailing list software, and the root user can connect to remote SMTP servers.

[Read more](https://docs.cpanel.net/whm/security-center/smtp-restrictions/)

* * *

#### [SSH Password Authorization Tweak](https://docs.cpanel.net/whm/security-center/ssh-password-authorization-tweak/)

This feature allows you to enable or disable password authorization for SSH authentication.

[Read more](https://docs.cpanel.net/whm/security-center/ssh-password-authorization-tweak/)

* * *

#### [Traceroute Enable/Disable](https://docs.cpanel.net/whm/security-center/traceroute-enable-disable/)

This interface allows you to discover possible weaknesses that attackers might exploit, or to troubleshoot problems with the network.

[Read more](https://docs.cpanel.net/whm/security-center/traceroute-enable-disable/)

* * *

#### [Two-Factor Authentication for WHM](https://docs.cpanel.net/whm/security-center/two-factor-authentication-for-whm/)

This interface allows you to configure two-factor authentication (2FA), a security measure for the login interface of cPanel & WHM.

[Read more](https://docs.cpanel.net/whm/security-center/two-factor-authentication-for-whm/)
