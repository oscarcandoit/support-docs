## Show Duplicate Licenses

Last modified: _2020 May 14_

* * *

## Overview

The _Show Duplicate Licenses_ interface displays any duplicate license attached to an IP address.

## View duplicate licenses

If duplicate licenses exist, the duplicate licenses table will appear. The duplicate licenses table displays the following information:

- _IP_ — The IP address associated with the license.
- _Message_ — The information regarding the license’s duplication.
- _Duplicated On_ — The date the license duplication took place.
