## Search Engine Optimization Change Log

Last modified: _2025 October 2_

* * *

### 1.7.0-1

* * *

##### **2025-09-30**

- DUCKS-4004: Expand Search Engine Optimization rollout to 10k additional servers.
  - Preserve all previously enabled servers.

### 1.6.0-2

* * *

##### **2025-08-05**

- DUCKS-3575: Release Search Engine Optimization to random 1k servers. The Search Engine Optimization feature is now available to a random selection of 1,000 cPanel Direct servers.

### 1.5.0-1

* * *

##### **2025-07-07**

- DUCKS-2927: Initial release of the Search Engine Optimization v1.5 for cPanel.
- cPanel’s new Search Engine Optimization feature enables users to
  - increase their websites’ visibility.
  - track keyword performance.
  - identify problems that could decrease traffic.
- Released for a limited audience.

### 1.4.0-1

* * *

#### **2025-04-17**

- DUCKS-2757: Disable Search Engine Optimization feature for cPanel version 128.

### 1.3.0-1

* * *

##### **2025-03-10**

- DUCKS-1824: Updates include:
  - Add tooling to reinitialize the feature on configuration changes, such as server license updates, that affect feature access.
  - Prevent users with bookmarked internal links to the Social Media
    Management page from accessing the page when it is disabled.

### 1.2.0-1

* * *

##### **2025-02-27**

- DUCKS-2305: Disabling Search Engine Optimization feature:
  - for cPanel version 126
  - for cPanel Direct 118 - 126

### 1.1.0-1

* * *

##### **2025-02-25**

- DUCKS-1956: Enable Search Engine Optimization feature by default for cPanel
versions 126 and above.
- Release Search Engine Optimization feature for additional partners.
  - cPanel Direct
  - Siteocity

### 1.0.0-1

* * *

##### **2025-02-03**

- DUCKS-1769: Initial release of the SEO plugin for select company ID.
