## Scripts and Utilities

* * *

#### [Autofixer](https://docs.cpanel.net/knowledge-base/scripts-and-utilities/autofixer/)

You can use the Autofixer script collection to fix some common issues from a web browser that you cannot otherwise access from the command line.

[Read more](https://docs.cpanel.net/knowledge-base/scripts-and-utilities/autofixer/)

* * *

#### [Cloud-Hosted Hostname Solution](https://docs.cpanel.net/knowledge-base/scripts-and-utilities/cloud-hosted-hostname-solution/)

The workaround scripts for the dhclient script may interfere with WHM's Change Hostname feature, which causes hostname configuration issues and a locked cPanel & WHM license.

[Read more](https://docs.cpanel.net/knowledge-base/scripts-and-utilities/cloud-hosted-hostname-solution/)

* * *

#### [How to Restore Database Grants](https://docs.cpanel.net/knowledge-base/scripts-and-utilities/how-to-restore-database-grants/)

If a database user loses access to databases, you can restore the user's grants to the database with the restoregrants utility.

[Read more](https://docs.cpanel.net/knowledge-base/scripts-and-utilities/how-to-restore-database-grants/)

* * *

#### [The scl Utility](https://docs.cpanel.net/knowledge-base/scripts-and-utilities/the-scl-utility/)

To manage software packages in a Software Collection Library (SCL) environment via the command line, use the scl utility.

[Read more](https://docs.cpanel.net/knowledge-base/scripts-and-utilities/the-scl-utility/)

* * *

#### [Troubleshooting Guide for Perl and CGI Scripts](https://docs.cpanel.net/knowledge-base/scripts-and-utilities/troubleshooting-guide-for-perl-and-cgi-scripts/)

Many users and system administrators encounter problems with CGI and Perl scripts. This troubleshooting guide lists many of the common problems and solutions.

[Read more](https://docs.cpanel.net/knowledge-base/scripts-and-utilities/troubleshooting-guide-for-perl-and-cgi-scripts/)
