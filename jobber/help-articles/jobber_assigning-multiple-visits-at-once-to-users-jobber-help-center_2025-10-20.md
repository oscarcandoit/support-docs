### Overview

If you need to assign multiple visits to a user for a particular day, the fastest way to do this is using the map view of the schedule. This comes in handy when an employee calls in sick, or when there are schedule changes.

### Assign a single visit

If a visit is not assigned, it will show the red **Unassigned** icon in the team section of the visit details. To assign a single visit, click on it, then click **Edit**. From the edit pop-up, select **Assign**.

![Screenshot 2024-03-22 at 4.08.34 PM.png](https://help.getjobber.com/hc/article_attachments/22277220422167)

![assign.png](https://help.getjobber.com/hc/article_attachments/22277223641495)

A box will come up, where you can click **Assign** and select your team. You can then select **Save** to update the visit.

![Screenshot 2024-03-22 at 4.12.28 PM.png](https://help.getjobber.com/hc/article_attachments/22277220436119)

### Assign multiple visits

To assign visits in the Map view, select the user's name at the top of the map. Then select the visit(s) that you would like to assign to them, either in the map or list. Once you have the tasks selected you can then click **Save** in the top right corner.

![clicking a team member name to highlight it, then selecting visits from the side bar, then selecting save to assign visits](https://help.getjobber.com/hc/article_attachments/22277223649687)

Once you save the visit selection, the map will have blue lines between appointments to represent the route for that user. You can use [Daily Route Optimization](https://help.getjobber.com/hc/en-us/articles/115009542508) to optimize the route for each user. Routing on the map only works for unscheduled visits. If a visit has a set time, it will not be part of the route.
