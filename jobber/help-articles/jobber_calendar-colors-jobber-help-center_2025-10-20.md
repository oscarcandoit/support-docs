### Overview

You can customize calendar colors on your schedule in Jobber.com to keep your calendar organized and easy to understand at a glance. Use colors to highlight tasks assigned to specific team members or those containing certain keywords in the title, making it simple to spot important details quickly. Please note that calendar colors are only available on Jobber.com and won't appear in the Jobber mobile app.

### Assign a calendar color

To assign a calendar color:

1\. Select the **Gear Icon**, then select **Settings**.

2\. From the settings menu, select **Schedule**.

3\. Locate the calendar colors section, then click **Assign a Color** to apply a new calendar color to a team member or keyword.

![calendar colors that have been assigned. There is a button for assign a color](https://help.getjobber.com/hc/article_attachments/22216883828631)

You'll see a prompt like this:

![color assignment settings where colors can be selected and when colors will be assigned](https://help.getjobber.com/hc/article_attachments/22216886068503)

You can apply the color to a certain _user_, by selecting from the first drop down **When Assigned**. Then in the second drop-down, you can select the user.

To apply the color to a certain _word_ in the visit title, select **When Visit Title** from the drop-down menu. In the box below you can enter the word or phrase you'd like the color associated with.

The visit title is the title given to visits on the calendar. You can edit the template for visit titles by going to the **Gear Icon** then **Settings** then **Work Settings**. For example, you can make the client's city or their postal code show in the template, so it displays in the calendar.

To edit a calendar color after you have assigned it, follow these steps:

1. Select the **Gear Icon**, then select **Settings**.
2. From the settings menu, select **Schedule**.
3. Locate the calendar colors section, then select one of the assigned colors.

From the list of calendar colors, select the color you wish to change. An edit screen will open where you can change the color as well as the assigned user or keyword.

![color assignment settings where colors can be selected and when colors will be assigned](https://help.getjobber.com/hc/article_attachments/22216886095255)

### Color order

Colors are prioritized from bottom to top in the list. You can rearrange the order by dragging colors using the 'drag' button next to each one. If a visit is assigned to multiple users with different colors, the calendar will display the color of the user who appears closest to the bottom of the list.
