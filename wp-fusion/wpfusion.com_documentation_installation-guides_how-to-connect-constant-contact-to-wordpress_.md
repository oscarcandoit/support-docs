In this guide, you will learn how to install WP Fusion and set up a direct connection to integrate your WordPress website with your Constant Contact account.

To get started install and activate the WP Fusion plugin, then navigate to Settings » WP Fusion.

From the ‘Select Your CRM’ list choose ‘Constant Contact ’.

[![](https://wpfusion.com/wp-content/uploads/2022/04/constant-contact-configuration-1024x633.png)](https://wpfusion.com/wp-content/uploads/2022/04/constant-contact-configuration.png)

Click ‘Authorize with Constant Contact’ to initialize the connection and generate access and refresh tokens for your website.

Once the connection is completed, click **Save Changes** below to save the changes.

That’s it! Your site is connected, and new WordPress users and customers will automatically be synced with Constant Contact in real time.

- For exporting historical data from before you installed WP Fusion, check out the [Exporting Data guide](https://wpfusion.com/documentation/tutorials/batch-operations/).
- For mapping and syncing additional custom fields with Constant Contact, see the [Syncing Contact Fields guide](https://wpfusion.com/documentation/getting-started/syncing-contact-fields/).
- For using Constant Contact lists to track activity in WordPress and personalize the user experience, check out the [Working with Tags](https://wpfusion.com/documentation/getting-started/working-with-tags/) and [Restricting Access to Content](https://wpfusion.com/documentation/getting-started/access-control/) guides.
