You can now add custom fields as selectable columns in your Contacts and Opportunities table widgets. This enhancement provides more flexibility and visibility, allowing you to tailor dashboards and reports to display the data that matters most to your business.

## **Key Capabilities**

•Custom Fields in Tables – Add your custom fields as columns in both Contacts and Opportunities widgets.

•Birthdate Field – The Birthdate field has been added as a selectable column in Contacts widgets.

•Dashboard & Reports Support – These fields are available in both Dashboards and Custom Reports.

## **How to Add Custom Fields**

1\. Enter Edit Mode

•Go to your dashboard or custom report.

•Click Edit to begin making changes.

2\. Add a Table Widget

•Click Add Widget.

•Choose either a Contacts or Opportunities table widget.

![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155052520145/original/TdX8Yc_dFXFtW3Rp7yLhN-LJlyCJ8gy3GQ.png?1756311263)

3\. Select Columns

•Open the column selector for the table widget.

•You’ll now see all custom fields available.

•Check the fields you want to display.

4\. Save and View

•Click Save to apply your changes.

•Your selected custom fields will now appear as columns in the table.

## **Why This Matters**

By enabling custom fields in dashboard tables, you can:

•Personalize dashboards and reports to match your business needs.

•Track unique data points directly from a single view.

•Improve visibility without needing to export or switch tools.

## **Frequently Asked Questions (FAQs)**

Q: Which widgets support custom fields?

Custom fields can be added to Contacts and Opportunities table widgets.

Q: Can I use these fields in both dashboards and reports?

Yes. Custom fields are available in both Dashboards and Custom Reports.

Q: Is Birthdate available as a column option?

Yes. Birthdate is now a selectable column specifically for Contacts widgets.

Q: Do I need admin permissions to add custom fields to a widget?

You only need permission to edit the dashboard or report. No additional setup is required.

Q: Will adding fields to a widget change the underlying data?

No. It only changes the way data is displayed in your dashboard or report.
