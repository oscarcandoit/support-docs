# How to set up appointment reminder workflow with Zoom location

Modified on: Fri, 21 Jan, 2022 at 2:09 PM

## When we set up the appointment reminder campaign, please be sure to add a wait step for a minute in order for us to process the location value if you are using [Zoom integration](https://help.gohighlevel.com/support/solutions/articles/48001179593-zoom-integration).
