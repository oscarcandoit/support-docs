# cPanel Documentation Repository

This repository contains comprehensive cPanel documentation scraped from `docs.cpanel.net`.

## Contents

**Total Files:** 1,629 documentation files  
**Size:** ~21MB  
**Format:** Markdown (.md) files  

## Documentation Coverage

- **cPanel Interface:** User guides and tutorials
- **WHM (Web Host Manager):** Administrative documentation
- **EasyApache 4 (EA4):** Apache and PHP configuration
- **Installation Guides:** System requirements and setup
- **Knowledge Base:** Troubleshooting and how-to articles
- **API Documentation:** Development references
- **Change Logs:** Version history and updates

## File Organization

All documentation files are stored in the `/docs/` directory with descriptive filenames based on their original URLs from docs.cpanel.net.

## Usage

This repository serves as a comprehensive offline reference for cPanel administration, development, and troubleshooting.
