This interface allows you to customize parts of the cPanel and Webmail interfaces.

* * *

#### [Database Map Tool](https://docs.cpanel.net/whm/sql-services/database-map-tool/)

This feature allows server administrators to grant ownership of database objects (databases and database users) to a cPanel user.

#### [Delete a DNS Zone](https://docs.cpanel.net/whm/dns-functions/delete-a-dns-zone/)

This feature allows you to delete DNS zones from your server.

[Read more](https://docs.cpanel.net/whm/dns-functions/delete-a-dns-zone/)

* * *

#### [Delete a Locale](https://docs.cpanel.net/whm/locales/delete-a-locale/)

This interface allows you to delete a locale.

[Read more](https://docs.cpanel.net/whm/locales/delete-a-locale/)

* * *

#### [Delete a Package](https://docs.cpanel.net/whm/packages/delete-a-package/)

This feature allows you to remove a package from your server.

[Read more](https://docs.cpanel.net/whm/packages/delete-a-package/)

* * *

#### [DirectoryIndex Priority](https://docs.cpanel.net/whm/service-configuration/directoryindex-priority/)

The DirectoryIndex Priority interface allows you to specify the filenames that Apache recognizes and displays as index pages.

[Read more](https://docs.cpanel.net/whm/service-configuration/directoryindex-priority/)

* * *

#### [DNS Cluster](https://docs.cpanel.net/whm/clusters/dns-cluster/)

This interface allows you to configure a DNS cluster and add servers to an existing DNS cluster.

[Read more](https://docs.cpanel.net/whm/clusters/dns-cluster/)

* * *

#### [DNS Zone Manager](https://docs.cpanel.net/whm/dns-functions/dns-zone-manager/)

This feature allows you to edit the records in a domain's DNS zone file.

[Read more](https://docs.cpanel.net/whm/dns-functions/dns-zone-manager/)

* * *

#### [EasyApache 4 Interface](https://docs.cpanel.net/whm/software/easyapache-4-interface/)

This interface allows you to update and install Apache, PHP, and other components of your web server.

[Read more](https://docs.cpanel.net/whm/software/easyapache-4-interface/)

* * *

#### [Edit a Locale](https://docs.cpanel.net/whm/locales/edit-a-locale/)

Use this interface to edit the locales on your server.

[Read more](https://docs.cpanel.net/whm/locales/edit-a-locale/)

* * *

#### [Edit a Package](https://docs.cpanel.net/whm/packages/edit-a-package/)

This feature allows you to edit existing packages.

[Read more](https://docs.cpanel.net/whm/packages/edit-a-package/)

* * *

#### [Edit Database Configuration](https://docs.cpanel.net/whm/database-services/edit-database-configuration/)

This interface allows you to make changes to your MariaDB or MySQL configuration.

[Read more](https://docs.cpanel.net/whm/database-services/edit-database-configuration/)

* * *

#### [Edit Reseller Nameservers and Privileges](https://docs.cpanel.net/whm/resellers/edit-reseller-nameservers-and-privileges/)

[Read more](https://docs.cpanel.net/whm/resellers/edit-reseller-nameservers-and-privileges/)

* * *

#### [Edit SQL Configuration](https://docs.cpanel.net/whm/sql-services/edit-sql-configuration/)

This interface allows you to make changes to your MariaDB or MySQL configuration.

* * *

#### [Edit System Mail Preferences](https://docs.cpanel.net/whm/server-contacts/edit-system-mail-preferences/)

[Read more](https://docs.cpanel.net/whm/server-contacts/edit-system-mail-preferences/)

* * *

#### [Edit Zone Templates](https://docs.cpanel.net/whm/dns-functions/edit-zone-templates/)

[Read more](https://docs.cpanel.net/whm/dns-functions/edit-zone-templates/)

* * *

#### [Email All Resellers](https://docs.cpanel.net/whm/resellers/email-all-resellers/)
