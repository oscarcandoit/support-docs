# accounts

* * *

#### [Show Accounts Over Quota](https://docs.cpanel.net/whm/account-information/show-accounts-over-quota/)

This interface allows you to list and manage accounts that exceed their disk quota.

[Read more](https://docs.cpanel.net/whm/account-information/show-accounts-over-quota/)

* * *

#### [Terminate Accounts](https://docs.cpanel.net/whm/account-functions/terminate-accounts/)

This interface allows you to simultaneously terminate multiple accounts.

[Read more](https://docs.cpanel.net/whm/account-functions/terminate-accounts/)

* * *

#### [The autossl\_check Script](https://docs.cpanel.net/whm/scripts/the-autossl_check-script/)

This script performs a manual AutoSSL check.

[Read more](https://docs.cpanel.net/whm/scripts/the-autossl_check-script/)

* * *

#### [The modify\_accounts Script](https://docs.cpanel.net/whm/scripts/the-modify_accounts-script/)

You can use this script to set existing cPanel accounts to use a new theme.

[Read more](https://docs.cpanel.net/whm/scripts/the-modify_accounts-script/)

* * *

#### [The removeacct Script](https://docs.cpanel.net/whm/scripts/the-removeacct-script/)

This script terminates a cPanel account from the command line.

[Read more](https://docs.cpanel.net/whm/scripts/the-removeacct-script/)

* * *

#### [The suspendacct Script](https://docs.cpanel.net/whm/scripts/the-suspendacct-script/)

This script allows you to suspend an account.

[Read more](https://docs.cpanel.net/whm/scripts/the-suspendacct-script/)

* * *

#### [The sync\_child\_accounts Script](https://docs.cpanel.net/whm/scripts/the-sync_child_accounts-script/)

This script synchronizes a distributed cPanel account between a parent node and its child nodes.

[Read more](https://docs.cpanel.net/whm/scripts/the-sync_child_accounts-script/)

* * *

#### [The transfer\_account\_as\_user Script](https://docs.cpanel.net/whm/scripts/the-transfer_account_as_user-script/)

This script transfers a cPanel account from a remote server to the local server.

[Read more](https://docs.cpanel.net/whm/scripts/the-transfer_account_as_user-script/)

* * *

#### [The transfer\_accounts\_as\_root Script](https://docs.cpanel.net/whm/scripts/the-transfer_accounts_as_root-script/)

This script transfers one or more cPanel accounts from a remote server to the local server.

[Read more](https://docs.cpanel.net/whm/scripts/the-transfer_accounts_as_root-script/)

* * *

#### [The unsuspendacct Script](https://docs.cpanel.net/whm/scripts/the-unsuspendacct-script/)

This script lets you suspend an account.

[Read more](https://docs.cpanel.net/whm/scripts/the-unsuspendacct-script/)

* * *

#### [The wwwacct Script](https://docs.cpanel.net/whm/scripts/the-wwwacct-script/)

This script creates a new cPanel account.

[Read more](https://docs.cpanel.net/whm/scripts/the-wwwacct-script/)

* * *

#### [Unsuspend Bandwidth Exceeders](https://docs.cpanel.net/whm/account-functions/unsuspend-bandwidth-exceeders/)

This feature allows you to simultaneously unsuspend all of the accounts that have exceeded their bandwidth.

[Read more](https://docs.cpanel.net/whm/account-functions/unsuspend-bandwidth-exceeders/)

* * *

#### [Upgrade/Downgrade an Account](https://docs.cpanel.net/whm/account-functions/upgrade-downgrade-an-account/)

This interface allows you to change an account’s package.

[Read more](https://docs.cpanel.net/whm/account-functions/upgrade-downgrade-an-account/)

* * *

#### [User Manager](https://docs.cpanel.net/cpanel/preferences/user-manager/)

This interface allows you to manage your cPanel account's subaccounts.

[Read more](https://docs.cpanel.net/cpanel/preferences/user-manager/)
