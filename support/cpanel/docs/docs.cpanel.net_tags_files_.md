# files

* * *

#### [Directory Privacy](https://docs.cpanel.net/cpanel/files/directory-privacy/)

This interface allows you to protect specific directories in your cPanel account's files.

[Read more](https://docs.cpanel.net/cpanel/files/directory-privacy/)

* * *

#### [DirectoryIndex Priority](https://docs.cpanel.net/whm/service-configuration/directoryindex-priority/)

The DirectoryIndex Priority interface allows you to specify the filenames that Apache recognizes and displays as index pages.

[Read more](https://docs.cpanel.net/whm/service-configuration/directoryindex-priority/)

* * *

#### [File and Directory Restoration for WHM](https://docs.cpanel.net/whm/backup/file-and-directory-restoration-for-whm/)

The File and Directory Restoration interface allows you to restore items from the local backups of your cPanel account users.

[Read more](https://docs.cpanel.net/whm/backup/file-and-directory-restoration-for-whm/)

* * *

#### [File Manager](https://docs.cpanel.net/cpanel/files/file-manager/)

Use the File Manager interface to manage and edit your files.

[Read more](https://docs.cpanel.net/cpanel/files/file-manager/)

* * *

#### [Hotlink Protection](https://docs.cpanel.net/cpanel/security/hotlink-protection/)

A hotlink occurs when someone embeds content from your site in another site and uses your bandwidth to serve the files. You can use this interface to prevent this issue.

[Read more](https://docs.cpanel.net/cpanel/security/hotlink-protection/)

* * *

#### [HTML Editor (Beta)](https://docs.cpanel.net/cpanel/files/html-editor/)

The HTML Editor (Beta) interface allows you to easily edit your HTML documents from the File Manager interface.

[Read more](https://docs.cpanel.net/cpanel/files/html-editor/)

* * *

#### [Images](https://docs.cpanel.net/cpanel/files/images/)

The Images interface allows you to modify and manage images that you saved to your account.

[Read more](https://docs.cpanel.net/cpanel/files/images/)

* * *

#### [Indexes](https://docs.cpanel.net/cpanel/advanced/indexes/)

Use the Indexes interface to define how the server displays a specific directory's index to a visitor, or to disable index display for a directory.

[Read more](https://docs.cpanel.net/cpanel/advanced/indexes/)

* * *

#### [Leech Protection](https://docs.cpanel.net/cpanel/security/leech-protection/)

The Leech Protection interface allows you to detect unusual levels of activity in password-restricted directories.

[Read more](https://docs.cpanel.net/cpanel/security/leech-protection/)

* * *

#### [MIME Types](https://docs.cpanel.net/cpanel/advanced/mime-types/)

MIME types relay information to the browser about how to handle file extensions.

[Read more](https://docs.cpanel.net/cpanel/advanced/mime-types/)

* * *

#### [Optimize Website](https://docs.cpanel.net/cpanel/software/optimize-website/)

This interface allows you to configure your server to automatically compress specified types of content when visitors access that content.

[Read more](https://docs.cpanel.net/cpanel/software/optimize-website/)

* * *

#### [Skeleton Directory](https://docs.cpanel.net/whm/account-functions/skeleton-directory/)

This interface displays the location of the skeleton directory.

[Read more](https://docs.cpanel.net/whm/account-functions/skeleton-directory/)

* * *

#### [Web Disk](https://docs.cpanel.net/cpanel/files/web-disk/)

Use this interface to manage and manipulate files on your server on multiple types of devices

[Read more](https://docs.cpanel.net/cpanel/files/web-disk/)

* * *

#### [Web Template Editor](https://docs.cpanel.net/whm/account-functions/web-template-editor/)

This interface allows you to create templates for placeholder web pages.

[Read more](https://docs.cpanel.net/whm/account-functions/web-template-editor/)
