* * *

the server for the Sitejet Beta features

### 4.0.0-1

* * *

##### **2025-06-23**

- DUCKS-2487: Release of Sitejet AI for cPanel.

This change is releasing the Sitejet AI to a select audience of partners.

Users not belonging to one of those partners will not see any of the Sitejet AI BETA feature.

This change includes supporting cPanel changes tied to the new Sitejet AI new website wizard:
  - Updated WHM feature showcase to include information about the feature for the WHM admin.
  - Updated cPanel Home Page feature education banner to reflect Sitejet AI feature changes.
  - Implemented direct navigation in Sitejet Builder, allowing users to choose between the Sitejet AI and Template workflows.
