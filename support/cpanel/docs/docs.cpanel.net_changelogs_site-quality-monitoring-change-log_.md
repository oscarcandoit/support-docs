## Site Quality Monitoring Change Log

Last modified: _2025 November 13_

* * *

### 2.1.2-0

* * *

##### **2025-11-13**

- SQM-51: Tests implemented / logout / translation
  - Minor UI improvements & fixes

### 2.1.1-2

* * *

##### **2025-10-29**

- CPCLOUD-5485: Implement advanced crawler for SQM users
  - Implemented new crawler for pro users
  - Fixed minor project alerts

### 2.1.0-1

* * *

##### **2025-08-27**

- DUCKS-3086: Update SQM to support temp domains
  - Implemented new SQM API calls to move domains.
  - Created a new UAPI function ‘update\_domain’ to fix SQM after domain change.
  - Added hooks to automate the process on domain change.
  - Added support for localization in SQM.

### 2.0.0-2

* * *

##### **2025-08-07**

- DUCKS-2844: Major feature and UI update for SQM v2.0.0
- Access your user settings and actions faster with the redesigned user menu, now available throughout the entire app.
- Quickly switch between your projects directly within any monitoring metric view for seamless comparison
- Improved input validation for domains and names when creating new projects.
- New incident indicator with active alert counter and redesigned incident tables including search and filter functionality for faster overview
- Enhanced automated URL Finder for Pro projects and a new job queue system to better track crawler and URL Finder processes
- Introduction of a global score for each project, alongside master scores per category.
- UI/UX Enhancements:
  - Improved feedback when no data is retrieved from the API
  - Feature guard improvements for user and feature-specific access
  - Instantly understand what each metric means with the new in-app advisor integrated throughout the tool
  - Context menus now open next to their activator for better usability
  - Enhanced automated URL Finder for Pro plans
  - Improved comparison table for plans
  - Better chart usability with zoom and pan options
  - Button for unfinished projects to quickly complete setup
- Fixes:
  - Resolved incomplete charts for plugin health data
  - Fixed translation key issues in charts
  - Addressed scrolling and visibility issues in project lists with many entries
  - Fixed menu visibility when scrolling beyond initial screen height
  - Corrected score visibility and weight calculation in menus

* * *

##### **2025-03-28**

- DUCKS-2326: Improved security by rotating JWT credentials to enhance authentication integrity and other minor security improvements. Changes to pricing and plans: Published new pricing and removed the Basic plan. Consolidated all users into a reduced price Pro plan.
