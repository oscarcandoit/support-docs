## Manage Ip Access

Last modified: _2020 March 25_

* * *

## Overview

The _Manage Ip Access_ interface allows you to revoke permission to log in to your account from an IP address in the removed IP range.

## Remove an IP address

To remove an IP address, find the desired IP address and then click _Remove_.
