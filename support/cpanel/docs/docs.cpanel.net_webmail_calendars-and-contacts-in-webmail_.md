## Overview

This section of the _Webmail_ interface has features that allow you to set up, manage, and share your calendars and contacts.

| Interface | Description |
| --- | --- |
| _Calendars and Contacts Management_ | Use this interface to display, create, delete, and rename calendars and address books. For more information, read our [_Calendars and Contacts Management_](https://docs.cpanel.net/cpanel/email/calendars-and-contacts-management/) documentation. |
| _Calendars and Contacts Configuration_ | Use this interface to get the information you need to set up your calendars and contacts on a mobile device or desktop system. For more information, read our [How to Set Up Calendars and Contacts](https://docs.cpanel.net/knowledge-base/webmail/how-to-set-up-calendars-and-contacts/) documentation. |
| _Calendars and Contacts Sharing_ | Use this interface to share calendars and address books with other email users in your domain. For more information, read our [_Calendars and Contacts Sharing_](https://docs.cpanel.net/cpanel/email/calendars-and-contacts-sharing/) documentation. |
