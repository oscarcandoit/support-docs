## Rejected License Requests

Last modified: _2024 February 22_

* * *

## Overview

The _Rejected License Requests_ interface displays the last 300 license requests that did not sync with the cPanel license server.

## License request information

You can view the following information for each rejected license request:

- _Ip Address_ — The IP address for the requested license.
- _Hostname_ — The hostname for the requested license.
- _Contact_ — The email address associated with the license.
- _cPanel Version_ — The version of cPanel & WHM for the requested license.
- _Time_ — The time WebPros International, LLC rejected the license.
- _Reason_ — The reason why WebPros International, LLC rejected the license request.
- _Server_ — The authentication server that rejected the license.

If no rejected license requests exist, the following message will appear:

```perl
There are no Recent License Rejections!
```
