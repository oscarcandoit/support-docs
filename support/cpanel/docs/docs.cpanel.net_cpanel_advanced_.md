## Advanced

* * *

#### [Apache Handlers](https://docs.cpanel.net/cpanel/advanced/apache-handlers/)

Apache handlers control how your site’s Apache web server software manages certain file types and file extensions.

[Read more](https://docs.cpanel.net/cpanel/advanced/apache-handlers/)

* * *

#### [API Shell for cPanel](https://docs.cpanel.net/cpanel/advanced/api-shell-for-cpanel/)

This interface allows you to run cPanel API functions interactively.

[Read more](https://docs.cpanel.net/cpanel/advanced/api-shell-for-cpanel/)

* * *

#### [Cron Jobs](https://docs.cpanel.net/cpanel/advanced/cron-jobs/)

Cron jobs are scheduled tasks that the system runs at predefined times or intervals.

[Read more](https://docs.cpanel.net/cpanel/advanced/cron-jobs/)

* * *

#### [Error Pages](https://docs.cpanel.net/cpanel/advanced/error-pages/)

Error pages inform visitors about problems when they attempt to access your site.

[Read more](https://docs.cpanel.net/cpanel/advanced/error-pages/)

* * *

#### [Indexes](https://docs.cpanel.net/cpanel/advanced/indexes/)

Use the Indexes interface to define how the server displays a specific directory's index to a visitor, or to disable index display for a directory.

[Read more](https://docs.cpanel.net/cpanel/advanced/indexes/)

* * *

#### [MIME Types](https://docs.cpanel.net/cpanel/advanced/mime-types/)

MIME types relay information to the browser about how to handle file extensions.

[Read more](https://docs.cpanel.net/cpanel/advanced/mime-types/)

* * *

#### [Terminal for cPanel](https://docs.cpanel.net/cpanel/advanced/terminal-in-cpanel/)

The Terminal interface allows you to access an in-browser terminal application for direct command-line access within a cPanel & WHM session.

[Read more](https://docs.cpanel.net/cpanel/advanced/terminal-in-cpanel/)

* * *

#### [Track DNS](https://docs.cpanel.net/cpanel/advanced/track-dns/)

This interface contains tools to help you retrieve network information.

[Read more](https://docs.cpanel.net/cpanel/advanced/track-dns/)

* * *

#### [Virus Scanner](https://docs.cpanel.net/cpanel/advanced/virus-scanner/)

The Virus Scanner interface scans your cPanel account for security threats.

[Read more](https://docs.cpanel.net/cpanel/advanced/virus-scanner/)
