## Security

* * *

#### [Hotlink Protection](https://docs.cpanel.net/cpanel/security/hotlink-protection/)

A hotlink occurs when someone embeds content from your site in another site and uses your bandwidth to serve the files. You can use this interface to prevent this issue.

[Read more](https://docs.cpanel.net/cpanel/security/hotlink-protection/)

* * *

#### [IP Blocker](https://docs.cpanel.net/cpanel/security/ip-blocker/)

The IP Blocker interface allows you to block access to your website for one or more IP addresses or fully qualified domain names (FQDNs).

[Read more](https://docs.cpanel.net/cpanel/security/ip-blocker/)

* * *

#### [Leech Protection](https://docs.cpanel.net/cpanel/security/leech-protection/)

The Leech Protection interface allows you to detect unusual levels of activity in password-restricted directories.

[Read more](https://docs.cpanel.net/cpanel/security/leech-protection/)

* * *

#### [Manage API Tokens in cPanel](https://docs.cpanel.net/cpanel/security/manage-api-tokens-in-cpanel/)

This feature lets you create, list, update, and revoke API tokens.

[Read more](https://docs.cpanel.net/cpanel/security/manage-api-tokens-in-cpanel/)

* * *

#### [ModSecurity®](https://docs.cpanel.net/cpanel/security/modsecurity/)

This interface allows you to enable or disable ModSecurity for your domains.

[Read more](https://docs.cpanel.net/cpanel/security/modsecurity/)

* * *

#### [Security Policy](https://docs.cpanel.net/cpanel/security/security-policy/)

This interface allows you to update and manage your cPanel account's security questions.

[Read more](https://docs.cpanel.net/cpanel/security/security-policy/)

* * *

#### [SSH Access](https://docs.cpanel.net/cpanel/security/ssh-access/)

This interface provides information about how to connect to another web server via the SSH (secure shell) network protocol.

[Read more](https://docs.cpanel.net/cpanel/security/ssh-access/)

* * *

#### [SSL/TLS](https://docs.cpanel.net/cpanel/security/ssl-tls/)

The features in this interface allow you to generate and manage SSL certificates, signing requests, and keys, which enhance your website's security.

[Read more](https://docs.cpanel.net/cpanel/security/ssl-tls/)

* * *

#### [SSL/TLS Status](https://docs.cpanel.net/cpanel/security/ssl-tls-status/)

This interface lets you view, upgrade, or renew your domains' Secure Sockets Layer (SSL) certificates.

[Read more](https://docs.cpanel.net/cpanel/security/ssl-tls-status/)

* * *

#### [SSL/TLS Wizard](https://docs.cpanel.net/cpanel/security/ssl-tls-wizard/)

This interface allows you to easily purchase and install SSL certificates for domains on your account.

[Read more](https://docs.cpanel.net/cpanel/security/ssl-tls-wizard/)

* * *

#### [Two-Factor Authentication for cPanel](https://docs.cpanel.net/cpanel/security/two-factor-authentication-for-cpanel/)

Two-factor authentication (2FA) is a security measure that requires two forms of identification.

[Read more](https://docs.cpanel.net/cpanel/security/two-factor-authentication-for-cpanel/)
