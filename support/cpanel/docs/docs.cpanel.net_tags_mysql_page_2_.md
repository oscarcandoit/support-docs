# mysql

* * *

#### [phpMyAdmin](https://docs.cpanel.net/cpanel/databases/phpmyadmin/)

Use this interface to manipulate MySQL® or MariaDB® databases.

[Read more](https://docs.cpanel.net/cpanel/databases/phpmyadmin/)

* * *

#### [Remote Database Access](https://docs.cpanel.net/cpanel/databases/remote-database-access/)

This feature allows remote hosts (servers) to access MySQL® and MariaDB® databases on your account.

[Read more](https://docs.cpanel.net/cpanel/databases/remote-database-access/)

* * *

#### [Repair a MySQL Database](https://docs.cpanel.net/whm/sql-services/repair-a-mysql-database/)

This interface allows you to repair a MySQL database.

[Read more](https://docs.cpanel.net/whm/sql-services/repair-a-mysql-database/)

* * *

#### [Repair Databases](https://docs.cpanel.net/whm/database-services/repair-databases/)

This interface allows you to repair a MySQL® or MariaDB® database.

[Read more](https://docs.cpanel.net/whm/database-services/repair-databases/)

* * *

#### [Show Database Processes](https://docs.cpanel.net/whm/database-services/show-database-processes/)

This interface lists all of the processes that currently run on any database on your server.

[Read more](https://docs.cpanel.net/whm/database-services/show-database-processes/)

* * *

#### [Show MySQL Processes](https://docs.cpanel.net/whm/sql-services/show-mysql-processes/)

This interface lists all of the processes that currently run on any database on your server.

[Read more](https://docs.cpanel.net/whm/sql-services/show-mysql-processes/)

* * *

#### [The auto-adjust-mysql-limits Script](https://docs.cpanel.net/whm/scripts/the-auto-adjust-mysql-limits-script/)

The /usr/local/cpanel/scripts/auto-adjust-mysql-limits script updates MySQL® configuration values.

[Read more](https://docs.cpanel.net/whm/scripts/the-auto-adjust-mysql-limits-script/)

* * *

#### [The build\_mysql\_conf Script](https://docs.cpanel.net/whm/scripts/the-build_mysql_conf-script/)

This script builds the default MySQL config file.

[Read more](https://docs.cpanel.net/whm/scripts/the-build_mysql_conf-script/)

* * *

#### [The manage\_mysql\_profiles Script](https://docs.cpanel.net/whm/scripts/the-manage_mysql_profiles-script/)

The script allows you to import and export MySQL® profiles.

[Read more](https://docs.cpanel.net/whm/scripts/the-manage_mysql_profiles-script/)

* * *

#### [The MysqlDump Stream](https://docs.cpanel.net/knowledge-base/sql/the-mysqldump-stream/)

The MysqlDump stream allows you to produce a dump of a MySQL® database via WebSocket.

[Read more](https://docs.cpanel.net/knowledge-base/sql/the-mysqldump-stream/)

* * *

#### [The securemysql Script](https://docs.cpanel.net/whm/scripts/the-securemysql-script/)

This script secures a cPanel account's MySQL® configuration.

[Read more](https://docs.cpanel.net/whm/scripts/the-securemysql-script/)

#### [Upgrade Database Version](https://docs.cpanel.net/whm/database-services/upgrade-database-version/)
