## View Yearly License Renewal Options

Last modified: _2021 November 24_

* * *

## Overview

This interface allows you to change license renewal options.

## Renew an expired license

The system uses this feature through the [_Add License_](https://docs.cpanel.net/manage2/licenses/add-license/) interface.

To renew an expired license, you **must** perform the following additional steps:

1. After you click _Add License_, a warning message will display. Verify the displayed IP addresses before you continue.
2. Click _Click Here for Forced Add_ to renew the license.
