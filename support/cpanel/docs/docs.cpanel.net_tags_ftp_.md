# ftp

* * *

#### [Anonymous FTP](https://docs.cpanel.net/cpanel/files/anonymous-ftp/)

Use this interface to define how anonymous users interact with your FTP server.

[Read more](https://docs.cpanel.net/cpanel/files/anonymous-ftp/)

* * *

#### [FTP Accounts](https://docs.cpanel.net/cpanel/files/ftp-accounts/)

You can use this interface to create and manage your website's File Transfer Protocol (FTP) accounts.

[Read more](https://docs.cpanel.net/cpanel/files/ftp-accounts/)

* * *

#### [FTP Connections](https://docs.cpanel.net/cpanel/files/ftp-connections/)

This interface displays information about current connections to your FTP server and allows you to terminate those connections.

[Read more](https://docs.cpanel.net/cpanel/files/ftp-connections/)

* * *

#### [FTP Server Configuration](https://docs.cpanel.net/whm/service-configuration/ftp-server-configuration/)

The FTP Server Configuration interface allows you to customize your chosen FTP server's behavior.

[Read more](https://docs.cpanel.net/whm/service-configuration/ftp-server-configuration/)

* * *

#### [FTP Server Selection](https://docs.cpanel.net/whm/service-configuration/ftp-server-selection/)

This interface allows you to select an FTP server daemon, or to disable FTP services.

[Read more](https://docs.cpanel.net/whm/service-configuration/ftp-server-selection/)

* * *

#### [How to Configure Your SFTP Client](https://docs.cpanel.net/knowledge-base/ftp/how-to-configure-your-sftp-client/)

This document details the information you will need to connect to your cPanel account via SFTP.

[Read more](https://docs.cpanel.net/knowledge-base/ftp/how-to-configure-your-sftp-client/)

#### [How to Deny FTP Access](https://docs.cpanel.net/knowledge-base/ftp/how-to-deny-ftp-access/)

#### [How to Edit FTP Server Configuration from the Command Line](https://docs.cpanel.net/knowledge-base/ftp/how-to-edit-ftp-server-configuration-from-the-command-line/)

This document explains how to manually edit the FTP server's configuration from the command line.

[Read more](https://docs.cpanel.net/knowledge-base/ftp/how-to-edit-ftp-server-configuration-from-the-command-line/)

* * *

#### [How to Enable FTP Passive Mode](https://docs.cpanel.net/knowledge-base/ftp/how-to-enable-ftp-passive-mode/)

This document explains how to use the active or passive mode to connect to a File Transfer Protocol (FTP) server.

[Read more](https://docs.cpanel.net/knowledge-base/ftp/how-to-enable-ftp-passive-mode/)

* * *

#### [How to Restore Missing FTP Interfaces in cPanel](https://docs.cpanel.net/knowledge-base/ftp/how-to-restore-missing-ftp-interfaces-in-cpanel/)

#### [How to Upload Files with FTP](https://docs.cpanel.net/knowledge-base/ftp/how-to-upload-files-with-ftp/)
