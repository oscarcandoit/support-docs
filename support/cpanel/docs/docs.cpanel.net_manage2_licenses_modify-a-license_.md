## Modify a License

Last modified: _2023 February 24_

* * *

## Overview

This interface allows you to modify information about your account’s license.

Note:

- You will only be able to modify license parameters for which you have the authority to do so. If you want to modify a parameter for which you do not have `editor` permissions, contact the owner of the Manage2 account to make the change(s) for you.
- You can access the _Modify a List_ interface through the _List Licenses_ interface as described above, or through the _Search_ feature.
- Not all fields are visible for every license type. For example, 360 Monitoring licenses will have the _Sites_ and _Servers_ fields but not an _IP_ field.

## Modify the License Parameter(s)

- Click the parameter that you want to modify.
- Enter the new information.
- Click `Update` to save the new information.






Note:




If your package has the maximum number of users allowed, the following message will appear: _The operation was not completed: The account volume on this license exceeds the package maximum._
