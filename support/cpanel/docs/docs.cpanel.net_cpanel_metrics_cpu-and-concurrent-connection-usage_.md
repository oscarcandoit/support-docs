## Overview

This interface allows you to view and monitor resource usage on your cPanel account. For more information, read [CloudLinux’s](https://docs.cloudlinux.com/lve_manager/#client-plugins) documentation.

Note:

This feature only exists on servers that run CloudLinux™.
