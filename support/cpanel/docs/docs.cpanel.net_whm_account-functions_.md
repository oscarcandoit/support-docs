## Account Functions

* * *

#### [Change Site's IP Address](https://docs.cpanel.net/whm/account-functions/change-site-ip-address/)

This interface allows you to change an account's IP address.

[Read more](https://docs.cpanel.net/whm/account-functions/change-site-ip-address/)

* * *

#### [Create a New Account](https://docs.cpanel.net/whm/account-functions/create-a-new-account/)

This interface allows you to create new cPanel or WHM accounts.

[Read more](https://docs.cpanel.net/whm/account-functions/create-a-new-account/)

* * *

#### [Email All Users](https://docs.cpanel.net/whm/account-functions/email-all-users/)

This interface enables you to email every cPanel user simultaneously.

[Read more](https://docs.cpanel.net/whm/account-functions/email-all-users/)

* * *

#### [Force Password Change](https://docs.cpanel.net/whm/account-functions/force-password-change/)

This feature allows you to force users to change their passwords.

[Read more](https://docs.cpanel.net/whm/account-functions/force-password-change/)

* * *

#### [Limit Bandwidth Usage](https://docs.cpanel.net/whm/account-functions/limit-bandwidth-usage/)

This interface allows you to change the allotted bandwidth for an account.

[Read more](https://docs.cpanel.net/whm/account-functions/limit-bandwidth-usage/)

* * *

#### [Manage Account Suspension](https://docs.cpanel.net/whm/account-functions/manage-account-suspension/)

This feature allows you to suspend and unsuspend accounts on your server.

[Read more](https://docs.cpanel.net/whm/account-functions/manage-account-suspension/)

* * *

#### [Manage Demo Mode](https://docs.cpanel.net/whm/account-functions/manage-demo-mode/)

This feature allows you to convert any cPanel account on your server to a demo account.

[Read more](https://docs.cpanel.net/whm/account-functions/manage-demo-mode/)

* * *

#### [Manage Shell Access](https://docs.cpanel.net/whm/account-functions/manage-shell-access/)

This interface allows you to manage which of your accounts can access your server remotely from the command line.

[Read more](https://docs.cpanel.net/whm/account-functions/manage-shell-access/)

* * *

#### [Modify an Account](https://docs.cpanel.net/whm/account-functions/modify-an-account/)

This interface allows you to make changes to an existing account.

[Read more](https://docs.cpanel.net/whm/account-functions/modify-an-account/)

* * *

#### [Password Modification](https://docs.cpanel.net/whm/account-functions/password-modification/)

This interface allows you to change the passwords for your reseller and cPanel accounts.

[Read more](https://docs.cpanel.net/whm/account-functions/password-modification/)

* * *

#### [Quota Modification](https://docs.cpanel.net/whm/account-functions/quota-modification/)

This interface allows you to change an account's disk space quota.

[Read more](https://docs.cpanel.net/whm/account-functions/quota-modification/)

* * *

#### [Raw Apache Log Download](https://docs.cpanel.net/whm/account-functions/raw-apache-log-download/)

This interface allows you to download Apache's raw access logs.

[Read more](https://docs.cpanel.net/whm/account-functions/raw-apache-log-download/)

* * *

#### [Raw FTP Log Download](https://docs.cpanel.net/whm/account-functions/raw-ftp-log-download/)

This interface allows you to download an FTP server's raw access logs.

[Read more](https://docs.cpanel.net/whm/account-functions/raw-ftp-log-download/)

* * *

#### [Raw NGINX® Log Download](https://docs.cpanel.net/whm/account-functions/raw-nginx-log-download/)

This interface allows you to download NGINX's raw access logs.

[Read more](https://docs.cpanel.net/whm/account-functions/raw-nginx-log-download/)

* * *

#### [Rearrange an Account](https://docs.cpanel.net/whm/account-functions/rearrange-an-account/)

This interface allows you to change which home directory stores an account.

[Read more](https://docs.cpanel.net/whm/account-functions/rearrange-an-account/)

* * *

#### [Reset Account Bandwidth Limit](https://docs.cpanel.net/whm/account-functions/reset-account-bandwidth-limit/)

If an account's bandwidth quota does not match the quota for the account's package, this interface enables you to reset the account's quota.

[Read more](https://docs.cpanel.net/whm/account-functions/reset-account-bandwidth-limit/)

* * *

#### [Skeleton Directory](https://docs.cpanel.net/whm/account-functions/skeleton-directory/)

This interface displays the location of the skeleton directory.

[Read more](https://docs.cpanel.net/whm/account-functions/skeleton-directory/)

* * *

#### [Terminate Accounts](https://docs.cpanel.net/whm/account-functions/terminate-accounts/)

This interface allows you to simultaneously terminate multiple accounts.

[Read more](https://docs.cpanel.net/whm/account-functions/terminate-accounts/)

* * *

#### [Unsuspend Bandwidth Exceeders](https://docs.cpanel.net/whm/account-functions/unsuspend-bandwidth-exceeders/)

This feature allows you to simultaneously unsuspend all of the accounts that have exceeded their bandwidth.

[Read more](https://docs.cpanel.net/whm/account-functions/unsuspend-bandwidth-exceeders/)

* * *

#### [Upgrade/Downgrade an Account](https://docs.cpanel.net/whm/account-functions/upgrade-downgrade-an-account/)

This interface allows you to change an account’s package.

[Read more](https://docs.cpanel.net/whm/account-functions/upgrade-downgrade-an-account/)
