## Overview

This interface allows you to view available locales, cross-referenced against the themes for which they are available.

## How to view available locales

(![check mark](https://docs.cpanel.net/img/viewavailablelocalescheck.png)) — A check mark indicates that the locale is available for a particular theme.

The _Provider_ column displays whether WebPros International, LLC provided the locale. The column displays either of the following icons:

- (![cPanel logo](https://docs.cpanel.net/img/viewavailablelocalescpanel.png)) — This icon indicates that WebPros International, LLC provided this locale.
- (![people icon](https://docs.cpanel.net/img/viewavailablelocalespeople.png)) — This icon indicates that WebPros International, LLC did **not** provide this locale.

Note:

To refresh the list of your system’s locales, click _rebuild the locale databases_ at the bottom of the interface to update the system.
