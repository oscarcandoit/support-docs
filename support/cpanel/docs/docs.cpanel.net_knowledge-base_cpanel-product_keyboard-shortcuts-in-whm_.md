## Keyboard Shortcuts in WHM

Last modified: _2021 October 13_

* * *

## Overview

We include keyboard shortcuts within [the WHM interface](https://docs.cpanel.net/whm/the-whm-interface/the-whm-interface/). These shortcuts help make the WHM interface more accessible and easier to navigate.

## Global shortcuts

The following global shortcuts are available in any part of WHM’s interface:

| Key | macOS® alternative | Action |
| --- | --- | --- |
| `Shift` \+ `8` |  | Toggle the menu between _Expand all_ and _Collapse all._ |
| `Ctrl` \+ `/` | `⌘` \+ `/` | Move the cursor to the Search text box. |

## Left navigation menu shortcuts

The following shortcuts only apply to the left navigation menu:

| Key | macOS alternative | Action |
| --- | --- | --- |
| `↑` |  | Navigate up the interface or menu. |
| `↓` |  | Navigate down the interface or menu. |
| `←` |  | - For left-to-right (LTR) locales, collapse the highlighted category.<br>- For right-to-left (RTL) locales, expand the highlighted category. |
| `→` |  | - For left-to-right (LTR) locales, expand the highlighted category.<br>- For right-to-left (RTL) locales, collapse the highlighted category. |
| `Enter` |  | - For the main category listings, expand/collapse the highlighted category.<br>- For applications under an expanded category, open the selected item. |
| `Shift` \+ `Tab` |  | Navigate up the menu. |
| `Tab` |  | Navigate down the menu. |
| `Home` | `Fn` \+ `←` | Select the first section in the menu. |
| `End` | `Fn` \+ `→` | Select the last item in the menu. |

## Set up browser shortcuts in macOS

Firefox® or Safari® browser users on the macOS may need to enable the use of the `Tab` key to properly navigate the left menu. The Google Chrome™ browser enables the use of `Tab` key navigation by default.

### Firefox browser

To enable the use of the `Tab` key in Firefox, perform the following steps:

1. Open _System Preferences_ in your macOS system.
2. Click _Keyboard_.
3. Select _Keyboard Shortcuts_.
4. Select _All controls_ under the _Full Keyboard Access_ section.

### Safari browser

To enable the use of the `Tab` key in Safari, perform the following steps:

1. Open Safari.
2. In the menu bar, click _Safari_ and then select _Preferences_ from the menu.
3. Select _Advanced_ from the _Preferences_ window.
4. Select the checkbox next to _Press Tab_ to highlight each item on a webpage.

### Google Chrome browser

To enable the use of the `Tab` key in Google Chrome, perform the following steps:

1. Open Chrome.
2. In the menu bar, click _Chrome_ and then select _Preferences_ from the menu.
3. Select _Appearance_ from the menu.
4. Toggle the _Pressing Tab on a webpage highlights links, as well as form fields_ to enabled.
