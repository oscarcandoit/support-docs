# whmui

* * *

#### [Add a Package](https://docs.cpanel.net/whm/packages/add-a-package/)

This feature allows you to create a new package that you can assign to any account on your server.

[Read more](https://docs.cpanel.net/whm/packages/add-a-package/)

* * *

#### [Additional MySQL Access Hosts](https://docs.cpanel.net/whm/sql-services/additional-mysql-access-hosts/)

This interface allows you to permit remote hosts (servers) to access MySQL® databases on your server, or to access a remote MySQL server that you configured via WHM's Manage MySQL Profiles interface.

[Read more](https://docs.cpanel.net/whm/sql-services/additional-mysql-access-hosts/)

* * *

#### [Apache Configuration](https://docs.cpanel.net/whm/service-configuration/apache-configuration/)

This collection of features allows you to configure Apache.

[Read more](https://docs.cpanel.net/whm/service-configuration/apache-configuration/)

* * *

#### [Apache mod\_userdir Tweak](https://docs.cpanel.net/whm/security-center/apache-mod_userdir-tweak/)

This interface allows you to disable the Apache \`mod\_userdir\` module's functionality for your users.

[Read more](https://docs.cpanel.net/whm/security-center/apache-mod_userdir-tweak/)

* * *

#### [Apache Status](https://docs.cpanel.net/whm/server-status/apache-status/)

This interface displays the status of your Apache server.

[Read more](https://docs.cpanel.net/whm/server-status/apache-status/)

* * *

#### [API Shell for WHM](https://docs.cpanel.net/whm/development/api-shell-for-whm/)

Allows you to call WHM API 1 functions from the UI.

[Read more](https://docs.cpanel.net/whm/development/api-shell-for-whm/)

* * *

#### [Apps Managed by AppConfig](https://docs.cpanel.net/whm/development/apps-managed-by-appconfig/)

This interface displays applications that you installed and registered with AppConfig.

[Read more](https://docs.cpanel.net/whm/development/apps-managed-by-appconfig/)

* * *

#### [Background Process Killer](https://docs.cpanel.net/whm/system-health/background-process-killer/)

This interface allows you to select processes that the system will terminate when the upcp script calls the system maintenance script every night.

[Read more](https://docs.cpanel.net/whm/system-health/background-process-killer/)

* * *

#### [Backup Configuration](https://docs.cpanel.net/whm/backup/backup-configuration/)

The Backup Configuration interface allows system administrators to customize their scheduled backups.

[Read more](https://docs.cpanel.net/whm/backup/backup-configuration/)

* * *

#### [Backup System Migration](https://docs.cpanel.net/whm/backup/backup-system-migration/)

The Backup System Migration interface automatically converts and migrates any Legacy Backup system files to the current Backup system.

[Read more](https://docs.cpanel.net/whm/backup/backup-system-migration/)

* * *

#### [Basic WebHost Manager Setup](https://docs.cpanel.net/whm/server-configuration/basic-webhost-manager-setup/)

Use this interface to configure basic settings for cPanel & WHM.

[Read more](https://docs.cpanel.net/whm/server-configuration/basic-webhost-manager-setup/)

* * *

#### [Change Database Root Password](https://docs.cpanel.net/whm/database-services/change-database-root-password/)

This interface allows you to set the MySQL® or MariaDB® root account password.

[Read more](https://docs.cpanel.net/whm/database-services/change-database-root-password/)

* * *

#### [Change Database User Password](https://docs.cpanel.net/whm/database-services/change-database-user-password/)

This interface allows you to change a database user's password.

[Read more](https://docs.cpanel.net/whm/database-services/change-database-user-password/)

* * *

#### [Change Hostname](https://docs.cpanel.net/whm/networking-setup/change-hostname/)

This interface allows you to change your server's hostname.

[Read more](https://docs.cpanel.net/whm/networking-setup/change-hostname/)

* * *

#### [Change Log](https://docs.cpanel.net/whm/cpanel/change-log/)

This interface will automatically redirect you to our Change Logs documentation.

[Read more](https://docs.cpanel.net/whm/cpanel/change-log/)

* * *

#### [Change Multiple Sites' IP Addresses](https://docs.cpanel.net/whm/multi-account-functions/change-multiple-sites-ip-addresses/)

This interface allows you to change the IP addresses of multiple accounts simultaneously.

[Read more](https://docs.cpanel.net/whm/multi-account-functions/change-multiple-sites-ip-addresses/)

* * *

#### [Change MySQL User Password](https://docs.cpanel.net/whm/sql-services/change-mysql-user-password/)

This interface allows you to change a database user's password.

[Read more](https://docs.cpanel.net/whm/sql-services/change-mysql-user-password/)

* * *

#### [Change Ownership of Multiple Accounts](https://docs.cpanel.net/whm/resellers/change-ownership-of-multiple-accounts/)

This feature allows you to transfer multiple accounts to a reseller.

[Read more](https://docs.cpanel.net/whm/resellers/change-ownership-of-multiple-accounts/)

* * *

#### [Change Root Password](https://docs.cpanel.net/whm/server-configuration/change-root-password/)

This interface allows you to change the root user's password.

[Read more](https://docs.cpanel.net/whm/server-configuration/change-root-password/)

* * *

#### [Change Site's IP Address](https://docs.cpanel.net/whm/account-functions/change-site-ip-address/)

This interface allows you to change an account's IP address.

[Read more](https://docs.cpanel.net/whm/account-functions/change-site-ip-address/)
