## Database Services

* * *

#### [Change Database Root Password](https://docs.cpanel.net/whm/database-services/change-database-root-password/)

This interface allows you to set the MySQL® or MariaDB® root account password.

[Read more](https://docs.cpanel.net/whm/database-services/change-database-root-password/)

* * *

#### [Change Database User Password](https://docs.cpanel.net/whm/database-services/change-database-user-password/)

This interface allows you to change a database user's password.

[Read more](https://docs.cpanel.net/whm/database-services/change-database-user-password/)

* * *

#### [Configure PostgreSQL](https://docs.cpanel.net/whm/database-services/configure-postgresql/)

This interface allows you to complete the PostgreSQL® installation process, create PostgreSQL users, and update the PostgreSQL password.

[Read more](https://docs.cpanel.net/whm/database-services/configure-postgresql/)

* * *

#### [Database Map Tool](https://docs.cpanel.net/whm/database-services/database-map-tool/)

This feature allows server administrators to grant ownership of database objects (databases and database users) to a cPanel user.

[Read more](https://docs.cpanel.net/whm/database-services/database-map-tool/)

* * *

#### [Edit Database Configuration](https://docs.cpanel.net/whm/database-services/edit-database-configuration/)

This interface allows you to make changes to your MariaDB or MySQL configuration.

[Read more](https://docs.cpanel.net/whm/database-services/edit-database-configuration/)

* * *

#### [Manage Database Access Hosts](https://docs.cpanel.net/whm/database-services/manage-database-access-hosts/)

This interface allows you to permit remote hosts (servers) to access MySQL® and MariaDB® databases on your server. It also allows you to access remote database servers that you configured via WHM's Manage Database Profiles interface.

[Read more](https://docs.cpanel.net/whm/database-services/manage-database-access-hosts/)

* * *

#### [Manage Database Profiles](https://docs.cpanel.net/whm/database-services/manage-database-profiles/)

This interface allows you to manage multiple MySQL® or MariaDB® profiles.

[Read more](https://docs.cpanel.net/whm/database-services/manage-database-profiles/)

* * *

#### [Manage Database Users](https://docs.cpanel.net/whm/database-services/manage-database-users/)

This interface allows server administrators to retrieve a list of MySQL® and MariaDB® database users, which account owns them, and their databases.

[Read more](https://docs.cpanel.net/whm/database-services/manage-database-users/)

* * *

#### [Manage Databases](https://docs.cpanel.net/whm/database-services/manage-databases/)

This interface allows you to list cPanel-managed databases on your server, each database's engine, and which user owns each database.

[Read more](https://docs.cpanel.net/whm/database-services/manage-databases/)

* * *

#### [Repair Databases](https://docs.cpanel.net/whm/database-services/repair-databases/)

This interface allows you to repair a MySQL® or MariaDB® database.

[Read more](https://docs.cpanel.net/whm/database-services/repair-databases/)

* * *

#### [Show Database Processes](https://docs.cpanel.net/whm/database-services/show-database-processes/)

This interface lists all of the processes that currently run on any database on your server.

[Read more](https://docs.cpanel.net/whm/database-services/show-database-processes/)

* * *

#### [Upgrade Database Version](https://docs.cpanel.net/whm/database-services/upgrade-database-version/)

This interface allows you to manage MySQL® or MariaDB® version upgrades.

[Read more](https://docs.cpanel.net/whm/database-services/upgrade-database-version/)
