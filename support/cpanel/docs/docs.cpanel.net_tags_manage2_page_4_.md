# manage2

* * *

#### [Search for a Group or Direct Account](https://docs.cpanel.net/manage2/direct/search-for-a-group-or-direct-account/)

This interface allows you to search for an individual Group or Direct Account.

[Read more](https://docs.cpanel.net/manage2/direct/search-for-a-group-or-direct-account/)

* * *

#### [Search for a License](https://docs.cpanel.net/manage2/licenses/search-for-a-license/)

This interface allows you to search the licenses in your account. By default, the search results include both active and expired licenses.

[Read more](https://docs.cpanel.net/manage2/licenses/search-for-a-license/)

* * *

#### [Security Setup](https://docs.cpanel.net/manage2/security/security-setup/)

This interface allows you to configure your Manage2 account's security questions.

[Read more](https://docs.cpanel.net/manage2/security/security-setup/)

* * *

#### [Setup Automatic ACH Payment](https://docs.cpanel.net/manage2/billing/setup-automatic-ach-payment/)

Use this interface to generate a form to enroll in ACH payments from your checking or savings account.

[Read more](https://docs.cpanel.net/manage2/billing/setup-automatic-ach-payment/)

* * *

#### [Show Duplicate Licenses](https://docs.cpanel.net/manage2/licenses/show-duplicate-licenses/)

This interface displays any duplicate license attached to an IP address.

[Read more](https://docs.cpanel.net/manage2/licenses/show-duplicate-licenses/)

* * *

#### [Submit a New Priority Support Request](https://docs.cpanel.net/manage2/support/submit-a-new-priority-support-request/)

This link automatically forwards you to cPanel's cPanel Customer Portal interface.

[Read more](https://docs.cpanel.net/manage2/support/submit-a-new-priority-support-request/)

* * *

#### [Support Usage Report](https://docs.cpanel.net/manage2/support/support-usage-report/)

This interface provides a detailed overview for support incident tickets usage.

[Read more](https://docs.cpanel.net/manage2/support/support-usage-report/)

* * *

#### [The Manage2 Dashboard](https://docs.cpanel.net/manage2/admin/the-manage2-dashboard/)

Manage2's Dashboard interface provides you with access to features that allow you to manage your cPanel & WHM licenses.

[Read more](https://docs.cpanel.net/manage2/admin/the-manage2-dashboard/)

* * *

#### [Transfer a license](https://docs.cpanel.net/manage2/licenses/transfer-a-license/)

This interface allows you to transfer an active license from the current IP address to a new IP address.

[Read more](https://docs.cpanel.net/manage2/licenses/transfer-a-license/)

* * *

#### [Update Company Information](https://docs.cpanel.net/manage2/company/update-company-information/)

This interface allows you to view or update your company information.

[Read more](https://docs.cpanel.net/manage2/company/update-company-information/)

* * *

#### [Update My Information](https://docs.cpanel.net/manage2/users/update-my-information/)

This interface allows you to update the information for your own user account.

[Read more](https://docs.cpanel.net/manage2/users/update-my-information/)

* * *

#### [Upload Forms](https://docs.cpanel.net/manage2/billing/upload-forms/)

This interface allows you to upload digital copies of authorization forms.

[Read more](https://docs.cpanel.net/manage2/billing/upload-forms/)

* * *

#### [View Contracts and Agreements](https://docs.cpanel.net/manage2/company/view-contracts-and-agreements/)

The View Contracts and Agreements interface allows you to download contracts and agreements associated with your account.

[Read more](https://docs.cpanel.net/manage2/company/view-contracts-and-agreements/)

* * *

#### [View Open and Closed Requests](https://docs.cpanel.net/manage2/support/view-open-and-closed-requests/)

This feature directs you to the Help Center interface and displays information for your support tickets.

[Read more](https://docs.cpanel.net/manage2/support/view-open-and-closed-requests/)

* * *

#### [View Yearly License Renewal Options](https://docs.cpanel.net/manage2/direct/view-yearly-license-renewal-options/)

This interface allows you to change license renewal options.

[Read more](https://docs.cpanel.net/manage2/direct/view-yearly-license-renewal-options/)
