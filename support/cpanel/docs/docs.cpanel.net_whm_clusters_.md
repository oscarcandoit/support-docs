## Clusters

* * *

#### [Configuration Cluster](https://docs.cpanel.net/whm/clusters/configuration-cluster/)

This interface allows you to link a master server to additional servers in the configuration cluster.

[Read more](https://docs.cpanel.net/whm/clusters/configuration-cluster/)

* * *

#### [DNS Cluster](https://docs.cpanel.net/whm/clusters/dns-cluster/)

This interface allows you to configure a DNS cluster and add servers to an existing DNS cluster.

[Read more](https://docs.cpanel.net/whm/clusters/dns-cluster/)

* * *

#### [Remote Access Key](https://docs.cpanel.net/whm/clusters/remote-access-key/)

You can use this feature to generate an access key, which is also known as an access hash.

[Read more](https://docs.cpanel.net/whm/clusters/remote-access-key/)
