# quotas

* * *

#### [Bandwidth](https://docs.cpanel.net/cpanel/metrics/bandwidth/)

The Bandwidth interface displays bandwidth usage information in several sets of graphs.

[Read more](https://docs.cpanel.net/cpanel/metrics/bandwidth/)

* * *

#### [Disk Usage](https://docs.cpanel.net/cpanel/files/disk-usage/)

This interface displays information about your files, your directories, and how you use your account's disk space.

[Read more](https://docs.cpanel.net/cpanel/files/disk-usage/)

* * *

#### [Email Disk Usage](https://docs.cpanel.net/cpanel/email/email-disk-usage/)

This interface allows you to view all of your mailboxes and remove old or large messages.

[Read more](https://docs.cpanel.net/cpanel/email/email-disk-usage/)

* * *

#### [Enable Quotas on a Virtuozzo VPS](https://docs.cpanel.net/whm/server-configuration/enable-quotas-on-a-virtuozzo-vps/)

If you run cPanel & WHM on a Virtuozzo® Virtual Private Server (VPS), and you disable quotas, you must perform extra steps to reenable quotas.

[Read more](https://docs.cpanel.net/whm/server-configuration/enable-quotas-on-a-virtuozzo-vps/)

* * *

#### [FTP Accounts](https://docs.cpanel.net/cpanel/files/ftp-accounts/)

You can use this interface to create and manage your website's File Transfer Protocol (FTP) accounts.

[Read more](https://docs.cpanel.net/cpanel/files/ftp-accounts/)

* * *

#### [How to Disable Filesystem Quotas](https://docs.cpanel.net/knowledge-base/accounts/how-to-disable-filesystem-quotas/)

This document explains how to disable filesystem quotas on your server.

[Read more](https://docs.cpanel.net/knowledge-base/accounts/how-to-disable-filesystem-quotas/)

* * *

#### [How to Fix Quotas](https://docs.cpanel.net/knowledge-base/general-systems-administration/how-to-fix-quotas/)

This document describes how to confirm whether you properly configured the disk space quotas on your system's devices.

[Read more](https://docs.cpanel.net/knowledge-base/general-systems-administration/how-to-fix-quotas/)

* * *

#### [Initial Quota Setup](https://docs.cpanel.net/whm/server-configuration/initial-quota-setup/)

The Initial Quota Setup interface scans your server to confirm that it uses disk space quotas on the directories in which your cPanel users store their files.

[Read more](https://docs.cpanel.net/whm/server-configuration/initial-quota-setup/)

* * *

#### [Limit Bandwidth Usage](https://docs.cpanel.net/whm/account-functions/limit-bandwidth-usage/)

This interface allows you to change the allotted bandwidth for an account.

[Read more](https://docs.cpanel.net/whm/account-functions/limit-bandwidth-usage/)

* * *

#### [Overselling](https://docs.cpanel.net/knowledge-base/cpanel-product/overselling/)

This document explains the concept of overselling. Overselling means that the system limits resellers via the resources that their cPanel accounts actually use instead of the resources that you allocate to them.

[Read more](https://docs.cpanel.net/knowledge-base/cpanel-product/overselling/)

* * *

#### [Quota Modification](https://docs.cpanel.net/whm/account-functions/quota-modification/)

This interface allows you to change an account's disk space quota.

[Read more](https://docs.cpanel.net/whm/account-functions/quota-modification/)

* * *

#### [Reset Account Bandwidth Limit](https://docs.cpanel.net/whm/account-functions/reset-account-bandwidth-limit/)

If an account's bandwidth quota does not match the quota for the account's package, this interface enables you to reset the account's quota.

[Read more](https://docs.cpanel.net/whm/account-functions/reset-account-bandwidth-limit/)

* * *

#### [Show Accounts Over Quota](https://docs.cpanel.net/whm/account-information/show-accounts-over-quota/)

This interface allows you to list and manage accounts that exceed their disk quota.

[Read more](https://docs.cpanel.net/whm/account-information/show-accounts-over-quota/)

* * *

#### [The balance\_linked\_node\_quotas Script](https://docs.cpanel.net/whm/scripts/the-balance_linked_node_quotas-script/)

This script lets you enforce account quotas across multiple servers.

[Read more](https://docs.cpanel.net/whm/scripts/the-balance_linked_node_quotas-script/)

* * *

#### [The Quota File Systems Configuration File](https://docs.cpanel.net/knowledge-base/cpanel-product/the-quota-file-systems-configuration-file/)

This file allows you to specify file system types that you wish to exclude from cPanel & WHM's quota system.

[Read more](https://docs.cpanel.net/knowledge-base/cpanel-product/the-quota-file-systems-configuration-file/)

* * *

#### [Unsuspend Bandwidth Exceeders](https://docs.cpanel.net/whm/account-functions/unsuspend-bandwidth-exceeders/)

This feature allows you to simultaneously unsuspend all of the accounts that have exceeded their bandwidth.

[Read more](https://docs.cpanel.net/whm/account-functions/unsuspend-bandwidth-exceeders/)
