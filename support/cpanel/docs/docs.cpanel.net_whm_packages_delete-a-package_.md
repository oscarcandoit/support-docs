## Overview

This feature allows you to remove a package from your server.

Warning:

- Exercise caution when you delete a package, because you **cannot** recover a deleted package; you will have to recreate it.
- You **cannot** delete a package that an account on your server currently uses.

## How to delete a package

To delete a package, perform the following steps:

1. Select the package that you wish to delete from the _Package_ menu.
2. Click _Delete_ to delete the package.
