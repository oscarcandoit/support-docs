## Support

* * *

#### [Create Support Ticket](https://docs.cpanel.net/whm/support/create-support-ticket/)

The interface allows you to open a ticket with cPanel's Technical Support.

[Read more](https://docs.cpanel.net/whm/support/create-support-ticket/)

* * *

#### [Grant cPanel Support Access](https://docs.cpanel.net/whm/support/grant-cpanel-support-access/)

This interface uses the server authentication details in your ticket to automatically provide cPanel Technical Support with SSH access to your server.

[Read more](https://docs.cpanel.net/whm/support/grant-cpanel-support-access/)

* * *

#### [Support Center](https://docs.cpanel.net/whm/support/support-center/)

The Support Center interface contains several links that allow you to access cPanel & WHM support features.

[Read more](https://docs.cpanel.net/whm/support/support-center/)
