## How to Purchase a Sitejet Commerce License

Last modified: _2024 September 5_

* * *

## Overview

Sitejet Commerce, part of our suite of products powered by [Sitejet](https://www.sitejet.io/en), helps you build and manage web stores.

## Purchase a Sitejet Commerce license from Manage2

Important:

To buy a Sitejet Commerce license through Manage2, you **must** be a cPanel Partner.

To purchase a Sitejet Commerce license, perform the following steps:

1. Log in to the [Manage2](https://manage2.cpanel.net/) interface.
2. In the _Licenses_ section, click _Add License_.
3. Select one of the following Commerce plans:

- `Sitejet Commerce Starter` — For web stores that manage up to 100 products.
- `Sitejet Commerce Professional` — For web stores that manage up to 2500 products.
- `Sitejet Commerce Unlimited` — For web stores that manage more than 2500 products.

4. Click _Add Licenses_.

A success message containing the activation link will appear at the top of the Manage2 interface.

For more information, read our [Add License](https://docs.cpanel.net/manage2/licenses/add-license) documentation.
