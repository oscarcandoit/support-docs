# packages

* * *

#### [Add a Package](https://docs.cpanel.net/whm/packages/add-a-package/)

This feature allows you to create a new package that you can assign to any account on your server.

[Read more](https://docs.cpanel.net/whm/packages/add-a-package/)

* * *

#### [Delete a Package](https://docs.cpanel.net/whm/packages/delete-a-package/)

This feature allows you to remove a package from your server.

[Read more](https://docs.cpanel.net/whm/packages/delete-a-package/)

* * *

#### [Edit a Package](https://docs.cpanel.net/whm/packages/edit-a-package/)

This feature allows you to edit existing packages.

[Read more](https://docs.cpanel.net/whm/packages/edit-a-package/)

* * *

#### [Feature Manager](https://docs.cpanel.net/whm/packages/feature-manager/)

This interface allows you to edit feature lists, which you can assign to packages that you apply to cPanel accounts.

[Read more](https://docs.cpanel.net/whm/packages/feature-manager/)

* * *

#### [Modify/Upgrade Multiple Accounts](https://docs.cpanel.net/whm/multi-account-functions/modify-upgrade-multiple-accounts/)

This interface allows you to simultaneously change the attributes of multiple accounts to use the same settings.

[Read more](https://docs.cpanel.net/whm/multi-account-functions/modify-upgrade-multiple-accounts/)

* * *

#### [The modify\_packages Script](https://docs.cpanel.net/whm/scripts/the-modify_packages-script/)

You can use this script to set existing packages to use a new cPanel theme.

[Read more](https://docs.cpanel.net/whm/scripts/the-modify_packages-script/)

* * *

#### [Upgrade/Downgrade an Account](https://docs.cpanel.net/whm/account-functions/upgrade-downgrade-an-account/)

This interface allows you to change an account’s package.

[Read more](https://docs.cpanel.net/whm/account-functions/upgrade-downgrade-an-account/)
