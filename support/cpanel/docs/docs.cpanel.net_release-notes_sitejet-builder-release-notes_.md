## Sitejet Builder Release Notes

Last modified: _2025 October 24_

* * *

## Sitejet Builder version 4.4.0-1

_Released October 2025_

##### Integrated Cloudflare Turnstile CAPTCHA

Sitejet Builder now includes the [Cloudflare® Turnstile](https://developers.cloudflare.com/turnstile/) CAPTCHA solution. For each CAPTCHA element, you can now choose either a built-in CAPTCHA or a Turnstile solution.

To set up Turnstile, you must first [create a Cloudflare account](https://www.cloudflare.com/lp/turnstile/). Then, read the [SiteJet Turnstile CAPTCHA](https://help.sitejet.io/hc/en-us/articles/32574598962967-Captcha#h_01K06Z58FCZMSTGCP97JN69P5H) documentation to learn how to create a Turnstile widget and include it in SiteJet Builder.

* * *

## Sitejet Builder version 4.2.0-1

_Released August 2025_

##### Fully Released the AI Website Generator

Sitejet Builder’s AI Website Generator is now out of beta. The AI Website Generator allows users to power their website design with AI in the [Sitejet Website Builder (Content Management System)](https://help.sitejet.io/). The AI generator will ask you a series of questions about your website, then create text and images for your website based on your answers. You can still make changes manually to your website if desired.

For more information, read cPanel’s [Sitejet Builder](https://docs.cpanel.net/cpanel/domains/sitejet-builder) documentation and the [Sitejet Website Builder](https://help.sitejet.io/hc/en-us/articles/24275952909463-Overview-Website-Builder-CMS) documentation.

## Sitejet Builder version 4.1.0-3

_Released July 2025_

##### Added AI Website Generator Availability to Sitejet Builder

The AI Website Generator adds AI website generation functionality to [Sitejet Website Builder (Content Management System)](https://help.sitejet.io/) website. The AI generator will ask you a series of questions about your website and generate text and images for your website based on your answers. You can still make changes manually to your website if desired.

This feature is in BETA and your [hosting provider](https://docs.cpanel.net/knowledge-base/cpanel-product/how-to-identify-your-hosting-provider/) **must** enable this feature for it to be available.

For more information, read cPanel’s [Sitejet Builder](https://docs.cpanel.net/cpanel/domains/sitejet-builder) documentation and the [Sitejet Website Builder](https://help.sitejet.io/hc/en-us/articles/24275952909463-Overview-Website-Builder-CMS) documentation.

##### Modified WHM API 1 functions

- [`local_enable`](https://api.docs.cpanel.net/openapi/whm/operation/local_enable/) — We added the `namespace` parameter.
- [`local_disable`](https://api.docs.cpanel.net/openapi/whm/operation/local_disable/) — We added the `namespace` parameter.

## Sitejet Builder version 3.2.0-1

_Released April 2025_

#### Added Sitejet Domains interface

We added the _Sitejet Domains_ interface to the Sitejet Builder feature. This updates the workflow for creating a Sitejet website. When you _Create a New Website_ in the _Sitejet Builder_ interface, the system redirects you to the _Sitejet Domains_ interface. Before, the system redirected you to the _Domains_ interface. For more information, read our [_Sitejet Builder_](https://docs.cpanel.net/cpanel/domains/sitejet-builder/#create-a-new-sitejet-website) interface documentation.

##### New UAPI functions

- [`Sitejet::can_create_domains`](https://api.docs.cpanel.net/openapi/cpanel/operation/can_create_domains/) \- This function returns whether a cPanel account can create new subdomains or addon domains.

## Sitejet Builder version 1.6.0

_Released October 2024_

##### New WHM API 1 functions

- [`get_commerce`](https://api.docs.cpanel.net/openapi/whm/operation/get_commerce/) – This function returns Sitejet Commerce data.
- [`set_commerce`](https://api.docs.cpanel.net/openapi/whm/operation/set_commerce/) – This function enables Sitejet Commerce for all users and allows the hosting provider to set a custom URL for their payment platform, which customers can use to purchase the Sitejet Commerce upgrade.

##### Deprecated API 1 functions

- `get_ecommerce` – We renamed this function [`get_commerce`](https://api.docs.cpanel.net/openapi/whm/operation/get_commerce/).
- `set_ecommerce` – We renamed this function [`set_commerce`](https://api.docs.cpanel.net/openapi/whm/operation/set_commerce/).

## Sitejet Builder version 1.5.0

_Released August 2024_

#### Updated French translations

We added an improved set of French translations. This will provide a better experience for users who prefer French.

## Sitejet Builder version 1.4.1-2

_Released July 2024_

#### Added ability to enable and disable Sitejet Commerce

We added the ability for hosting providers to enable and disable [Sitejet Commerce](http://www.sitejet.io/en/commerce) using WHM’s [`get_ecommerce`](https://api.docs.cpanel.net/openapi/whm/operation/get_ecommerce/) and [`set_ecommerce`](https://api.docs.cpanel.net/openapi/whm/operation/set_ecommerce/) functions. Sitejet Commerce allows users to create, design, and manage a store on their website.

##### New WHM API 1 functions

- [`get_ecommerce`](https://api.docs.cpanel.net/openapi/whm/operation/get_ecommerce/) – This function returns Sitejet Commerce data.
- [`set_ecommerce`](https://api.docs.cpanel.net/openapi/whm/operation/set_ecommerce/) – This function enables Sitejet Commerce for all users and allows the hosting provider to set a custom URL for their payment platform, which customers can use to purchase the Sitejet Commerce upgrade.

## Sitejet Builder version 1.4.0

_Released April 2024_

#### Updated suppported languages

We added language support for Spanish (Central American), Indonesian, Portuguese (Brazilian), Arabic, German, French, Japanese, and Turkish .

## Sitejet Builder version 1.3.0

_Released March 2024_

#### Added the ability to move a Sitejet website

We added the ability to [change a Sitejet website’s](https://docs.cpanel.net/cpanel/domains/sitejet-builder/#move-a-sitejet-website) document root.

#### Added progress bar

We added a progress bar to the _Website Details_ interface to show the publishing status of your Sitejet website.

## Sitejet Builder version 1.2.0

_Released March 2024_

#### Added a restore point

We added the ability to create a [restore point](https://docs.cpanel.net/knowledge-base/cpanel-product/cpanel-glossary/#restore-point). When you publish a website with Sitejet for the first time on a domain that currently has an existing website, we will give you the option to create a restore point. This feature allows the user to recover their non-Sitejet website at a future date.

#### Added a Continue Editing feature

We added the _Continue Editing_ button. This feature allows the user to continue editing an unpublished Sitejet website.

#### Renamed Build with Sitejet

We changed the name of the _Build with Sitejet_ button to _Try Sitejet_. The functionality is the same.

##### New UAPI functions

- [`Sitejet::create_restore_point`](https://api.docs.cpanel.net/openapi/cpanel/operation/create_restore_point/) – This function creates a restore point for a domain’s document root and removes the domain’s files.
- [`Sitejet::restore_document_root`](https://api.docs.cpanel.net/openapi/cpanel/operation/restore_document_root/) – This function reverts the document root to the restore point.

##### Modified UAPI functions

- [`Sitejet::get_all_user_sitejet_info`](https://api.docs.cpanel.net/openapi/cpanel/operation/get_all_user_sitejet_info/) – Added the `is_restore_point_available` field.

## Sitejet Builder version 1.1.0-3

_Released February 2024_

#### Added an interactive website preview

We added the ability to interact with the website preview. You can now scroll, click, and explore the site functionality as it will behave when published.

## Sitejet Builder version 1.0.1-45

_Released January 2024_

#### Added the Sitejet Builder interface

With the [_Sitejet Builder_](https://docs.cpanel.net/cpanel/domains/sitejet-builder) plugin, you can create and design a website directly in your web browser for your domain. Sitejet Builder is fully integrated with the [Sitejet Website Builder (CMS)](https://help.sitejet.io/) website. You can use the Sitejet Website Builder (CMS) to create a website for your domain. Then, you can publish it with your cPanel account.

Sitejet Website Builder (CMS) allows you to personalize the appearance of your website with a drag-drop editor or start from scratch and build your website from the ground up. Every template is fully customizable.

The [_Sitejet Builder_](https://docs.cpanel.net/cpanel/domains/sitejet-builder) interface ( _cPanel » Home » Domains » Sitejet Builder_) lists all the domains associated with your account. Use this interface to view and manage the websites created for your domains with the Sitejet Website Builder (CMS). If you have an existing website, you can use this interface to build a new website for your domain with Sitejet.

##### New UAPI functions

- [`Sitejet::create_account`](https://api.docs.cpanel.net/openapi/cpanel/operation/create_sitejet_account/) – This function creates a Sitejet API token for a cPanel account.
- [`Sitejet::get_templates`](https://api.docs.cpanel.net/openapi/cpanel/operation/get_sitejet_templates/) – This function fetches the list of available Sitejet templates.
- [`Sitejet::set_template`](https://api.docs.cpanel.net/openapi/cpanel/operation/set_sitejet_template/) – This function sets a Sitejet website’s template.
- [`Sitejet::get_api_token`](https://api.docs.cpanel.net/openapi/cpanel/operation/get_sitejet_api_token/) – This function returns the cPanel account’s Sitejet API token.
- [`Sitejet::add_api_token`](https://api.docs.cpanel.net/openapi/cpanel/operation/add_sitejet_api_token/) – This function adds an existing Sitejet API token to a cPanel user.
- [`Sitejet::create_website`](https://api.docs.cpanel.net/openapi/cpanel/operation/create_sitejet_website/) – This function creates a Sitejet ID for the specified domain.
- [`Sitejet::start_publish`](https://api.docs.cpanel.net/openapi/cpanel/operation/start_publish_sitejet/) – This function publishes a domain’s Sitejet website.
- [`Sitejet::get_all_user_sitejet_info`](https://api.docs.cpanel.net/openapi/cpanel/operation/get_all_user_sitejet_info/) – This function returns the Sitejet domains’ information for the cPanel account.
- [`Sitejet::get_sso_link`](https://api.docs.cpanel.net/openapi/cpanel/operation/get_sitejet_sso_link/) – This function returns a Sitejet website’s SSO URL.
- [`Sitejet::get_preview_url`](https://api.docs.cpanel.net/openapi/cpanel/operation/get_sitejet_preview_url/) – This function returns a Sitejet website’s preview URL.
