## Overview

The _Support Center_ interface contains several links that allow you to access cPanel & WHM support features.

## Resources

You can click on the following resources to help you administer your system and resolve issues:

- [_Support Forums_](https://forums.cpanel.net/) — The cPanel Support Forums.






Note:





You **must** register with the cPanel Support Forums and log in to post content in the cPanel Support Forums.

- [_WHM Documentation_](https://docs.cpanel.net/whm/) — Documentation for the WHM interface.
- [_cPanel Documentation_](https://docs.cpanel.net/cpanel/) — Documentation for the cPanel interface.
- [_cPanel Release Notifications_](https://cpanel.net/mailing-list/) — Add your name to the WebPros International, LLC mailing lists.
- [_cPanel News_](https://news.cpanel.com/) — Review the latest information about WebPros International, LLC.
- [_cPanel University_](https://university.cpanel.net/) — The cPanel University website.

## Contact cPanel

Use the following links to contact WebPros International, LLC to help you resolve issues:

- [_Submit a Support Request_](https://docs.cpanel.net/whm/support/create-support-ticket/) — Click to open WHM’s [_Create Support Ticket_](https://docs.cpanel.net/whm/support/create-support-ticket) interface ( _WHM » Home » Support » Create Support Ticket_). This interface allows you to create a new cPanel support request.






Note:





For more information about how to log in to the cPanel Customer Portal and update your support requests, read our [Contact cPanel](https://docs.cpanel.net/knowledge-base/technical-support-services/contact-cpanel/) documentation.

- [_Grant cPanel Support Access_](https://docs.cpanel.net/whm/support/grant-cpanel-support-access/) — Click to open WHM’s [_Grant cPanel Support Access_](https://docs.cpanel.net/whm/support/create-support-ticket) interface ( _WHM » Home » Support » Grant cPanel Support Access_). This interface allows you to configure access to your server for an open cPanel support request.
