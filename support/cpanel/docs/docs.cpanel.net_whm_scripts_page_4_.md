## Scripts

* * *

#### [The restorepkg Script](https://docs.cpanel.net/whm/scripts/the-restorepkg-script/)

This script allows you to restore a cPanel account from a backup file.

[Read more](https://docs.cpanel.net/whm/scripts/the-restorepkg-script/)

* * *

#### [The secureit Script](https://docs.cpanel.net/whm/scripts/the-secureit-script/)

The secureit script secures permissions on your server

[Read more](https://docs.cpanel.net/whm/scripts/the-secureit-script/)

* * *

#### [The securemysql Script](https://docs.cpanel.net/whm/scripts/the-securemysql-script/)

This script secures a cPanel account's MySQL® configuration.

[Read more](https://docs.cpanel.net/whm/scripts/the-securemysql-script/)

* * *

#### [The securetmp Script](https://docs.cpanel.net/whm/scripts/the-securetmp-script/)

This script secures the /tmp and /var/tmp directories.

[Read more](https://docs.cpanel.net/whm/scripts/the-securetmp-script/)

* * *

#### [The servicedomains Script](https://docs.cpanel.net/whm/scripts/the-servicedomains-script/)

This script lets you to create DNS records for service subdomains.

[Read more](https://docs.cpanel.net/whm/scripts/the-servicedomains-script/)

* * *

#### [The set\_hostname Utility](https://docs.cpanel.net/whm/scripts/the-set_hostname-utility/)

This script updates the server's hostname and performs all of the necessary actions to apply this change throughout the system.

[Read more](https://docs.cpanel.net/whm/scripts/the-set_hostname-utility/)

* * *

#### [The set-tls-settings Script](https://docs.cpanel.net/whm/scripts/the-set-tls-settings-script/)

This script configures a server's Secure Socket Layer (SSL) and Transport Layer Security (TLS) cipher suites and protocols.

[Read more](https://docs.cpanel.net/whm/scripts/the-set-tls-settings-script/)

* * *

#### [The setup\_greylist\_db Script](https://docs.cpanel.net/whm/scripts/the-setup_greylist_db-script/)

The setup\_grelist\_db script sets up the SQLite database for the cPanel Greylisting service.

[Read more](https://docs.cpanel.net/whm/scripts/the-setup_greylist_db-script/)

* * *

#### [The setupftpserver script](https://docs.cpanel.net/whm/scripts/the-setupftpserver-script/)

The setupftpserver script allows you to select your server's FTP server, configure anonymous FTP services, or disable FTP services.

[Read more](https://docs.cpanel.net/whm/scripts/the-setupftpserver-script/)

* * *

#### [The setupmailserver Script](https://docs.cpanel.net/whm/scripts/the-setupmailserver-script/)

This script lets you select a mail server.

[Read more](https://docs.cpanel.net/whm/scripts/the-setupmailserver-script/)

* * *

#### [The slurp\_exim\_mainlog Script](https://docs.cpanel.net/whm/scripts/the-slurp_exim_mainlog-script/)

This script imports all unarchived Exim /var/log/exim\_mainlog files into the eximstats database.

[Read more](https://docs.cpanel.net/whm/scripts/the-slurp_exim_mainlog-script/)

* * *

#### [The snapshot\_prep Script](https://docs.cpanel.net/whm/scripts/the-snapshot_prep-script/)

This script prepares your server to create a deployment image.

[Read more](https://docs.cpanel.net/whm/scripts/the-snapshot_prep-script/)

* * *

#### [The spf\_installer Script](https://docs.cpanel.net/whm/scripts/the-spf_installer-script/)

The /usr/local/cpanel/bin/spf\_installer script allows you to add an SPF record to all of a user's domains' zone files from the command line.

[Read more](https://docs.cpanel.net/whm/scripts/the-spf_installer-script/)

* * *

#### [The suspendacct Script](https://docs.cpanel.net/whm/scripts/the-suspendacct-script/)

This script allows you to suspend an account.

[Read more](https://docs.cpanel.net/whm/scripts/the-suspendacct-script/)

* * *

#### [The swapip Script](https://docs.cpanel.net/whm/scripts/the-swapip-script/)

Use this script to update a domain's IP address in the zone file.

[Read more](https://docs.cpanel.net/whm/scripts/the-swapip-script/)

* * *

#### [The sync\_child\_accounts Script](https://docs.cpanel.net/whm/scripts/the-sync_child_accounts-script/)

This script synchronizes a distributed cPanel account between a parent node and its child nodes.

[Read more](https://docs.cpanel.net/whm/scripts/the-sync_child_accounts-script/)

* * *

#### [The transfer\_account\_as\_user Script](https://docs.cpanel.net/whm/scripts/the-transfer_account_as_user-script/)

This script transfers a cPanel account from a remote server to the local server.

[Read more](https://docs.cpanel.net/whm/scripts/the-transfer_account_as_user-script/)

* * *

#### [The transfer\_accounts\_as\_root Script](https://docs.cpanel.net/whm/scripts/the-transfer_accounts_as_root-script/)

This script transfers one or more cPanel accounts from a remote server to the local server.

[Read more](https://docs.cpanel.net/whm/scripts/the-transfer_accounts_as_root-script/)

* * *

#### [The try-later Script](https://docs.cpanel.net/whm/scripts/the-try-later-script/)

This script runs a user-defined check command at a specified interval to check it can run a user-defined action command.

[Read more](https://docs.cpanel.net/whm/scripts/the-try-later-script/)

* * *

#### [The uninstall\_plugin Script](https://docs.cpanel.net/whm/scripts/the-uninstall_plugin-script/)

This script uninstalls a cPanel plugin.

[Read more](https://docs.cpanel.net/whm/scripts/the-uninstall_plugin-script/)
