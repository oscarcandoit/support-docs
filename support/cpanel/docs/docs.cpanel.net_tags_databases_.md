# databases

* * *

#### [Additional MySQL Access Hosts](https://docs.cpanel.net/whm/sql-services/additional-mysql-access-hosts/)

This interface allows you to permit remote hosts (servers) to access MySQL® databases on your server, or to access a remote MySQL server that you configured via WHM's Manage MySQL Profiles interface.

[Read more](https://docs.cpanel.net/whm/sql-services/additional-mysql-access-hosts/)

* * *

#### [Change Database Root Password](https://docs.cpanel.net/whm/database-services/change-database-root-password/)

This interface allows you to set the MySQL® or MariaDB® root account password.

[Read more](https://docs.cpanel.net/whm/database-services/change-database-root-password/)

* * *

#### [Change Database User Password](https://docs.cpanel.net/whm/database-services/change-database-user-password/)

This interface allows you to change a database user's password.

[Read more](https://docs.cpanel.net/whm/database-services/change-database-user-password/)

* * *

#### [Change MySQL User Password](https://docs.cpanel.net/whm/sql-services/change-mysql-user-password/)

This interface allows you to change a database user's password.

[Read more](https://docs.cpanel.net/whm/sql-services/change-mysql-user-password/)

* * *

#### [Configure PostgreSQL](https://docs.cpanel.net/whm/database-services/configure-postgresql/)

This interface allows you to complete the PostgreSQL® installation process, create PostgreSQL users, and update the PostgreSQL password.

[Read more](https://docs.cpanel.net/whm/database-services/configure-postgresql/)

* * *

#### [Configure PostgreSQL](https://docs.cpanel.net/whm/sql-services/configure-postgresql/)

This interface allows you to complete the PostgreSQL® installation process, create PostgreSQL users, and update the PostgreSQL password.

[Read more](https://docs.cpanel.net/whm/sql-services/configure-postgresql/)

* * *

#### [Create an Amazon® RDS™ Instance](https://docs.cpanel.net/knowledge-base/web-services/create-an-amazon-rds-instance/)

This document explains how to create an Amazon® RDS™ instance.

[Read more](https://docs.cpanel.net/knowledge-base/web-services/create-an-amazon-rds-instance/)

* * *

#### [Database Map Tool](https://docs.cpanel.net/whm/database-services/database-map-tool/)

This feature allows server administrators to grant ownership of database objects (databases and database users) to a cPanel user.

[Read more](https://docs.cpanel.net/whm/database-services/database-map-tool/)

* * *

#### [Database Map Tool](https://docs.cpanel.net/whm/sql-services/database-map-tool/)

This feature allows server administrators to grant ownership of database objects (databases and database users) to a cPanel user.

[Read more](https://docs.cpanel.net/whm/sql-services/database-map-tool/)

* * *

#### [Database Wizard](https://docs.cpanel.net/cpanel/databases/database-wizard/)

This wizard guides you through the setup of a MySQL® or MariaDB® database, user accounts, and user privileges.

[Read more](https://docs.cpanel.net/cpanel/databases/database-wizard/)

* * *

#### [Edit Database Configuration](https://docs.cpanel.net/whm/database-services/edit-database-configuration/)

This interface allows you to make changes to your MariaDB or MySQL configuration.

[Read more](https://docs.cpanel.net/whm/database-services/edit-database-configuration/)

* * *

#### [Edit SQL Configuration](https://docs.cpanel.net/whm/sql-services/edit-sql-configuration/)

This interface allows you to make changes to your MariaDB or MySQL configuration.

[Read more](https://docs.cpanel.net/whm/sql-services/edit-sql-configuration/)

* * *

#### [How to Delete a MySQL® Database](https://docs.cpanel.net/knowledge-base/sql/how-to-delete-a-mysql-database/)

This document describes how to delete a MySQL® database from a cPanel & WHM server.

[Read more](https://docs.cpanel.net/knowledge-base/sql/how-to-delete-a-mysql-database/)

* * *

#### [How to Enable the Slow Query Log in MySQL® or MariaDB](https://docs.cpanel.net/knowledge-base/sql/how-to-enable-the-slow-query-log-in-mysql-or-mariadb/)

This document describes how to enable the MySQL® or MariaDB Slow Query Log.

[Read more](https://docs.cpanel.net/knowledge-base/sql/how-to-enable-the-slow-query-log-in-mysql-or-mariadb/)

* * *

#### [How to Restore a User's Database Access](https://docs.cpanel.net/knowledge-base/sql/how-to-restore-a-user-database-access/)

You can restore the user's grants to the database with the restoregrants utility.

[Read more](https://docs.cpanel.net/knowledge-base/sql/how-to-restore-a-user-database-access/)

* * *

#### [How to Restore Database Grants](https://docs.cpanel.net/knowledge-base/scripts-and-utilities/how-to-restore-database-grants/)

If a database user loses access to databases, you can restore the user's grants to the database with the restoregrants utility.

[Read more](https://docs.cpanel.net/knowledge-base/scripts-and-utilities/how-to-restore-database-grants/)

* * *

#### [Manage Database Access Hosts](https://docs.cpanel.net/whm/database-services/manage-database-access-hosts/)

This interface allows you to permit remote hosts (servers) to access MySQL® and MariaDB® databases on your server. It also allows you to access remote database servers that you configured via WHM's Manage Database Profiles interface.

[Read more](https://docs.cpanel.net/whm/database-services/manage-database-access-hosts/)

* * *

#### [Manage Database Profiles](https://docs.cpanel.net/whm/database-services/manage-database-profiles/)

This interface allows you to manage multiple MySQL® or MariaDB® profiles.

[Read more](https://docs.cpanel.net/whm/database-services/manage-database-profiles/)

* * *

#### [Manage Database Users](https://docs.cpanel.net/whm/database-services/manage-database-users/)

This interface allows server administrators to retrieve a list of MySQL® and MariaDB® database users, which account owns them, and their databases.

[Read more](https://docs.cpanel.net/whm/database-services/manage-database-users/)

* * *

#### [MySQL® Upgrade Information Center](https://docs.cpanel.net/knowledge-base/sql/mysql-upgrade-information-center/)

Due to various support and security reasons, we strongly recommend that you upgrade MySQL® 5.5 to MySQL 5.7.

[Read more](https://docs.cpanel.net/knowledge-base/sql/mysql-upgrade-information-center/)
