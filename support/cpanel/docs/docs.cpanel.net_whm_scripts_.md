## Scripts

* * *

#### [The servers\_queue Script](https://docs.cpanel.net/whm/scripts/the-servers-queue-script/)

This script allows you to manage the task queue system.

[Read more](https://docs.cpanel.net/whm/scripts/the-servers-queue-script/)

* * *

#### [The auto-adjust-mysql-limits Script](https://docs.cpanel.net/whm/scripts/the-auto-adjust-mysql-limits-script/)

The /usr/local/cpanel/scripts/auto-adjust-mysql-limits script updates MySQL® configuration values.

[Read more](https://docs.cpanel.net/whm/scripts/the-auto-adjust-mysql-limits-script/)

* * *

#### [The autossl\_check Script](https://docs.cpanel.net/whm/scripts/the-autossl_check-script/)

This script performs a manual AutoSSL check.

[Read more](https://docs.cpanel.net/whm/scripts/the-autossl_check-script/)

* * *

#### [The backup Script](https://docs.cpanel.net/whm/scripts/the-backup-script/)

Use this script to run a manual backup in WHM via the command line.

[Read more](https://docs.cpanel.net/whm/scripts/the-backup-script/)

* * *

#### [The backup\_restore\_manager Script](https://docs.cpanel.net/whm/scripts/the-backup_restore_manager-script/)

This script manages the backup restoration queue.

[Read more](https://docs.cpanel.net/whm/scripts/the-backup_restore_manager-script/)

* * *

#### [The backups\_clean\_metadata\_for\_missing\_backups Script](https://docs.cpanel.net/whm/scripts/the-backups_clean_metadata_for_missing_backups-script/)

This script removes the metadata for backup files that a user manually removed from the system.

[Read more](https://docs.cpanel.net/whm/scripts/the-backups_clean_metadata_for_missing_backups-script/)

* * *

#### [The backups\_create\_metadata Script](https://docs.cpanel.net/whm/scripts/the-backups_create_metadata-script/)

This script allows a user to create metadata files for all account backup types.

[Read more](https://docs.cpanel.net/whm/scripts/the-backups_create_metadata-script/)

* * *

#### [The backups\_list\_user\_files Script](https://docs.cpanel.net/whm/scripts/the-backups_list_user_files-script/)

This script returns a list of a user's backup files in Comma-Separated Value (CSV) format.

[Read more](https://docs.cpanel.net/whm/scripts/the-backups_list_user_files-script/)

* * *

#### [The balance\_linked\_node\_quotas Script](https://docs.cpanel.net/whm/scripts/the-balance_linked_node_quotas-script/)

This script lets you enforce account quotas across multiple servers.

[Read more](https://docs.cpanel.net/whm/scripts/the-balance_linked_node_quotas-script/)

* * *

#### [The build\_maxemails\_config Script](https://docs.cpanel.net/whm/scripts/the-build_maxemails_config-script/)

This script sets limits on the amount of email that users can send from domains on your server.

[Read more](https://docs.cpanel.net/whm/scripts/the-build_maxemails_config-script/)

* * *

#### [The build\_mysql\_conf Script](https://docs.cpanel.net/whm/scripts/the-build_mysql_conf-script/)

This script builds the default MySQL config file.

[Read more](https://docs.cpanel.net/whm/scripts/the-build_mysql_conf-script/)

* * *

#### [The ccs-check Script](https://docs.cpanel.net/whm/scripts/the-ccs-check-script/)

This script lets the Calendar and Contacts Server (CCS) plugin run using your server's SSL certificate.

[Read more](https://docs.cpanel.net/whm/scripts/the-ccs-check-script/)

* * *

#### [The check\_plugin\_pkgs Script](https://docs.cpanel.net/whm/scripts/the-check_plugin_pkgs-script/)

This script verifies the integrity of installed plugin packages.

[Read more](https://docs.cpanel.net/whm/scripts/the-check_plugin_pkgs-script/)

* * *

#### [The check\_security\_advice\_changes Script](https://docs.cpanel.net/whm/scripts/the-check_security_advice_changes-script/)

This script monitors the state of WHM's Security Advisor.

[Read more](https://docs.cpanel.net/whm/scripts/the-check_security_advice_changes-script/)

* * *

#### [The check\_users\_my\_cnf Script](https://docs.cpanel.net/whm/scripts/the-check_users_my_cnf-script/)

This script checks cPanel accounts for any invalid ~/.my.cnf files and renames them.

[Read more](https://docs.cpanel.net/whm/scripts/the-check_users_my_cnf-script/)

* * *

#### [The checkallsslcerts Script](https://docs.cpanel.net/whm/scripts/the-checkallsslcerts-script/)

The system runs several certificate-related functions during the nightly cPanel & WHM update (upcp) process, and when you purchase or add a license.

[Read more](https://docs.cpanel.net/whm/scripts/the-checkallsslcerts-script/)

* * *

#### [The clean\_user\_php\_sessions Script](https://docs.cpanel.net/whm/scripts/the-clean_user_php_sessions-script/)

This script purges expired PHP sessions from a secure temporary directory.

[Read more](https://docs.cpanel.net/whm/scripts/the-clean_user_php_sessions-script/)

* * *

#### [The cleandns Script](https://docs.cpanel.net/whm/scripts/the-cleandns-script/)

This script removes orphaned and duplicate DNS zone definitions.

[Read more](https://docs.cpanel.net/whm/scripts/the-cleandns-script/)

* * *

#### [The convert\_accesshash\_to\_token Script](https://docs.cpanel.net/whm/scripts/the-convert_accesshash_to_token-script/)

This script converts a WHM user's access hash to an API token.

[Read more](https://docs.cpanel.net/whm/scripts/the-convert_accesshash_to_token-script/)

* * *

#### [The convert\_maildir\_to\_mdbox Script](https://docs.cpanel.net/whm/scripts/the-convert_maildir_to_mdbox-script/)

This script converts mailboxes from maildir to mdbox format.

[Read more](https://docs.cpanel.net/whm/scripts/the-convert_maildir_to_mdbox-script/)
