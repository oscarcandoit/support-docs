## Domains

* * *

#### [Domains](https://docs.cpanel.net/cpanel/domains/domains/)

Use this interface to create and manage multiple domains from a single cPanel account.

[Read more](https://docs.cpanel.net/cpanel/domains/domains/)

* * *

#### [Dynamic DNS](https://docs.cpanel.net/cpanel/domains/dynamic-dns/)

Use this interface to manage your dynamic DNS (DDNS) domains.

[Read more](https://docs.cpanel.net/cpanel/domains/dynamic-dns/)

* * *

#### [Redirects](https://docs.cpanel.net/cpanel/domains/redirects/)

This interface allows you to send all of the visitors of a domain or particular page to a different URL.

[Read more](https://docs.cpanel.net/cpanel/domains/redirects/)

* * *

#### [Search Engine Optimization](https://docs.cpanel.net/cpanel/domains/search-engine-optimization/)

This interface lets you audit your websites for how easy they make it for search engines like Google to process the site's content. This is so that search engines can deliver your content to users who search for related terms.

[Read more](https://docs.cpanel.net/cpanel/domains/search-engine-optimization/)

* * *

#### [Site Publisher](https://docs.cpanel.net/cpanel/domains/site-publisher/)

This interface enables you to quickly create a simple website, even if you have never created a website before.

[Read more](https://docs.cpanel.net/cpanel/domains/site-publisher/)

* * *

#### [Sitejet Builder](https://docs.cpanel.net/cpanel/domains/sitejet-builder/)

Use this interface to view and manage your Sitejet domains.

[Read more](https://docs.cpanel.net/cpanel/domains/sitejet-builder/)

* * *

#### [Social Media Management (Powered by SocialBee)](https://docs.cpanel.net/cpanel/domains/social-media-management/)

Use this interface to manage social media for your websites.

[Read more](https://docs.cpanel.net/cpanel/domains/social-media-management/)

* * *

#### [WP Toolkit for cPanel](https://docs.cpanel.net/cpanel/domains/wp-toolkit-for-cpanel/)

This interface allows you to manage the WordPress® installations on your cPanel account.

[Read more](https://docs.cpanel.net/cpanel/domains/wp-toolkit-for-cpanel/)

* * *

#### [Zone Editor](https://docs.cpanel.net/cpanel/domains/zone-editor/)

This feature allows you to create, edit, and delete Domain Name System (DNS) zone records.

[Read more](https://docs.cpanel.net/cpanel/domains/zone-editor/)
