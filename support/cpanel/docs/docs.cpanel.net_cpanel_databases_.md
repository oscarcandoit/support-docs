## Databases

* * *

#### [Database Wizard](https://docs.cpanel.net/cpanel/databases/database-wizard/)

This wizard guides you through the setup of a MySQL® or MariaDB® database, user accounts, and user privileges.

[Read more](https://docs.cpanel.net/cpanel/databases/database-wizard/)

* * *

#### [Manage My Databases](https://docs.cpanel.net/cpanel/databases/manage-my-databases/)

Use this interface to create, manage, and delete MySQL® or MariaDB® databases and database users.

[Read more](https://docs.cpanel.net/cpanel/databases/manage-my-databases/)

* * *

#### [MySQL® Database Wizard](https://docs.cpanel.net/cpanel/databases/mysql-database-wizard/)

This wizard guides you through the setup of a MySQL® database, user accounts, and user privileges.

[Read more](https://docs.cpanel.net/cpanel/databases/mysql-database-wizard/)

* * *

#### [MySQL® Databases](https://docs.cpanel.net/cpanel/databases/mysql-databases/)

Use this interface to create, manage, and delete MySQL databases and database users.

[Read more](https://docs.cpanel.net/cpanel/databases/mysql-databases/)

* * *

#### [MySQL® Manager](https://docs.cpanel.net/cpanel/databases/mysql-manager/)

Use this interface to create, manage, and delete MySQL® databases and database users.

[Read more](https://docs.cpanel.net/cpanel/databases/mysql-manager/)

* * *

#### [phpMyAdmin](https://docs.cpanel.net/cpanel/databases/phpmyadmin/)

Use this interface to manipulate MySQL® or MariaDB® databases.

[Read more](https://docs.cpanel.net/cpanel/databases/phpmyadmin/)

* * *

#### [phpPgAdmin](https://docs.cpanel.net/cpanel/databases/phppgadmin/)

Use this interface to manipulate PostgreSQL® databases.

[Read more](https://docs.cpanel.net/cpanel/databases/phppgadmin/)

* * *

#### [PostgreSQL Database Wizard](https://docs.cpanel.net/cpanel/databases/postgresql-database-wizard/)

[Read more](https://docs.cpanel.net/cpanel/databases/postgresql-database-wizard/)

* * *

#### [PostgreSQL Databases](https://docs.cpanel.net/cpanel/databases/postgresql-databases/)

Use this interface to create, manage, and delete PostgreSQL® databases and database users.

[Read more](https://docs.cpanel.net/cpanel/databases/postgresql-databases/)

* * *

#### [Remote Database Access](https://docs.cpanel.net/cpanel/databases/remote-database-access/)

This feature allows remote hosts (servers) to access MySQL® and MariaDB® databases on your account.

[Read more](https://docs.cpanel.net/cpanel/databases/remote-database-access/)
