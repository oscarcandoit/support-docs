## Update My Information

Last modified: _2022 January 19_

* * *

## Overview

This interface allows you to update the information for your own user account. Use this interface to hide unused features from your _Dashboard_ or to limit your own list of Groups and Packages.

## Update your information

For detailed information about the available options, read the [_Edit Users_](https://docs.cpanel.net/manage2/users/edit-users) documentation.

Warning:

For security reasons, the interface does **not** display your password in the _Password_ text box. If you do **not** wish to change your password, leave this text box blank when you update other information.
