## Overview

cPanel-provided statistics software supports a variety of languages. If the software’s default map of locales to languages does not fit your customers’ needs, the _Application Locale Configuration_ interface allows you to customize the language in which the system displays a customer’s preferred locale. For example, you can associate the cPanel locale option _de_ with the Webalizer option _german_. This will allow users to view Webalizer statistics in German.

To update the list of locales on the system, click _rebuild the locale databases_ at the bottom of the interface.

## Application Locale Configuration

To change your application locale configuration, perform the following steps:

1. Select a value from the _Analog_ menu.
2. Select a value from the _Awstats_ menu.
3. Select a value from the _Webalizer_ menu.
4. Click _Save_.

Note:

The system will **not** implement the changes that you make to the applications’ locale configurations until after the next statistics update.
