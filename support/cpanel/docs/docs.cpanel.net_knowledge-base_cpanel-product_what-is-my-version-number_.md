Last modified: _2022 February 16_

## Overview

The cPanel & WHM version that your server runs impacts its functionality.

More:

For more information about cPanel & WHM versions and updates, read our [Product Versions and the Release Process](https://docs.cpanel.net/knowledge-base/cpanel-product/product-versions-and-the-release-process) documentation.
