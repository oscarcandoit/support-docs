# whmui

* * *

#### [Service Manager](https://docs.cpanel.net/whm/service-configuration/service-manager/)

This interface lists your server's services and it allows you to enable and monitor them.

[Read more](https://docs.cpanel.net/whm/service-configuration/service-manager/)

* * *

#### [Service Status](https://docs.cpanel.net/whm/server-status/service-status/)

This interface displays a list of the server's monitored services.

[Read more](https://docs.cpanel.net/whm/server-status/service-status/)

* * *

#### [Set Zone Time to Live (TTL)](https://docs.cpanel.net/whm/dns-functions/set-zone-time-to-live/)

This interface allows you to set the value for time to live (TTL) in your server's DNS zones.

[Read more](https://docs.cpanel.net/whm/dns-functions/set-zone-time-to-live/)

* * *

#### [Setup/Edit Domain Forwarding](https://docs.cpanel.net/whm/dns-functions/setup-edit-domain-forwarding/)

This feature allows you to redirect one domain to another domain.

[Read more](https://docs.cpanel.net/whm/dns-functions/setup-edit-domain-forwarding/)

* * *

#### [Shell Fork Bomb Protection](https://docs.cpanel.net/whm/security-center/shell-fork-bomb-protection/)

This feature helps you protect your server from users with terminal access (SSH or Telnet) who may inadvertently or intentionally cause a fork bomb.

[Read more](https://docs.cpanel.net/whm/security-center/shell-fork-bomb-protection/)

* * *

#### [Show Current Disk Usage](https://docs.cpanel.net/whm/system-health/show-current-disk-usage/)

This feature displays your server's current partition and hard disk usage.

[Read more](https://docs.cpanel.net/whm/system-health/show-current-disk-usage/)

* * *

#### [Show Current Running Processes](https://docs.cpanel.net/whm/system-health/show-current-running-processes/)

This interface displays a table of all of the active processes on your server.

[Read more](https://docs.cpanel.net/whm/system-health/show-current-running-processes/)

* * *

#### [Show Database Processes](https://docs.cpanel.net/whm/database-services/show-database-processes/)

This interface lists all of the processes that currently run on any database on your server.

[Read more](https://docs.cpanel.net/whm/database-services/show-database-processes/)

* * *

#### [Show IP Address Usage](https://docs.cpanel.net/whm/ip-functions/show-ip-address-usage/)

This feature lists how your system uses IP addresses.

[Read more](https://docs.cpanel.net/whm/ip-functions/show-ip-address-usage/)

* * *

#### [Show MySQL Processes](https://docs.cpanel.net/whm/sql-services/show-mysql-processes/)

This interface lists all of the processes that currently run on any database on your server.

[Read more](https://docs.cpanel.net/whm/sql-services/show-mysql-processes/)

* * *

#### [Show Reseller Accounts](https://docs.cpanel.net/whm/resellers/show-reseller-accounts/)

This interface lists the server's reseller accounts.

[Read more](https://docs.cpanel.net/whm/resellers/show-reseller-accounts/)

* * *

#### [Show/Edit Reserved IPs](https://docs.cpanel.net/whm/ip-functions/show-edit-reserved-ips/)

This feature allows you to reserve an IP address that is in your address pool.

[Read more](https://docs.cpanel.net/whm/ip-functions/show-edit-reserved-ips/)

* * *

#### [Skeleton Directory](https://docs.cpanel.net/whm/account-functions/skeleton-directory/)

This interface displays the location of the skeleton directory.

[Read more](https://docs.cpanel.net/whm/account-functions/skeleton-directory/)

* * *

#### [SMTP Restrictions](https://docs.cpanel.net/whm/security-center/smtp-restrictions/)

This interface allows you to configure your server so that only the mail transport agent (MTA), Mailman mailing list software, and the root user can connect to remote SMTP servers.

[Read more](https://docs.cpanel.net/whm/security-center/smtp-restrictions/)

* * *

#### [Spamd Startup Configuration](https://docs.cpanel.net/whm/email/spamd-startup-configuration/)

Use the Spamd Startup Configuration interface to configure the startup options for the Apache SpamAssassin daemon.

[Read more](https://docs.cpanel.net/whm/email/spamd-startup-configuration/)

* * *

#### [SSH Password Authorization Tweak](https://docs.cpanel.net/whm/security-center/ssh-password-authorization-tweak/)

This feature allows you to enable or disable password authorization for SSH authentication.

[Read more](https://docs.cpanel.net/whm/security-center/ssh-password-authorization-tweak/)

* * *

#### [SSL Storage Manager](https://docs.cpanel.net/whm/ssl-tls/ssl-storage-manager/)

This interface allows system administrators to view detailed information about their SSL resources.

[Read more](https://docs.cpanel.net/whm/ssl-tls/ssl-storage-manager/)

* * *

#### [SSL/TLS Configuration](https://docs.cpanel.net/whm/ssl-tls/ssl-tls-configuration/)

This interface lets you modify your server's SSL/TLS-related configurations.

[Read more](https://docs.cpanel.net/whm/ssl-tls/ssl-tls-configuration/)

* * *

#### [Statistics Software Configuration](https://docs.cpanel.net/whm/server-configuration/statistics-software-configuration/)

This interface displays information about your server’s statistics and statistics software.

[Read more](https://docs.cpanel.net/whm/server-configuration/statistics-software-configuration/)

* * *

#### [Support Center](https://docs.cpanel.net/whm/support/support-center/)

The Support Center interface contains several links that allow you to access cPanel & WHM support features.

[Read more](https://docs.cpanel.net/whm/support/support-center/)
