# whmui

* * *

#### [Rearrange an Account](https://docs.cpanel.net/whm/account-functions/rearrange-an-account/)

This interface allows you to change which home directory stores an account.

[Read more](https://docs.cpanel.net/whm/account-functions/rearrange-an-account/)

* * *

#### [Rebuild RPM Database](https://docs.cpanel.net/whm/software/rebuild-rpm-database/)

This interface allows you to rebuild the RPM database.

[Read more](https://docs.cpanel.net/whm/software/rebuild-rpm-database/)

* * *

#### [Rebuild the IP Address Pool](https://docs.cpanel.net/whm/ip-functions/rebuild-the-ip-address-pool/)

This feature scans the server's bound IP addresses in order to locate IP addresses that are not linked to a domain.

[Read more](https://docs.cpanel.net/whm/ip-functions/rebuild-the-ip-address-pool/)

* * *

#### [Remote Access Key](https://docs.cpanel.net/whm/clusters/remote-access-key/)

You can use this feature to generate an access key, which is also known as an access hash.

[Read more](https://docs.cpanel.net/whm/clusters/remote-access-key/)

* * *

#### [Repair a MySQL Database](https://docs.cpanel.net/whm/sql-services/repair-a-mysql-database/)

This interface allows you to repair a MySQL database.

[Read more](https://docs.cpanel.net/whm/sql-services/repair-a-mysql-database/)

* * *

#### [Repair Databases](https://docs.cpanel.net/whm/database-services/repair-databases/)

This interface allows you to repair a MySQL® or MariaDB® database.

[Read more](https://docs.cpanel.net/whm/database-services/repair-databases/)

* * *

#### [Repair Mailbox Permissions](https://docs.cpanel.net/whm/email/repair-mailbox-permissions/)

This interface allows you to repair mailbox permissions.

[Read more](https://docs.cpanel.net/whm/email/repair-mailbox-permissions/)

* * *

#### [Reseller Center](https://docs.cpanel.net/whm/resellers/reseller-center/)

This interface adds or removes reseller status for existing accounts.

[Read more](https://docs.cpanel.net/whm/resellers/reseller-center/)

* * *

#### [Reserved IP Address Editor](https://docs.cpanel.net/whm/service-configuration/reserved-ip-address-editor/)

The Reserved IP Address Editor interface allows you to configure Apache to ignore HTTP requests on certain IP addresses.

[Read more](https://docs.cpanel.net/whm/service-configuration/reserved-ip-address-editor/)

* * *

#### [Reset a Mailman Password](https://docs.cpanel.net/whm/cpanel/reset-a-mailman-password/)

This feature allows you to reset the password for your server's Mailman mailing list software and mailing lists.

[Read more](https://docs.cpanel.net/whm/cpanel/reset-a-mailman-password/)

* * *

#### [Reset Account Bandwidth Limit](https://docs.cpanel.net/whm/account-functions/reset-account-bandwidth-limit/)

If an account's bandwidth quota does not match the quota for the account's package, this interface enables you to reset the account's quota.

[Read more](https://docs.cpanel.net/whm/account-functions/reset-account-bandwidth-limit/)

* * *

#### [Reset Resellers](https://docs.cpanel.net/whm/resellers/reset-resellers/)

This feature allows you to reset package permissions or apply Access Control Lists (ACLs) to resellers.

[Read more](https://docs.cpanel.net/whm/resellers/reset-resellers/)

* * *

#### [Resolver Configuration](https://docs.cpanel.net/whm/networking-setup/resolver-configuration/)

The Resolver Configuration interface allows you to use a configuration wizard to edit the /etc/resolv.conf file.

[Read more](https://docs.cpanel.net/whm/networking-setup/resolver-configuration/)

* * *

#### [Restart Services](https://docs.cpanel.net/whm/restart-services/restart-services/)

Use these interfaces to restart services within the WHM interface.

[Read more](https://docs.cpanel.net/whm/restart-services/restart-services/)

* * *

#### [Review Transfers and Restores](https://docs.cpanel.net/whm/transfers/review-transfers-and-restores/)

This interface allows you to review your server's transfer and restore logs.

[Read more](https://docs.cpanel.net/whm/transfers/review-transfers-and-restores/)

* * *

#### [Ruby Gem Installer](https://docs.cpanel.net/whm/software/ruby-gem-installer/)

This section of the Module Installers interface allows you to manage Ruby gems from the RubyGems.org repository.

[Read more](https://docs.cpanel.net/whm/software/ruby-gem-installer/)

* * *

#### [Security Advisor](https://docs.cpanel.net/whm/security-center/security-advisor/)

This interface runs a security scan on your server, and it advises you about how to resolve any security issues that it finds.

[Read more](https://docs.cpanel.net/whm/security-center/security-advisor/)

* * *

#### [Security Questions](https://docs.cpanel.net/whm/security-center/security-questions/)

This interface allows you to update and manage your WHM account's security questions.

[Read more](https://docs.cpanel.net/whm/security-center/security-questions/)

* * *

#### [Server Information](https://docs.cpanel.net/whm/server-status/server-information/)

This interface displays information about your hardware configuration, such as the speed of your processor.

[Read more](https://docs.cpanel.net/whm/server-status/server-information/)

* * *

#### [Server Profile](https://docs.cpanel.net/whm/server-configuration/server-profile/)

This interface allows you to select a server's profile, based on the service that you want the server to provide.

[Read more](https://docs.cpanel.net/whm/server-configuration/server-profile/)
