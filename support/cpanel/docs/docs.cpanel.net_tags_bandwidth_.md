#### [Bandwidth](https://docs.cpanel.net/cpanel/metrics/bandwidth/)

The Bandwidth interface displays bandwidth usage information in several sets of graphs.

[Read more](https://docs.cpanel.net/cpanel/metrics/bandwidth/)

* * *

#### [Hotlink Protection](https://docs.cpanel.net/cpanel/security/hotlink-protection/)

A hotlink occurs when someone embeds content from your site in another site and uses your bandwidth to serve the files. You can use this interface to prevent this issue.

[Read more](https://docs.cpanel.net/cpanel/security/hotlink-protection/)

* * *

#### [Leech Protection](https://docs.cpanel.net/cpanel/security/leech-protection/)

The Leech Protection interface allows you to detect unusual levels of activity in password-restricted directories.

[Read more](https://docs.cpanel.net/cpanel/security/leech-protection/)

* * *

#### [Limit Bandwidth Usage](https://docs.cpanel.net/whm/account-functions/limit-bandwidth-usage/)

This interface allows you to change the allotted bandwidth for an account.

[Read more](https://docs.cpanel.net/whm/account-functions/limit-bandwidth-usage/)

* * *

#### [Overselling](https://docs.cpanel.net/knowledge-base/cpanel-product/overselling/)

This document explains the concept of overselling. Overselling means that the system limits resellers via the resources that their cPanel accounts actually use instead of the resources that you allocate to them.

[Read more](https://docs.cpanel.net/knowledge-base/cpanel-product/overselling/)

* * *

#### [Reset Account Bandwidth Limit](https://docs.cpanel.net/whm/account-functions/reset-account-bandwidth-limit/)

If an account's bandwidth quota does not match the quota for the account's package, this interface enables you to reset the account's quota.

[Read more](https://docs.cpanel.net/whm/account-functions/reset-account-bandwidth-limit/)

* * *

#### [Unsuspend Bandwidth Exceeders](https://docs.cpanel.net/whm/account-functions/unsuspend-bandwidth-exceeders/)

This feature allows you to simultaneously unsuspend all of the accounts that have exceeded their bandwidth.

[Read more](https://docs.cpanel.net/whm/account-functions/unsuspend-bandwidth-exceeders/)

* * *

#### [View Bandwidth Usage](https://docs.cpanel.net/whm/account-information/view-bandwidth-usage/)

This interface monitors traffic from HTTP, FTP, SMTP, and POP3/IMAP.

[Read more](https://docs.cpanel.net/whm/account-information/view-bandwidth-usage/)

* * *

#### [View Reseller Usage and Manage Account Status](https://docs.cpanel.net/whm/resellers/view-reseller-usage-and-manage-account-status/)

These features allow you to view your resellers’ disk space and bandwidth usage. You can also use this set of features to suspend, unsuspend, and terminate reseller accounts.

[Read more](https://docs.cpanel.net/whm/resellers/view-reseller-usage-and-manage-account-status/)
