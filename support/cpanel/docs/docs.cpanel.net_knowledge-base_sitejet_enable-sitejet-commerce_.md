## Enable Sitejet Commerce

Last modified: _2024 December 26_

* * *

## Overview

[Sitejet Commerce](https://go.cpanel.net/sitejet-commerce) is an optional subscription that helps users build and manage online stores.

Sitejet Commerce is available to cPanel & WHM partners and resellers.

### Enable Sitejet Commerce

#### cPanel & WHM partners

For some partners, Sitejet Commerce is automatically enabled.

If you are a partner and you would like to enable Sitejet Commerce, [submit your request to cPanel & WHM](https://docs.cpanel.net/manage2/support/submit-a-new-priority-support-request/).

#### Resellers

To enable Sitejet Commerce as a reseller, perform the following steps:

1. Use the WHM API [`set_commerce` function](https://api.docs.cpanel.net/openapi/whm/operation/set_commerce/) to activate Sitejet Commerce. This function also lets you set a URL that links to your store.
2. Add the Sitejet plans as products to your store using the following SKUs:
   - _Starter plan_ – SJ-COM-STA-1M
   - _Professional plan_ – SJ-COM-PRO-1M
   - _Unlimited plan_ – SJ-COM-UNL-1M
