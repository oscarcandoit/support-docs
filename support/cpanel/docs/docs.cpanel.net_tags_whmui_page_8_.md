[Read more](https://docs.cpanel.net/whm/software/multiphp-manager-for-whm/)

#### [MultiPHP Manager for WHM — User Domain Settings](https://docs.cpanel.net/whm/software/multiphp-manager-for-whm-user-domain-settings/)

[Read more](https://docs.cpanel.net/whm/software/multiphp-manager-for-whm-user-domain-settings/)

#### [MySQL Root Password](https://docs.cpanel.net/whm/sql-services/mysql-root-password/)

[Read more](https://docs.cpanel.net/whm/sql-services/mysql-root-password/)

* * *

#### [MySQL/MariaDB Upgrade](https://docs.cpanel.net/whm/sql-services/mysql-or-mariadb-upgrade/)

This interface allows you to manage MySQL® or MariaDB® version upgrades (for example, when you upgrade from MySQL 5.6 to the latest version).

[Read more](https://docs.cpanel.net/whm/sql-services/mysql-or-mariadb-upgrade/)

#### [Nameserver Record Report](https://docs.cpanel.net/whm/dns-functions/nameserver-record-report/)

* * *

#### [Nameserver Selection](https://docs.cpanel.net/whm/service-configuration/nameserver-selection/)

This interface allows you to select your server's nameserver software or to disable DNS services.

[Read more](https://docs.cpanel.net/whm/service-configuration/nameserver-selection/)

* * *

#### [NGINX® Manager](https://docs.cpanel.net/whm/software/nginx-manager/)

WHM's NGINX® Manager allows you to install, uninstall, and manage your NGINX with reverse proxy and caching server.

[Read more](https://docs.cpanel.net/whm/software/nginx-manager/)

* * *

#### [Park a Domain](https://docs.cpanel.net/whm/dns-functions/park-a-domain/)

A parked domain (or alias) is a secondary domain that points to the user’s primary domain.

[Read more](https://docs.cpanel.net/whm/dns-functions/park-a-domain/)

* * *

#### [PHP Extensions and Applications Package](https://docs.cpanel.net/whm/software/php-extensions-and-applications-package/)

Use this interface to install and manage PHP packages via the PHP Extensions and Applications Repository (PEAR).

[Read more](https://docs.cpanel.net/whm/software/php-extensions-and-applications-package/)

* * *

#### [PHP PECL](https://docs.cpanel.net/whm/software/php-pecl/)

#### [Piped Log Configuration](https://docs.cpanel.net/whm/service-configuration/piped-log-configuration/)

#### [Raw Apache Log Download](https://docs.cpanel.net/whm/account-functions/raw-apache-log-download/)
