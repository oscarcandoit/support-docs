# whmadvanced

* * *

#### [The manage\_mysql\_profiles Script](https://docs.cpanel.net/whm/scripts/the-manage_mysql_profiles-script/)

The script allows you to import and export MySQL® profiles.

[Read more](https://docs.cpanel.net/whm/scripts/the-manage_mysql_profiles-script/)

* * *

#### [The migrate\_legacy\_wordpress\_to\_modern\_wordpress Script](https://docs.cpanel.net/whm/scripts/the-migrate_legacy_wordpress_to_modern_wordpress-script/)

This script converts legacy cPAddon WordPress® installations to the RPM-based WordPress cPAddon.

[Read more](https://docs.cpanel.net/whm/scripts/the-migrate_legacy_wordpress_to_modern_wordpress-script/)

* * *

#### [The modify\_accounts Script](https://docs.cpanel.net/whm/scripts/the-modify_accounts-script/)

You can use this script to set existing cPanel accounts to use a new theme.

[Read more](https://docs.cpanel.net/whm/scripts/the-modify_accounts-script/)

* * *

#### [The modify\_packages Script](https://docs.cpanel.net/whm/scripts/the-modify_packages-script/)

You can use this script to set existing packages to use a new cPanel theme.

[Read more](https://docs.cpanel.net/whm/scripts/the-modify_packages-script/)

* * *

#### [The modsec\_vendor Script](https://docs.cpanel.net/whm/scripts/the-modsec_vendor-script/)

The /usr/local/cpanel/scripts/modsec\_vendor script manages the vendors in your server's ModSecurity® installation.

[Read more](https://docs.cpanel.net/whm/scripts/the-modsec_vendor-script/)

* * *

#### [The MysqlDump Stream](https://docs.cpanel.net/knowledge-base/sql/the-mysqldump-stream/)

The MysqlDump stream allows you to produce a dump of a MySQL® database via WebSocket.

[Read more](https://docs.cpanel.net/knowledge-base/sql/the-mysqldump-stream/)

* * *

#### [The onboot\_handler Script](https://docs.cpanel.net/whm/scripts/the-onboot_handler-script/)

This script runs all of the scripts in the /var/cpanel/on\_boot and the /usr/local/cpanel/libexec/on\_boot directories.

[Read more](https://docs.cpanel.net/whm/scripts/the-onboot_handler-script/)

* * *

#### [The php\_fpm\_config Script](https://docs.cpanel.net/whm/scripts/the-php_fpm_config-script/)

The php\_fpm\_config script installs configuration files when you manually edit the PHP-FPM configuration files.

[Read more](https://docs.cpanel.net/whm/scripts/the-php_fpm_config-script/)

* * *

#### [The pkgacct Script](https://docs.cpanel.net/whm/scripts/the-pkgacct-script/)

You can use this script to create a cpmove archive for an account.

[Read more](https://docs.cpanel.net/whm/scripts/the-pkgacct-script/)

* * *

#### [The post\_snapshot Script](https://docs.cpanel.net/whm/scripts/the-post_snapshot-script/)

This script updates configurations on a system created from an image.

[Read more](https://docs.cpanel.net/whm/scripts/the-post_snapshot-script/)

* * *

#### [The process\_site\_templates Script](https://docs.cpanel.net/whm/scripts/the-process_site_templates-script/)

The process\_site\_templates script processes template files and user-provided data to create a Site Publisher website.

[Read more](https://docs.cpanel.net/whm/scripts/the-process_site_templates-script/)

* * *

#### [The rebuild\_dbmap Script](https://docs.cpanel.net/whm/scripts/the-rebuild_dbmap-script/)

If the database map is corrupt for a cPanel user, you can use the rebuild\_dbmap script to rebuild that user's database map.

[Read more](https://docs.cpanel.net/whm/scripts/the-rebuild_dbmap-script/)

* * *

#### [The rebuildhttpdconf Script](https://docs.cpanel.net/whm/scripts/the-rebuildhttpdconf-script/)

You can use the rebuildhttpdconf script to rebuild Apache's main configuration file.

[Read more](https://docs.cpanel.net/whm/scripts/the-rebuildhttpdconf-script/)

* * *

#### [The remove\_dovecot\_index\_files Script](https://docs.cpanel.net/whm/scripts/the-remove_dovecot_index_files-script/)

Use the /usr/local/cpanel/scripts/remove\_dovecot\_index\_files script to remove the Dovecot® index logs and cache files from cPanel mail accounts.

[Read more](https://docs.cpanel.net/whm/scripts/the-remove_dovecot_index_files-script/)

* * *

#### [The removeacct Script](https://docs.cpanel.net/whm/scripts/the-removeacct-script/)

This script terminates a cPanel account from the command line.

[Read more](https://docs.cpanel.net/whm/scripts/the-removeacct-script/)

* * *

#### [The rescan\_user\_dovecot\_fts Script](https://docs.cpanel.net/whm/scripts/the-rescan_user_dovecot_fts-script/)

Use the /usr/local/cpanel/scripts/rescan\_user\_dovecot\_fts script to perform a full text search (FTS) scan on an account's mailboxes.

[Read more](https://docs.cpanel.net/whm/scripts/the-rescan_user_dovecot_fts-script/)

* * *

#### [The restartsrv Script](https://docs.cpanel.net/whm/scripts/the-restartsrv-script/)

This script restarts a service.

[Read more](https://docs.cpanel.net/whm/scripts/the-restartsrv-script/)

* * *

#### [The securetmp Script](https://docs.cpanel.net/whm/scripts/the-securetmp-script/)

This script secures the /tmp and /var/tmp directories.

[Read more](https://docs.cpanel.net/whm/scripts/the-securetmp-script/)

* * *

#### [The servicedomains Script](https://docs.cpanel.net/whm/scripts/the-servicedomains-script/)

This script lets you to create DNS records for service subdomains.

[Read more](https://docs.cpanel.net/whm/scripts/the-servicedomains-script/)

* * *

#### [The set-tls-settings Script](https://docs.cpanel.net/whm/scripts/the-set-tls-settings-script/)

This script configures a server's Secure Socket Layer (SSL) and Transport Layer Security (TLS) cipher suites and protocols.

[Read more](https://docs.cpanel.net/whm/scripts/the-set-tls-settings-script/)
