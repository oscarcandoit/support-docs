# whmui

* * *

#### [Email All Users](https://docs.cpanel.net/whm/account-functions/email-all-users/)

This interface enables you to email every cPanel user simultaneously.

[Read more](https://docs.cpanel.net/whm/account-functions/email-all-users/)

* * *

#### [Email Deliverability in WHM](https://docs.cpanel.net/whm/email/email-deliverability-in-whm/)

Use this interface to identify problems with your server's mail-related DNS records.

[Read more](https://docs.cpanel.net/whm/email/email-deliverability-in-whm/)

* * *

#### [Email DNS Record Manager](https://docs.cpanel.net/whm/dns-functions/email-dns-record-manager/)

Use this interface to add a default DMARC policy to your server's domains.

[Read more](https://docs.cpanel.net/whm/dns-functions/email-dns-record-manager/)

* * *

#### [Email Routing Configuration](https://docs.cpanel.net/whm/dns-functions/email-routing-configuration/)

This interface lets you configure the entry for each destination mail server.

[Read more](https://docs.cpanel.net/whm/dns-functions/email-routing-configuration/)

* * *

#### [Enable DKIM/SPF Globally](https://docs.cpanel.net/whm/dns-functions/enable-dkim-spf-globally/)

This feature creates DKIM and SPF records for all accounts on the server.

[Read more](https://docs.cpanel.net/whm/dns-functions/enable-dkim-spf-globally/)

* * *

#### [Exim Configuration Manager](https://docs.cpanel.net/whm/service-configuration/exim-configuration-manager/)

Use the features in the Exim Configuration Manager interface to modify your Exim configuration.

[Read more](https://docs.cpanel.net/whm/service-configuration/exim-configuration-manager/)

* * *

#### [Exim Configuration Manager Basic Editor](https://docs.cpanel.net/whm/service-configuration/exim-configuration-manager-basic-editor/)

Select the Basic Editor tab in the Exim Configuration Manager interface to modify your server's Exim configuration settings.

[Read more](https://docs.cpanel.net/whm/service-configuration/exim-configuration-manager-basic-editor/)

* * *

#### [Feature Manager](https://docs.cpanel.net/whm/packages/feature-manager/)

This interface allows you to edit feature lists, which you can assign to packages that you apply to cPanel accounts.

[Read more](https://docs.cpanel.net/whm/packages/feature-manager/)

* * *

#### [File and Directory Restoration for WHM](https://docs.cpanel.net/whm/backup/file-and-directory-restoration-for-whm/)

The File and Directory Restoration interface allows you to restore items from the local backups of your cPanel account users.

[Read more](https://docs.cpanel.net/whm/backup/file-and-directory-restoration-for-whm/)

* * *

#### [Filter Incoming Emails by Country](https://docs.cpanel.net/whm/email/filter-incoming-emails-by-country/)

Use this interface to block specific countries from sending email messages to your server.

[Read more](https://docs.cpanel.net/whm/email/filter-incoming-emails-by-country/)

* * *

#### [Filter Incoming Emails by Domain](https://docs.cpanel.net/whm/email/filter-incoming-emails-by-domain/)

Use this interface to block email message from certain domains.

[Read more](https://docs.cpanel.net/whm/email/filter-incoming-emails-by-domain/)

* * *

#### [Force Password Change](https://docs.cpanel.net/whm/account-functions/force-password-change/)

This feature allows you to force users to change their passwords.

[Read more](https://docs.cpanel.net/whm/account-functions/force-password-change/)

* * *

#### [FTP Server Configuration](https://docs.cpanel.net/whm/service-configuration/ftp-server-configuration/)

The FTP Server Configuration interface allows you to customize your chosen FTP server's behavior.

[Read more](https://docs.cpanel.net/whm/service-configuration/ftp-server-configuration/)

* * *

#### [FTP Server Selection](https://docs.cpanel.net/whm/service-configuration/ftp-server-selection/)

This interface allows you to select an FTP server daemon, or to disable FTP services.

[Read more](https://docs.cpanel.net/whm/service-configuration/ftp-server-selection/)

* * *

#### [Getting Started in WHM](https://docs.cpanel.net/whm/the-whm-interface/getting-started/)

When you log in to WHM for the first time on a new cPanel & WHM server, the system will display some steps to help you get started with WHM.

[Read more](https://docs.cpanel.net/whm/the-whm-interface/getting-started/)

* * *

#### [Global Configuration](https://docs.cpanel.net/whm/service-configuration/global-configuration/)

This feature allows you to adjust several of the Apache® web server's advanced features.

[Read more](https://docs.cpanel.net/whm/service-configuration/global-configuration/)

* * *

#### [Grant cPanel Support Access](https://docs.cpanel.net/whm/support/grant-cpanel-support-access/)

This interface uses the server authentication details in your ticket to automatically provide cPanel Technical Support with SSH access to your server.

[Read more](https://docs.cpanel.net/whm/support/grant-cpanel-support-access/)

* * *

#### [Greylisting](https://docs.cpanel.net/whm/email/greylisting/)

This interface allows you to configure Greylisting, a service that protects your server against unwanted email or spam.

[Read more](https://docs.cpanel.net/whm/email/greylisting/)

* * *

#### [Host Access Control](https://docs.cpanel.net/whm/security-center/host-access-control/)

Use this interface to allow, reject, or drop access to services for specific IP addresses.

[Read more](https://docs.cpanel.net/whm/security-center/host-access-control/)

* * *

#### [How to Create a Demo cPanel Account](https://docs.cpanel.net/knowledge-base/accounts/how-to-create-a-demo-cpanel-account/)

Demo accounts allow users, such as potential customers, to navigate the features of cPanel & WHM.

[Read more](https://docs.cpanel.net/knowledge-base/accounts/how-to-create-a-demo-cpanel-account/)
