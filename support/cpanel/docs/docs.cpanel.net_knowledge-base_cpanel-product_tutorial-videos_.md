## cPanel Basics Playlist

The [cPanel Basics playlist](https://www.youtube.com/watch?v=i9hsWsE9vLA&list=PLZk46idJS6s7VPzwPQXNtwIsU23D3TqRV&t=72s) includes videos to help new cPanel users take the first steps in their cPanel accounts. All of these videos also appear in the cPanel Tutorials playlist.

This playlist includes videos on the following topics:

- How to log in to your cPanel account.
- How to navigate the main cPanel interface.
- How to perform some simple tasks in cPanel (for example, how to create email accounts or upload files).

## cPanel Tutorials Playlist

The [cPanel Tutorials playlist](https://www.youtube.com/watch?v=K19xlT8g7ys&list=PLZk46idJS6s54hAX8K79_AY8brgBwPtHd) includes videos to help new and experienced cPanel users manage their cPanel accounts’ features successfully.

This playlist includes videos on the following topics:

- How to log in to your cPanel account.
- How to navigate the main cPanel interface.
- How to perform simple and advanced tasks in many of cPanel’s interfaces.

## WHM Tutorials Playlist

The [WHM Tutorials playlist](https://www.youtube.com/watch?v=3XpB4FGyr9k&list=PLZk46idJS6s4S2PqAeStUXZYd09YgmCPh) includes videos to help new and experienced WHM users manage their servers.

This playlist includes videos on the following topics:

- How to log in to your WHM account.
- How to configure WHM’s basic settings.
- How to use WHM’s interface to perform basic and advanced system administration tasks.
