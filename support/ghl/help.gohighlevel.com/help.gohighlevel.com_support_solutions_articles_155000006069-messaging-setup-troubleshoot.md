# Messaging setup & troubleshoot

Modified on: Wed, 20 Aug, 2025 at 4:31 AM

This guide will help you connect your **Facebook Page** and **Instagram Business Account** so that you can manage conversations directly from the CRM.

**TABLE OF CONTENTS**

- [FB page messaging setup](https://help.leadconnectorhq.com/support/solutions/articles/155000005226-messaging-setup-troubleshoot#FB-page-messaging-setup%3A)
- [Instagram Messaging setup](https://help.leadconnectorhq.com/support/solutions/articles/155000005226-messaging-setup-troubleshoot#Instagram-Messaging-setup%3A)
- [Other facebook/instagram related support guides](https://help.leadconnectorhq.com/support/solutions/articles/155000005226-messaging-setup-troubleshoot#Other-facebook/instagram-related-support-guides%3A)

# FB page messaging setup:

1\. Leadconnector should have access to send message and access message for the page ( [https://www.facebook.com/settings?tab=business\_tools&ref=settings](https://www.facebook.com/settings?tab=business_tools&ref=settings))

# [![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155051975375/original/YUqz2KdURT-9YHgb010cyxhuZSV0WJ7A0g.jpeg?1755682293)](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155046611201/original/560dNU3lKrXB3ADHgcxA9uUfY8O2B6oMvw.png?1747220054)

2\. User account should have full access to page and should have message access. ( Link: [https://business.facebook.com/latest/settings](https://business.facebook.com/latest/settings))

[![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155051975374/original/cp0IkVvwNzY-Ovfo4YuShCC4JyALOaiK9Q.jpeg?1755682293)](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155046618449/original/jD5YyOakvOAc3JTp5SfO-NdLt6qeSc-1vQ.png?1747225602)

3\. FB messenger should be enabled in Settings >> Integration >> Facebook card >> Settings

[![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155051975370/original/ubh0Lx2BwLcsbdL_e6QNzWcPkwidz_0A-Q.jpeg?1755682293)](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155046618680/original/fXU2j4i8_yMPm5b_vp4sgqbteIBBnFBnIw.png?1747225743)

# Instagram Messaging setup:

1\. Leadconnector should have access to send message and access message for the page ( [https://www.facebook.com/settings?tab=business\_tools&ref=settings](https://www.facebook.com/settings?tab=business_tools&ref=settings))

# [![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155051975373/original/3e4HfLHBuS2vcIzHqOPPlKkwyYesU5fDKg.jpeg?1755682293)](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155046611201/original/560dNU3lKrXB3ADHgcxA9uUfY8O2B6oMvw.png?1747220054)

2\. Verify that you have access to send and receive message on instagram under instagram accounts ( [https://business.facebook.com/latest/settings](https://business.facebook.com/latest/settings))

[![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155051975369/original/nhErqFoGV5OIegyHIHcNcbDrovfqJf9NCQ.jpeg?1755682293)](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155046820921/original/tLINkNzirrxoXfTSJdGxULjH4cx8k9f3Ug.png?1747628845)

3\. Verify instagram account is present in connected assets list in the page connected.

[![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155051975371/original/rhD9Rep3yoErxPaN6MV6KiE2NpVJ1UIaRw.jpeg?1755682293)](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155046820913/original/rThbgLELyFW3QsFfS1WTuVteuNKAyl2-ag.png?1747628832)

4\. Instagram messaging is enabled in Settings >> Integration >> Facebook card >> Settings

[![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155051975372/original/DAyCnypW3ZQUyFnsnFIGJUt3SbX3qGNf5w.jpeg?1755682293)](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/155046820968/original/rYafkZPj6wQIXvvuNGOOqY4P2wQWSO_S0Q.png?1747629000)
