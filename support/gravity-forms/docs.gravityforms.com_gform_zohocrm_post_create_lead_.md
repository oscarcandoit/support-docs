## Description

Allows custom actions to be performed after creating a lead with a Zoho CRM feed.

## Usage

|     |
| --- |
| `add_action(``'gform_zohocrm_post_create_lead'``,``'action_after_lead_creation'``, 10, 5 );`<br>`function``action_after_lead_creation(``$lead_record``,``$lead``,``$feed``,``$entry``,``$form``) {`<br>`// Do your stuff`<br>`}` |

## Parameters

- **$lead\_record** [array](https://docs.gravityforms.com/array)


The response from Zoho CRM after a feed has successfully created a lead, with the lead id injected.



Sample array:


|     |
| --- |
| `array`<br>`(`<br>```'code'``=>``'SUCCESS'`<br>```'duplicate_field'``=>``''`<br>```'action'``=>``'insert'`<br>```'details'``=>``array`<br>```(`<br>```'Modified_Time'``=>``'2019-08-08T12:15:47-07:00'``//System-generated modification timestamp`<br>```'Modified_By'``=>``array``// Name and ID of the user who modified the lead.`<br>```(`<br>```'name'``=>``'Silvia Samples'`<br>```'id'``=> 3451042000000292017`<br>```)`<br>`​`<br>```'Created_Time'``=>``'2019-08-08T12:15:47-07:00'``// System-generated creation timestamp.`<br>```'id'``=> 3351042000000745001``// Unique ID of the lead`<br>```'Created_By'``=>``array``// Name and ID of the user who created the lead.`<br>```(`<br>```'name'``=>``'Silvia Samples'`<br>```'id'``=> 3451042000000292017`<br>```)`<br>`​`<br>```)`<br>`​`<br>```'message'``=>``'record added'`<br>```'status'``=>``'success'`<br>`)` |

- **$lead** [array](https://docs.gravityforms.com/array)


The lead arguments that you sent to Zoho CRM to initiate the creation process. It includes entry fields as per your Zoho feed field mappings, and the [feed options](https://docs.gravityforms.com/creating-a-feed-for-the-zoho-crm-add-on/#additional-options) you have set.
**Note regarding duplicateCheck**:

`duplicateCheck` value of `1`is equivalent to the feed options of _Update Contact If Already Exists = false_, and _Allow Duplicate = false_. That is, feed will only create contact if it does not exist already.

`duplicateCheck` value of `2` is equivalent to the feed options of _Update Contact If Already Exists = true_, and _Allow Duplicate = false_. That is, feed will update a matching contact if it exists already.



A sample array may be:

| `array`<br>`(`<br>```'Email_Opt_Out'``=>``''``//feed setting`<br>```'Description'``=>``''``//Lead description, as per Zoho definitions.`<br>```'Lead_Source'``=>``'Partner'``//Source from which the lead was created, as per Zoho definitions.`<br>```'Lead_Status'``=>``'Rating'``=>``'Contact in Future'``//Status of the lead, as per Zoho definitions.`<br>```'Rating'``=>``'Contact in Future'``//The rating of the lead, as per Zoho definitions.`<br>``<br>```'options'``=> '``array`<br>```(`<br>```'duplicateCheck'``=> 1``//value of 1 or 2. See note below.`<br>```'isApproval'``=>``''``//the Approval Mode from the feed settings`<br>```'wfTrigger'``=>``''``//the Workflow Mode from the feed settings`<br>```)`<br>`​`<br>```// Fields that have been mapped in the Zoho feed, and their values collected from the entry.`<br>```'Company'``=>``'Sample Company'`<br>```'Email'``=>``'test@sample.com'`<br>```'First_Name'``=>``'Silvia'`<br>```'Last_Name'``=>``'Samples'`<br>`)` |
