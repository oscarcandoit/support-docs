## Description

Allows setting the maximum number of choices shown in the form editor for choice based fields ( [Radio Buttons](https://docs.gravityforms.com/radio-buttons/), [Checkbox](https://docs.gravityforms.com/checkbox-field/), [Image Choice](https://docs.gravityforms.com/image-choice-field/), and [Multiple Choice](https://docs.gravityforms.com/multiple-choice-field/)).

## Usage

|     |     |
| --- | --- |
| 1 | `add_filter(``'gform_field_choices_max_count_visible'``,``'your_function_name'``, 10, 2 );` |

## Parameters

- **$max\_choices\_visible\_count** [string](https://docs.gravityforms.com/string)

The default number of choices visible is 5.
- **$field** [object](https://docs.gravityforms.com/object)

The current field object

## Examples

Sets the maximum choices shown in the form editor to 1 for all choice based fields.

|     |     |
| --- | --- |
| 1<br>2<br>3 | `add_filter(``'gform_field_choices_max_count_visible'``,``function``(``$max_choices_visible_count``,``$field``) {`<br>```return``1;`<br>`}, 10, 2 );` |

Sets the maximum choices shown in the form editor for the [Image Choice](https://docs.gravityforms.com/image-choice-field/) fields to 1 and keeps the default for Radio Buttons, Checkbox, and Multiple Choice fields.

|     |     |
| --- | --- |
| 1<br>2<br>3<br>4<br>5<br>6 | `add_filter(``'gform_field_choices_max_count_visible'``,``function``(``$max_choices_visible_count``,``$field``) {`<br>```if``(``$field``[``'type'``] ===``'image_choice'``) {`<br>```return``1;`<br>```}`<br>```return``$max_choices_visible_count``;`<br>`}, 10, 2 );` |

## Placement

This code can be used in the [functions.php](https://developer.wordpress.org/themes/basics/theme-functions/) file of the active theme, a custom functions plugin, a custom add-on, or with a code snippets plugin.

See also the PHP section in this article: [Where Do I Put This Code?](https://docs.gravityforms.com/where-do-i-put-this-code/)

## Since

This filter was added in Gravity Forms 2.9.
