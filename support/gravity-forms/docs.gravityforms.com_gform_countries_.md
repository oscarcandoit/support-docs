## Description

This filter can be used to add or remove countries from the address field Country drop down.

## Usage

|     |
| --- |
| `add_filter(``'gform_countries'``,``'your_function_name'``);` |

## Parameters

- **$countries** [array](https://docs.gravityforms.com/array)
The array to be filtered. It contains The list of countries in a standard array (see sample below).


|     |
| --- |
| `array``(``'Argentina'``,``'Brazil'``,``'Netherlands'``,``'United States'``,``'United Kingdom'``, ... );` |

## Examples

### Limit to Specific Countries

This example demonstrates how to limit the country drop down to only a few select countries.

|     |     |
| --- | --- |
| 1<br>2<br>3 | `add_filter(``'gform_countries'``,``function``(``$countries``) {`<br>```return``array``(``'Brazil'``,``'United States'``,``'Netherlands'``,``'United Kingdom'``);`<br>`} );` |

### Add New Countries

This example demonstrates how to add new countries to the list.

|     |     |
| --- | --- |
| 1<br>2<br>3<br>4<br>5<br>6<br>7<br>8 | `add_filter(``'gform_countries'``,``function``(``$countries``) {`<br>```$countries``[] =``'Country 1'``;`<br>```$countries``[] =``'Country 2'``;`<br>``<br>```asort(``$countries``);`<br>``<br>```return``$countries``;`<br>`} );` |

This example demonstrates how to add new countries to the list when used in conjunction with the [Use Country Code as Value](https://docs.gravityforms.com/gform_countries/#h-use-country-code-as-value) snippet.

|     |     |
| --- | --- |
| 1<br>2<br>3<br>4<br>5<br>6<br>7<br>8 | `add_filter(``'gform_countries'``,``function``(``$countries``) {`<br>```$countries``[``'COUNTRY1'``] =``'Country 1'``;`<br>```$countries``[``'COUNTRY2'``] =``'Country 2'``;`<br>``<br>```asort(``$countries``);`<br>``<br>```return``$countries``;`<br>`} );` |

### Use Country Code as Value

These examples show how you can update the countries list to use the country code as the choice value.

The following example requires Gravity Forms 2.4.19.3+.

|     |     |
| --- | --- |
| 1<br>2<br>3<br>4<br>5<br>6 | `add_filter(``'gform_countries'``,``function``() {`<br>```$countries``= GF_Fields::get(``'address'``)->get_default_countries();`<br>```asort(``$countries``);`<br>```return``$countries``;`<br>`} );` |

The following example works with Gravity Forms 1.9+.

|     |     |
| --- | --- |
| 1<br>2<br>3<br>4<br>5<br>6<br>7<br>8<br>9<br>10 | `add_filter(``'gform_countries'``,``function``(``$countries``) {`<br>```$new_countries``=``array``();`<br>```foreach``(``$countries``as``$country``) {`<br>```$code``= GF_Fields::get(``'address'``)->get_country_code(``$country``);`<br>```$new_countries``[``$code``] =``$country``;`<br>```}`<br>```return``$new_countries``;`<br>`} );` |

### Improved Sorting of Translated Countries

The standard PHP sorting functions struggle with strings that have accented and special characters. To solve this issue, you can use the PHP Collator class, which provides better support for sorting such strings. If your site doesn’t have access to the PHP Collator class, the following function will return the default countries list.

|     |     |
| --- | --- |
| 1<br>2<br>3<br>4<br>5<br>6<br>7<br>8<br>9<br>10 | `add_filter(``'gform_countries'``,``function``(``$countries``) {`<br>```if``( !``class_exists``(``'Collator'``) ) {`<br>```return``$countries``;`<br>```}`<br>```$collator``=``new``Collator( get_user_locale() );`<br>```$collator``->asort(``$countries``);`<br>```return``$countries``;`<br>`} );` |

## Placement

This code should be placed in the [functions.php](http://codex.wordpress.org/Theme_Development#Functions_File) file of your active theme.

## Source Code

This filter is located in GF\_Field\_Address::get\_countries() in _includes/fields/class-gf-field-address.php_.
