# Pricing Fields

## Articles

[**2Checkout Field** \\
How to Configure the 2Checkout Field used in the Gravity Forms 2Checkout Add-On](https://docs.gravityforms.com/2checkout-field/ "2Checkout Field") [**Coupon Field** \\
The Coupon field allows using coupons created with the Coupons Add-On. Coupons can be used to offer discounts on products and services.](https://docs.gravityforms.com/coupon-field/ "Coupon Field") [**Credit Card** \\
The Credit Card field makes it easy to capture credit card information and is available for the Authorize.Net, PayPal Payments Pro, and Stripe add-ons.](https://docs.gravityforms.com/credit-card-field/ "Credit Card") [**Mollie Field** \\
The Mollie field provides Credit Card and Cardholder input fields tied to integrating with the Mollie payment service through the Mollie Add-On.](https://docs.gravityforms.com/mollie-field/ "Mollie Field") [**Option** \\
The Option field allows the creation of options for products created by a Product field. Option fields have special functionality which displays how much selecting the option will add (or subtract) from the total.](https://docs.gravityforms.com/option/ "Option") [**PayPal Field** \\
The PayPal Field is a pricing field available when using the Gravity Forms PayPal Checkout Add-On, and appears in the Pricing Field section.](https://docs.gravityforms.com/paypal-field/ "PayPal Field") [**Product** \\
The Product field allows the creation of products in the form. There are several options for the type of field that should be used for the Product field: Single Product, Drop Down, Radio Buttons and User Defined Price.](https://docs.gravityforms.com/product/ "Product") [**Quantity** \\
The Quantity field allows a quantity to be specified for the Product field.](https://docs.gravityforms.com/quantity/ "Quantity") [**Shipping** \\
The Shipping field allows a shipping fee to be added to the form total.](https://docs.gravityforms.com/shipping/ "Shipping") [**Square Field** \\
The Square Field is a multi-input payment method input field provided with our official Square Add-On. This article shows you where to find it and any special settings.](https://docs.gravityforms.com/square-field/ "Square Field") 1 [2](https://docs.gravityforms.com/category/user-guides/pricing-fields/page/2/) [Next](https://docs.gravityforms.com/category/user-guides/pricing-fields/page/2/)
