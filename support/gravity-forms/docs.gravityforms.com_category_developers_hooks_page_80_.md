## Articles

[**gform\_square\_authorization\_only** \\

This filter from the official Gravity Forms Square Add-On allows you to control the look of the Square credit card details input.](https://docs.gravityforms.com/gform_square_card_details_style/ "gform_square_card_details_style") [**gform\_square\_card\_style** \\
This filter was removed from the Square Add-On and is no longer supported. From Gravity Forms Square Add-On version 1.7, the look of the Square field can be managed via the gform\_square\_card\_details\_style filter.](https://docs.gravityforms.com/gform_square_card_style/ "gform_square_card_style") [**gform\_square\_data\_sync\_enabled** \\
Allows data sync to be disabled for Square Add-On. Default is Enabled.](https://docs.gravityforms.com/gform_square_data_sync_enabled/ "gform_square_data_sync_enabled") [**gform\_square\_payment\_data** \\
The gform\_square\_payment\_data filter in version 1.0 of the Square add-on allows you to modify the payment data before it is sent to Square.](https://docs.gravityforms.com/gform_square_payment_data/ "gform_square_payment_data") [**gform\_square\_show\_custom\_app\_settings** \\
The gform\_square\_show\_custom\_app\_settings filter allows displaying the custom app settings link in the Square add-on settings page.](https://docs.gravityforms.com/gform_square_show_custom_app_settings/ "gform_square_show_custom_app_settings") [**gform\_stripe\_api\_mode** \\
This filter allows the API mode to be set to “live” or “test”.](https://docs.gravityforms.com/gform_stripe_api_mode/ "gform_stripe_api_mode") [**gform\_stripe\_cancel\_url** \\
