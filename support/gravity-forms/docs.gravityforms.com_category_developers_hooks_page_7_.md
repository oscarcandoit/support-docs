This hook has been deprecated in v. 1.7. Please use gform\_notification instead.](https://docs.gravityforms.com/gform_admin_notification_attachments/ "gform_admin_notification_attachments") [**gform\_admin\_pre\_render** \\
This filter is executed before the entry detail is displayed and can be used to manipulate the Form Object prior to rendering the entry.](https://docs.gravityforms.com/gform_admin_pre_render/ "gform_admin_pre_render") [**gform\_advanced\_settings** \\
This hook has been deprecated. Please use gform\_form\_settings instead.](https://docs.gravityforms.com/gform_advanced_settings/ "gform_advanced_settings") [**gform\_advancedpostcreation\_args\_pre\_get\_custom\_post\_types** \\
