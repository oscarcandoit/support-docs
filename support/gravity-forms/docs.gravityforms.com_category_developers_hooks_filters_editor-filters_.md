[**gform\_add\_field\_buttons** \\
This filter can be used to add/edit/remove the "add field" buttons from the form editor's floating toolbox.](https://docs.gravityforms.com/gform_add_field_buttons/ "gform_add_field_buttons") [**gform\_disable\_ajax\_save** \\
The filter gform\_disable\_ajax\_save can be used to turn off AJAX saving in the form editor.](https://docs.gravityforms.com/gform_disable_ajax_save/ "gform_disable_ajax_save") [**gform\_editor\_sidebar\_panels** \\
The gform\_editor\_sidebar\_panels filter enables custom panels to be registered for display in the form editor sidebar.](https://docs.gravityforms.com/gform_editor_sidebar_panels/ "gform_editor_sidebar_panels") [**gform\_embed\_edit\_post\_link** \\
The gform\_embed\_edit\_post\_link filter is used to customize the edit post link template for the Embed Form flyout.](https://docs.gravityforms.com/gform_embed_edit_post_link/ "gform_embed_edit_post_link") [**gform\_embed\_post\_types** \\
The gform\_embed\_post\_types allows users to modify the post types presented in the Embed Form flyout.](https://docs.gravityforms.com/gform_embed_post_types/ "gform_embed_post_types") [**gform\_field\_groups\_form\_editor** \\
