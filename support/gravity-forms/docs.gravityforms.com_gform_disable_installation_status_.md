- [Description](https://docs.gravityforms.com/gform_disable_installation_status/#h-description)

- [Parameters](https://docs.gravityforms.com/gform_disable_installation_status/#h-parameters)

- [Since](https://docs.gravityforms.com/gform_disable_installation_status/#h-since)

## Description

This filter is used to disable the display of the **Installation Status** section on the Forms > Settings page.

## Usage

|     |     |
| --- | --- |
| 1 | `add_filter(``'gform_disable_installation_status'``,``'__return_true'``);` |

## Parameters

This hook has no parameters.

## Placement

This code should be placed in the **functions.php** file of your active theme.

## Source Code

This filter is located in GFSettings::gravityforms\_settings\_page() in _settings.php_

## Since

This filter was added in Gravity Forms 1.9.11.1
