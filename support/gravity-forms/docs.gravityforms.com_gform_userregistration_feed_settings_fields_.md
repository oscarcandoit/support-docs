## Description

This filter can be used to modify the setting fields that appear on the feed page.

## Usage

|     |
| --- |
| `add_filter(``'gform_userregistration_feed_settings_fields'``,``'your_function_name'``, 10, 2 );` |

## Parameters

- **$fields** [array](https://docs.gravityforms.com/array)

An array of fields, with the name property corresponding with the property which appears in the [User Registration Feed Meta](https://docs.gravityforms.com/user-registration-feed-meta). See the [Settings API](https://docs.gravityforms.com/settings-api) for more details.
- **$form** [Form Object](https://docs.gravityforms.com/form-object)

The form currently being processed.

## Remove a Setting

This example shows how to remove the “Role” dropdown from the User Registration feed for any user without the Administrator role.

| 1<br>2<br>3<br>4<br>5<br>6<br>7<br>8<br>9<br>10<br>11<br>12<br>13<br>14<br>15<br>16<br>17<br>18 | `add_filter(``'gform_userregistration_feed_settings_fields'``,``'remove_role_assignment'``, 10, 2 );`<br>`function``remove_role_assignment(``$fields``,``$form``) {`<br>```// Return early if the user is an administrator`<br>```if``( current_user_can(``'administrator'``) ) {`<br>```return``$fields``;`<br>```}`<br>```foreach``(``$fields``[``'user_settings'``][``'fields'``]``as``$index``=>``$field``) {`<br>```if``( isset(``$field``[``'name'``] ) &&``$field``[``'name'``] ===``'role'``) {`<br>```unset(``$fields``[``'user_settings'``][``'fields'``][``$index``] );`<br>```}`<br>```}`<br>```// Re-index the array`<br>```$fields``[``'user_settings'``][``'fields'``] =``array_values``(``$fields``[``'user_settings'``][``'fields'``] );`<br>```return``$fields``;`<br>`}` |
