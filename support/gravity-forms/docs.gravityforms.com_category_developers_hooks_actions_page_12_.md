[**gform\_uninstalling** \\

The gform\_update\_feed\_active action fires immediately after a feed’s active status is updated.](https://docs.gravityforms.com/gform_update_feed_active/ "gform_update_feed_active") [**gform\_update\_PROPERTY\_NAME** \\
Use this action hook to perform logic when entries’ basic information is updated.](https://docs.gravityforms.com/gform_update_status/ "gform_update_PROPERTY_NAME") [**gform\_update\_status** \\
The “gform\_update\_status” action hook is a dynamically generated filter name.](https://docs.gravityforms.com/gform_update_status-dynamic/ "gform_update_status") [**gform\_updates** \\
