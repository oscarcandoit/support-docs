## Articles

[**gform\_incomplete\_submission\_post\_get** \\
Allows the draft submission for Save and Continue to be overridden after it is retrieved from the database but before it used to populate the form.](https://docs.gravityforms.com/gform_incomplete_submission_post_get/ "gform_incomplete_submission_post_get") [**gform\_incomplete\_submission\_pre\_save** \\
Allows the draft submission for Save and Continue to be overridden before saved to the database.](https://docs.gravityforms.com/gform_incomplete_submission_pre_save/ "gform_incomplete_submission_pre_save") [**gform\_incomplete\_submissions\_expiration\_days** \\
Overrides the number of days until incomplete submissions are purged.](https://docs.gravityforms.com/gform_incomplete_submissions_expiration_days/ "gform_incomplete_submissions_expiration_days") [**gform\_init\_scripts\_footer** \\
This filter is executed during form load. When set to true, the form init scripts are loaded in the footer of the site, instead of the default location of which is in the page body immediately after the form.](https://docs.gravityforms.com/gform_init_scripts_footer/ "gform_init_scripts_footer") [**gform\_input\_mask\_script** \\
Use this filter to change the initialization script for the input mask script. Can be used to specify different initialization parameters.](https://docs.gravityforms.com/gform_input_mask_script/ "gform_input_mask_script") [**gform\_input\_masks** \\
Use this filter to edit the list of built-in input masks that are displayed in the Text Field input mask setting.](https://docs.gravityforms.com/gform_input_masks/ "gform_input_masks") [**gform\_ip\_address** \\
Allows the IP address of the client to be modified. Use this filter if the server is behind a proxy.](https://docs.gravityforms.com/gform_ip_address/ "gform_ip_address") [**gform\_is\_asynchronous\_notifications\_enabled** \\
The gform\_is\_asynchronous\_notifications\_enabled filter can be used to enable or disable async (background) processing of notifications.](https://docs.gravityforms.com/gform_is_asynchronous_notifications_enabled/ "gform_is_asynchronous_notifications_enabled") [**gform\_is\_delayed\_pre\_process\_feed** \\
This filter can be used to delay feed processing for Add-On Framework based add-ons.](https://docs.gravityforms.com/gform_is_delayed_pre_process_feed/ "gform_is_delayed_pre_process_feed") [**gform\_is\_disabled\_duplicate\_submissions\_protection** \\
