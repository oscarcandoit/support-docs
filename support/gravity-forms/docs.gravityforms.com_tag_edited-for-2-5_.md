[**{embed\_post} Merge Tag** \\
Displays the member-variable data specified for the singular post or page from which the entry was submitted.](https://docs.gravityforms.com/embed_post-merge-tag/ "{embed_post} Merge Tag") [**Reviewing Spam Submissions** \\
The following features are available on the Entries List and Entry Detail pages when entries have been marked as spam by a compatible anti-spam plugin.](https://docs.gravityforms.com/reviewing-spam-submissions/ "Reviewing Spam Submissions") [**Entry Creator Merge Tag** \\
The entry creator merge tag can be used in notifications, confirmations, and with some add-ons to output properties of the user who was logged in at the time the entry was saved.](https://docs.gravityforms.com/entry-creator-merge-tag/ "Entry Creator Merge Tag") [**Entry Date Merge Tags** \\
The entry date merge tags can be used in notifications, confirmations, and with some add-ons to output the date and/or time the entry was created, updated, and paid.](https://docs.gravityforms.com/entry-date-merge-tags/ "Entry Date Merge Tags") [**Field Merge Tags** \\
Displays the value submitted for the field with the specified field\_id. If no value was submitted, the merge tag will not display anything.](https://docs.gravityforms.com/field-merge-tags/ "Field Merge Tags") [**Post Image Merge Tags** \\
Will display either the uploaded post image or metadata about the image.](https://docs.gravityforms.com/post-image-merge-tags/ "Post Image Merge Tags") [**Exporting Form Entries** \\
Learn how to export all of your form entries (submissions) to enable further analysis or manipulation in external tools.](https://docs.gravityforms.com/exporting-form-entries/ "Exporting Form Entries") [**Entry Trash** \\
With Gravity Forms version 1.6, unwanted entries are no longer deleted immediately. Instead these entries are moved to the Trash.](https://docs.gravityforms.com/trash/ "Entry Trash") [**Printing an Entry** \\
This option shows a print preview for your entry, displaying all of the form field names as well as the values that have been entered.](https://docs.gravityforms.com/print/ "Printing an Entry") [**Bulk Print Multiple Entries** \\
