## Articles

[**gform\_force\_hooks\_js\_output** \\
The gform\_force\_hooks\_js\_output filter can be used to prevent the Gravity Forms scripts from being added to all pages.](https://docs.gravityforms.com/gform_force_hooks_js_output/ "gform_force_hooks_js_output") [**gform\_form\_actions** \\
Add custom form actions which display below the form title in the Form List view](https://docs.gravityforms.com/gform_form_actions/ "gform_form_actions") [**gform\_form\_after\_open** \\
This filter allows markup to be added directly after the opening form wrapper.](https://docs.gravityforms.com/gform_form_after_open/ "gform_form_after_open") [**gform\_form\_args** \\
The gform\_form\_args filter provides the ability to modify the options used to display the form.](https://docs.gravityforms.com/gform_form_args/ "gform_form_args") [**gform\_form\_export\_filename** \\
Allows the form export filename to be changed.](https://docs.gravityforms.com/gform_form_export_filename/ "gform_form_export_filename") [**gform\_form\_list\_column\_COLUMN\_NAME** \\
This filter enabled you to display custom column content on the Form List page after creating a custom column with the gform\_form\_list\_columns filter.](https://docs.gravityforms.com/gform_form_list_column_column_name/ "gform_form_list_column_COLUMN_NAME") [**gform\_form\_list\_columns** \\
Allows the column headers displayed on the Form List page to be modified.](https://docs.gravityforms.com/gform_form_list_columns/ "gform_form_list_columns") [**gform\_form\_list\_count** \\
This filter allows the form list count array to be modified. This is useful when filtering the form count list.](https://docs.gravityforms.com/gform_form_list_count/ "gform_form_list_count") [**gform\_form\_list\_forms** \\
