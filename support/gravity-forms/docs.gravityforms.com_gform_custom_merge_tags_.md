## Description

The `gform_custom_merge_tags` filter allows you to add additional merge tags that you have defined to the drop down menus where merge tags are available. **Important:** This filter only **adds** the merge tags to the drop down menus; it does not create them. They must have already been created using the [gform\_replace\_merge\_tags](https://docs.gravityforms.com/gform_replace_merge_tags/) filter.

## Usage

The `gform_custom_merge_tags` The filter has both [JavaScript](https://docs.gravityforms.com/gform_custom_merge_tags/#h-javascript-version) and [PHP versions](https://docs.gravityforms.com/gform_custom_merge_tags/#h-php-version). These are used in different contexts based on how the merge tag selector is presented.

The JavaScript version filters available merge tags shown in selectors for all field settings rendered in the form editor.

|     |
| --- |
| `gform.addFilter(``'gform_custom_merge_tags'``,``function``( mergeTags, formId, fields, elementId ) {`<br>```// do stuff`<br>``<br>```return``mergeTags;`<br>`} );` |

The PHP version filters the merge tag selector on all other settings pages.

|     |
| --- |
| `add_filter(``'gform_custom_merge_tags'``,``'your_function_name'``, 10, 4 );` |

## JavaScript Version

### Parameters

- `$customMergeTags` [array](https://docs.gravityforms.com/array/)

The custom merge tags.
- `$formId` [string](https://docs.gravityforms.com/int/)

The ID of the current form.
- `$fields` [Field Object](https://docs.gravityforms.com/field-object/)

An array of fields objects.
- `$elementId` [string](https://docs.gravityforms.com/string/)

The ID of the input field.

## Placement

This code can be used in the [functions.php](https://developer.wordpress.org/themes/basics/theme-functions/) file of the active theme, a custom functions plugin, a custom add-on, or with a code snippets plugin.

See also the PHP section in this article: [Where Do I Put This Code?](https://docs.gravityforms.com/where-do-i-put-this-code/)

## Since

This PHP filter was added in Gravity Forms version 1.6.2. The JavaScript version was added in version 2.8.5.2.

## Source Code

The PHP filter is located in:

GFCommon::get\_merge\_tags() in _common.php_

GFCommon::insert\_calculation\_variables() in _common.php_

The JavaScript filter is located in _assets/js/src/admin/components/merge-tags/utils.js_
