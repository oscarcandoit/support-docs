# gform\_authorizenet\_amount\_pre\_authorize

- [Source Code](https://docs.gravityforms.com/gform_authorizenet_amount_pre_authorize/#h-source-code)

## Description

This filter can be used to modify the authorization amount before it is sent to Authorize.net.

## Usage

The filter which would run for all ‘product and services’ type Authorize.net feeds can be used like so:

|     |     |
| --- | --- |
| 1 | `add_filter(``'gform_authorizenet_amount_pre_authorize'``,``'your_function_name'``, 10, 6 );` |

## Parameters

- **$auth\_amount** [float](https://docs.gravityforms.com/float)
The authorization amount. Defaults to value of $config\[‘amount’\].

- **$transaction** [object](https://docs.gravityforms.com/object)

The Authorize.net [transaction object](http://developer.authorize.net/api/reference/#payment-transactions-authorize-a-credit-card).

- **$form\_data** [Form Data](https://docs.gravityforms.com/form-data)

An associative array containing the form title, billing address, payment amount, setup fee amount, line items created using the submitted pricing field values and any discounts from coupons.

- **$config** [Authorize Net Config](https://docs.gravityforms.com/authorize-net-config)

The feed which is currently being processed.

- **$form** [Form Object](https://docs.gravityforms.com/form-object)

The form which is currently being processed.

- **$entry** [Entry Object](https://docs.gravityforms.com/entry-object)

The entry which is currently being processed. Since version 2.1.8.


## Example

The following example shows how you can override the authorization amount.

|     |     |
| --- | --- |
| 1<br>2<br>3<br>4<br>5<br>6<br>7<br>8 | `add_filter(``'gform_authorizenet_amount_pre_authorize'``,``'change_amount'``, 10, 4 );`<br>`function``change_amount(``$auth_amount``,``$transaction``,``$form_data``,``$config``,``$form``) {`<br>`if``(``$form``[``'id'``] == 10 ) {`<br>`$auth_amount``= 1;`<br>`}`<br>`return``$auth_amount``;`<br>`}` |

## Placement

Your code snippet should be placed in the [functions.php](https://codex.wordpress.org/Theme_Development#Functions_File) file of your active theme.

## Source Code

This filter is located in GFAuthorizeNet::authorize() in _class-gf-authorizenet.php_.
