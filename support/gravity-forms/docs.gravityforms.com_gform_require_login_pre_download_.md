## Description

By default the real location of an uploaded file is hidden. The download URL will be generated with a security token to prevent guessing or enumeration attacks to discover the location of other files. The gform\_require\_login\_pre\_download filter can be used to require the user be logged in before the download URL will allow access to the file.

## Usage

The following would apply to all forms.

|     |
| --- |
| `add_filter(``'gform_require_login_pre_download'``,``'your_function_name'``, 10, 3 );` |

## Parameters

- **$require\_login** [bool](https://docs.gravityforms.com/bool)
Does the user need to be logged in to access the file? Default false.

- **$form\_id** [int](https://docs.gravityforms.com/int)

The ID of the form used to upload the requested file.

- **$field\_id** [int](https://docs.gravityforms.com/int)

The ID of the field used to upload the requested file.
