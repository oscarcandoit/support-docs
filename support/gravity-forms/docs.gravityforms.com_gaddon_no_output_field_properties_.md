## Description

Some non-standard HTML properties are not output to the field HTML by default. This filter allows you to allow or deny specific properties to be added to the field HTML.

## Usage

The following would apply to all forms.

|     |     |
| --- | --- |
| 1 | `add_filter(``'gaddon_no_output_field_properties'``,``'your_function_name'``);` |

## Parameters

- **$properties** [array](https://docs.gravityforms.com/array)
Array of properties not allowed.

- **$field** [Form Object](https://docs.gravityforms.com/form-object)

The current field meta to be parsed.


## Placement

This code should be placed in the [functions.php](http://codex.wordpress.org/Theme_Development#Functions_File) file of your active theme.
