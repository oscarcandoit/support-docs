## Description

The gform\_signature\_init\_options filter can be used to customize the Signature Pad initialization options.

## Usage

The following would apply to all forms.

|     |     |
| --- | --- |
| 1 | `add_filter(``'gform_signature_init_options'``,``'your_function_name'``, 10, 3 );` |

## Parameters

- **$init\_options** [array](https://docs.gravityforms.com/array)

The options to be used when initializing Signature Pad for this field.

Refer to [Signature Pad](https://github.com/szimek/signature_pad?tab=readme-ov-file#options) documentation for more information about the options.

|     |     |
| --- | --- |
| 1<br>2<br>3<br>4<br>5<br>6<br>7<br>8<br>9<br>10 | `$init_options``=``array``(`<br>```'dotSize'``=> 2,``// Radius of a single dot`<br>```'minWidth'``=>``empty``(``$this``->penSize ) ? 0.5 :``$this``->penSize,``// Minimum width of a line`<br>```'maxWidth'``=> rgblank(``$this``->boxWidth ) ? 2.5 :``$this``->boxWidth,``// Maximum width of a line`<br>```'throttle'``=> 16,``// Max milliseconds between points`<br>```'minDistance'``=> 5,``// Minimum distance between points`<br>```'backgroundColor'``=>``empty``(``$this``->backgroundColor ) ?``'rgba(255,255,255,1)'``:``$this``->backgroundColor,``// Background color`<br>```'penColor'``=>``empty``(``$this``->penColor ) ?``'black'``:``$this``->penColor,``// Pen color`<br>```'velocityFilterWeight'``=> 0.7,``// Smoothing for velocity`<br>`);` |

- **$field** [Field Object](https://docs.gravityforms.com/field-object) The current field object.
- **$form** [Form Object](https://docs.gravityforms.com/form-object) The current form object.

## Examples

### Set Background and Pen Color

|     |     |
| --- | --- |
| 1<br>2<br>3<br>4<br>5<br>6<br>7<br>8<br>9<br>10 | `add_filter(``'gform_signature_init_options'``,``'customize_signature_options'``, 10, 3 );`<br>`function``customize_signature_options(``$init_options``,``$field``,``$form``) {`<br>```// Target specific form and field`<br>```if``(``$form``[``'id'``] == 5 &&``$field``->id == 3 ) {``// Adjust form ID and field ID as needed`<br>```$init_options``[``'backgroundColor'``] =``'rgba(255,255,255,1)'``;``// White background`<br>```$init_options``[``'penColor'``] =``'rgb(0,0,0)'``;``// Black pen color`<br>```}`<br>```return``$init_options``;`<br>`}` |

### Customized Line Width and Colors

|     |     |
| --- | --- |
| 1<br>2<br>3<br>4<br>5<br>6<br>7<br>8<br>9 | `add_filter(``'gform_signature_init_options'``,``'customize_signature_options'``, 10, 3 );`<br>`function``customize_signature_options(``$init_options``,``$field``,``$form``) {`<br>```$init_options``[``'minWidth'``] = 1;``// Thicker minimum line width`<br>```$init_options``[``'maxWidth'``] = 4;``// Thicker maximum line width`<br>```$init_options``[``'throttle'``] = 10;``// Increased responsiveness`<br>```$init_options``[``'velocityFilterWeight'``] = 0.9;``// Smoother line drawing`<br>```return``$init_options``;`<br>`}` |

## Since

This filter was added in Gravity Forms Signature Add-On 3.0.2.

## Placement

This code should be placed in the [functions.php](http://codex.wordpress.org/Theme_Development#Functions_File) file of your active theme.

## Source Code

This filter is located in GF\_Field\_Signature::GF\_Field\_Signature::get\_signaturepad\_init\_options() in _class-gf-field-signature.php_.

It was previously located in GF\_Field\_Signature::get\_supersignature\_init\_options().
