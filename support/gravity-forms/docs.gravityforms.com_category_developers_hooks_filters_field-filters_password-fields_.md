Use this filter to change the Password Field sub-label.](https://docs.gravityforms.com/gform_password/ "gform_password") [**gform\_password\_confirm** \\
Use this filter to change the sub-label for the Confirm Password Field.](https://docs.gravityforms.com/gform_password_confirm/ "gform_password_confirm")
