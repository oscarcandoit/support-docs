## Description

This filter is fired every time Gravity Forms requests the list of products saved in an entry.

Use this filter to manipulate the list of products displayed in the entry detail and sent to third party services via Add-Ons such as the PayPal Add-On.

Note that this filter only applies to the creation of new entries, and will not fire when loading existing cached entries. To manipulate existing entries, you may need to [delete the cached entry data](https://docs.gravityforms.com/deleting-cached-entry-data/) first.

## Usage

The following would apply to all forms using pricing fields.

|     |     |
| --- | --- |
| 1 | `add_filter(``'gform_product_info'``,``'your_function_name'``, 10, 3 );` |

To limit the scope of the function to a specific form simply append the form id after the hook name. (format: gform\_product\_info\_FORMID)

|     |     |
| --- | --- |
| 1 | `add_filter(``'gform_product_info_6'``,``'your_function_name'``, 10, 3 );` |

## Examples

### 1\. Add an extra line item

This example adds an Extra Fee line item if the quantity is greater than 200.

|     |     |
| --- | --- |
| 1<br>2<br>3<br>4<br>5<br>6<br>7<br>8<br>9<br>10 | `add_filter(``'gform_product_info'``,``'add_fee'``, 10, 3 );`<br>`function``add_fee(``$product_info``,``$form``,``$lead``) {`<br>```$quantity``=``$product_info``[``'products'``][1][``'quantity'``];`<br>```if``(``$quantity``> 200 ) {`<br>```$product_info``[``'products'``][``'extra_fee'``] =``array``(``'name'``=>``'Extra Fee'``,``'price'``=> 50,``'quantity'``=> 1 );`<br>```}`<br>```return``$product_info``;`<br>`}` |
