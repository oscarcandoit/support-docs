# Obtaining Your Agile CRM API Key

When using the Agile CRM Add-On for Gravity Forms, you will need to be able to authenticate with Agile CRM from your site. In this article, we will show you how to obtain your Agile CRM API key.

1. Log into your [Agile CRM account](https://my.agilecrm.com/enter-domain?to=login).
2. From the top right of your main Agile CRM dashboard page, click on your name/icon, then click on **Admin Settings**.
3. From the left-hand menu, click on **Developers & API**. In the API Key section, you will see your REST API key and JavaScript API key for Analytics Tracking.
