## Description

This filter can be used to modify a value before it is sent to a third-party by one of the [Add-On Framework](https://docs.gravityforms.com/category/developers/php-api/add-on-framework) based add-ons. If you want to filter the value for a specific add-on you can use [gform\_short\_slug\_field\_value](https://docs.gravityforms.com/gform_short_slug_field_value).

## Usage

The base filter which would run for all forms and all fields would be used like so:

|     |     |
| --- | --- |
| 1 | `add_filter(``'gform_addon_field_value'``,``'your_function_name'``, 10, 5 )` |

To target a specific form append the form id to the hook name. (format: gform\_addon\_field\_value\_FORMID)

|     |     |
| --- | --- |
| 1 | `add_filter(``'gform_addon_field_value_10'``,``'your_function_name'``, 10, 5 );` |

To target a specific field append both the form id and the field id to the hook name. (format: gform\_addon\_field\_value\_FORMID\_FIELDID)

|     |     |
| --- | --- |
| 1 | `add_filter(``'gform_addon_field_value_10_3'``,``'your_function_name'``, 10, 5 );` |

## Parameters

| Parameter | Type | Description |
| --- | --- | --- |
| **$field\_value** | mixed | The current value of the field before being filtered. |
| **$form** | [Form Object](https://docs.gravityforms.com/form-object/) | The form currently being processed. |
| **$entry** | [Entry Object](https://docs.gravityforms.com/entry-object/) | The entry currently being processed. |
| **$field\_id** | [string](https://docs.gravityforms.com/string/) | The ID of the field being processed. |
| **$slug** | [string](https://docs.gravityforms.com/string/) | The add-on slug, used to namespace the filter hook. |

## Examples

### 1\. Change Value of Specific Field

This example shows how you can change the value of field 3 on form 10 before it is passed to any of the feed add-ons.

|     |     |
| --- | --- |
| 1<br>2<br>3 | `add_filter(``'gform_addon_field_value_10_3'``,``function``(``$field_value``,``$form``,``$entry``,``$field_id``) {`<br>```return``'your new value'``;`<br>`}, 10, 4 );` |

### 2\. Use Choice Text Instead of Value

This example shows how you can replace the value of a choice based survey field with the choice text.

|     |     |
| --- | --- |
| 1<br>2<br>3<br>4<br>5<br>6<br>7<br>8<br>9<br>10 | `add_filter(``'gform_addon_field_value'``,``'gf_get_choice_text'``, 10, 5 );`<br>`function``gf_get_choice_text(``$field_value``,``$form``,``$entry``,``$field_id``,``$slug``) {`<br>```$field``= RGFormsModel::get_field(``$form``,``$field_id``);`<br>```if``(``is_object``(``$field``) &&``$field``->type ==``'survey'``) {`<br>```$field_value``=``$field``->get_value_export(``$entry``,``$field_id``, true );`<br>```}`<br>```return``$field_value``;`<br>`}` |
