- [Source Code](https://docs.gravityforms.com/gform_helpscout_tags/#h-source-code)

## Description

This filter is used to dynamically alter the tags being created for a Help Scout conversation.

## Usage

Applies to all forms:

|     |     |
| --- | --- |
| 1 | `add_filter(``'gform_helpscout_tags'``,``'your_function_name'``, 10, 4 );` |

Applies to a specific form:

|     |     |
| --- | --- |
| 1 | `add_filter(``'gform_helpscout_tags_4'``,``'your_function_name'``, 10, 4 );` |

## Parameters

| Parameter | Type | Description |
| --- | --- | --- |
| **$tags** | [array](https://docs.gravityforms.com/array) | An array of tags to be assigned to the conversation. |
| **$feed** | [Feed Object](https://docs.gravityforms.com/feed-object) | The current feed object. |
| **$entry** | [Entry Object](https://docs.gravityforms.com/entry-object) | The current entry object. |
| **$form** | [Form Object](https://docs.gravityforms.com/form-object) | The current form object. |

## Examples

This example retrieves the license type a user has associated to their user meta and adds it as a tag.

|     |     |
| --- | --- |
| 1<br>2<br>3<br>4<br>5<br>6<br>7<br>8<br>9<br>10<br>11<br>12 | `add_filter(``'gform_helpscout_tags'``,``'add_license_tag'``, 10, 4 );`<br>`function``add_license_tag(``$tags``,``$feed``,``$entry``,``$form``) {`<br>```global``$current_user``;`<br>```get_currentuserinfo();`<br>```$license_type``= get_user_meta(``$current_user``->ID,``'license_type'``,``'No License'``);`<br>```$tags``[] =``$license_type``;`<br>```return``$tags``;`<br>`}` |

## Placement

This code can be used in the [functions.php](https://developer.wordpress.org/themes/basics/theme-functions/) file of the active theme, a custom functions plugin, a custom add-on, or with a code snippets plugin.

See also the PHP section in this article: [Where Do I Put This Code?](https://docs.gravityforms.com/where-do-i-put-this-code/)
