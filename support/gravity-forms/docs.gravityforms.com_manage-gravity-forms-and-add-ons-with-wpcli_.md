By using the CLI Add-On and the following root commands you can easily install and update Gravity Forms and official Add-Ons from the command line.

## Pre-Requisites

- [Download](https://docs.gravityforms.com/downloading-a-gravity-forms-add-on) and [install](https://docs.gravityforms.com/installing-a-gravity-forms-add-on/) [Gravity Forms CLI](https://www.gravityforms.com/my-account/licenses/)

## Installing Gravity Forms or an Add-On

`wp gf install`

### Examples

|     |
| --- |
| `# Using the install command without supplying a license key.`<br>`$ wp gf``install`<br>`Error: A valid license key must be specified either``in``the GF_LICENSE_KEY constant or the --key option.` |

|     |
| --- |
| `# Install Gravity Forms, replacing the installed version.`<br>`$ wp gf``install``--force`<br>`Downloading installation package from http:``//``...`<br>`Unpacking the package...`<br>`Installing the plugin...`<br>`Removing the old version of the plugin...`<br>`Plugin updated successfully.`<br>`Success: Installed 1 of 1 plugins.` |

|     |
| --- |
| `# Installing Gravity Forms using the specified license key.`<br>`$ wp gf``install``--key=abc123`<br>`Downloading installation package from http:``//``...`<br>`Unpacking the package...`<br>`Installing the plugin...`<br>`Plugin installed successfully.`<br>`Success: Installed 1 of 1 plugins.` |

|     |
| --- |
| `# Install the Gravity Forms Polls Add-On using the specified license key.`<br>`$ wp gf``install``gravityformspolls --key=abc123`<br>`Downloading installation package from http:``//``...`<br>`Unpacking the package...`<br>`Installing the plugin...`<br>`Plugin installed successfully.`<br>`Success: Installed 1 of 1 plugins.` |

|     |
| --- |
| `# Using an invalid add-on slug or license key.`<br>`$ wp gf``install``something --key=abc123`<br>`Error: There was a problem retrieving the download URL, please check the key.` |

### Parameters

| Argument | Description |
| --- | --- |
| `[<slug>]` | The plugin slug.<br>Accepted values: _gravityforms_ or a [Gravity Forms Add-On slug](https://docs.gravityforms.com/gravity-forms-add-on-slugs/).<br>Default: _gravityforms_. |
| `[--key=<value>]` | A valid Gravity Forms license key. |
| `[--version=<value>]` | The version to be installed.<br>Accepted values: _auto-update_, _hotfix_, or _beta_.<br>Default: _hotfix_. |
| `[--force]` | Overwrites the installed version of the plugin, without prompting for confirmation. |
| `[--activate]` | Activates the plugin immediately after install. |
| `[--activate-network]` | Network activates the plugin immediately after install. |

## Updating Gravity Forms or an Add-On

`wp gf update`

### Examples

|     |
| --- |
| `# Attempting to update without Gravity Forms being active.`<br>`$ wp gf update`<br>`Error: Gravity Forms is not active.` |

|     |
| --- |
| `# Attempting to update Gravity Forms without a license key.`<br>`$ wp gf update`<br>`Error: A valid license key must be saved``in``the settings or specified``in``the GF_LICENSE_KEY constant or the --key option.` |

|     |
| --- |
| `# Attempting to update Gravity Forms when an update is not available.`<br>`$ wp gf update`<br>`Success: Plugin already updated` |

|     |
| --- |
| `# Successfully updating Gravity Forms using the specified license key.`<br>`$ wp gf update --key=abc123`<br>`Downloading installation package from http:``//``...`<br>`Unpacking the package...`<br>`Installing the plugin...`<br>`Removing the old version of the plugin...`<br>`Plugin updated successfully.`<br>`Success: Installed 1 of 1 plugins.`<br>`Success: Database upgraded.` |

|     |
| --- |
| `# Successfully updating the Gravity Forms Polls Add-On.`<br>`$ wp gf update gravityformspolls`<br>`Downloading installation package from http:``//``...`<br>`Unpacking the package...`<br>`Installing the plugin...`<br>`Removing the old version of the plugin...`<br>`Plugin updated successfully.`<br>`Success: Installed 1 of 1 plugins.`<br>`Success: setup gravityformspolls` |

### Parameters

| Argument | Description |
| --- | --- |
| `[<slug>]` | The plugin slug.<br>Accepted values: _gravityforms_ or a [Gravity Forms Add-On slug](https://docs.gravityforms.com/gravity-forms-add-on-slugs/).<br>Default: _gravityforms_. |
| `[--key=<value>]` | A valid Gravity Forms license key. |
| `[--version=<value>]` | The version to be installed.<br>Accepted values: _auto-update_, _hotfix_, or _beta_.<br>Default: _hotfix_. |

## Running the Database Setup

`wp gf setup`

### Examples

|     |
| --- |
| `# Successfully running the database setup for a new install.`<br>`$ wp gf setup`<br>`Success: setup gravityforms` |

|     |
| --- |
| `# Failing to run the database setup for an existing install.`<br>`$ wp gf setup`<br>`Error: Use the --force flag to force the database setup.` |

|     |
| --- |
| `# Successfully running the database setup for an existing install.`<br>`$ wp gf setup --force`<br>`Success: Database upgraded.` |

|     |
| --- |
| `# Successfully running the database setup for the Gravity Forms Polls Add-On.`<br>`$ wp gf setup gravityformspolls`<br>`Success: setup gravityformspolls` |

|     |
| --- |
| `# Forcing the setup on a multisite network`<br>`$``for``url``in``$(wp site list --fields=url --``format``=csv |``tail``-n +2);``do``echo``Running setup``for``site:  $url; wp gf setup --force --url=$url;``done`<br>`Running setup``for``site: https:``//one``.site.``local``/`<br>`Success: Database upgraded.`<br>`Running setup``for``site: https:``//two``.site.``local``/`<br>`Success: Database upgraded.` |

### Parameters

| Argument | Description |
| --- | --- |
| `[<slug>]` | The plugin slug.<br>Accepted values: _gravityforms_ or a [Gravity Forms Add-On slug](https://docs.gravityforms.com/gravity-forms-add-on-slugs/).<br>Default: _gravityforms_. |
| `[--force]` | Runs the setup regardless of whether it has been run before. |

## Checking for Updates

`wp gf check-update`

### Examples

|     |
| --- |
| `# Successfully getting the available Gravity Forms versions.`<br>`$ wp gf check-update`<br>`+-------------+---------+`<br>`|``type``| version |`<br>`+-------------+---------+`<br>`| auto-update | 2.3     |`<br>`| hotfix      | 2.3.0.4 |`<br>`+-------------+---------+` |

|     |
| --- |
| `# Successfully getting the available Gravity Forms Polls Add-On versions in yaml format.`<br>`$ wp gf check-update gravityformspolls --``format``=yaml`<br>`---`<br>`-`<br>```type``: auto-update`<br>```version: 3.1`<br>`-`<br>```type``: hotfix`<br>```version: 3.1.4` |

### Parameters

| Argument | Description |
| --- | --- |
| `[<slug>]` | The plugin slug.<br>Accepted values: _gravityforms_ or a [Gravity Forms Add-On slug](https://docs.gravityforms.com/gravity-forms-add-on-slugs/).<br>Default: _gravityforms_. |
| `[--format=<value>]` | Defines the format in which the updates will be listed.<br>Accepted values: _table_, _csv_, _json_, _yaml_, and _count_.<br>Default: _table_. |

## Getting the Version

`wp gf version`

### Examples

|     |
| --- |

|     |

| `# Successfully getting the Gravity Forms version.`<br>`$ wp gf version`<br>`2.3.0.4` |

|     |
| --- |
| `# Attempting to get the add-on version using an invalid slug.`<br>`$ wp gf version something`<br>`Error: Invalid plugin slug: something` |

|     |
| --- |
| `# Successfully getting the Gravity Forms Polls Add-On version.`<br>`$ wp gf version gravityformspolls`<br>`3.1.4` |

### Parameters

| Argument | Description |
| --- | --- |
| `[<slug>]` | The plugin slug.<br>Accepted values: _gravityforms_ or a [Gravity Forms Add-On slug](https://docs.gravityforms.com/gravity-forms-add-on-slugs/).<br>Default: _gravityforms_. |
