# Extending the Dropbox Add-On

The Dropbox Add-On provides hooks that can be used to modify the default functionality or extend it. For more information, review the [Dropbox](https://docs.gravityforms.com/category/developers/hooks/filters/filters-addons/dropbox-filters/) category.

Looking for information about what properties the Dropbox Upload field uses in the [Field Object](https://docs.gravityforms.com/field-object) or what format it’s value is stored in the [Entry Object](https://docs.gravityforms.com/entry-object) and how you can access it? Then take a look at the [GF\_Field\_Dropbox](https://docs.gravityforms.com/gf_field_dropbox) article.

Because the Dropbox Add-On is built using the [Add-On Framework](https://docs.gravityforms.com/category/developers/php-api/add-on-framework), it also inherits any hooks available in the framework methods it uses such as:

- [gform\_short\_slug\_field\_value](https://docs.gravityforms.com/gform_short_slug_field_value) for changing the Dropbox Upload field value before it is used to upload the files to Dropbox.
- [gform\_short\_slug\_error](https://docs.gravityforms.com/gform_short_slug_error) for performing a custom action when the add-on experiences an error processing a feed.
