## Description

The gform\_field\_settings\_tabs filter enables new tabs to be registered for display in the field settings panel in the form editor. Use the [gform\_field\_settings\_tab\_content](https://docs.gravityforms.com/gform_field_settings_tab_content) action hook to echo the tab content.

**Note:** the filter can only be used to add new tabs and there is no provision to remove tabs.

## Usage

The filter which would run for all forms would be used like so:

|     |     |
| --- | --- |
| 1 | `add_filter(``'gform_field_settings_tabs'``,``'your_function_name'``, 10, 2 );` |

You can also target a specific form by adding the form id after the hook name.

|     |     |
| --- | --- |
| 1 | `add_filter(``'gform_field_settings_tabs_6'``,``'your_function_name'``, 10, 2 );` |

## Parameters

- **$tabs** [array](https://docs.gravityforms.com/array)

An array of custom field settings tabs.
- **$form** [Form Object](https://docs.gravityforms.com/form-object)

The form currently being edited.

## Examples

### 1\. Add a new tab

|     |     |
| --- | --- |
| 1<br>2<br>3<br>4<br>5<br>6<br>7<br>8<br>9<br>10 | `add_filter(``'gform_field_settings_tabs'``,``function``(``$tabs``,``$form``) {`<br>```// Add a custom tab`<br>```$tabs``[] =``array``(`<br>```'id'``=>``'my_custom_tab_1'``,``// The Tab ID`<br>```'title'``=>``'My Custom Tab'``,``// The Tab Title`<br>```'toggle_classes'``=>``array``(``'my_toggle_class_1'``,``'my_toggle_class_2'``),`<br>```'body_classes'``=>``array``(``'my_body_class_1'``),`<br>```);`<br>```return``$tabs``;`<br>`}, 10, 2);` |

## Placement

This code should be placed in the [functions.php](http://codex.wordpress.org/Theme_Development#Functions_File) file of your active theme or a custom functions plugin.

## Since

This filter was added in Gravity Forms v2.5.

## Source Code

This filter is located in GFFormDetail::forms\_page() in _form\_detail.php_
