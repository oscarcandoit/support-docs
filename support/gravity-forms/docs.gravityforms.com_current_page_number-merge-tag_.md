# {current\_page\_number} Merge Tag

- [Summary](https://docs.gravityforms.com/current_page_number-merge-tag/#h-summary)
- [Usage](https://docs.gravityforms.com/current_page_number-merge-tag/#h-usage)
- [Output](https://docs.gravityforms.com/current_page_number-merge-tag/#h-output)

## Summary

The `{current_page_number}` merge tag is available for use in **Form Settings → Pagination Tracking → Event Parameters** when using the [Google Analytics Add-On](https://www.gravityforms.com/add-ons/google-analytics/).

## Usage

`{current_page_number}`

## Output

This merge tag returns the number of the page the user is currently visiting.
