## Description

Allows modifying the spinner URL when the gformAddSpinner() function is called directly.

If the spinner URL is not implemented using the gformAddSpinner function, the URL should be changed using the [gform\_ajax\_spinner\_url](https://docs.gravityforms.com/gform_ajax_spinner_url) filter.

## Usage

|     |     |
| --- | --- |
| 1<br>2<br>3<br>4<br>5 | `gform.addFilter(``'gform_spinner_url'``,``'change_spinner'``);`<br>`function``change_spinner(){ spinnerUrl, formId )`<br>```//do something here`<br>```return``spinnerUrl;`<br>`}` |

## Parameters

- **spinnerUrl** [string](https://docs.gravityforms.com/string)


The URL to the image used for the spinner.

- **formId** [int](https://docs.gravityforms.com/int)


The current form ID.


## Example

|     |     |
| --- | --- |
| 1<br>2<br>3<br>4<br>5 | `gform.addFilter(``'gform_spinner_url'``,``'change_spinner'``);`<br>`function``change_spinner(){ spinnerUrl, formId )`<br>```spinnerUrl =``'https://your-domain-here.test/wp-content/uploads/spinner.svg'``;`<br>```return``spinnerUrl;`<br>`}` |

## Placement

This code should be placed in a custom JavaScript file used by your theme.

## Since

This filter was added in Gravity Forms version 1.8.

## Source Code

This filter is located in gformAddSpinner() in **js/gravityforms.js**.
