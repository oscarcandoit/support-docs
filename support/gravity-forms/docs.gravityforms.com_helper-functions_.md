# Helper Functions

[get\_capabilities()](https://docs.gravityforms.com/get_capabilities)

The get\_capabilities function retrieves all capabilities or a specific capability for the add-on.

[gf\_do\_action()](https://docs.gravityforms.com/gf_do_action)

The gf\_do\_action function allows additional actions based on form and field ID to be defined easily.

[gform\_add\_meta()](https://docs.gravityforms.com/gform_add_meta)

The gform\_add\_meta function adds the metadata associated with an entry in the Entry Meta table.

[gform\_delete\_meta()](https://docs.gravityforms.com/gform_delete_meta)

The gform\_delete\_meta function deletes the meta key from the Entry Meta table.

[gform\_get\_meta()](https://docs.gravityforms.com/gform_get_meta)

The gform\_get\_meta function retrieves the meta value of the specified meta key from the Entry Meta table.

[gform\_update\_meta()](https://docs.gravityforms.com/gform_update_meta)

The gform\_update\_meta function updates the meta value of the specified meta key in the Entry Meta table.

[gravity\_form\_enqueue\_scripts](https://docs.gravityforms.com/gravity_form_enqueue_scripts)

The gravity\_form\_enqueue\_scripts function will enqueue the necessary styles and scripts for the specified Gravity Form.

[rgar()](https://docs.gravityforms.com/rgar)

The rgar function is used to retrieve a value from an array.

[rgars()](https://docs.gravityforms.com/rgars)

The rgars function is used to retrieve a value from a multi level array.

[rgblank()](https://docs.gravityforms.com/rgblank)

The rblank function can be used to validate if a string is empty.

[rgempty()](https://docs.gravityforms.com/rgempty)

The rgempty function is used to validate if a variable or array key is empty, null, or false.

[rgexplode()](https://docs.gravityforms.com/rgexplode)

The rgexplode function explodes a string into an array of strings.

[rgget()](https://docs.gravityforms.com/rgget)

The rgget function is used to retrieve a value from the $\_GET array.

[rgobj()](https://docs.gravityforms.com/rgobj)

The rgobj function is used to retrieve a property value from an object.

[rgpost()](https://docs.gravityforms.com/rgpost)

The rgpost function is used to retrieve a value from the $\_POST array.
