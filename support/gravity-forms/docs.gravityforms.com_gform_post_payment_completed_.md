## Description

Triggers when a payment has been completed through the form.

## Usage

|     |
| --- |
| `add_action(``'gform_post_payment_completed'``,``'my_function'``, 10, 2 );` |

## Parameters

- **$entry** [Entry Object](https://docs.gravityforms.com/entry-object)

The entry object that was created.
- **$action** [array](https://docs.gravityforms.com/array)

The action that occurred. Contains things such as the amount, what occurred, and the transaction ID.

|     |
| --- |
| `$action``=``array``(`<br>```'type'``=>``''``,`<br>```'amount'``=>``''``,`<br>```'transaction_type'``=>``''``,`<br>```'transaction_id'``=>``''``,`<br>```'subscription_id'``=>``''``,`<br>```'entry_id'``=>``''``,`<br>```'payment_status'``=>``''``,`<br>```'note'``=>``''``,`<br>`);` |
