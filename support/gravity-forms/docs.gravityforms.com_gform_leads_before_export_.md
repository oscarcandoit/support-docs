# gform\_leads\_before\_export

- [Description](https://docs.gravityforms.com/gform_leads_before_export/#h-description)
- [Usage](https://docs.gravityforms.com/gform_leads_before_export/#h-usage)
- [Parameters](https://docs.gravityforms.com/gform_leads_before_export/#h-parameters)
- [Examples](https://docs.gravityforms.com/gform_leads_before_export/#h-examples)
- [Placement](https://docs.gravityforms.com/gform_leads_before_export/#h-placement)
- [Source Code](https://docs.gravityforms.com/gform_leads_before_export/#h-source-code)

## Description

Allows entries to be changed before export is executed.

## Usage

Apply to all forms.

|     |     |
| --- | --- |
| 1 | `add_filter(``'gform_leads_before_export'``,``'my_modify_entries_before_export_function'``, 10, 3 );` |

Apply to a specific form.

|     |     |
| --- | --- |
| 1 | `add_filter(``'gform_leads_before_export_{$form_id}'``,``'my_modify_entries_before_export_function'``, 10, 3 );` |

## Parameters

- **$entries** [array](https://docs.gravityforms.com/array)

An array of [entry objects](https://docs.gravityforms.com/entry-object) to be exported
- **$form** [array](https://docs.gravityforms.com/array)

The current [form object](https://docs.gravityforms.com/form-object/)
- **$paging** [array](https://docs.gravityforms.com/array)

The current paging for the export. Includes the following properties:

  - **offset** [integer](https://docs.gravityforms.com/integer)

    Increases by 1 with each “page” of entries the export has processed
  - **page\_size** [integer](https://docs.gravityforms.com/integer)

    Number of entries per page. Defaults to 20
