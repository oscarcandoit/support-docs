## Description

Modify update (and other non-error) messages displayed by Gravity Forms in the WordPress admin.

## Usage

|     |     |
| --- | --- |
| 1 | `add_action( &#039;gform_admin_messages&#039;, &#039;my_custom_function&#039; );` |

## Parameters

- **$messages** [array](https://docs.gravityforms.com/array)
An array of messages to be displayed below the title on Gravity Form pages.


## Examples

This example demonstrates how we can append a ‘Preview Form’ link to any update message where a form ID is available in the query string.

| --- | --- |
| 1<br>2<br>3<br>4<br>5<br>6<br>7<br>8<br>9<br>10<br>11<br>12<br>13<br>14<br>15<br>16<br>17<br>18 | `add_action( &#039;gform_admin_messages&#039;, &#039;my_append_preview_form_link&#039; );`<br>`function``my_append_preview_form_link(``$messages``) {`<br>`// get the form ID from the query string`<br>`$form_id``= rgget( &#039;id&#039; );`<br>`// if no form ID is available, let&#039;s not add the &#039;Preview Form&#039; link`<br>`if``( !``$form_id``) {`<br>`return``;`<br>`}`<br>`// loop through our error messages and append the &#039;Preview Form&#039; link`<br>`foreach``(``$messages``as``&amp;amp;``$message``) {`<br>`$message``.= &#039; &lt;a href=&quot;&#039; . site_url() . &#039;/?gf_page=preview&amp;amp;id=&#039; .``$form_id``. &#039;&quot;&gt;Preview Form&lt;/a&gt;&#039;;`<br>`}`<br>`return``$messages``;`<br>`}` |

## Placement

This code should be placed in the [functions.php](http://codex.wordpress.org/Theme_Development#Functions_File) file of your active theme.

## Source Code

This filter is located in GFCommon::display\_admin\_message() in _common.php_
