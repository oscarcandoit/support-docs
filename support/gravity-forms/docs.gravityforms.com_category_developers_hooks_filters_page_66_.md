[**gform\_turnstile\_enable\_no\_conflict** \\
Use this filter to enable Turnstile’s No-Conflict mode, which dequeues other CAPTCHA scripts to avoid conflicts.](https://docs.gravityforms.com/gform_turnstile_enable_no_conflict/ "gform_turnstile_enable_no_conflict") [**gform\_twilio\_message** \\
This filter is used to modify the SMS message arguments before it is sent.](https://docs.gravityforms.com/gform_twilio_message/ "gform_twilio_message") [**gform\_updates\_list** \\
Filters the available updates displayed on the Forms -> System Status -> Updates page.](https://docs.gravityforms.com/gform_updates_list/ "gform_updates_list") [**gform\_upload\_path** \\
This filter is executed before uploading a file (by the File Upload or Post Image fields). It can be used to change the location where uploaded files are stored.](https://docs.gravityforms.com/gform_upload_path/ "gform_upload_path") [**gform\_upload\_root\_htaccess\_rules** \\
A filter to allow to turn off or modify rules being added to the .htaccess file created for the base upload path. It runs daily during the regeneration of the .htaccess file.](https://docs.gravityforms.com/gform_upload_root_htaccess_rules/ "gform_upload_root_htaccess_rules") [**gform\_us\_states** \\
This filter can be used to modify the choices listed in the US states drop down.](https://docs.gravityforms.com/gform_us_states/ "gform_us_states") [**gform\_user\_registered** \\
This action is used to trigger an event once a user has been registered through the User Registration Add-on.](https://docs.gravityforms.com/gform_user_registered/ "gform_user_registered") [**gform\_user\_registration\_add\_option\_group** \\
