## Articles

[**gform\_user\_registration\_pending\_activation\_expiration** \\
Allows the expiration period for pending activations to be overridden.Note that expiration doesn't invalidate or delete the pending activation, expiration time only means the data submitted (username and email address) is no longer reserved for the user who submitted the form initially.](https://docs.gravityforms.com/gform_user_registration_pending_activation_expiration/ "gform_user_registration_pending_activation_expiration") [**gform\_user\_registration\_prepared\_value** \\
This filter can be used to modify a field value before it is saved to the user meta.](https://docs.gravityforms.com/gform_user_registration_prepared_value/ "gform_user_registration_prepared_value") [**gform\_user\_registration\_save\_config** \\
Used to save custom settings/meta specified using the gform\_user\_registration\_add\_option\_section or gform\_user\_registration\_add\_option\_group action hooks.](https://docs.gravityforms.com/gform_user_registration_save_config/ "gform_user_registration_save_config") [**gform\_user\_registration\_signup\_meta** \\
This filter allows the modification of the signup meta data.](https://docs.gravityforms.com/gform_user_registration_signup_meta/ "gform_user_registration_signup_meta") [**gform\_user\_registration\_update\_user\_id** \\
This filter can be used to override which user is used to populate the form and updated when the form is submitted.](https://docs.gravityforms.com/gform_user_registration_update_user_id/ "gform_user_registration_update_user_id") [**gform\_user\_registration\_user\_data\_pre\_populate** \\
This filter can be used to modify a value before it is populated into the field.](https://docs.gravityforms.com/gform_user_registration_user_data_pre_populate/ "gform_user_registration_user_data_pre_populate") [**gform\_user\_registration\_user\_meta\_options** \\
This filter allows the options for the “Other User Meta” group to be set instead of pulling the meta keys from the database.](https://docs.gravityforms.com/gform_user_registration_user_meta_options/ "gform_user_registration_user_meta_options") [**gform\_user\_registration\_username** \\
The gform\_user\_registration\_username filter has been deprecated. Use the gform\_username hook instead.](https://docs.gravityforms.com/gform_user_registration_username/ "gform_user_registration_username") [**gform\_user\_registration\_validation** \\
This filter is used to validate submissions that have a User Registration feed attached to the submitted form.](https://docs.gravityforms.com/gform_user_registration_validation/ "gform_user_registration_validation") [**gform\_user\_registration\_validation\_message** \\
