# Constants

## Articles

[**GFORM\_AUTO\_DB\_MIGRATION\_DISABLED** \\
Prevents entries from being migrated in the background when the database schema has changed.](https://docs.gravityforms.com/gform_auto_db_migration_disabled/ "GFORM_AUTO_DB_MIGRATION_DISABLED") [**GFORM\_DB\_MIGRATION\_BATCH\_SIZE** \\
Allows the batch size to be changed when upgrading the database from a version older than 2.3.](https://docs.gravityforms.com/gform_db_migration_batch_size/ "GFORM_DB_MIGRATION_BATCH_SIZE") [**GFORM\_DISABLE\_AUTO\_UPDATE** \\
This Gravity Forms constant provides on/off control of automatic updates. It will disable them if set with a boolean value of ‘True’.](https://docs.gravityforms.com/gform_disable_auto_update-constant/ "GFORM_DISABLE_AUTO_UPDATE") [**IS\_ADMIN** \\
Indicates if the current page is a WordPress admin page.](https://docs.gravityforms.com/is_admin/ "IS_ADMIN") [**RG\_CURRENT\_PAGE** \\
The current page.](https://docs.gravityforms.com/rg_current_page/ "RG_CURRENT_PAGE") [**RG\_CURRENT\_VIEW** \\
The RG\_CURRENT\_VIEW constant in Gravity Forms defines the current view and is used in legacy versions of Gravity Forms before the database schema change in version 2.3.](https://docs.gravityforms.com/rg_current_view/ "RG_CURRENT_VIEW") [Previous](https://docs.gravityforms.com/category/developers/php-api/constants/) [1](https://docs.gravityforms.com/category/developers/php-api/constants/) 2
