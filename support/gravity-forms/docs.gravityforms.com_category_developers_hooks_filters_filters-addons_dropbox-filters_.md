# Dropbox Filters

## Articles

[**Dropbox Feed Meta** \\
The Feed Meta object for the Dropbox Add-On is an associative array containing the properties which determine how the add-on should process the form submission.](https://docs.gravityforms.com/dropbox-feed-meta/ "Dropbox Feed Meta") [**gform\_dropbox\_file\_name** \\
This filter can be used to override the filename before the file is uploaded to Dropbox.](https://docs.gravityforms.com/gform_dropbox_file_name/ "gform_dropbox_file_name") [**gform\_dropbox\_folder\_path** \\
This filter can be used to override the destination folder configured on the Dropbox feed.](https://docs.gravityforms.com/gform_dropbox_folder_path/ "gform_dropbox_folder_path") [**gform\_dropbox\_link\_type** \\
This filter can be used to change the type of link returned by Dropbox.](https://docs.gravityforms.com/gform_dropbox_link_type/ "gform_dropbox_link_type") [**gform\_dropbox\_post\_upload** \\
This hook is fired after a Dropbox feed has been processed and files have been uploaded.](https://docs.gravityforms.com/gform_dropbox_post_upload/ "gform_dropbox_post_upload") [**gform\_dropbox\_request\_timeout** \\
This filter in the Gravity Forms Dropbox Add-On allows the request timeout length for the feed processing request to be modified.](https://docs.gravityforms.com/gform_dropbox_request_timeout/ "gform_dropbox_request_timeout") [**gform\_dropbox\_shareable\_link\_settings** \\
This filter can be used to modify the settings before a Dropbox shareable link is generated, allowing for the link to expire after a certain time or change who is allowed to access it.](https://docs.gravityforms.com/gform_dropbox_shareable_link_settings/ "gform_dropbox_shareable_link_settings") [**gform\_dropbox\_should\_upload\_file** \\
The gform\_dropbox\_should\_upload\_file filter can be used to determine if a file for a fileupload or dropbox type field is uploaded to Dropbox.](https://docs.gravityforms.com/gform_dropbox_should_upload_file/ "gform_dropbox_should_upload_file") [**gform\_dropbox\_ssl\_compatibility** \\
The filter hook allows the SSL compatibility checks within the Dropbox add-on to be overridden.](https://docs.gravityforms.com/gform_dropbox_ssl_compatibility/ "gform_dropbox_ssl_compatibility") [**gform\_dropbox\_store\_local\_version** \\
This filter can be used to prevent a local copy of the file being retained once it has been uploaded to Dropbox.](https://docs.gravityforms.com/gform_dropbox_store_local_version/ "gform_dropbox_store_local_version") 1 [2](https://docs.gravityforms.com/category/developers/hooks/filters/filters-addons/dropbox-filters/page/2/) [Next](https://docs.gravityforms.com/category/developers/hooks/filters/filters-addons/dropbox-filters/page/2/)
