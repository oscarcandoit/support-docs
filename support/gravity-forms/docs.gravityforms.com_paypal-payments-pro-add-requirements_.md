# PayPal Payments Pro Add-On Requirements

- [Requirements](https://docs.gravityforms.com/paypal-payments-pro-add-requirements/#h-requirements)

PayPal has announced that from September 2024, certain [legacy APIs](https://developer.paypal.com/api/rest/deprecated-resources/) they provide will stop receiving support and customers using them may experience a loss of functionality. If you are using any of our Gravity Forms legacy PayPal Add-Ons ( [PayPal Payments Pro Add-On](https://docs.gravityforms.com/category/add-ons-gravity-forms/paypal-add-ons/paypa-legacy-add-ons/paypal-payments-pro-add-on/), [PayPal Payments Standard Add-On](https://docs.gravityforms.com/category/add-ons-gravity-forms/paypal-add-ons/paypa-legacy-add-ons/paypal-payments-standard-add-on/), [PayPal Pro Add-On](https://docs.gravityforms.com/category/add-ons-gravity-forms/paypal-add-ons/paypa-legacy-add-ons/paypal-pro-add-on/)), we recommend you evaluate the feasibility of migrating to the PayPal Checkout Add-On to avoid possible disruption of services.

There are various things required when using the PayPal Payments Pro Add-On. The biggest of these being a valid SSL certificate installed and configured on your server. If you’re not sure about this, contact your web host for more information.

## Requirements

- Gravity Forms v1.9.16+
- Latest version of WordPress
- SSL Certificate Installed and Configured
- PCI Compliance
- [PayPal Payments Pro account](https://www.paypal.com/webapps/mpp/paypal-payments-pro) with **PayFlow API** support.
