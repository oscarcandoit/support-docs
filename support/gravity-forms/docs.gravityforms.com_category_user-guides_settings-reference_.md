# Settings Reference

## Articles

[**Form Settings** \\
Explore the available form settings for Gravity Forms.](https://docs.gravityforms.com/form-settings/ "Form Settings") [**Gravity Forms Block Settings** \\
This article details the settings available for the Gravity Forms block once it has been inserted into your WP page or post using the block editor.](https://docs.gravityforms.com/gravity-forms-block-settings/ "Gravity Forms Block Settings") [**Personal Data Settings** \\
Added in Gravity Forms version 2.4, the Personal Data tab of the Form Settings provides integration with the WordPress Export Personal Data and Erase Personal Data tools. Data for both entries and draft submissions (save & continue) are supported.](https://docs.gravityforms.com/personal-data-settings/ "Personal Data Settings") [**Role and Capability Management** \\
Gravity Forms introduces new capabilities to the WordPress Role system. These capabilities are only manageable via a plugin that makes use of the WordPress Role Management system.](https://docs.gravityforms.com/role-management-guide/ "Role and Capability Management") [**Save and Continue** \\
Save and Continue functionality is a critical feature that many individuals rely on when building large forms. By utilizing Save and Continue functionality, users are able to save their place and come back to the form without the need to start over again.](https://docs.gravityforms.com/save-continue-gravity-forms/ "Save and Continue") [**Submit Button Settings** \\
The Submit Button allows a user to trigger submission of the form, and its settings are available under the Field Settings tab.](https://docs.gravityforms.com/submit-button/ "Submit Button Settings") [**WP Config Options** \\
Configuration options for Gravity Forms within WordPress.](https://docs.gravityforms.com/wp-config-options/ "WP Config Options")
