# Admin Dashboard Filters

## Articles

[**gform\_admin\_error\_messages** \\
Modify error messages displayed by Gravity Forms in the WordPress admin.](https://docs.gravityforms.com/gform_admin_error_messages/ "gform_admin_error_messages") [**gform\_admin\_messages** \\
Modify update (and other non-error) messages displayed by Gravity Forms in the WordPress admin.](https://docs.gravityforms.com/gform_admin_messages/ "gform_admin_messages") [**gform\_bypass\_template\_library** \\
The gform\_bypass\_template\_library filter can be used to bypass the template library when creating new forms.](https://docs.gravityforms.com/gform_bypass_template_library/ "gform_bypass_template_library") [**gform\_dashboard\_title** \\
This filter allows the title of the forms statistics section displayed on the Dashboard to be changed.](https://docs.gravityforms.com/gform_dashboard_title/ "gform_dashboard_title") [**gform\_disable\_installation\_status** \\
This filter is used to disable the display of the Installation Status section on the Forms > Settings page.](https://docs.gravityforms.com/gform_disable_installation_status/ "gform_disable_installation_status") [**gform\_display\_disable\_css\_setting** \\
The filter gform\_display\_disable\_css\_setting indicates if the Output Default CSS setting should be displayed or not.](https://docs.gravityforms.com/gform_display_disable_css_setting/ "gform_display_disable_css_setting") [**gform\_form\_summary** \\
Filters through and allows modification of the forms displayed in the Dashboard Forms widget.](https://docs.gravityforms.com/gform_form_summary/ "gform_form_summary") [**gform\_form\_switcher\_forms** \\
Filters through and allows modification of the forms displayed in the Form Switcher dropdown.](https://docs.gravityforms.com/gform_form_switcher_forms/ "gform_form_switcher_forms") [**gform\_has\_admin\_notices** \\
The gform\_has\_admin\_notices filter is used to determine whether or not to display the “WordPress has new notifications.” notice at the top of Gravity Forms admin pages.](https://docs.gravityforms.com/gform_has_admin_notices/ "gform_has_admin_notices") [**gform\_menu\_position** \\
Modify the position the Gravity Forms menu should appear in the WordPress admin menu.](https://docs.gravityforms.com/gform_menu_position/ "gform_menu_position") 1 [2](https://docs.gravityforms.com/category/developers/hooks/filters/admin-dashboard-filters/page/2/) [Next](https://docs.gravityforms.com/category/developers/hooks/filters/admin-dashboard-filters/page/2/)
