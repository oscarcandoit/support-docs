[**gform\_mailchimp\_override\_empty\_fields** \\

This filter allows the request arguments to be modified before sent to Mailchimp.](https://docs.gravityforms.com/gform_mailchimp_request_args/ "gform_mailchimp_request_args") [**gform\_mailchimp\_subscription** \\
This filter modifies the subscription object before it is executed.](https://docs.gravityforms.com/gform_mailchimp_subscription/ "gform_mailchimp_subscription") [**gform\_mailerlite\_subscriber** \\
Allows the subscriber properties to be overridden before the upsert request is made.](https://docs.gravityforms.com/gform_mailerlite_subscriber/ "gform_mailerlite_subscriber") [**gform\_mailgun\_email** \\
Allows modification of the email arguments which are passed to the Mailgun API.](https://docs.gravityforms.com/gform_mailgun_email/ "gform_mailgun_email") [**gform\_mailgun\_send\_email\_failed** \\
The gform\_mailgun\_send\_email\_failed action hook is triggered when MailGun API reports it is unable to send the email.](https://docs.gravityforms.com/gform_mailgun_send_email_failed/ "gform_mailgun_send_email_failed") [**gform\_max\_async\_feed\_attempts** \\
