## Articles

[**gform\_quiz\_show\_choice\_values** \\
This filter controls whether the values for the answer choices may be modified in the Form Editor.](https://docs.gravityforms.com/gform_quiz_show_choice_values/ "gform_quiz_show_choice_values") [**gform\_recaptcha\_callback** \\
The gform\_recaptcha\_callback filter in allows a custom callback function to be executed when the user successfully submits the captcha](https://docs.gravityforms.com/gform_recaptcha_callback/ "gform_recaptcha_callback") [**gform\_recaptcha\_enterprise\_keys\_page\_size** \\
ReCAPTCHA’s Enterprise API can only retrieve 100 ReCAPTCHA keys at a time. The gform\_recaptcha\_enterprise\_keys\_page\_size filter allows developers to set the pageSize to something smaller.](https://docs.gravityforms.com/gform_recaptcha_enterprise_keys_page_size/ "gform_recaptcha_enterprise_keys_page_size") [**gform\_recaptcha\_init\_script** \\
This filter is no longer available since Gravity Forms 2.0.](https://docs.gravityforms.com/gform_recaptcha_init_script/ "gform_recaptcha_init_script") [**gform\_recaptcha\_language**\\
This filter has been removed. The recaptcha language can be configured in the form editor by editing the field or by using the gform\_pre\_render filter to set the $field->captchaLanguage property.](https://docs.gravityforms.com/gform_recaptcha_language/ "gform_recaptcha_language") [**gform\_recaptcha\_spam\_score\_threshold** \\
This gform\_recaptcha\_spam\_score\_threshold filter allows control over the spam score threshold used by the reCAPTCHA Add-On.](https://docs.gravityforms.com/gform_recaptcha_spam_score_threshold/ "gform_recaptcha_spam_score_threshold") [**gform\_recaptcha\_valid\_hostnames** \\
The filter gform\_recaptcha\_valid\_hostnames allows developers to define the set of hostnames considered valid for reCAPTCHA verification.](https://docs.gravityforms.com/gform_recaptcha_valid_hostnames/ "gform_recaptcha_valid_hostnames") [**gform\_register\_init\_scripts** \\
Tap into Gravity Forms “initialization scripts” functionality and register your own scripts to run inline with Gravity Forms.](https://docs.gravityforms.com/gform_register_init_scripts/ "gform_register_init_scripts") [**gform\_repeater\_item\_pre\_add** \\
