# Styling/Theme/CSS Filters

## Articles

[**gform\_default\_styles** \\
The gform\_default\_styles filter can be used to filter the global form theme styles.](https://docs.gravityforms.com/gform_default_styles/ "gform_default_styles") [**gform\_disable\_css** \\
The filter gform\_disable\_css can be used to disable all CSS files from being loaded on the Front End.](https://docs.gravityforms.com/gform_disable_css/ "gform_disable_css") [**gform\_disable\_form\_legacy\_css** \\
The gform\_disable\_form\_legacy\_css filter allows the CSS to be disabled for forms using the legacy markup, usually forms created with Gravity Forms 2.4 and earlier.](https://docs.gravityforms.com/gform_disable_form_legacy_css/ "gform_disable_form_legacy_css") [**gform\_disable\_form\_theme\_css** \\
The gform\_disable\_form\_theme\_css filter allows the default form theme used by forms created with Gravity Forms 2.5 and greater to be disabled.](https://docs.gravityforms.com/gform_disable_form_theme_css/ "gform_disable_form_theme_css") [**gform\_field\_css\_class** \\
This filter can be used to dynamically add/remove CSS classes to a field.](https://docs.gravityforms.com/gform_field_css_class/ "gform_field_css_class") [**gform\_form\_theme\_slug** \\
The gform\_form\_theme\_slug filter can be used to filter the form theme.](https://docs.gravityforms.com/gform_form_theme_slug/ "gform_form_theme_slug") [**gform\_noconflict\_styles** \\
Use this filter to “register” your css file with Gravity Forms, making sure it gets enqueued in the form editor page when no-conflict mode is enabled.](https://docs.gravityforms.com/gform_noconflict_styles/ "gform_noconflict_styles") [**gform\_preview\_styles** \\
Use this filter to specify styles to be enqueued on Gravity Forms preview pages.](https://docs.gravityforms.com/gform_preview_styles/ "gform_preview_styles") [**gform\_print\_styles** \\
Use this filter to add custom stylesheets to the print entry screen.](https://docs.gravityforms.com/gform_print_styles/ "gform_print_styles")
