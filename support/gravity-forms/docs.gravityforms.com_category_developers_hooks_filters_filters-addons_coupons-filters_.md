# Coupons Filters

## Articles

[**Coupons Feed Meta** \\
The Feed Meta object for the Coupons add-On is an associative array containing the properties which determine the type of coupon and discount available.](https://docs.gravityforms.com/coupons-feed-meta/ "Coupons Feed Meta") [**GFCoupons** \\
GFCoupons is the class which houses the main functionality of the Gravity Forms Coupons Add-on; it extends the GFFeedAddOn class which is part of the add-on framework.](https://docs.gravityforms.com/gfcoupons/ "GFCoupons") [**gform\_coupons\_can\_apply\_coupon** \\
This PHP filter allows custom logic to be used to determine if the coupon code can be applied.](https://docs.gravityforms.com/gform_coupons_can_apply_coupon/ "gform_coupons_can_apply_coupon") [**gform\_coupons\_discount\_amount** \\
This filter can be used to modify the coupon discount before it is applied to the form total.](https://docs.gravityforms.com/gform_coupons_discount_amount/ "gform_coupons_discount_amount") [**gform\_coupons\_is\_valid\_code** \\
Overrides coupon code alphanumeric checking.](https://docs.gravityforms.com/gform_coupons_is_valid_code/ "gform_coupons_is_valid_code") [**gform\_coupons\_post\_delete\_coupon** \\
This JavaScript action fires after a coupon is deleted from a field, allowing further actions to be performed.](https://docs.gravityforms.com/gform_coupons_post_delete_coupon/ "gform_coupons_post_delete_coupon")
