## Description

This filter can be used to modify a value before it is sent to Zapier.

## Usage

|     |     |
| --- | --- |
| 1 | `add_filter(``'gform_zapier_field_value'``,``'your_function_name'``, 10, 4 );` |

## Parameters

- **$value** [string](https://docs.gravityforms.com/string/)

The value to be modified.
- **$form\_id** [integer](https://docs.gravityforms.com/integer/)

The ID of the form being processed.
- **$field\_id** [string](https://docs.gravityforms.com/string/)

The ID of the field being processed.
- **$entry** [Entry Object](https://docs.gravityforms.com/entry-object/)

The entry currently being processed.

## Examples

### 1\. Use GF\_Field::get\_value\_export

This example shows how you can have the Zapier add-on use GF\_Field::get\_value\_export() to format the field value for export. **Requires a minimum of Gravity Forms 1.9.12.4.**

| 1<br>2<br>3<br>4<br>5<br>6<br>7<br>8<br>9<br>10 | `add_filter(``'gform_zapier_field_value'``,``'format_entry_value'``, 10, 4 );`<br>`function``format_entry_value(``$value``,``$form_id``,``$field_id``,``$entry``) {`<br>```$field``= GFAPI::get_field(``$form_id``,``$field_id``);`<br>```if``(``is_object``(``$field``) ) {`<br>```$value``=``$field``->get_value_export(``$entry``,``$field_id``,``$use_choice_text``= true );`<br>```}`<br>```return``$value``;`<br>`}` |
