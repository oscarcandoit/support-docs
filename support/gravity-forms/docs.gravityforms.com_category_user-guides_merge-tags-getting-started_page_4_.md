# Merge Tags

## Articles

[**{user\_agent} Merge Tag** \\
Displays the browser and platform information of the machine from which the entry was submitted.](https://docs.gravityforms.com/user_agent-merge-tag/ "{user_agent} Merge Tag") [**{user} Merge Tag** \\
Displays the value of the user meta field for the logged in user, based on the modifier used.](https://docs.gravityforms.com/user-merge-tag/ "{user} Merge Tag") [**Conditional Shortcode** \\
The conditional shortcode allows you to perform conditional logic in the message content of your Admin and User Notification emails as well as the Confirmation Message that is displayed when a form is submitted.](https://docs.gravityforms.com/conditional-shortcode/ "Conditional Shortcode") [**Custom Field Merge Tags** \\
Display a custom field value from the post/page the form was submitted from. These fields could be added through WordPress Custom Fields, through register\_meta, or through a plugin like ACF, Pods or MetaBox.](https://docs.gravityforms.com/custom-field-merge-tags/ "Custom Field Merge Tags") [**Entry Creator Merge Tag** \\
The entry creator merge tag can be used in notifications, confirmations, and with some add-ons to output properties of the user who was logged in at the time the entry was saved.](https://docs.gravityforms.com/entry-creator-merge-tag/ "Entry Creator Merge Tag") [**Entry Date Merge Tags** \\
The entry date merge tags can be used in notifications, confirmations, and with some add-ons to output the date and/or time the entry was created, updated, and paid.](https://docs.gravityforms.com/entry-date-merge-tags/ "Entry Date Merge Tags") [**Field Merge Tags** \\
Displays the value submitted for the field with the specified field\_id. If no value was submitted, the merge tag will not display anything.](https://docs.gravityforms.com/field-merge-tags/ "Field Merge Tags") [**Merge Tags Reference** \\
Gravity Forms uses merge tags to dynamically insert submitted field values and other relevant information into notifications, confirmations, post content templates, and more. This article offers a quick reference guide and links to detailed documentation for each tag.](https://docs.gravityforms.com/merge-tags-reference/ "Merge Tags Reference") [**Post Image Merge Tags** \\
Will display either the uploaded post image or metadata about the image.](https://docs.gravityforms.com/post-image-merge-tags/ "Post Image Merge Tags") [Previous](https://docs.gravityforms.com/category/user-guides/merge-tags-getting-started/page/3/) [1](https://docs.gravityforms.com/category/user-guides/merge-tags-getting-started/) [2](https://docs.gravityforms.com/category/user-guides/merge-tags-getting-started/page/2/) [3](https://docs.gravityforms.com/category/user-guides/merge-tags-getting-started/page/3/) 4
