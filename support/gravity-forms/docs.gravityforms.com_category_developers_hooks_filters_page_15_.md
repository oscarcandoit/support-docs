[**gform\_creditcard\_types** \\
Use this filter to change the default list of supported credit card types.](https://docs.gravityforms.com/gform_creditcard_types/ "gform_creditcard_types") [**gform\_currencies** \\
The gform\_currencies filter can be used to update existing currencies and/or create new currencies.](https://docs.gravityforms.com/gform_currencies/ "gform_currencies") [**gform\_currency** \\
Use this filter to override the currency configured on the Forms > Settings page. It will affect all forms.](https://docs.gravityforms.com/gform_currency/ "gform_currency") [**gform\_currency\_disabled** \\
Use this filter to enable/disable the currency drop down on the settings page.](https://docs.gravityforms.com/gform_currency_disabled/ "gform_currency_disabled") [**gform\_currency\_pre\_save\_entry** \\
This filter can be used to override the default currency code before the entry is saved.](https://docs.gravityforms.com/gform_currency_pre_save_entry/ "gform_currency_pre_save_entry") [**gform\_custom\_merge\_tags** \\
The gform\_custom\_merge\_tags filter allows you to add additional merge tags that you have defined to the drop down menus where merge tags are available.](https://docs.gravityforms.com/gform_custom_merge_tags/ "gform_custom_merge_tags") [**gform\_dashboard\_title** \\
This filter allows the title of the forms statistics section displayed on the Dashboard to be changed.](https://docs.gravityforms.com/gform_dashboard_title/ "gform_dashboard_title") [**gform\_date\_max\_year** \\
Use this filter to specify the maximum year displayed in the date field’s year drop down and the HTML5 max attribute for the date field’s year input.](https://docs.gravityforms.com/gform_date_max_year/ "gform_date_max_year") [**gform\_date\_min\_year** \\
