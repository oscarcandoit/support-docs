## Articles

[**gform\_form\_settings\_initial\_values** \\
The gform\_form\_settings\_initial\_values filter allows the initial values that will be populated into the form settings to be overridden.](https://docs.gravityforms.com/gform_form_settings_initial_values/ "gform_form_settings_initial_values") [**gform\_form\_settings\_menu** \\
The gform\_form\_settings\_menu filter can be used to add new or modify default menu items which appear in the form settings section menu.](https://docs.gravityforms.com/gform_form_settings_menu/ "gform_form_settings_menu") [**gform\_form\_settings\_page\_gravityformszapier** \\
The “gform\_form\_settings\_page\_gravityformszapier” action in the Gravity Forms Zapier Add-On is a hook name that is dynamically generated.](https://docs.gravityforms.com/gform_form_settings_page_gravityformszapier/ "gform_form_settings_page_gravityformszapier") [**gform\_form\_settings\_page\_VIEW** \\
Adds custom pages (i.e. “views”) to the Form Settings section.](https://docs.gravityforms.com/gform_form_settings_page_view/ "gform_form_settings_page_VIEW") [**gform\_form\_summary** \\
Filters through and allows modification of the forms displayed in the Dashboard Forms widget.](https://docs.gravityforms.com/gform_form_summary/ "gform_form_summary") [**gform\_form\_switcher\_forms** \\
Filters through and allows modification of the forms displayed in the Form Switcher dropdown.](https://docs.gravityforms.com/gform_form_switcher_forms/ "gform_form_switcher_forms") [**gform\_form\_tag** \\
This filter is executed when the form is displayed and can be used to completely change the form tag (i.e. <form method="post">).](https://docs.gravityforms.com/gform_form_tag/ "gform_form_tag") [**gform\_form\_theme\_slug** \\
The gform\_form\_theme\_slug filter can be used to filter the form theme.](https://docs.gravityforms.com/gform_form_theme_slug/ "gform_form_theme_slug") [**gform\_form\_trash\_link** \\
This filter was removed in Gravity Forms v2.5 with no replacement.](https://docs.gravityforms.com/gform_form_trash_link/ "gform_form_trash_link") [**gform\_form\_update\_meta** \\
