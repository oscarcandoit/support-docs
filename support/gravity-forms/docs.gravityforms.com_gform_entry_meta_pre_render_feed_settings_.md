## Description

Allows the JavaScript entry\_meta variable to be populated on the feed configuration page.

## Usage

Applies to all forms:

|     |
| --- |
| `add_filter(``'gform_entry_meta_pre_render_feed_settings'``,``'your_function_name'``, 10, 3 );` |

## Parameters

- **$entry\_meta** [array](https://docs.gravityforms.com/array)

An empty array or an array of entry meta fields to be assigned to the JavaScript entry\_meta variable.
- **$form** [Form Object](https://docs.gravityforms.com/array/)

The form the feed is being created or edited for.
- **$addon** [Object](https://docs.gravityforms.com/array/)

The current add-on instance.
