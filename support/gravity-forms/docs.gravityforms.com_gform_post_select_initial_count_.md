## Description

The `gform_post_select_initial_count` filter in Gravity Forms allows setting the default number of posts to retrieve when the [Settings API](https://docs.gravityforms.com/category/developers/php-api/add-on-framework/settings-api/) renders a field of type `post_select`.

## Usage

```
add_filter( 'gform_post_select_initial_count', 'your_function_name' );
```

## Parameters

- **$count** [integer](https://docs.gravityforms.com/integer)

The number of posts to retrieve. Default value is 5.

## Examples

This example sets the initial number of posts to 10.

```
add_filter( 'gform_post_select_initial_count', function( $count ) {
    return 10;
} );
```

## Placement

This code can be used in the [functions.php](https://developer.wordpress.org/themes/basics/theme-functions/) file of the active theme, a custom functions plugin, a custom add-on, or with a code snippets plugin.

See also the PHP section in this article: [Where Do I Put This Code?](https://docs.gravityforms.com/where-do-i-put-this-code/)

## Since

This filter was added in Gravity Forms version 2.6.2.

## Source Code

This filter is located in GF\_Settings\_Config\_Admin::data() in _includes/settings/config/class-gf-settings-config-admin.php_.
