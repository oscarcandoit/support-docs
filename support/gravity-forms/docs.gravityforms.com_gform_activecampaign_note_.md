## Description

Filter the note that will be created and associated with the given contact. Supports filtering which contact and/or list the note should be associated with.

## Usage

The following would apply to all forms.

|     |     |
| --- | --- |
| 1 | `add_filter(``'gform_activecampaign_note'``,``'your_function_name'``, 10, 3 );` |

The following would apply to a specific form.

|     |     |
| --- | --- |
| 1 | `add_filter(``'gform_activecampaign_note_FORMID'``,``'your_function_name'``, 10, 3 );` |

## Parameters

- **$note** [array](https://docs.gravityforms.com/array)
Properties that will be used to create and assign the note.

  - **$contact\_id** [int](https://docs.gravityforms.com/int)
    Contact ID to associate the note with.

  - **$list\_id** [int](https://docs.gravityforms.com/int)

    List ID to associate the note with.

  - **$note** [string](https://docs.gravityforms.com/string)

    Actual note content. HTML will be stripped.
- **$feed** [array](https://docs.gravityforms.com/array)

Current Active Campaign feed being processed.

- **$entry** [array](https://docs.gravityforms.com/array)

Current entry being processed.

- **$form** [array](https://docs.gravityforms.com/array)

Current form being processed.


## Examples

### 1\. Customize the note.

The following example shows how you can customize the note.

|     |     |
| --- | --- |
| 1<br>2<br>3<br>4 | `add_filter(``'gform_activecampaign_note'``,``function``(``$note``) {`<br>```$note``[``'note'``] =``'My custom note.'``;`<br>```return``$note``;`<br>`} );` |

## Placement

This code should be placed in the [functions.php](http://codex.wordpress.org/Theme_Development#Functions_File) file of your active theme or a custom functions plugin.

## Since

This filter was added in Gravity Forms Active Campaign 1.6.

## Source Code

This filter is located in GFActiveCampaign::process\_feed() in _includes/class-gf-activecampaign.php_.
