## Introduction

The [Feed Meta object](https://docs.gravityforms.com/feed-object) for the Zoho CRM Add-On is an associative array containing the properties which determine how the add-on should process the form submission.

|     |     |
| --- | --- |
| 1<br>2<br>3<br>4<br>5<br>6<br>7<br>8 | `$feed``[``'meta'``] =``array``(`<br>```'feedName'``=>``'Zoho CRM Feed 1'``,`<br>```'message'``=>``''``,`<br>```'feed_condition_conditional_logic'``=> true,`<br>```'feed_condition_conditional_logic_object'``=>``array``(`<br>```'conditionalLogic'``=>``array``(),`<br>```),`<br>`);` |

## Usage

We recommend accessing the $feed meta using the rgar() or rgars() functions, e.g.:

|     |     |
| --- | --- |
| 1<br>2<br>3<br>4<br>5<br>6<br>7<br>8<br>9<br>10<br>11<br>12<br>13 | `$conditional_logic_enabled``= rgars(``$feed``,``'meta/feedCondition'``);`<br>`if``(``$conditional_logic_enabled``) {`<br>```// Conditional logic is enabled for this feed.`<br>```$logic_rules``= rgars(``$feed``,``'meta/feed_condition_conditional_logic_object/conditionalLogic/rules'``);`<br>```foreach``(``$logic_rules``as``$rule``) {`<br>```// Process each conditional logic rule.`<br>```$field_id``= rgar(``$rule``,``'fieldId'``);`<br>```$operator``= rgar(``$rule``,``'operator'``);`<br>```$value``= rgar(``$rule``,``'value'``);`<br>```// Example: Use $field_id, $operator, and $value in your logic.`<br>```}`<br>`}` |

## Properties

- **feedName** [string](https://docs.gravityforms.com/string)
The feed name which appears on the add-ons feeds tab.

- **action** [string](https://docs.gravityforms.com/string)

The action which will occur when this feed is processed. _Possible values: contact or lead._

- **feed\_condition\_conditional\_logic** [boolean](https://docs.gravityforms.com/boolean)

Is the feed condition (conditional logic) setting enabled. Default is false.

- **feed\_condition\_conditional\_logic\_object** [array](https://docs.gravityforms.com/array)

An associative array containing the conditional logic rules. See the [Conditional Logic Object](https://docs.gravityforms.com/conditional-logic-object) for more details.


## Contact Properties

The following properties are only applicable when the **action** is set to **contact**.

- **contactStandardFields\_Email** [string](https://docs.gravityforms.com/string)
The ID of the form field containing the contact’s email.

- **contactStandardFields\_First\_Name** [string](https://docs.gravityforms.com/string)

The ID of the form field containing the contact’s first name.

- **contactStandardFields\_Last\_Name** [string](https://docs.gravityforms.com/string)

The ID of the form field containing the contact’s last name.

- **contactCustomFields** [array](https://docs.gravityforms.com/array)

A multidimensional array containing the contact custom fields. See [Custom Field Properties](https://docs.gravityforms.com/zoho-crm-feed-meta/#custom-field-properties).

- **contactOwner** [string](https://docs.gravityforms.com/string)

The Zoho CRM user ID to be assigned to this contact.

- **contactLeadSource** [string](https://docs.gravityforms.com/string)

The source from which the contact is created. _Default values: Advertisement, Cold Call, Employee Referral, External Referral, OnlineStore, Partner, Public Relations, Sales Mail Alias, Seminar Partner, Seminar-Internal, Trade Show, Web Download, Web Research, and Chat._

- **contactDescription** [string](https://docs.gravityforms.com/string)

The contact description. Merge tags supported.

- **contactAttachments** [array](https://docs.gravityforms.com/array)


|     |     |
| --- | --- |
| 1<br>2<br>3<br>4 | `array``(`<br>```'10'``=> true,`<br>```'11'``=> false,`<br>`);` |

An array of file upload fields. The field ID is the key with a boolean value indicating if the files for that field should be attached.

- **contactApprovalMode** [boolean](https://docs.gravityforms.com/boolean)

Is approval mode enabled? Default is false.
- **contactWorkflowMode** [boolean](https://docs.gravityforms.com/boolean)

Should workflow rules be triggered when this contact is added? Default is false.
- **contactEmailOptOut** [boolean](https://docs.gravityforms.com/boolean)

Remove contact from your mailing list so that they will not receive any emails from your Zoho CRM account. Default is false.
- **contactUpdate** [boolean](https://docs.gravityforms.com/boolean)

Update contact if existing? Default is false.

## Lead Properties

The following properties are only applicable when the **action** is set to **lead**.

- **leadStandardFields\_Company** [string](https://docs.gravityforms.com/string)


The ID of the form field containing the lead’s company.

- **leadStandardFields\_Email** [string](https://docs.gravityforms.com/string) The ID of the form field containing the lead’s email.

- **leadStandardFields\_First\_Name** [string](https://docs.gravityforms.com/string) The ID of the form field containing the lead’s first name.

- **leadStandardFields\_Last\_Name** [string](https://docs.gravityforms.com/string) The ID of the form field containing the lead’s last name.

- **leadCustomFields** [array](https://docs.gravityforms.com/array) A multidimensional array containing the lead custom fields. See [Custom Field Properties](https://docs.gravityforms.com/zoho-crm-feed-meta/#custom-field-properties).

- **leadOwner** [string](https://docs.gravityforms.com/string) The Zoho CRM user ID to be assigned to this lead.

- **leadRating** [string](https://docs.gravityforms.com/string) The lead rating. _Default values: Acquired, Active, Market Failed, Project Cancelled, and ShutDown._

- **leadSource** [string](https://docs.gravityforms.com/string) The source from which the lead is generated. _Default values: Advertisement, Cold Call, Employee Referral, External Referral, OnlineStore, Partner, Public Relations, Sales Mail Alias, Seminar Partner, Seminar-Internal, Trade Show, Web Download, Web Research, and Chat._

- **leadStatus** [string](https://docs.gravityforms.com/string) The lead status. _Default values: Attempted to Contact, Contact in Future, Contacted, Junk Lead, Lost Lead, Not Contacted, and Pre Qualified._

- **leadDescription** [string](https://docs.gravityforms.com/string) The lead description. Merge tags supported.

- **leadAttachments** [array](https://docs.gravityforms.com/array)

- **leadApprovalMode** [boolean](https://docs.gravityforms.com/boolean)

Is approval mode enabled? Default is false.
- **leadWorkflowMode** [boolean](https://docs.gravityforms.com/boolean)

Should workflow rules be triggered when this contact is added? Default is false.
- **leadEmailOptOut** [boolean](https://docs.gravityforms.com/boolean)

- **leadUpdate** [boolean](https://docs.gravityforms.com/boolean)

Update lead if existing? Default is false.

## Task Properties

- **createTask** [boolean](https://docs.gravityforms.com/boolean) Should a task be created? Default is false.

- **taskSubject** [string](https://docs.gravityforms.com/string) The task subject. Merge tags supported.

- **taskDueDate** [integer](https://docs.gravityforms.com/integer) The number of days until the task is due.

- **taskOwner** [string](https://docs.gravityforms.com/string) The Zoho CRM user ID to be assigned to this contact.

- **taskStatus** [string](https://docs.gravityforms.com/string) The task status. _Default values: Not Started, Deferred, In Progress, Completed, and Waiting on someone else._

The ID of the form field or entry meta item containing the value for this custom field.

- **custom\_key** [string](https://docs.gravityforms.com/string) The custom key. Only used when the custom field **key** is set to _gf\_custom_.
