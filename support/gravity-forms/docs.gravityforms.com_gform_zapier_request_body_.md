## Description

Allows the request body sent to Zapier to be filtered.

## Usage

The following would apply to all forms:

|     |     |
| --- | --- |
| 1 | `add_filter(``'gform_zapier_request_body'``,``'your_function_name'``, 10, 4);` |

To target a specific form, append the form id to the hook name.

Format: **_gform\_zapier\_request\_body\_FORMID_**

|     |     |
| --- | --- |
| 1 | `add_filter(``'gform_zapier_request_body_1'``,``'your_function_name'``, 10, 4);` |

## Parameters

| Name | Type | Note |
| --- | --- | --- |
| $body | [array](https://docs.gravityforms.com/array) | An associative array containing the request body that will be sent to Zapier. |
| $feed | [Feed Object](https://docs.gravityforms.com/feed-object) | The feed object currently being processed. |
| $entry | [Entry Object](https://docs.gravityforms.com/entry-object) | The entry object currently being processed. |
| $form | [Form Object](https://docs.gravityforms.com/form-object) | The form object currently being processed. |

## Examples

### 1\. Change Date format

|     |     |
| --- | --- |
| 1<br>2<br>3<br>4<br>5 | `add_filter(``'gform_zapier_request_body'``,``'change_date_format'``, 10, 4);`<br>```function``change_date_format(``$body``,``$feed``,``$entry``,``$form``){`<br>```$body``[``'Entry Date'``] =``gmdate``(``'Y-m-d'``,``strtotime``(``$entry``[``'date_created'``] ) );`<br>```return``$body``;`<br>```}` |

### 2\. Send Score for Survey fields

|     |     |
| --- | --- |
| 1<br>2<br>3<br>4<br>5<br>6<br>7<br>8<br>9<br>10 | `add_filter(``'gform_zapier_request_body'``,``'zapier_single_score_export'``, 10, 4 );`<br>`function``zapier_single_score_export(``$body``,``$feed``,``$entry``,``$form``) {`<br>```$survey_fields``= GFAPI::get_fields_by_type(``$form``,``array``(``'survey'``) );`<br>```foreach``(``$survey_fields``as``$field``) {`<br>```$body_key``= GF_Zapier::get_body_key(``$body``,``'Survey Score: '``.``$field``->label );`<br>```$body``[``$body_key``] = gf_survey()->get_field_score(``$field``,``$entry``);`<br>```}`<br>```return``$body``;`<br>`}` |

### 3\. Adjust GMT Entry Date to Local Time

Gravity Forms stores the `date_created` in GMT. Use this snippet to adjust the date/time to the local timezone based on the [WordPress Timezone setting](https://wordpress.com/support/settings/time-settings/).

|     |
| --- |
| `add_filter(``'gform_zapier_request_body'``,``function``(``$body``,``$feed``,``$entry``,``$form``) {`<br>```$body``[``'Entry Date'``] = GFCommon::format_date(``$entry``[``'date_created'``], false,``'Y-m-d H:i:s'``, false );`<br>```return``$body``;`<br>`}, 10, 4 );` |

## Placement

This code should be placed in the [functions.php](http://codex.wordpress.org/Theme_Development#Functions_File) file of your active theme.

## Since

This filter was added in the Gravity Forms Zapier Add-On v3.1.1.

This filter is located in `GF_Zapier::get_body()` method in _zapier.php_.
