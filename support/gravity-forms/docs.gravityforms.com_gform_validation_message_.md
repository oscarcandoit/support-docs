## Description

This filter is executed when a form fails validation, before the validation message is displayed. Use this filter to change the default validation message.

## Usage

Applies to all forms.

|     |     |
| --- | --- |
| 1 | `add_filter(``'gform_validation_message'``,``'your_function_name'``, 10, 2 );` |

To limit the scope of your function to a specific form, append the form id to the end of the hook name. (format: gform\_validation\_message\_FORMID)

|     |     |
| --- | --- |
| 1 | `add_filter(``'gform_validation_message_5'``,``'your_function_name'``, 10, 2 );` |

## Parameters

- **$message** [string](https://docs.gravityforms.com/string)
The validation message to be filtered.





|     |     |
| --- | --- |
| 1 | `"<div class='validation_error'>"``. esc_html__(``'There was a problem with your submission.'``,``'gravityforms'``) .``' '``. esc_html__(``'Errors have been highlighted below.'``,``'gravityforms'``) .``'</div>'` |

- **$form** [Form Object](https://docs.gravityforms.com/form-object)

The current form.
