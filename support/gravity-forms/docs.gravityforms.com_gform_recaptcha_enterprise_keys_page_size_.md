## Description

reCAPTCHA’s Enterprise API can only retrieve 100 reCAPTCHA keys at a time. The **gform\_recaptcha\_enterprise\_keys\_page\_size** filter allows developers to set the page size to something smaller.

## Usage

|     |     |
| --- | --- |
| 1 | `add_filter(``'gform_recaptcha_enterprise_keys_page_size'``,``'my_custom_page_size'``);` |

## Parameters

| Parameter | Type | Description |
| --- | --- | --- |
| **$page\_size** | [integer](https://docs.gravityforms.com/integer/) | The number of reCAPTCHA Enterprise keys to retrieve in each API call. The default is 100 and the maximum allowed per API call is 100. |

## Examples

### Decrease page size to 25 per API call

|     |     |
| --- | --- |
| 1<br>2<br>3 | `add_filter(``'gform_recaptcha_enterprise_keys_page_size'``,``function``(``$page_size``) {`<br>```return``25;``// or whatever number of keys you want the API to retrieve before making additional requests. Max 100`<br>`} );` |

## Additional Information

You can also set the reCAPTCHA API Enterprise keys using constants in your wp-config.php. See [this article](https://docs.gravityforms.com/wp-config-options/#h-add-on-constants) for more information.

## Source Code

This filter is located in includes/class-recaptcha-api.php

## Since

reCAPTCHA Add-On v1.8.0
