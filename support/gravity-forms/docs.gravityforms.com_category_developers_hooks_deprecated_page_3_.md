# Deprecated

## Articles

[**gform\_delete\_lead** \\
This hook has been deprecated and gform\_delete\_entry should be used instead.](https://docs.gravityforms.com/gform_delete_lead/ "gform_delete_lead") [**gform\_disable\_admin\_notification** \\
This hook has been deprecated. Please use gform\_disable\_notification instead.](https://docs.gravityforms.com/gform_disable_admin_notification/ "gform_disable_admin_notification") [**gform\_enable\_entry\_info\_payment\_details** \\
The “gform\_enable\_entry\_info\_payment\_details” action enables the payment details panel for the current entry. This hook has been deprecated because the payment details are now visible on the entry if the payment status is set.](https://docs.gravityforms.com/gform_enable_entry_info_payment_details/ "gform_enable_entry_info_payment_details") [**gform\_enable\_field\_label\_visibility\_settings** \\
As of Gravity Forms 2.4, this functionality is part of the Form Editor and the hook has been deprecated.](https://docs.gravityforms.com/gform_enable_field_label_visibility_settings/ "gform_enable_field_label_visibility_settings") [**gform\_form\_delete\_link** \\
This action hook has been deprecated. Use the gform\_post\_payment\_action hook instead.](https://docs.gravityforms.com/gform_form_delete_link/ "gform_form_delete_link") [**gform\_form\_settings** \\
This filter was deprecated in Gravity Forms v2.5. Form settings are now defined via the gform\_form\_settings\_fields filter.](https://docs.gravityforms.com/gform_form_settings/ "gform_form_settings") [**gform\_form\_trash\_link** \\
This filter was removed in Gravity Forms v2.5 with no replacement.](https://docs.gravityforms.com/gform_form_trash_link/ "gform_form_trash_link") [**gform\_mailchimp\_args\_pre\_subscribe** \\
The “gform\_mailchimp\_args\_pre\_subscribe” filter in the Gravity Forms MailChimp add-on has been deprecated. Use the gform\_mailchimp\_subscription filter instead.](https://docs.gravityforms.com/gform_mailchimp_args_pre_subscribe/ "gform_mailchimp_args_pre_subscribe") [**gform\_notification\_email** \\
This filter hook has been deprecated. Use the gform\_notification hook instead.](https://docs.gravityforms.com/gform_notification_email/ "gform_notification_email") [**gform\_notification\_format** \\
This filter has been deprecated since Gravity Forms version 1.7. Use the filter gform\_notication instead.](https://docs.gravityforms.com/gform_notification_format/ "gform_notification_format") [Previous](https://docs.gravityforms.com/category/developers/hooks/deprecated/page/2/) [1](https://docs.gravityforms.com/category/developers/hooks/deprecated/) [2](https://docs.gravityforms.com/category/developers/hooks/deprecated/page/2/) 3 [4](https://docs.gravityforms.com/category/developers/hooks/deprecated/page/4/) [Next](https://docs.gravityforms.com/category/developers/hooks/deprecated/page/4/)
