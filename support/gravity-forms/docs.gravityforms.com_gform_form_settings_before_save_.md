## Description

Modify the form object before saving on the Form Settings page. Useful for saving custom settings added via the [gform\_form\_settings](https://docs.gravityforms.com/gform_form_settings) filter.

## Usage

|     |     |
| --- | --- |
| 1 | `add_filter(``'gform_form_settings_before_save'``,``'my_custom_function'``);` |

## Parameters

| Parameter | Type | Description |
| --- | --- | --- |
| **$form** | [Form Object](https://docs.gravityforms.com/form-object) | The current form object being edited. |
