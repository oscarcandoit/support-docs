## Description

This filter is executed before the form is displayed and can be used to insert a review page before the final form submission.

## Usage

The following would apply to all forms.

|     |     |
| --- | --- |
| 1 | `add_filter(``'gform_review_page'``,``'your_function_name'``);` |

To limit the scope of your function to a specific form, append the form id to the end of the hook name. (format: gform\_review\_page\_FORMID)

|     |     |
| --- | --- |
| 1 | `add_filter(``'gform_review_page_6'``,``'your_function_name'``);` |

## Parameters

- **$review\_page** [array](https://docs.gravityforms.com/array)

The review page to be created:

| 1<br>2<br>3<br>4<br>5<br>6<br>7<br>8<br>9<br>10<br>11<br>12<br>13<br>14<br>15<br>16<br>17 | `array``(`<br>```'content'``=>``''``,`<br>```'cssClass'``=>``''``,`<br>```'is_enabled'``=> false,`<br>```'nextButton'``=>``array``(`<br>```'type'``=>``'text'``,`<br>```'text'``=> __(``'Review Form'``,``'gravityforms'``),`<br>```'imageUrl'``=>``''``,`<br>```'imageAlt'``=>``''``,`<br>```),`<br>```'previousButton'``=>``array``(`<br>```'type'``=>``'text'``,`<br>```'text'``=> __(``'Previous'``,``'gravityforms'``),`<br>```'imageUrl'``=>``''``,`<br>```'imageAlt'``=>``''``,`<br>```),`<br>`);` |

- **$form** [form object](https://docs.gravityforms.com/form-object/)

- **$entry** [entry object](https://docs.gravityforms.com/entry-object/)

False, or the current entry.
