# gform\_merge\_tag\_list\_exclude

- [Description](https://docs.gravityforms.com/gform_merge_tag_list_exclude/#h-description)

- [Parameters](https://docs.gravityforms.com/gform_merge_tag_list_exclude/#h-parameters)
- [Examples](https://docs.gravityforms.com/gform_merge_tag_list_exclude/#h-examples)

- [Source Code](https://docs.gravityforms.com/gform_merge_tag_list_exclude/#h-source-code)

## Description

Use this filter to exclude field types from the merge tag drop-downs that are displayed throughout the admin pages.

## Usage

|     |     |
| --- | --- |
| 1 | `add_filter(``'gform_merge_tag_list_exclude'``,``'exclude_hidden'``, 10, 3 );` |

## Parameters

- **$excluded\_fields** [array](https://docs.gravityforms.com/array)
Array to be filtered. Contains a list of field types that will be excluded from the merge tag drop down

- **$element\_id** [string](https://docs.gravityforms.com/string)

HTML ID attribute of the merge tag drop-down.

- **$fields** [array](https://docs.gravityforms.com/array)

Array containing all form [Field Object](https://docs.gravityforms.com/field-object)


## Examples

This example adds a message below the currency drop down.

|     |     |
| --- | --- |
| 1<br>2<br>3<br>4 | `add_filter(``'gform_merge_tag_list_exclude'``,``'currency_message'``, 10, 3 );`<br>`function``currency_message(``$message``) {`<br>```return``'US Dollars is the only supported currency by this payment gateway.'``;`<br>`}` |

## Placement

This code should be placed in the [functions.php](http://codex.wordpress.org/Theme_Development#Functions_File) file of your active theme.

## Source Code

This filter is located in GFCommon::insert\_variables() in _common.php_
