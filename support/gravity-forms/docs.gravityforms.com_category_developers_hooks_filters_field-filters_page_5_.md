## Articles

[**gform\_field\_choices** \\
This filter is executed when creating the checkbox items. It can be used to manipulate the item’s string before it gets added to the checkbox list.](https://docs.gravityforms.com/gform_field_choices/ "gform_field_choices") [**gform\_field\_choices\_max\_count\_visible** \\
Allows setting the maximum number of choices shown in the form editor for choice based fields (Radio Buttons, Checkbox, Image Choice, and Multiple Choice).](https://docs.gravityforms.com/gform_field_choices_max_count_visible/ "gform_field_choices_max_count_visible") [**gform\_field\_container** \\
This filter can be used to modify the markup, the li, used for the field container.](https://docs.gravityforms.com/gform_field_container/ "gform_field_container") [**gform\_field\_content** \\
This filter is executed before creating the field's content, allowing users to completely modify the way the field is rendered.](https://docs.gravityforms.com/gform_field_content/ "gform_field_content") [**gform\_field\_input** \\
Description.](https://docs.gravityforms.com/gform_field_input/ "gform_field_input") [**gform\_field\_size\_choices** \\
The gform\_field\_size\_choices filter allows the choices for Field Size setting in the form editor to be customized.](https://docs.gravityforms.com/gform_field_size_choices/ "gform_field_size_choices") [**gform\_field\_type\_title** \\
When adding a new field type, use this filter to assign a title to the new type.](https://docs.gravityforms.com/gform_field_type_title/ "gform_field_type_title") [**gform\_field\_types\_delete\_files** \\
Allows additional files to be deleted when file deletion occurs by adding additional field types.](https://docs.gravityforms.com/gform_field_types_delete_files/ "gform_field_types_delete_files") [**gform\_file\_path\_pre\_delete\_file** \\
This filter is executed before an uploaded file (from the File Upload or Post Image fields) is deleted from the entry.](https://docs.gravityforms.com/gform_file_path_pre_delete_file/ "gform_file_path_pre_delete_file") [**gform\_file\_permission** \\
