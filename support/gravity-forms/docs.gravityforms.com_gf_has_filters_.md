## Description

The function **gf\_has\_filters()** determines if a callback has been registered for the specified filter.

## Usage

|     |
| --- |
| `gf_has_filters(``array``(``'filter_name'``,``$modifier1``) );` |

This helper function also has an alias, **gf\_has\_filter()**, which can be used like so:

|     |
| --- |
| `gf_has_filter(``array``(``'filter_name'``,``$modifier1``) );` |

## Parameters

| Parameter | Type | Description |
| --- | --- | --- |
| **$filter** | [array](https://docs.gravityforms.com/int) | Array containing the filter tag as well as the possible modifiers. |
| **$function\_to\_check** | [bool](https://docs.gravityforms.com/int) \|callable | The optional callback to check for. |

## Returns

Returns boolean for whether the hook has any callbacks registered or if a specific callback is registered.

## Examples

|     |
| --- |
| `$result``= gf_has_filters(``array``(``'gform_pre_render'``,``$form_id``) );` |

|     |
| --- |
| `$result``= gf_has_filter(``array``(``'gform_pre_render'``,``$form_id``) );` |

|     |
| --- |
| `$result``= gf_has_filters(``array``(``'gform_disable_view_counter'``,``$form_id``),``'__return_true'``);` |

## Since

The gf\_has\_filters and gf\_has\_filter functions were added in Gravity Forms version 2.4.18.

## Source Code

These helper functions are located in _gravityforms.php_
