## Description

This filter can be used to modify the choices listed in the US states drop down.

## Usage

|     |     |
| --- | --- |
| 1 | `add_filter(``'gform_us_states'``,``'your_function_name'``);` |

## Parameters

| Parameter | Type | Description |
| --- | --- | --- |
| **$states** | [array](https://docs.gravityforms.com/array) | The array to be filtered. It contains the list of states in a standard array |

|     |     |
| --- | --- |
| 1<br>2<br>3<br>4<br>5<br>6<br>7<br>8 | `$states``=``array``(`<br>```'Alabama'``,`<br>```'Alaska'``,`<br>```'Arizona'``,`<br>```'Arkansas'``,`<br>```'California'`<br>```// ... more states`<br>`);` |

## Examples

### 1\. Use State Code

This example demonstrates how to use the two letter state code as the choice value.

|     |     |
| --- | --- |
| 1<br>2<br>3<br>4<br>5<br>6<br>7<br>8<br>9 | `add_filter(``'gform_us_states'``,``'us_states'``);`<br>`function``us_states(``$states``) {`<br>```$new_states``=``array``();`<br>```foreach``(``$states``as``$state``) {`<br>```$new_states``[ GF_Fields::get(``'address'``)->get_us_state_code(``$state``) ] =``$state``;`<br>```}`<br>```return``$new_states``;`<br>`}` |

**Note:** This example does not change the display value, just the stored value. This will store the 2-letter state code in the entry and is required for some add-ons.
