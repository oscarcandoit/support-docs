# How to connect to Salesforce using manual configuration

- [Introduction](https://docs.gravityforms.com/salesforce-add-on-manual-configuration/#h-introduction)
- [Pre-Requisites](https://docs.gravityforms.com/salesforce-add-on-manual-configuration/#h-pre-requisites)
- [Setup](https://docs.gravityforms.com/salesforce-add-on-manual-configuration/#h-setup)

## Introduction

This method allows for the manual configuration of the Salesforce Add-On connection. It is particularly useful when managing a client’s site or when logging into the Salesforce account to establish the connection is not feasible.

**Note**: To use the manual configuration connection method, you must first enable the [gform\_salesforce\_allow\_manual\_configuration](https://docs.gravityforms.com/?p=38945) filter to display the related settings.

## Pre-Requisites

Manual configuration requires a Connected App in Salesforce that supports the Client Credentials OAuth flow. Using an External Client App is not recommended and may cause authentication errors. See [Salesforce: Set Up the Client Credentials Flow for a Connected App](https://help.salesforce.com/s/articleView?id=sf.connected_app_client_credentials_setup.htm&type=5).

## Setup

1. Log into your WordPress admin dashboard.
2. On the left side navigation menu, hover over **Forms** and click on **Settings**.
3. From this page, click the **Salesforce** tab.
4. Enter your **Consumer Key**, **Consumer Secret,** and your **Salesforce Domain**. For help finding your Salesforce domain, refer to [this article.](https://help.salesforce.com/s/articleView?id=sf.faq_domain_name_what.htm&type=5)
5. Click **Update Settings** and a green checkmark confirms success, otherwise verify details and retry.

[![Image showing Manual Configuration Settings](https://docs.gravityforms.com/wp-content/uploads/2024/05/Captura-de-pantalla-2024-05-22-a-las-12.29.44%E2%80%AFp.-m-1024x444.png)](https://docs.gravityforms.com/wp-content/uploads/2024/05/Captura-de-pantalla-2024-05-22-a-las-12.29.44%E2%80%AFp.-m.png)

You are now ready to [create a feed](https://docs.gravityforms.com/creating-a-feed-salesforce-add-on/) for the Salesforce Add-On.

**Note**: For issues establishing the connection or locating the Callback URL, navigate to **Forms > Settings > Salesforce** in the WordPress admin dashboard. Copy and paste the URL from the browser’s address bar. The URL should resemble this format: `admin.php?page=gf_settings&subview=gravityformssalesforce`
