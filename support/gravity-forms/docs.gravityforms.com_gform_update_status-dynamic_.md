# gform\_update\_status

- [Description](https://docs.gravityforms.com/gform_update_status-dynamic/#h-description)

## Description

The “gform\_update\_status” action hook is a dynamically generated filter name. The “status” part of the hook name is an entry property and processed by the hook [gform\_update\_PROPERTY\_NAME](https://docs.gravityforms.com/gform_update_status/).

See the hook [gform\_update\_PROPERTY\_NAME](https://docs.gravityforms.com/gform_update_status) for more details.
