## Description

The “gform\_form\_list\_count” filter in Gravity Forms allows the form list count array to be modified. This is useful when filtering the form count list. This filter should be used with the [gform\_form\_list\_forms](https://docs.gravityforms.com/gform_form_list_forms) hook.

## Usage

|     |     |
| --- | --- |
| 1 | `add_filter(``'gform_form_list_count'``,``'your_function_name'``, 10, 1 );` |

## Parameters

- **$form\_count** [array](https://docs.gravityforms.com/array)
The form count by filter name.


## Examples

This example simply sets the active form count to 20.

|     |     |
| --- | --- |
| 1<br>2<br>3<br>4<br>5 | `add_filter(``'gform_form_list_count'``,``'change_list_count'``, 10, 1 );`<br>`function``change_list_count(``$form_count``){`<br>```$form_count``[``'active'``] = 20;`<br>```return``$form_count``;`<br>`}` |

This example reduces the form count by half and reflects the change in active or inactive. Trash is not counted in the total.

|     |     |
| --- | --- |
| 1<br>2<br>3<br>4<br>5<br>6<br>7<br>8<br>9<br>10<br>11<br>12<br>13<br>14<br>15<br>16<br>17<br>18<br>19<br>20<br>21<br>22<br>23<br>24<br>25<br>26<br>27<br>28<br>29<br>30<br>31<br>32<br>33<br>34<br>35<br>36<br>37<br>38<br>39<br>40<br>41<br>42<br>43<br>44<br>45<br>46<br>47<br>48 | `add_filter(``'gform_form_list_count'``,``'change_list_count'``, 10, 1 );`<br>`function``change_list_count(``$form_count``){`<br>```if``( ! isset(``$_GET``[``'filter'``] ) ){`<br>```$filter``=``''``;`<br>```}`<br>```else``{`<br>```$filter``=``$_GET``[``'filter'``];`<br>```}`<br>```switch``(``$filter``) {`<br>```case``'active'``:`<br>```//remove half of forms that are active, adjust total`<br>```$orig_count``=``$form_count``[``'active'``];`<br>```$form_count``[``'active'``] =``round``(``$form_count``[``'active'``]/2 );`<br>```$form_count``[``'total'``] =``$form_count``[``'total'``] - (``$orig_count``-``$form_count``[``'active'``] );`<br>```break``;`<br>```case``'inactive'``:`<br>```//remove half of forms that are inactive, adjust total`<br>```$orig_count``=``$form_count``[``'inactive'``];`<br>```$form_count``[``'inactive'``] =``round``(``$form_count``[``'inactive'``]/2 );`<br>```$form_count``[``'total'``] =``$form_count``[``'total'``] - (``$orig_count``-``$form_count``[``'inactive'``] );`<br>```break``;`<br>```case``'trash'``:`<br>```//remove half of forms that are trash, do NOT adjust total because trash is not counted in total`<br>```$form_count``[``'trash'``] =``round``(``$form_count``[``'trash'``]/2 );`<br>```break``;`<br>```default``:`<br>```//all forms, remove half and adjust counts`<br>```$orig_count``=``$form_count``[``'total'``];`<br>```$active_count``=``$form_count``[``'active'``];`<br>```$inactive_count``=``$form_count``[``'inactive'``];`<br>```$form_count``[``'total'``] =``round``(``$form_count``[``'total'``]/2 );`<br>```$number_to_remove``=``$orig_count``-``$form_count``[``'total'``];`<br>```//active and inactive counts need to be modified since the form array will be removing active/inactive items`<br>```//remove active forms first, if there are less active forms than the new form total, need to remove from inactive as well`<br>```if``(``$active_count``<``$number_to_remove``){`<br>```//remove active`<br>```$form_count``[``'active'``] = 0;`<br>```//remove inactive`<br>```$form_count``[``'inactive'``] =``$inactive_count``- (``$number_to_remove``-``$active_count``);`<br>```}`<br>```else``{`<br>```//just remove active`<br>```$form_count``[``'active'``] =``$active_count``-``$number_to_remove``;`<br>```}`<br>```}`<br>```return``$form_count``;`<br>`}` |

## Placement

This code should be placed in the [functions.php](http://codex.wordpress.org/Theme_Development#Functions_File) file of your active theme.

## Since

This filter was added in Gravity Forms version 2.3.

## Source Code

This filter is located in GF\_Form\_List\_Table::get\_views() in _form\_list.php_.
