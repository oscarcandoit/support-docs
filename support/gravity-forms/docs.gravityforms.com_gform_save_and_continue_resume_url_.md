## Description

Use this filter to change the resume link generated when an incomplete submission is saved.

## Usage

|     |     |
| --- | --- |
| 1 | `add_filter(``'gform_save_and_continue_resume_url'``,``'your_function_name'``, 10, 4 );` |

## Parameters

- **$resume\_url** [string](https://docs.gravityforms.com/string)

The current resume url.
- **$form** [Form Object](https://docs.gravityforms.com/form-object)

The form that is currently being processed.
- **$token** [string](https://docs.gravityforms.com/string)

The unique token generated when the incomplete submission was saved.
- **$email** [string](https://docs.gravityforms.com/string)

The email the resume link will be sent to.

## Placement

This code should be placed in the [functions.php file](https://codex.wordpress.org/Theme_Development#Functions_File) of your active theme.

## Source Code
