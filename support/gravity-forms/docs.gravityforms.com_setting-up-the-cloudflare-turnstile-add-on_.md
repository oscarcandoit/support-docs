# Setting up the Cloudflare Turnstile Add-On

- [Prerequisites](https://docs.gravityforms.com/setting-up-the-cloudflare-turnstile-add-on/#h-prerequisites)
- [Settings](https://docs.gravityforms.com/setting-up-the-cloudflare-turnstile-add-on/#h-settings)
- [Create a widget](https://docs.gravityforms.com/setting-up-the-cloudflare-turnstile-add-on/#create-a-widget)
- [Adding your Site Key and Site Secret](https://docs.gravityforms.com/setting-up-the-cloudflare-turnstile-add-on/#h-adding-your-site-key-and-site-secret)
- [Additional Settings](https://docs.gravityforms.com/setting-up-the-cloudflare-turnstile-add-on/#h-additional-settings)

## Prerequisites

- [Download](https://docs.gravityforms.com/downloading-a-gravity-forms-add-on) and [install](https://docs.gravityforms.com/installing-a-gravity-forms-add-on/) the add-on.
- Create a [Cloudflare](https://www.cloudflare.com/) account.

Cloudflare offers a free account for those planning to use Turnstile. It is not necessary to use Cloudflare’s caching service or have a paid account.

## Settings

For a full list of all the Cloudflare Turnstile Add-On settings, refer to [this guide](https://docs.gravityforms.com/cloudflare-turnstile-add-on-settings-reference/).

## Create a widget

1. In the Cloudflare dashboard, go to the **[Turnstile](https://dash.cloudflare.com/?to=/:account/turnstile)** page.
2. Select **Add widget**.

[![screenshot of the Add Site flow in the Cloudflare Turnstile dashboard](https://docs.gravityforms.com/wp-content/uploads/2025/11/image-8-1024x408.png)](https://docs.gravityforms.com/wp-content/uploads/2025/11/image-8-scaled.png)

3. Fill out the required information:

– **Widget name**: A descriptive name for your widget.

**– Hostname management**: Domains where the widget will be used.

– (Optional) Configure **Pre-clearance support** for single-page applications.

[![Image showing Cloudflare Turnstile Widget management](https://docs.gravityforms.com/wp-content/uploads/2025/11/image-10-1024x595.png)](https://docs.gravityforms.com/wp-content/uploads/2025/11/image-10-scaled.png) Image showing Cloudflare Turnstile Widget management[![Image showing Cloudflare Turnstile Widget Hostname management](https://docs.gravityforms.com/wp-content/uploads/2025/11/image-9-1024x635.png)](https://docs.gravityforms.com/wp-content/uploads/2025/11/image-9-scaled.png) Image showing Cloudflare Turnstile Widget Hostname management

4. Select **Create** to save your widget.

[![Image showing Cloudflare Turnstile Widget management](https://docs.gravityforms.com/wp-content/uploads/2025/11/image-11-1024x165.png)](https://docs.gravityforms.com/wp-content/uploads/2025/11/image-11-scaled.png)

5. Copy your sitekey and secret key, and store the secret key securely.

[![Image showing Cloudflare Turnstile Widget management](https://docs.gravityforms.com/wp-content/uploads/2025/11/image-12-1024x498.png)](https://docs.gravityforms.com/wp-content/uploads/2025/11/image-12-1024x498.png)

## Adding your Site Key and Site Secret

1. Go to **Forms → Settings → Cloudflare Turnstile**.
2. Copy the **Site Key** and **Secret Key** from your Cloudflare dashboard.
3. Paste both keys into the **Turnstile Credentials** section.
4. Under **Field Options → Theme**, choose your preferred theme (defaults to **Auto**).
5. Click **Save Settings** to apply your changes. Turnstile is now ready to use in your forms.

[![Image showing Gravity Forms Turnstile Add-On settings page.](https://docs.gravityforms.com/wp-content/uploads/2025/11/image-13-1024x640.png)](https://docs.gravityforms.com/wp-content/uploads/2025/11/image-13.png)
