# Zapier Filters

## Articles

[**gform\_form\_settings\_page\_gravityformszapier** \\
The “gform\_form\_settings\_page\_gravityformszapier” action in the Gravity Forms Zapier Add-On is a hook name that is dynamically generated.](https://docs.gravityforms.com/gform_form_settings_page_gravityformszapier/ "gform_form_settings_page_gravityformszapier") [**gform\_save\_zapier\_button** \\
This filter allows the Zapier Feed save button to be modified.](https://docs.gravityforms.com/gform_save_zapier_button/ "gform_save_zapier_button") [**gform\_zap\_before\_save** \\
This filter allows changes to be made to the Zapier feed setup before it is saved to the database.](https://docs.gravityforms.com/gform_zap_before_save/ "gform_zap_before_save") [**gform\_zapier\_actions** \\
This filter allows the modification of the feed actions allowed (delete and edit).](https://docs.gravityforms.com/gform_zapier_actions/ "gform_zapier_actions") [**gform\_zapier\_feed\_conditional\_logic** \\
This filter allows the feed conditional logic rule to be overridden during submission, allowing multiple rules to be defined.](https://docs.gravityforms.com/gform_zapier_feed_conditional_logic/ "gform_zapier_feed_conditional_logic") [**gform\_zapier\_field\_value** \\
This filter can be used to modify a value before it is sent to Zapier.](https://docs.gravityforms.com/gform_zapier_field_value/ "gform_zapier_field_value") [**gform\_zapier\_products** \\
This filter allows the products array to be modified before it is sent to Zapier.](https://docs.gravityforms.com/gform_zapier_products/ "gform_zapier_products") [**gform\_zapier\_request\_body** \\
Allows the request body sent to Zapier to be filtered.](https://docs.gravityforms.com/gform_zapier_request_body/ "gform_zapier_request_body") [**gform\_zapier\_sample\_field\_value** \\
This filter can be used to modify the sample field value before it is sent to Zapier.](https://docs.gravityforms.com/gform_zapier_sample_field_value/ "gform_zapier_sample_field_value") [**gform\_zapier\_use\_stored\_body** \\
This filter is used to determine if the already stored body should be used.](https://docs.gravityforms.com/gform_zapier_use_stored_body/ "gform_zapier_use_stored_body") 1 [2](https://docs.gravityforms.com/category/developers/hooks/filters/filters-addons/zapier-filters/page/2/) [Next](https://docs.gravityforms.com/category/developers/hooks/filters/filters-addons/zapier-filters/page/2/)
