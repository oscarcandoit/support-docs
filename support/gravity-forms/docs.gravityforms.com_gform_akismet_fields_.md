## Description

Use this filter to specify the fields that are sent to the Akismet anti-spam service.

## Usage

|     |
| --- |
| `add_filter(``'gform_akismet_fields'``,``'set_akismet_fields'``, 10, 4 );` |

You can also target a specific form by adding the form ID after the hook name.

|     |
| --- |
| `add_filter(``'gform_akismet_fields_6'``,``'set_akismet_fields'``, 10, 4 );` |

## Parameters

| Parameter | Type | Description |
| --- | --- | --- |
| **$akismet\_fields** | [array](https://docs.gravityforms.com/array) | The data to be sent to Akismet, in the following format: |

|     |     |
| --- | --- |
| 1<br>2<br>3<br>4<br>5<br>6<br>7<br>8<br>9<br>10<br>11<br>12<br>13<br>14 | `array``(`<br>```'comment_type'``=>``'gravity_form'``,`<br>```'comment_author'``=>``'Author name'``,`<br>```'comment_author_email'``=>``'email@test.com'``,`<br>```'comment_author_url'``=>``'http://www.someurl.com'``,`<br>```'comment_content'``=>``'Some text'``,`<br>```'contact_form_subject'``=>``'Contact Us'``,`<br>```'comment_author_ip'``=>``'127.0.0.1'``,`<br>```'permalink'``=>``'http://www.someurl.com/contactus'``,`<br>```'user_ip'``=>``'127.0.0.1'``,`<br>```'user_agent'``=>``'user agent string'``,`<br>```'referrer'``=>``'http://www.referrer.com'``,`<br>```'blog'``=>``'http://www.someurl.com'``,`<br>`);` |

| Property | Default Source |
| --- | --- |
| comment\_author | The first Name type field found on the form |
| comment\_author\_email | The first Email type field found on the form |
| comment\_author\_url | The first Website type field found on the form |
| comment\_content | The first Paragraph type field found on the form |
| contact\_form\_subject | The form title |

| Parameter | Type | Description |
| --- | --- | --- |
| **$form** | [Form Object](https://docs.gravityforms.com/form-object) | The form which created the entry. |
| **$entry** | [Entry Object](https://docs.gravityforms.com/entry-object) | The entry being processed. |
| **$action** | [string](https://docs.gravityforms.com/string) | The action being performed. |

The action triggering the Akismet request: **submit**, **spam**, or **ham**. Since version 2.4.18.5.

## Examples

### 1\. Override the comment\_content

This example overrides the value in the “comment\_content” Akismet field so that it is pulled from another field. (field ID: 1).

|     |     |
| --- | --- |
| 1<br>2<br>3<br>4<br>5<br>6 | `add_filter(``'gform_akismet_fields'``,``'set_akismet_fields'``, 10, 3 );`<br>`function``set_akismet_fields(``$akismet_fields``,``$form``,``$entry``) {`<br>```$akismet_fields``[``'comment_content'``] = rgar(``$entry``,``'1'``);`<br>```return``$akismet_fields``;`<br>`}` |

### 2\. Provide the IP address

This example shows how you can pass the IP address to Akismet when it has been removed from the entry using the [gform\_ip\_address](https://docs.gravityforms.com/gform_ip_address) filter.

|     |     |
| --- | --- |
| 1<br>2<br>3<br>4<br>5<br>6<br>7<br>8<br>9<br>10<br>11<br>12<br>13<br>14 | `add_filter(``'gform_akismet_fields'``,``'set_akismet_fields'``, 10, 3 );`<br>`function``set_akismet_fields(``$akismet_fields``,``$form``,``$entry``) {`<br>```// Remove any filters that block the IP address`<br>```remove_filter(``'gform_ip_address'``,``'__return_empty_string'``);`<br>```// Retrieve the user's IP address`<br>```$ip``= GFFormsModel::get_ip();`<br>```// Add the IP address to the Akismet fields`<br>```$akismet_fields``[``'comment_author_IP'``] =``$ip``;`<br>```$akismet_fields``[``'user_ip'``] = preg_replace(``'/[^0-9., ]/'``,``''``,``$ip``);`<br>```return``$akismet_fields``;`<br>`}` |

## Placement

This code can be used in the [functions.php](https://developer.wordpress.org/themes/basics/theme-functions/) file of the active theme, a custom functions plugin, a custom add-on, or with a code snippets plugin.

See also the PHP section in this article: [Where Do I Put This Code?](https://docs.gravityforms.com/where-do-i-put-this-code/)

## Source Code

Since Akismet Add-On version 1.1, this filter is located in `\Gravity_Forms\Gravity_Forms_Akismet\Akismet_Fields_Filter::get_fields()` in `includes/class-akismet-fields-filter.php`. It was previously located in Gravity Forms, in `GFCommon::get_akismet_fields()` in `common.php`.
