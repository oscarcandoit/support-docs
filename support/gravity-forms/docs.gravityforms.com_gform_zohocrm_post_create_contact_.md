## Description

Allows custom actions to be performed after creating the contact.

## Usage

|     |     |
| --- | --- |
| 1<br>2<br>3<br>4<br>5 | ```add_action(``'gform_zohocrm_post_create_contact'``,``$contact_record``,``$contact``,``$feed``,``$entry``,``$form``);`<br>```return``$contact_id``;`<br>`}` |

## Parameters

- **$contact\_record** [array](https://docs.gravityforms.com/array)
The contact record.

- **$contact** [array](https://docs.gravityforms.com/array)

The contact arguments.

- **$feed** [array](https://docs.gravityforms.com/array)

The feed object.

- **$entry** [array](https://docs.gravityforms.com/array)

The entry object.

- **$form** [array](https://docs.gravityforms.com/array)

The form object.

|     |     |
| --- | --- |

This code should be placed in the [functions.php](http://codex.wordpress.org/Theme_Development#Functions_File) file of your active theme.
