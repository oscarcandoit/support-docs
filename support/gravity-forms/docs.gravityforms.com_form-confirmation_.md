# Form Confirmation CSS Selectors

- [Inline Confirmation Message](https://docs.gravityforms.com/form-confirmation/#h-inline-confirmation-message)

## Inline Confirmation Message

Displays on same page upon successful completion of the form

- **example:** inline confirmation message (div) – applies to all forms

|     |     |
| --- | --- |
| 1 | `body .gform_confirmation_wrapper .gform_confirmation_message  {``border``:``1px``solid``red``;}` |

**example:** inline confirmation message (div) – applies to form id 1

|     |     |
| --- | --- |
| 1 | `body #gform_confirmation_wrapper_``1``.gform_confirmation_message  {``color``:``black``;}` |
