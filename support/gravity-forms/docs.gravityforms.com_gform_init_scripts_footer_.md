# gform\_init\_scripts\_footer

- [Description](https://docs.gravityforms.com/gform_init_scripts_footer/#h-description)
- [Usage](https://docs.gravityforms.com/gform_init_scripts_footer/#h-usage)
- [Placement](https://docs.gravityforms.com/gform_init_scripts_footer/#h-placement)
- [Source Code](https://docs.gravityforms.com/gform_init_scripts_footer/#h-source-code)

## Description

This filter is executed during form load. When set to true, the form init scripts are loaded in the footer of the site, instead of the default location of which is in the page body immediately after the form.

**Note**: This filter does not occur when AJAX is being used within the form.

**Note**: As of Gravity Forms 2.5 scripts are included in the footer by default.

## Usage

|     |     |
| --- | --- |
| 1 | `add_filter(``'gform_init_scripts_footer'``,``'__return_true'``);` |

## Placement

This code should be placed in the [functions.php](http://codex.wordpress.org/Theme_Development#Functions_File) file of your active theme.

## Source Code

This filter is located in GFFormDisplay::get\_form() in _form\_display.php_.
