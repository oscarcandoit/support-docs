## Description

A filter to determine if a form has image choices.

## Usage

|     |     |
| --- | --- |
| 1 | `add_filter(``'gform_has_image_choices'``,``'your_function_name'``, 10, 2 );` |

## Parameters

| Parameter | Type | Description |
| --- | --- | --- |
| $has\_image\_choices | bool | Result of Gravity Forms’ automatic detection. Default true or false. |
| $form | array | The current form object. |

## Examples

### Return Whether a Form Has Image Choice Fields.

|     |     |
| --- | --- |
| 1<br>2<br>3<br>4<br>5 | `add_filter(``'gform_has_image_choices'``,``'check_image_choices'``, 10, 2 );`<br>`function``check_image_choices(``$has_image_choices``,``$form``) {`<br>```return``$has_image_choices``? true : false;`<br>`}` |

## Placement

This code can be used in the [functions.php](https://developer.wordpress.org/themes/basics/theme-functions/) file of the active theme, a custom functions plugin, a custom add-on, or with a code snippets plugin.

See also the PHP section in this article: [Where Do I Put This Code?](https://docs.gravityforms.com/where-do-i-put-this-code/)

## Since

Gravity Forms 2.8

## Source Code

Located in GFFormDisplay::has\_image\_choices() inside form\_display.php.
