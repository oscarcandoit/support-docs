## Introduction

The [Feed Meta Object](https://docs.gravityforms.com/feed-object) for the Kit Add-On is an associative array with properties determining how the add-on should process the form submission.

| --- | --- |
| 1<br>2<br>3<br>4<br>5<br>6<br>7<br>8<br>9<br>10<br>11<br>12<br>13<br>14<br>15<br>16<br>17<br>18<br>19<br>20<br>21 | `$feed``[``'meta'``] =``array``(`<br>```'feed_name'``=>``'Kit Feed'``,`<br>```'form_id'``=>``'5500223'``,`<br>```'tag_id'``=>``'4282109'``,`<br>```'convertkit_custom_fields'``=>``array``(`<br>```0 =>``array``(`<br>```'key'``=>``''``,`<br>```'custom_key'``=>``''``,`<br>```'value'``=>``''``,`<br>```'custom_value'``=>``''``,`<br>```),`<br>```),`<br>```'feed_condition_conditional_logic_object'``=>``array``(),`<br>```'feed_condition_conditional_logic'``=>``'0'``,`<br>```'field_map_email'``=>``''``,`<br>```'field_map_name'``=>``''``,`<br>```'field_map_tag'``=>``''``,`<br>```'conditions'``=>``''``,`<br>```'trial_product'``=> NULL,`<br>```'trial_amount'``=> NULL,`<br>`);` |

## Usage

We recommend accessing the $feed meta using the rgar() or rgars() functions, e.g.:

|     |     |
| --- | --- |
| 1<br>2<br>3<br>4<br>5<br>6<br>7<br>8<br>9<br>10<br>11<br>12<br>13 | `$conditional_logic_enabled``= rgars(``$feed``,``'meta/feedCondition'``);`<br>`if``(``$conditional_logic_enabled``) {`<br>```// Conditional logic is enabled for this feed.`<br>```$logic_rules``= rgars(``$feed``,``'meta/feed_condition_conditional_logic_object/conditionalLogic/rules'``);`<br>```foreach``(``$logic_rules``as``$rule``) {`<br>```// Process each conditional logic rule.`<br>```$field_id``= rgar(``$rule``,``'fieldId'``);`<br>```$operator``= rgar(``$rule``,``'operator'``);`<br>```$value``= rgar(``$rule``,``'value'``);`<br>```// Example: Use $field_id, $operator, and $value in your logic.`<br>```}`<br>`}` |

## Properties

| Property | Type | Description |
| --- | --- | --- |
| **feed\_name** | string | A unique name for the Kit feed. |
| **form\_id** | string | The ID of the form associated with the feed. |
| **tag\_id** | string | The ID of the tag to be assigned. |
| **convertkit\_custom\_fields** | array | An array of [custom fields](https://docs.gravityforms.com/kit-feed-meta/#h-custom-fields) to map additional data. |
| **feed\_condition\_conditional\_logic\_object** | array | Stores the conditional logic rules for the feed. Empty if no rules are defined. |
| **feed\_condition\_conditional\_logic** | boolean | Determines whether conditional logic is enabled for the feed. |
| **field\_map\_email** | integer | Specifies the field ID used to map the email address. |
| **field\_map\_name** | string | Specifies the field ID used to map the name. |
| **field\_map\_tag** | string | Specifies the field ID used to map the tag. |
| **conditions** | string | Specifies additional conditions for running the feed. |
| **trial\_product** | null | Specifies the trial product. NULL if not applicable. |
| **trial\_amount** | null | Defines the trial amount. NULL if not applicable. |
