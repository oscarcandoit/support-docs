## Description

By default, the “Add Form” button and legacy shortcode UI will only be available on the Post or Page edit screens using the classic editor. The gform\_display\_add\_form\_button filter can be used to enable it for other pages or scenarios.

**Note:** The “Add Form” button requires a TinyMCE input, so in order to display it, you must first include a TinyMCE input to your page.

Indicates if the Add Form button and shortcode UI should be included for the current admin page.
