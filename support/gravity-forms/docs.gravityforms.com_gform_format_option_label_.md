## Description

This filter is executed when calculating and displaying the pricing on product option fields or shipping fields set as drop down field or radio button type. Use this filter to change the format of the choice labels for these field types.

## Usage

|     |     |
| --- | --- |
| 1<br>2<br>3<br>4<br>5<br>6<br>7<br>8 | `// Simply implement the function with the name "gform_format_option_label"`<br>`// to activate the JavaScript filter. The value returned by this function`<br>`// will be used as the option label.`<br>`function``gform_format_option_label(fullLabel, fieldLabel, priceLabel, selectedPrice, price, formId, fieldId) {`<br>```// Disabling option pricing. Simply return the field label.`<br>```return``fieldLabel;`<br>`}` |

## Parameters

- **fullLabel** [string](https://docs.gravityforms.com/string)

The default label that will be displayed for the current option. It has the field choice label as well as the calculated price. (i.e. My Option +$5.00)
- **fieldLabel** [string](https://docs.gravityforms.com/string)

The field/option label without the price. (i.e. My Option)
- **priceLabel** [string](https://docs.gravityforms.com/string)

The price text without the field label. (i.e. +$5.00)
- **selectedPrice** [float](https://docs.gravityforms.com/float)

The price of the currently selected option.
- **price** [float](https://docs.gravityforms.com/float)

The price of this option. (The option that this label applies to)
- **formId** [integer](https://docs.gravityforms.com/integer)

The current form ID.
- **fieldId** [integer](https://docs.gravityforms.com/integer)

The current field ID.
