## Description

The filter `gform_helpscout_display_enable_custom_app_setting` indicates if the enable custom app setting should be displayed or hidden. Defaults to the license validation state.

## Usage

|     |     |
| --- | --- |
| 1 | `add_filter(``'gform_helpscout_display_enable_custom_app_setting'``,``'your_function_name'``);` |

## Parameters

- **$gform\_helpscout\_display\_enable\_custom\_app\_setting** [bool](https://docs.gravityforms.com/bool/)

Indicates if the Custom App setting is shown at **Forms → Settings → Help Scout**. Default matches the Gravity Forms license validation state.


## Examples

### Always show the option to connect via a custom app

This example will show the Help Scout custom app setting.

|     |     |
| --- | --- |
| 1 | `add_filter(``'gform_helpscout_display_enable_custom_app_setting'``,``'__return_true'``);` |

## Placement

This code can be used in the [functions.php](https://developer.wordpress.org/themes/basics/theme-functions/) file of the active theme, a custom functions plugin, a custom add-on, or with a code snippets plugin.

See also the PHP section in this article: [Where Do I Put This Code?](https://docs.gravityforms.com/where-do-i-put-this-code/)

This filter was added in Gravity Forms v2.2.0.
