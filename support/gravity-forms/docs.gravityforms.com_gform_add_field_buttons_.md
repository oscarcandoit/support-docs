## Description

This filter can be used to add/edit/remove the “add field” buttons from the form editor’s floating toolbox

## Usage

|     |     |
| --- | --- |
| 1 | `add_filter(``'gform_add_field_buttons'``,``'add_map_field'``);` |

### Example One

This example adds a “Map” field button to the advanced group

| --- | --- |
| 1<br>2<br>3<br>4<br>5<br>6<br>7<br>8<br>9<br>10<br>11<br>12<br>13<br>14<br>15<br>16 | `add_filter(``'gform_add_field_buttons'``,``'add_map_field'``);`<br>`function``add_map_field(``$field_groups``) {`<br>`foreach``(``$field_groups``as``&``$group``) {`<br>`if``(``$group``[``'name'``] ==``'advanced_fields'``) {`<br>`$group``[``'fields'``][] =``array``(`<br>`'class'``=>``'button'``,`<br>`'data-type'``=>``'map'``,`<br>`'value'``=> __(``'Map'``,``'gravityforms'``),`<br>`'onclick'``=>``"StartAddField('map');"`<br>`);`<br>`break``;`<br>`}`<br>`}`<br>`return``$field_groups``;`<br>`}` |

### Example Two

This example removes the file upload button and the entire post field group from the form editor toolbox
