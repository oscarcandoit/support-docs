## Articles

[**gform\_file\_upload\_tmp\_dir** \\

This filter allows the disabling of file upload whitelisting. This filter applies to all forms.](https://docs.gravityforms.com/gform_file_upload_whitelisting_disabled/ "gform_file_upload_whitelisting_disabled") [**gform\_fileupload\_entry\_value\_file\_path** \\
Allows for the filtering of the file path before output.](https://docs.gravityforms.com/gform_fileupload_entry_value_file_path/ "gform_fileupload_entry_value_file_path") [**gform\_filter\_links\_entry\_list** \\
This filter allows the list of filters at the top of the Entry List page to be modified.](https://docs.gravityforms.com/gform_filter_links_entry_list/ "gform_filter_links_entry_list") [**gform\_filter\_ui** \\
Allows modifying the existing filter layout or styling.](https://docs.gravityforms.com/gform_filter_ui/ "gform_filter_ui") [**gform\_filters\_get\_users** \\
Use this filter to control the list of users available in the filters on the entry list, export entries conditional logic and results pages.](https://docs.gravityforms.com/gform_filters_get_users/ "gform_filters_get_users") [**gform\_filters\_pre\_results** \\
Use this filter to modify the filters used in the drop down list of fields in the admin results page.](https://docs.gravityforms.com/gform_filters_pre_results/ "gform_filters_pre_results") [**gform\_footer\_init\_scripts\_filter** \\
Allows the scripts that fire in the footer to be modified.](https://docs.gravityforms.com/gform_footer_init_scripts_filter/ "gform_footer_init_scripts_filter") [**gform\_force\_hooks\_js\_output** \\
