## Description

This filter can be used to keep a local copy of the file being retained once it has been uploaded to Dropbox.

## Usage

The following would apply to all forms:

|     |     |
| --- | --- |
| 1 | `add_filter(``'gform_dropbox_store_local_version'``,``'your_function_name'``, 10, 6 );` |

To target a specific form append the form id to the hook name. (format: gform\_dropbox\_store\_local\_version\_FORMID)

|     |     |
| --- | --- |
| 1 | `add_filter(``'gform_dropbox_store_local_version_10'``,``'your_function_name'``, 10, 6 );` |

To target a specific field append both the form id and the field id to the hook name. (format: gform\_dropbox\_store\_local\_version\_FORMID\_FIELDID)

|     |     |
| --- | --- |
| 1 | `add_filter(``'gform_dropbox_store_local_version_10_3'``,``'your_function_name'``, 10, 6 );` |

## Parameters

- **$store\_local\_version** [boolean](https://docs.gravityforms.com/boolean)
Should a local copy of the file be retained? Default is false.

- **$file** [array](https://docs.gravityforms.com/array)

The file properties.





|     |     |
| --- | --- |
| 1<br>2<br>3<br>4<br>5<br>6 | `array``(`<br>```'name'``=>``basename``(``$entry``[ rgar(``$field``,``'id'``) ] ),`<br>```'path'``=>``str_replace``( WP_CONTENT_URL, WP_CONTENT_DIR,``$entry``[ rgar(``$field``,``'id'``) ] ),`<br>```'url'``=>``$entry``[``$field``[``'id'``] ],`<br>```'destination'``=> rgars(``$feed``,``'meta/destinationFolder'``),`<br>`);` |

- **$field\_id** [string](https://docs.gravityforms.com/string)

The ID of the field currently being processed.

- **$form** [Form Object](https://docs.gravityforms.com/form-object)

The form currently being processed.

- **$entry** [Entry Object](https://docs.gravityforms.com/entry-object)

The entry currently being processed.

- **$feed** [Feed Object](https://docs.gravityforms.com/feed-object)

The feed currently being processed.
