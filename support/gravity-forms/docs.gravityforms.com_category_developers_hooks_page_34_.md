## Articles

[**gform\_field\_size\_choices** \\
The gform\_field\_size\_choices filter allows the choices for Field Size setting in the form editor to be customized.](https://docs.gravityforms.com/gform_field_size_choices/ "gform_field_size_choices") [**gform\_field\_standard\_settings** \\
Use this filter to create a new field settings under the Standard tab. Useful when implementing a new custom field type that requires custom settings.](https://docs.gravityforms.com/gform_field_standard_settings/ "gform_field_standard_settings") [**gform\_field\_type\_title** \\
When adding a new field type, use this filter to assign a title to the new type.](https://docs.gravityforms.com/gform_field_type_title/ "gform_field_type_title") [**gform\_field\_types\_delete\_files** \\
Allows additional files to be deleted when file deletion occurs by adding additional field types.](https://docs.gravityforms.com/gform_field_types_delete_files/ "gform_field_types_delete_files") [**gform\_field\_validation** \\
Use this filter to create custom validation logic for a field.](https://docs.gravityforms.com/gform_field_validation/ "gform_field_validation") [**gform\_field\_value\_$parameter\_name** \\
This filter is executed before displaying each field and can be used to dynamically populate fields with a default value.](https://docs.gravityforms.com/gform_field_value_parameter_name/ "gform_field_value_$parameter_name") [**gform\_file\_path\_pre\_delete\_file** \\
This filter is executed before an uploaded file (from the File Upload or Post Image fields) is deleted from the entry.](https://docs.gravityforms.com/gform_file_path_pre_delete_file/ "gform_file_path_pre_delete_file") [**gform\_file\_permission** \\
This filter is executed when uploading a file. It can be used to change the default permissions of uploaded files.](https://docs.gravityforms.com/gform_file_permission/ "gform_file_permission") [**gform\_file\_upload\_markup** \\
