## Description

Perform a custom action when the partial entry has been saved or updated.

## Usage

### Entry Saved

The following would run for all forms with the partial entries feature enabled when the entry\_saved event occurs.

|     |
| --- |
| `add_action(``'gform_partialentries_post_entry_saved'``,``'your_function_name'``, 10, 2 );` |

### Entry Updated

The following would run for all forms with the partial entries feature enabled when the entry\_updated event occurs.

|     |
| --- |
| `add_action(``'gform_partialentries_post_entry_updated'``,``'your_function_name'``, 10, 2 );` |

## Parameters

- **$partial\_entry** [Entry Object](https://docs.gravityforms.com/entry-object)


The partial entry object.

- **$form** [Form Object](https://docs.gravityforms.com/form-object)


The current form object.
