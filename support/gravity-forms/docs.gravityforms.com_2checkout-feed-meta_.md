## Introduction

The [Feed Meta Object](https://docs.gravityforms.com/feed-object) for the 2Checkout Add-On is an associative array with properties determining how the add-on should process the form submission.

| --- | --- |
| 1<br>2<br>3<br>4<br>5<br>6<br>7<br>8<br>9<br>10<br>11<br>12<br>13<br>14<br>15<br>16<br>17<br>18<br>19<br>20<br>21<br>22<br>23<br>24<br>25 | `$feed``[``'meta'``] =``array``(`<br>```'feedName'``=>``'Example Feed Name'``,`<br>```'transactionType'``=>``'product'``,`<br>```'paymentAmount'``=>``'10'``,`<br>```'billingCycle_length'``=>``'1'``,`<br>```'billingCycle_unit'``=>``'month'``,`<br>```'recurringTimes'``=>``'12'``,`<br>```'setupFee_enabled'``=> true,`<br>```'setupFee'``=>``'2'``,`<br>```'feedCondition'``=> true,`<br>```'feed_condition_object'``=>``array``(`<br>```'conditionalLogic'``=>``array``(`<br>```'actionType'``=>``'show'``,`<br>```'logicType'``=>``'all'``,`<br>```'rules'``=>``array``(`<br>```array``(`<br>```'fieldId'``=>``'9'``,`<br>```'operator'``=>``'is'``,`<br>```'value'``=>``'Approved'``,`<br>```),`<br>```),`<br>```),`<br>```),`<br>```'subscription_name'``=>``'Name'``,`<br>`);` |

## Usage

We recommend accessing the $feed meta using the rgar() or rgars() functions, e.g.:

|     |     |
| --- | --- |
| 1<br>2<br>3<br>4<br>5<br>6<br>7<br>8<br>9<br>10<br>11<br>12<br>13 | `$conditional_logic_enabled``= rgars(``$feed``,``'meta/feedCondition'``);`<br>`if``(``$conditional_logic_enabled``) {`<br>```// Conditional logic is enabled for this feed.`<br>```$logic_rules``= rgars(``$feed``,``'meta/feed_condition_conditional_logic_object/conditionalLogic/rules'``);`<br>```foreach``(``$logic_rules``as``$rule``) {`<br>```// Process each conditional logic rule.`<br>```$field_id``= rgar(``$rule``,``'fieldId'``);`<br>```$operator``= rgar(``$rule``,``'operator'``);`<br>```$value``= rgar(``$rule``,``'value'``);`<br>```// Example: Use $field_id, $operator, and $value in your logic.`<br>```}`<br>`}` |

## Properties

| Property | Type | Description |
| --- | --- | --- |
| **feedName** | string | A unique name for the feed. |
| **transactionType** | string | The type of transaction: ‘product’ or ‘subscription’. |
| **paymentAmount** | string | Specifies the field or merge tag to use for the payment amount. |
| **billingCycle\_length** | string | Specifies the length of the billing cycle for subscriptions. |
| **billingCycle\_unit** | string | Specifies the unit of time for the billing cycle: ‘week’, ‘month’, or ‘year’. |
| **recurringTimes** | string | Number of billing cycles for a subscription (or leave empty for ongoing). |
| **setupFee\_enabled** | boolean | Enables a one-time setup fee for subscriptions. |
| **setupFee** | string | Specifies the field or merge tag to use for the setup fee amount. |
| **feedCondition** | boolean | Enables conditional logic to determine if the feed should run. |
| **feed\_condition\_object** | array | Stores the conditional logic rules. |
| **subscription\_name** | string | Specifies a custom name for the subscription. |
