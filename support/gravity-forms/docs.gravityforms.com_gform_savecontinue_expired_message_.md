## Description

You can use this filter to modify the error message displayed when an expired or invalid **Save and Continue** link is used.

## Usage

|     |     |
| --- | --- |
| 1 | `add_filter(``'gform_savecontinue_expired_message'``,``'your_function_name'``, 10, 2 );` |

## Parameters

- **$message** [string](https://docs.gravityforms.com/string)

Save & Continue expired/invalid link error message.
- **$form** [Form Object](https://docs.gravityforms.com/form-object)

The form that is currently being processed.

## Placement

This code should be placed in the [functions.php](http://codex.wordpress.org/Theme_Development#Functions_File) file of your active theme.
