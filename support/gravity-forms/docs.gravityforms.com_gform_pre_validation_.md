## Description

This filter can be used to manipulate the [Form Object](https://docs.gravityforms.com/form-object) used during the form validation process.

## Usage

The following would apply to all forms.

|     |
| --- |
| `add_filter(``'gform_pre_validation'``,``'your_function_name'``);` |

To limit the scope of your function to a specific form, append the form id to the end of the hook name. (format: gform\_pre\_validation\_FORMID)

|     |
| --- |
| `add_filter(``'gform_pre_validation_6'``,``'your_function_name'``);` |

## Parameters

- **$form** [Form Object](https://docs.gravityforms.com/form-object)

The current form to be filtered.

## Examples

### 1\. Conditionally Required Field

The following example shows how you can conditionally require a field based on the value of another field.

|     |
| --- |
| `add_filter(``'gform_pre_render'``,``'set_conditional_requirement'``);`<br>`add_filter(``'gform_pre_validation'``,``'set_conditional_requirement'``);`<br>`function``set_conditional_requirement(``$form``) {`<br>```$value``= rgpost(``'input_2'``);`<br>```if``(``$value``==``'no'``) {`<br>```return``$form``;`<br>```}`<br>``<br>```foreach``(``$form``[``'fields'``]``as``&``$field``) {`<br>```if``(``$field``->id == 1 ) {`<br>```$field``->isRequired = true;`<br>```}`<br>```}`<br>```return``$form``;`<br>`}` |

### 2\. Populate Choices – Drop Down

This example dynamically populates a drop down field with posts that are in the Business category.

| --- |
| `add_filter(``'gform_pre_validation'``,``'populate_dropdown'``);`<br>`function``populate_dropdown(``$form``) {`<br>``<br>```//only populating drop down for form id 5`<br>```if``(``$form``[``'id'``] != 5 ) {`<br>```return``$form``;`<br>```}`<br>``<br>```//Reading posts for "Business" category;`<br>```$posts``= get_posts(``'category='``. get_cat_ID(``'Business'``) );`<br>``<br>```//Creating drop down item array.`<br>```$items``=``array``();`<br>``<br>```//Adding initial blank value.`<br>```$items``[] =``array``(``'text'``=>``''``,``'value'``=>``''``);`<br>``<br>```//Adding post titles to the items array`<br>```foreach``(``$posts``as``$post``) {`<br>```$items``[] =``array``(``'value'``=>``$post``->post_title,``'text'``=>``$post``->post_title );`<br>```}`<br>``<br>```//Adding items to field id 8. ### 3\. Dynamically add a reCAPTCHA field to all forms

The following example shows how you can dynamically add reCAPTCHA to all your forms.

|     |
| --- |
| `add_filter(``'gform_pre_render'``,``'add_captcha_field'``);`<br>`add_filter(``'gform_pre_validation'``,``'add_captcha_field'``);`<br>`function``add_captcha_field(``$form``) {`<br>```if``(``empty``(``$form``[``'fields'``] ) || !``is_array``(``$form``[``'fields'``] ) ) {`<br>```return``$form``;`<br>```}`<br>``<br>```$page_number``= 1;`<br>```$has_captcha_field``= false;`<br>``<br>```foreach``(``$form``[``'fields'``]``as``$field``) {`<br>```if``(``$field``->type ==``'captcha'``) {`<br>```$has_captcha_field``= true;`<br>```}`<br>``<br>```if``(``$field``->type ==``'page'``) {`<br>```$page_number``++;`<br>```}`<br>```}`<br>``<br>```if``( !``$has_captcha_field``) {`<br>```$form``[``'fields'``][] = GF_Fields::create(``array``(`<br>```'type'``=>``'captcha'``,`<br>```'id'``=> GFFormsModel::get_next_field_id(``$form``[``'fields'``] ),`<br>```'label'``=>``'Captcha'``,`<br>```'displayOnly'``=> true,`<br>```'formId'``=>``$form``[``'id'``],`<br>```'pageNumber'``=>``$page_number``,`<br>```'visibility'``=>``'visible'``,`<br>```) );`<br>```}`<br>``<br>```return``$form``;`<br>`}` |

## Placement

This code should be placed in the [functions.php](http://codex.wordpress.org/Theme_Development#Functions_File) file of your active theme.
