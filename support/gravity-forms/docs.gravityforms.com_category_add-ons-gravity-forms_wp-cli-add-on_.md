# WP-CLI Add-On

With the [Gravity Forms CLI Add-On](https://wordpress.org/plugins/gravityformscli/), you can use [WP-CLI](https://make.wordpress.org/cli/handbook/) to install Gravity Forms, manage your license, forms, entries and more from the command line.

Don't already have the add-on installed? Install it today using the following WP CLI command:

`wp plugin install gravityformscli --activate`
