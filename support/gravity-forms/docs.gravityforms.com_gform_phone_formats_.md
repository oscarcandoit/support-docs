## Description

The `gform_phone_formats` filter can be used to add new formats to those listed within the [Phone field](https://docs.gravityforms.com/phone/).

## Usage

The following would apply to all forms.

|     |
| --- |
| `add_filter(``'gform_phone_formats'``,``'your_function_name'``, 10, 2 );` |

To limit the scope of your function to a specific form, append the form id to the end of the hook name. (format: gform\_phone\_formats\_FORMID)

|     |
| --- |
| `add_filter(``'gform_phone_formats_5'``,``'your_function_name'``, 10, 2 );` |

## Parameters

- **$phone\_formats** [array](https://docs.gravityforms.com/array)

The phone formats.
- **$form\_id** [integer](https://docs.gravityforms.com/integer)

The ID of the current form.

### $phone\_formats

The following shows the default declaration of the `$phone_formats` array.

|     |
| --- |
| `array``(`<br>```'standard'``=>``array``(`<br>```'label'``=>``'(###) ###-####'``,`<br>```'mask'``=>``'(999) 999-9999'``,`<br>```'regex'``=>``'/^\D?(\d{3})\D?\D?(\d{3})\D?(\d{4})$/'``,`<br>```'instruction'``=>``'(###) ###-####'``,`<br>```'type'``=>``'standard'``,`<br>```),`<br>```'international'``=>``array``(`<br>```'label'``=> __(``'International'``,``'gravityforms'``),`<br>```'mask'``=> false,`<br>```'regex'``=> false,`<br>```'instruction'``=> false,`<br>```'type'``=>``'international'``,`<br>```),`<br>`)` |

Each phone format consists of four properties.

- **label** [string](https://docs.gravityforms.com/string)

The label which will be displayed for the choice in the _Phone Format_ setting, on the fields properties tab, in the form editor.
- **mask** [string](https://docs.gravityforms.com/string) \| [boolean](https://docs.gravityforms.com/boolean)

The input mask to be applied to the input when the form is displayed, or `false` if an input mask should not be used. See the [Input Mask](https://docs.gravityforms.com/input-mask) article for some usage examples.
- **regex** [string](https://docs.gravityforms.com/string) \| [boolean](https://docs.gravityforms.com/boolean)

The regular expression which should be used when validating the field value on submission, or `false` if validation of the value should not be performed.
- **instruction** [string](https://docs.gravityforms.com/string) \| [boolean](https://docs.gravityforms.com/boolean)

The text you want to display beneath the field input if the value fails the validation performed using the above `regex`, (i.e. _`Phone format: (###) ###-####`_) or `false` if you don’t want to display the instruction.

## Examples

### 1\. Add a UK phone format for all forms

The following example adds a UK format, including validation regex. It does not enable an input mask and does not display any instruction text if the field fails validation.

|     |     |
| --- | --- |
| 1<br>2<br>3<br>4<br>5<br>6<br>7<br>8<br>9<br>10<br>11 | `add_filter(``'gform_phone_formats'``,``'uk_phone_format'``);`<br>`function``uk_phone_format(``$phone_formats``) {`<br>```$phone_formats``[``'uk'``] =``array``(`<br>```'label'``=>``'UK'``,`<br>```'mask'``=> false,`<br>```'regex'``=>``'/^(((\+44\s?\d{4}|\(?0\d{4}\)?)\s?\d{3}\s?\d{3})|((\+44\s?\d{3}|\(?0\d{3}\)?)\s?\d{3}\s?\d{4})|((\+44\s?\d{2}|\(?0\d{2}\)?)\s?\d{4}\s?\d{4}))(\s?\#(\d{4}|\d{3}))?$/'``,`<br>```'instruction'``=> false,`<br>```);`<br>```return``$phone_formats``;`<br>`}` |

### 2\. Add a Spanish phone format for all forms

The following example adds a Spanish format, including validation regex that will accept only Spanish numbers in local format (no international prefix). It does not enable an input mask and does not display any instruction text if the field fails validation.

|     |     |
| --- | --- |
| 1<br>2<br>3<br>4<br>5<br>6<br>7<br>8<br>9<br>10<br>11 | `add_filter(``'gform_phone_formats'``,``'es_phone_format'``);`<br>`function``es_phone_format(``$phone_formats``) {`<br>```$phone_formats``[``'es'``] =``array``(`<br>```'label'``=>``'Spain'``,`<br>```'mask'``=>``'999999999'``,`<br>```'regex'``=>``'/^[9|6|7|8][0-9]{8}$/'``,`<br>```'instruction'``=>``'Introduce los 9 dígitos sin guiones ni espacios.'``,`<br>```);`<br>```return``$phone_formats``;`<br>`}` |

### 2\. Add a 10 digits phone format for all forms

The following example adds a _10 Digits_ format, including input mask and validation regex.

|     |     |
| --- | --- |
| 1<br>2<br>3<br>4<br>5<br>6<br>7<br>8<br>9<br>10 | `add_filter(``'gform_phone_formats'``,``function``(``$phone_formats``) {`<br>```$phone_formats``[``'only_numbers'``] =``array``(`<br>```'label'``=>``'10 Digits'``,`<br>```'mask'``=>``'9999999999'``,`<br>```'regex'``=>``'/^[0-9]{10}$/'``,`<br>```'instruction'``=> false,`<br>```);`<br>``<br>```return``$phone_formats``;`<br>`} );` |

## Placement

This code can be used in the [functions.php](https://developer.wordpress.org/themes/basics/theme-functions/) file of the active theme, a custom functions plugin, a custom add-on, or with a code snippets plugin.

See also the PHP section in this article: [Where Do I Put This Code?](https://docs.gravityforms.com/where-do-i-put-this-code/)
