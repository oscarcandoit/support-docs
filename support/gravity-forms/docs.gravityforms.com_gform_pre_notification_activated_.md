## Description

The “gform\_pre\_notification\_activated” action is triggered before a notification is activated, allowing further actions to be performed.

## Usage

|     |     |
| --- | --- |
| 1 | `add_action(``'gform_pre_notification_activated'``,``'my_function'``, 10, 2 );` |

## Parameters

- **$notification** [Notifications Object](https://docs.gravityforms.com/notifications-object)

The notifications object.
- **$form** [array](https://docs.gravityforms.com/array)

The form object.
