## Description

This filter is executed before displaying each field and can be used to dynamically populate fields with a default value.

This filter can also be used to pre-select or pre-check drop down, radio button, and checkbox items. When pre-selecting or pre-checking choice-based fields, the value returned by your function must match the value of the label _exactly_ or, if you are using the option to ‘ _Show Values_‘, the value returned by the function must match the **value** exactly.

**Please note**: This filter requires that the “ [Allow field to be populated dynamically](https://docs.gravityforms.com/allow-field-to-be-populated-dynamically/)” option is checked in the field editor’s advanced tab. **Also:** avoid using a parameter name from this [list of reserved WordPress terms](https://developer.wordpress.org/reference/functions/register_taxonomy/#reserved-terms).

## Parameters

| Parameter | Type | Description |
| --- | --- | --- |
| **$value** | [string](https://docs.gravityforms.com/string) | The string containing the current field value to be filtered. |
| **$field** | [Field Object](https://docs.gravityforms.com/field-object) | The current field being processed. |
| **$name** | [string](https://docs.gravityforms.com/string) | The parameter name of the field or input being processed. |
