## Description

The “gform\_post\_form\_trashed” action is triggered after a form is moved to trash in Gravity Forms, allowing further actions to be performed.

## Usage

|     |     |
| --- | --- |
| 1 | `add_action(``'gform_post_form_trashed'``,``'my_function'``, 10, 1 );` |

## Parameters

- **$form\_id** [int](https://docs.gravityforms.com/int)
The ID of the form being moved to trash.
