# Entry Actions

## Articles

[**gform\_update\_status** \\
The “gform\_update\_status” action hook is a dynamically generated filter name.](https://docs.gravityforms.com/gform_update_status-dynamic/ "gform_update_status") [**gform\_delete\_entry** \\
The “gform\_delete\_entry” action fires right before an entry is deleted and is used to perform actions when an entry is deleted.](https://docs.gravityforms.com/gform_delete_entry/ "gform_delete_entry") [**gform\_post\_send\_entry\_note** \\
The “gform\_post\_send\_entry\_note” action is triggered after an entry note is emailed, allowing further actions to be performed.](https://docs.gravityforms.com/gform_post_send_entry_note/ "gform_post_send_entry_note") [**gform\_update\_PROPERTY\_NAME** \\
Use this action hook to perform logic when entries’ basic information is updated.](https://docs.gravityforms.com/gform_update_status/ "gform_update_PROPERTY_NAME") [**gform\_entries\_column** \\
Use this action to inject markup to any non-first column of every entry in the entry list grid.](https://docs.gravityforms.com/gform_entries_column/ "gform_entries_column") [**gform\_after\_update\_entry** \\
This hook fires after the entry has been updated via the entry detail page.](https://docs.gravityforms.com/gform_after_update_entry/ "gform_after_update_entry")
