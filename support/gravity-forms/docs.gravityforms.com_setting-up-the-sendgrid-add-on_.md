# Setting Up the SendGrid Add-On

- [Pre-Requisites](https://docs.gravityforms.com/setting-up-the-sendgrid-add-on/#h-pre-requisites)
- [Setup](https://docs.gravityforms.com/setting-up-the-sendgrid-add-on/#h-setup)

## Pre-Requisites

- [Download](https://docs.gravityforms.com/downloading-a-gravity-forms-add-on) and [install](https://docs.gravityforms.com/installing-a-gravity-forms-add-on/) the add-on
- A [SendGrid](https://sendgrid.com/) account

## Setup

Before you can begin setting up your notifications, you’ll need do a quick bit of setup so that Gravity Forms can authenticate with your SendGrid account. To set up the SendGrid Add-On, perform the following steps:

1. Log into your WordPress admin dashboard.
2. On the left side navigation menu, hover over **Forms** and click on **Settings**.
3. From this page, click the **SendGrid** tab.
4. Enter your **SendGrid API Key**. If you need help getting your API key, you can follow our article on [obtaining your SendGrid API key](https://docs.gravityforms.com/sendgrid-api-key/).
5. Click **Update Settings**. If you’ve entered your account information correctly, you will be notified by a green check mark beside the box. If you do not see a green check mark, please double check this information and try again.

Once you have entered your API Key, you’re ready to begin [sending form notifications using SendGrid](https://docs.gravityforms.com/form-notifications-sendgrid/).
