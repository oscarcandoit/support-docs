## Articles

[**gform/conditionalLogic/applyRules/start** \\
The gform/conditionalLogic/applyRules/start event allows you to run custom logic right before conditional rules are applied. It’s useful for prepping UI changes, showing loaders, or managing field state in conversational forms.](https://docs.gravityforms.com/gform-conditionallogic-applyrules-start/ "gform/conditionalLogic/applyRules/start") [**gform/post\_init** \\
This event is intended for use by add-ons that depend on core scripts. It fires once per form and provides the form ID as a parameter.](https://docs.gravityforms.com/gform-post_init/ "gform/post_init") [**gform/post\_render** \\
The gform/post\_render event can be used to initialize scripts when a form loads. It is a direct replacement for gform\_post\_render.](https://docs.gravityforms.com/gform-post_render/ "gform/post_render") [**gform/products/product\_field\_changed** \\
This JavaScript event is triggered when any price changes.](https://docs.gravityforms.com/gform-products-product_field_changed/ "gform/products/product_field_changed") [**gform/submission/pre\_submission** \\
The gform/submission/pre\_submission filter should be used to perform custom actions between a form button (e.g. submit, next, previous) being clicked and the request being posted to the server.](https://docs.gravityforms.com/gform-submission-pre_submission/ "gform/submission/pre_submission") [**gform/theme/scripts\_loaded** \\
The gform/theme/scripts\_loaded event can be used to initialize scripts that depend on Gravity Forms front-end form scripts, including the JavaScript gform object.](https://docs.gravityforms.com/gform-theme-scripts_loaded/ "gform/theme/scripts_loaded") [**GFSignature** \\
GFSignature is the class which houses some of the functionality included by the Gravity Forms Signature Add-on.](https://docs.gravityforms.com/gfsignature/ "GFSignature") [**GFStripe** \\
GFStripe is the class which houses the main functionality of the Gravity Forms Stripe Add-on, it extends the GFPaymentAddOn class which is part of the add-on framework.](https://docs.gravityforms.com/gfstripe/ "GFStripe") [**Google Analytics Feed Meta** \\
The Feed Meta Object for the Google Analytics Add-On is an associative array with properties determining how the add-on should process the form submission.](https://docs.gravityforms.com/google-analytics-feed-meta/ "Google Analytics Feed Meta") [**gquiz\_answer\_indicator** \\
