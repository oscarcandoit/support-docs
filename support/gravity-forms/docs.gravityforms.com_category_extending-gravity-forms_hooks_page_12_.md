## Articles

[**gform\_authorizenet\_transaction\_pre\_authorize** \\
Allows the transaction to be modified before authorization occurs.](https://docs.gravityforms.com/gform_authorizenet_transaction_pre_authorize/ "gform_authorizenet_transaction_pre_authorize") [**gform\_authorizenet\_transaction\_pre\_capture** \\
This filter can be used to modify the transaction object before it is sent to Authorize.net.](https://docs.gravityforms.com/gform_authorizenet_transaction_pre_capture/ "gform_authorizenet_transaction_pre_capture") [**gform\_authorizenet\_transaction\_pre\_capture\_setup\_fee** \\
This filter can be used to modify the transaction object for the subscription setup fee before it is sent to Authorize.net.](https://docs.gravityforms.com/gform_authorizenet_transaction_pre_capture_setup_fee/ "gform_authorizenet_transaction_pre_capture_setup_fee") [**gform\_authorizenet\_validation\_message** \\
This filter can be used to modify the validation message added to the credit card field when Authorize.net returns an error during card authorization.](https://docs.gravityforms.com/gform_authorizenet_validation_message/ "gform_authorizenet_validation_message") [**gform\_autoresponder\_email** \\
This hook has been deprecated in v.1.7. Please use gform\_notification instead.](https://docs.gravityforms.com/gform_autoresponder_email/ "gform_autoresponder_email") [**gform\_aweber\_ad\_tracking** \\
This filter can be used to modify the ad\_tracking parameter value before it is sent to AWeber.](https://docs.gravityforms.com/gform_aweber_ad_tracking/ "gform_aweber_ad_tracking") [**gform\_aweber\_args\_pre\_subscribe** \\
This filter can be used to modify the subscription parameters (arguments) before they are sent to AWeber.](https://docs.gravityforms.com/gform_aweber_args_pre_subscribe/ "gform_aweber_args_pre_subscribe") [**gform\_aweber\_field\_value** \\
