# Installing a Gravity Forms Add-On

- [Installing Gravity Forms Add-Ons using the Add-On Browser](https://docs.gravityforms.com/installing-a-gravity-forms-add-on/#h-installing-gravity-forms-add-ons-using-the-add-on-browser)
- [Installing Gravity Forms Add-Ons using the WordPress dashboard](https://docs.gravityforms.com/installing-a-gravity-forms-add-on/#h-installing-gravity-forms-add-ons-using-the-wordpress-dashboard)
- [Installing Gravity Forms Add-Ons using FTP](https://docs.gravityforms.com/installing-a-gravity-forms-add-on/#h-installing-gravity-forms-add-ons-using-ftp)
- [Activate the Add-On and Review the Settings](https://docs.gravityforms.com/installing-a-gravity-forms-add-on/#h-activate-the-add-on-and-review-the-settings)

Installing Gravity Forms Add-Ons is as simple as installing Gravity Forms. The simplest method is using the Add-On browser. In this article, we will show you each of the possible methods that can be used for installation.

## Installing Gravity Forms Add-Ons using the Add-On Browser

1. [![installing-gf-addons-1](https://docs.gravityforms.com/wp-content/uploads/2015/03/installing-gf-addons-1-300x177.png)](https://docs.gravityforms.com/wp-content/uploads/2015/03/installing-gf-addons-1.png) Log into your WordPress admin dashboard.
2. [![installing-gf-addons-2](https://docs.gravityforms.com/wp-content/uploads/2015/03/installing-gf-addons-2-296x300.png)](https://docs.gravityforms.com/wp-content/uploads/2015/03/installing-gf-addons-2.png) Hover over **Forms** and click on **Add-Ons**.
3. [![installing-gf-addons-3](https://docs.gravityforms.com/wp-content/uploads/2015/03/installing-gf-addons-3-300x174.png)](https://docs.gravityforms.com/wp-content/uploads/2015/03/installing-gf-addons-3.png) Here you will see a list of all available add-ons. To install one, simply click the **Install** button. Once the _Install_ button is clicked, WordPress will handle the download and installation of the Gravity Forms Add-On. Be sure to click **Activate Plugin** to activate the add-on.

## Installing Gravity Forms Add-Ons using the WordPress dashboard

1. [![installing-gravity-forms-addons-1](https://docs.gravityforms.com/wp-content/uploads/2015/03/installing-gravity-forms-addons-1-300x271.png)](https://docs.gravityforms.com/wp-content/uploads/2015/03/installing-gravity-forms-addons-1.png) Download the zip file for your desired add-on. Access the [Gravity Forms Add-On download page](https://www.gravityforms.com/my-account/downloads/), click the add-on that you want to install, and download the zip file for it.
2. [![installing-gravity-forms-addons-2](https://docs.gravityforms.com/wp-content/uploads/2015/03/installing-gravity-forms-addons-2-300x177.png)](https://docs.gravityforms.com/wp-content/uploads/2015/03/installing-gravity-forms-addons-2.png) Log into your WordPress admin dashboard.
3. [![installing-gravity-forms-addons-3](https://docs.gravityforms.com/wp-content/uploads/2015/03/installing-gravity-forms-addons-3-300x198.png)](https://docs.gravityforms.com/wp-content/uploads/2015/03/installing-gravity-forms-addons-3.png) Hover over **Plugins** on the left side admin menu and click on **Add New**.
4. [![installing-gravity-forms-addons-4](https://docs.gravityforms.com/wp-content/uploads/2015/03/installing-gravity-forms-addons-4-300x171.png)](https://docs.gravityforms.com/wp-content/uploads/2015/03/installing-gravity-forms-addons-4.png) At the top of this page, click on **Upload Plugin**. Then click on **Choose file** and locate the zip file containing the add-on. Click the **Install Now** button.
5. [![installing-gravity-forms-addons-5](https://docs.gravityforms.com/wp-content/uploads/2015/03/installing-gravity-forms-addons-5-300x176.png)](https://docs.gravityforms.com/wp-content/uploads/2015/03/installing-gravity-forms-addons-5.png) WordPress will automatically handle the unpacking and installation for your Gravity Forms add-on. Once complete, click on **Activate Plugin** to activate it.

## Installing Gravity Forms Add-Ons using FTP

1. Download the add-on zip file from the [Gravity Forms add-on download page](https://www.gravityforms.com/my-account/downloads/) and unpack it.
2. Once you have your add-on files unzipped, connect to your server over FTP and navigate to the _wp\_content/plugins_ folder and upload the add-on files that you have unpacked.
3. Once the files have been successfully uploaded to the server, log into your WordPress admin dashboard.
4. Click on **Plugins** from the left side navigation menu.
5. You should see an entry for the add-on that you have uploaded. Click **Activate** to activate it.

## Activate the Add-On and Review the Settings

Now that you have successfully installed an add-on for Gravity Forms, don’t forget to go to your **WordPress admin Plugins page** and activate it. There may also be settings required to enable the add-on’s functionality. Further information on individual add-ons can be found in their respective articles under the [Add-Ons document category](https://docs.gravityforms.com/category/add-ons-gravity-forms/).
