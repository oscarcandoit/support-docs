## Description

Use this filter to override the field value before it is included in the CSV export. This filter can be used in conjunction with the [gform\_export\_fields](https://docs.gravityforms.com/gform_export_fields) filter to include custom columns in the export.

## Usage

|     |
| --- |
| `add_filter(``'gform_export_field_value'``,``'your_function_name'``, 10, 4 );` |

## Parameters

- **$value** [string](https://docs.gravityforms.com/string)
Value of the field being exported.

- **$form\_id** [integer](https://docs.gravityforms.com/integer)
ID of the current form.

- **$field\_id** [integer](https://docs.gravityforms.com/integer)

ID of the current field.

- **$entry** [Entry Object](https://docs.gravityforms.com/entry-object)

The current entry.
