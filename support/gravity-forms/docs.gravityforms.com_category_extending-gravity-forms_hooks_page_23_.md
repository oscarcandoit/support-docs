## Articles

[**gform\_display\_disable\_css\_setting** \\
The filter gform\_display\_disable\_css\_setting indicates if the Output Default CSS setting should be displayed or not.](https://docs.gravityforms.com/gform_display_disable_css_setting/ "gform_display_disable_css_setting") [**gform\_display\_field\_select\_columns\_entry\_list** \\
Allows fields to be added or removed from the \_select columns\_ UI on the entry list.](https://docs.gravityforms.com/gform_display_field_select_columns_entry_list/ "gform_display_field_select_columns_entry_list") [**gform\_display\_product\_summary** \\
This filter can be used to prevent the pricing summary being included in the output generated when using the {all\_fields} merge tag.](https://docs.gravityforms.com/gform_display_product_summary/ "gform_display_product_summary") [**gform\_dropbox\_file\_name** \\
This filter can be used to override the filename before the file is uploaded to Dropbox.](https://docs.gravityforms.com/gform_dropbox_file_name/ "gform_dropbox_file_name") [**gform\_dropbox\_folder\_path** \\
This filter can be used to override the destination folder configured on the Dropbox feed.](https://docs.gravityforms.com/gform_dropbox_folder_path/ "gform_dropbox_folder_path") [**gform\_dropbox\_link\_type** \\
This filter can be used to change the type of link returned by Dropbox.](https://docs.gravityforms.com/gform_dropbox_link_type/ "gform_dropbox_link_type") [**gform\_dropbox\_post\_upload** \\
This hook is fired after a Dropbox feed has been processed and files have been uploaded.](https://docs.gravityforms.com/gform_dropbox_post_upload/ "gform_dropbox_post_upload") [**gform\_dropbox\_request\_timeout** \\
This filter in the Gravity Forms Dropbox Add-On allows the request timeout length for the feed processing request to be modified.](https://docs.gravityforms.com/gform_dropbox_request_timeout/ "gform_dropbox_request_timeout") [**gform\_dropbox\_shareable\_link\_settings** \\
