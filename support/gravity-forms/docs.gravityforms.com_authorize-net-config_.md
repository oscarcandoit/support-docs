- [Customer Fields Properties](https://docs.gravityforms.com/authorize-net-config/#h-customer-fields-properties)

## Introduction

The Authorize.Net Config ($config) is an associative array; it is populated by the properties from the [Feed Object](https://docs.gravityforms.com/feed-object). It is used to ensure hooks which were available in the pre-framework versions of the plugin remain backwards compatible, simply making the new [Feed Object](https://docs.gravityforms.com/feed-object) available to these hooks instead could break existing code.

|     |     |
| --- | --- |
| 1<br>2<br>3<br>4<br>5<br>6 | `$config``=``array``(`<br>```'id'``=> 5,`<br>```'form_id'``=> 10,`<br>```'is_active'``=> true,`<br>```'meta'``=>``array``(),`<br>`);` |

## Usage

We recommend accessing the $config properties using the rgar() or rgars() functions, e.g.:

|     |     |
| --- | --- |
| 1<br>2 | `$meta``= rgar(``$config``,``'meta'``);`<br>`$conditional_logic_enabled``= rgars(``$config``,``'meta/authorizenet_conditional_enabled'``);` |

## Properties

- **id** [integer](https://docs.gravityforms.com/integer)
The feed ID.

- **form\_id** [integer](https://docs.gravityforms.com/integer)

The form ID this feed was created for.

- **is\_active** [boolean](https://docs.gravityforms.com/boolean)

Is the feed active or inactive. Default is true.

- **meta** [array](https://docs.gravityforms.com/array)

Config meta is an associative array. See below for the available properties.


## Meta Properties

- **type** [string](https://docs.gravityforms.com/string)

The type of transaction which should occur when the feed is processed. _Possible values: product or subscription._

- **customer\_fields** [array](https://docs.gravityforms.com/array)

An associative array containing the [Customer Fields Properties](https://docs.gravityforms.com/authorize-net-config/#customer-fields-properties).

- **enable\_receipt** [boolean](https://docs.gravityforms.com/boolean)

Send Authorize.Net email receipt? Default is false.

- **api\_settings\_enabled** [boolean](https://docs.gravityforms.com/boolean)

Override the default API settings? Default is false.

- **api\_mode** [string](https://docs.gravityforms.com/string)

Which mode should this feed use when overriding the default API settings. _Possible values: production or test._

- **api\_login** [string](https://docs.gravityforms.com/string)

The API login ID to use when overriding the default API settings.

- **api\_key** [string](https://docs.gravityforms.com/string)

The API transaction key to use when overriding the default API settings.

- **authorizenet\_conditional\_enabled** [boolean](https://docs.gravityforms.com/boolean)

Is the feed condition (conditional logic) setting enabled. Default is false.

- **authorizenet\_conditional\_field\_id** [integer](https://docs.gravityforms.com/integer)

Target field Id. Field that will have it’s value compared with the _authorizenet\_conditional\_value_ property to determine if this rule is a match.

- **authorizenet\_conditional\_operator** [string](https://docs.gravityforms.com/string)

Operator to be used when evaluating this rule. _Possible values: is, isnot, >, <, contains, starts\_with, or ends\_with._

- **authorizenet\_conditional\_value** [string](https://docs.gravityforms.com/string)

The value to compare with field specified by _authorizenet\_conditional\_field\_id_.


## Subscription Transaction Meta Properties

The following properties are only applicable when the **type** is set to **subscription**.

- **recurring\_amount\_field** [string](https://docs.gravityforms.com/string)
What to use for the recurring amount. _Possible values: form\_total or a product field ID_

- **billing\_cycle\_number** [integer](https://docs.gravityforms.com/integer)

How often should the recurring payment occur. _Minimum values: 7 days or 1 month_. _Maximum values: 365 days or 12 months._

- **billing\_cycle\_type** [string](https://docs.gravityforms.com/string)

What unit to use with the **billing\_cycle\_number**. _Possible values: day or month._

- **recurring\_times** [integer](https://docs.gravityforms.com/integer)

How many times the recurring payment should be made. The default (0) is to bill the customer until the subscription is canceled.

- **setup\_fee\_enabled** [boolean](https://docs.gravityforms.com/boolean)

Should a setup fee be charged? Default is false.

- **setup\_fee\_amount\_field** [string](https://docs.gravityforms.com/string)

The ID of the product field containing the amount the user should be charged for the setup fee.

- **trial\_period\_enabled** [boolean](https://docs.gravityforms.com/boolean)

Should the recurring charge occur immediately or after a trial period? Default false.

- **trial\_period\_number** [integer](https://docs.gravityforms.com/integer)

The duration of the trial. _Default value: 1._

- **trial\_amount** [string](https://docs.gravityforms.com/string)

The amount the user should be charged during the trial if **enter\_amount** is selected for the **trial\_product**.

- **update\_post\_action** [string](https://docs.gravityforms.com/string)

Change the post status when a subscription is canceled? _Possible values: draft or delete._


## Customer Fields Properties

- **email** [string](https://docs.gravityforms.com/string)

The ID of the field containing the customer’s email.

- **address1** [string](https://docs.gravityforms.com/string)

The ID of the field containing the first line of the customer’s address.

- **address2** [string](https://docs.gravityforms.com/string)

The ID of the field containing the second line of the customer’s address.

- **city** [string](https://docs.gravityforms.com/string)

The ID of the field containing the city of the customer’s address.

- **state** [string](https://docs.gravityforms.com/string)

The ID of the field containing the state of the customer’s address.

- **zip** [string](https://docs.gravityforms.com/string)

The ID of the field containing the zip of the customer’s address.

- **country** [string](https://docs.gravityforms.com/string)

The ID of the field containing the country of the customer’s address.
