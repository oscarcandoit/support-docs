## Articles

[**gform\_notification\_ui\_settings** \\
This filter has been deprecated. Use the gform\_notification\_settings\_fields filter instead.](https://docs.gravityforms.com/gform_notification_ui_settings/ "gform_notification_ui_settings") [**gform\_notification\_validation** \\
This filter was removed in Gravity Forms v2.5 and is no longer supported. From Gravity Forms 2.5, notification settings and their validation callbacks are now defined via the gform\_notification\_settings\_fields filter.](https://docs.gravityforms.com/gform_notification_validation/ "gform_notification_validation") [**gform\_order\_label** \\
The "gform\_order\_label" filter in Gravity Forms is executed when displaying the "Order" heading on the entry detail page or when sending the order information via email.](https://docs.gravityforms.com/gform_order_label/ "gform_order_label") [**gform\_order\_summary** \\
Filters the markup of the order summary which appears on the Entry Detail, the {all\_fields} merge tag and the {pricing\_fields} merge tag.](https://docs.gravityforms.com/gform_order_summary/ "gform_order_summary") [**gform\_other\_choice\_value** \\
Change the default “Other” placeholder text on Radio Button fields.](https://docs.gravityforms.com/gform_other_choice_value/ "gform_other_choice_value") [**gform\_page\_size\_form\_list** \\
The gform\_page\_size\_form\_list filter allows developers to modify the number of forms displayed per page in the Forms list.](https://docs.gravityforms.com/gform_page_size_form_list/ "gform_page_size_form_list") [**gform\_partialentries\_abort\_save** \\
The gform\_partialentries\_abort\_save filter allows you to stop a partial entry from being saved or updated. You can use it to prevent partial entries from being saved under specific conditions during form processing.](https://docs.gravityforms.com/gform_partialentries_abort_save/ "gform_partialentries_abort_save") [**gform\_partialentries\_post\_EVENT** \\
Perform a custom action when the partial entry has been saved or updated.](https://docs.gravityforms.com/gform_partialentries_post_event/ "gform_partialentries_post_EVENT") [**gform\_partialentries\_pre\_update** \\
The gform\_partialentries\_pre\_update filter is used to modify an existing partial entry before the database is updated.](https://docs.gravityforms.com/gform_partialentries_pre_update/ "gform_partialentries_pre_update") [**gform\_partialentries\_warning\_message** \\
