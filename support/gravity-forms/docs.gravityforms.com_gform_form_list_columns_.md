## Description

Allows the column headers displayed on the Form List page to be modified.

## Usage

|     |     |
| --- | --- |
| 1 | `add_filter(``'gform_form_list_columns'``,``'your_function_name'``, 10, 1 );` |

## Parameters

- **$columns**


An array of the columns to display.






|     |     |
| --- | --- |
| 1<br>2<br>3<br>4<br>5<br>6<br>7<br>8<br>9<br>10 | `//default values`<br>`array``(`<br>```'cb'``=>``'<input type="checkbox" />'``,`<br>```'is_active'``=>``''``,`<br>```'title'``=> esc_html__(``'Title'``,``'gravityforms'``),`<br>```'id'``=> esc_html__(``'ID'``,``'gravityforms'``),`<br>```'entry_count'``=> esc_html__(``'Entries'``,``'gravityforms'``),`<br>```'view_count'``=> esc_html__(``'Views'``,``'gravityforms'``),`<br>```'conversion'``=> esc_html__(``'Conversion'``,``'gravityforms'``),`<br>```);` |


## Examples

This example renames **Title** to **Form Title**, removes the column for the checkbox to apply actions, removes entry count, removes view count, and removes the conversion.

| 1<br>2<br>3<br>4<br>5<br>6<br>7<br>8<br>9 | `add_filter(``'gform_form_list_columns'``,``'change_columns'``, 10, 1 );`<br>`function``change_columns(``$columns``){`<br>```$columns``=``array``(`<br>```'is_active'``=>``''``,`<br>```'id'``=> esc_html__(``'ID'``,``'gravityforms'``),`<br>```'title'``=> esc_html__(``'Form Title'``,``'gravityforms'``),`<br>```);`<br>```return``$columns``;`<br>`}` |

## Placement

This code should be placed in the [functions.php](http://codex.wordpress.org/Theme_Development#Functions_File) file of your active theme.

## Since

This filter was added in Gravity Forms version 2.0.

## Source Code

This filter is located in GF\_Form\_List\_Table::get\_columns() in _form\_list.php_.
