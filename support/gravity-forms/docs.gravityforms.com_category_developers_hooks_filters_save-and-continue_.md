# Save and Continue Filters

## Articles

[**gform\_get\_form\_save\_confirmation\_filter** \\
This filter allows the form save confirmation text to be programmatically changed before it is rendered to the page.](https://docs.gravityforms.com/gform_get_form_save_confirmation_filter/ "gform_get_form_save_confirmation_filter") [**gform\_get\_form\_save\_email\_confirmation\_filter** \\
Allows the Save and Continue confirmation message to be changed.](https://docs.gravityforms.com/gform_get_form_save_email_confirmation_filter/ "gform_get_form_save_email_confirmation_filter") [**gform\_incomplete\_submission\_post\_get** \\
Allows the draft submission for Save and Continue to be overridden after it is retrieved from the database but before it used to populate the form.](https://docs.gravityforms.com/gform_incomplete_submission_post_get/ "gform_incomplete_submission_post_get") [**gform\_incomplete\_submission\_pre\_save** \\
Allows the draft submission for Save and Continue to be overridden before saved to the database.](https://docs.gravityforms.com/gform_incomplete_submission_pre_save/ "gform_incomplete_submission_pre_save") [**gform\_save\_and\_continue\_resume\_url** \\
Use this filter to change the resume link generated when an incomplete submission is saved.](https://docs.gravityforms.com/gform_save_and_continue_resume_url/ "gform_save_and_continue_resume_url") [**gform\_savecontinue\_expired\_message** \\
Use this filter to modify the error message displayed when an expired or invalid Save and Continue link is used.](https://docs.gravityforms.com/gform_savecontinue_expired_message/ "gform_savecontinue_expired_message") [**gform\_savecontinue\_link** \\
Filters the save and continue link, allowing the tag to be customized.](https://docs.gravityforms.com/gform_savecontinue_link/ "gform_savecontinue_link")
