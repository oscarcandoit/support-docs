# gform\_user\_registration\_add\_option\_section

- [Description](https://docs.gravityforms.com/gform_user_registration_add_option_section/#h-description)
- [Usage](https://docs.gravityforms.com/gform_user_registration_add_option_section/#h-usage)
- [Parameters](https://docs.gravityforms.com/gform_user_registration_add_option_section/#h-parameters)
- [Examples](https://docs.gravityforms.com/gform_user_registration_add_option_section/#h-examples)
- [Source Code](https://docs.gravityforms.com/gform_user_registration_add_option_section/#h-source-code)

**Removed:** This hook was removed in version 3.0 and replaced by [gform\_userregistration\_feed\_settings\_fields](https://docs.gravityforms.com/gform_userregistration_feed_settings_fields).

## Description

Used to add a custom option section to the User Registration Feed (similar to how BuddyPress and Multisite are integrated with the User Registration add-on).

## Usage

Applies to all forms.

|     |     |
| --- | --- |
| 1 | `add_action(``'gform_user_registration_add_option_section'``,``'your_function_name'``, 10, 3 );` |

## Parameters

- **$config** [array](https://docs.gravityforms.com/array/)

The User Registration configuration array.
- **$form** [Form Object](https://docs.gravityforms.com/form-object/)

The Form Object for which the current user registration feed is for.
- **$is\_validation\_error** [boolean](https://docs.gravityforms.com/boolean/)

Boolean value indicating whether there was a validation error with the User Registration configuration.
