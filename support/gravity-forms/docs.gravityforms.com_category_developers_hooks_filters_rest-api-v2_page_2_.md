# REST API v2 Filters

## Articles

[**gform\_rest\_api\_capability\_post\_forms** \\
Filters the capability required to create forms via the REST API v2.](https://docs.gravityforms.com/gform_rest_api_capability_post_forms/ "gform_rest_api_capability_post_forms") [**gform\_rest\_api\_capability\_put\_entries** \\
Filters the capability required to update entries via the REST API v2.](https://docs.gravityforms.com/gform_rest_api_capability_put_entries/ "gform_rest_api_capability_put_entries") [**gform\_rest\_api\_capability\_put\_feeds** \\
Filters the capability required to update feeds via the REST API v2.](https://docs.gravityforms.com/gform_rest_api_capability_put_feeds/ "gform_rest_api_capability_put_feeds") [**gform\_rest\_api\_capability\_put\_forms** \\
Filters the capability required to update forms via the REST API v2.](https://docs.gravityforms.com/gform_rest_api_capability_put_forms/ "gform_rest_api_capability_put_forms") [**gform\_rest\_api\_retrieve\_form\_totals** \\
The gform\_rest\_api\_retrieve\_form\_totals filter allows the entries property (entry totals) to be omitted from the GET /forms (REST API v2) response.](https://docs.gravityforms.com/gform_rest_api_retrieve_form_totals/ "gform_rest_api_retrieve_form_totals") [**gform\_webapi\_get\_users\_settings\_page** \\
Use this filter to control the list of users available to be selected in the “Impersonate account” setting. Note that this doesn’t control access to the API, it just controls the list of users available on the settings page.](https://docs.gravityforms.com/gform_webapi_get_users_settings_page/ "gform_webapi_get_users_settings_page") [**gform\_webapi\_key\_user\_capabilities** \\
The gform\_webapi\_key\_user\_capabilities filter allows the list of capabilities which are used to determine if a user is included in the User drop down menu when creating a new REST API Key to be overridden.](https://docs.gravityforms.com/gform_webapi_key_user_capabilities/ "gform_webapi_key_user_capabilities") [Previous](https://docs.gravityforms.com/category/developers/hooks/filters/rest-api-v2/) [1](https://docs.gravityforms.com/category/developers/hooks/filters/rest-api-v2/) 2
