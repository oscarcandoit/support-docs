## Description

This filter is executed when the form is displayed and can be used to completely change the form tag (i.e. <form method=”post”>).

## Usage

The following would apply to all forms.

|     |
| --- |
| `add_filter(``'gform_form_tag'``,``'your_function_name'``, 10, 2 );` |

To limit the scope of your function to a specific form, append the form id to the end of the hook name. (format: gform\_form\_tag\_FORMID)

|     |
| --- |
| `add_filter(``'gform_form_tag_6'``,``'your_function_name'``, 10, 2 );` |

## Parameters

- **$form\_tag** [string](https://docs.gravityforms.com/string)
The string containing the <form> tag

- **$form** [Form Object](https://docs.gravityforms.com/form-object)
The current form.
