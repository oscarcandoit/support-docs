# Form Component Filters

## Articles

[**gform\_next\_button** \\
This filter allows the markup for the next button to be changed, giving the user control over how it looks.](https://docs.gravityforms.com/gform_next_button/ "gform_next_button") [**gform\_pre\_process** \\
Allows filtering the Form Object before the form submission has begun processing.](https://docs.gravityforms.com/gform_pre_process/ "gform_pre_process") [**gform\_previous\_button** \\
Allows the markup for the previous button to be changed.](https://docs.gravityforms.com/gform_previous_button/ "gform_previous_button") [**gform\_progress\_bar** \\
Use this filter to modify or replace the default Gravity Forms progress bar (for multi-page forms).](https://docs.gravityforms.com/gform_progress_bar/ "gform_progress_bar") [**gform\_progress\_steps** \\
The gform\_progress\_steps filter can be used to modify or replace the default progress steps markup for multi-page forms.](https://docs.gravityforms.com/gform_progress_steps/ "gform_progress_steps") [**gform\_progressbar\_start\_at\_zero** \\
Use this filter to change the progress bar on multi-page forms to start at zero percent.](https://docs.gravityforms.com/gform_progressbar_start_at_zero/ "gform_progressbar_start_at_zero") [**gform\_required\_legend** \\
The gform\_required\_legend filter enables the legend displayed at the bottom of the form header, which explains how required fields are indicated, to be overridden.](https://docs.gravityforms.com/gform_required_legend/ "gform_required_legend") [**gform\_shortcode\_ACTION** \\
This filter is used to implement custom shortcode actions.](https://docs.gravityforms.com/gform_shortcode_form_property/ "gform_shortcode_ACTION") [**gform\_shortcode\_builder\_forms** \\
The gform\_shortcode\_builder\_forms filter allowss the list of available forms displayed in the shortcode builder to be overridden.](https://docs.gravityforms.com/gform_shortcode_builder_forms/ "gform_shortcode_builder_forms") [**gform\_submit\_button** \\
This filter is executed when the form is displayed and can be used to completely change the form button tag.](https://docs.gravityforms.com/gform_submit_button/ "gform_submit_button") [Previous](https://docs.gravityforms.com/category/developers/hooks/filters/form-component-filters/) [1](https://docs.gravityforms.com/category/developers/hooks/filters/form-component-filters/) 2
