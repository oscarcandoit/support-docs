## Description

The gform\_editor\_field\_settings filter allows the editor settings that are used for the current field, including those inherited from the inputType, to be overridden.

## Usage

The filter which would run for all forms and fields would be used like so:

|     |     |
| --- | --- |
| 1<br>2<br>3<br>4<br>5 | `gform.addFilter(``'gform_editor_field_settings'``,``function``( settingsArray, field ) {`<br>```// do stuff`<br>```return``settingsArray;`<br>`} );` |

## Parameters

- **settingsArray** [array](https://docs.gravityforms.com/array)
The current settings for the field. See [Field Settings](https://docs.gravityforms.com/category/developers/php-api/field-framework/field-framework-settings/) for available settings.





|     |     |
| --- | --- |
| 1<br>2<br>3<br>4<br>5<br>6<br>7<br>8<br>9<br>10<br>11<br>12<br>13<br>14<br>15<br>16<br>17<br>18<br>19<br>20 | `[`<br>```".conditional_logic_field_setting"``,`<br>```".prepopulate_field_setting"``,`<br>```".error_message_setting"``,`<br>```".label_setting"``,`<br>```".label_placement_setting"``,`<br>```".admin_label_setting"``,`<br>```".size_setting"``,`<br>```".input_mask_setting"``,`<br>```".maxlen_setting"``,`<br>```".password_field_setting"``,`<br>```".rules_setting"``,`<br>```".visibility_setting"``,`<br>```".duplicate_setting"``,`<br>```".default_value_setting"``,`<br>```".placeholder_setting"``,`<br>```".description_setting"``,`<br>```".css_class_setting"``,`<br>```".autocomplete_setting"`<br>`]` |

- **field** [Javascript Object](https://docs.gravityforms.com/javascript-object) \| [Field Object](https://docs.gravityforms.com/field-object)

The field being modified.

### 1\. Remove a setting

This example shows how a setting can be removed from a specific field type.

| 1<br>2<br>3<br>4<br>5<br>6<br>7<br>8<br>9<br>10<br>11<br>12<br>13<br>14<br>15<br>16<br>17<br>18 | `gform.addFilter(``'gform_editor_field_settings'``,``function``( settings, field ) {`<br>```if``( field.type !==``'quiz'``) {`<br>```return``settings;`<br>```}`<br>```// Find the index of the choices setting`<br>```var``i = settings.indexOf(``'.choices_setting'``);`<br>```// Doesn't exist; bail.`<br>```if``( i === -1 ) {`<br>```return``settings;`<br>```}`<br>```// Remove the choices_setting value.`<br>```settings.splice( i, 1 );`<br>```return``settings;`<br>`} );` |

## Placement

This code should be placed in a JavaScript file included in the admin by your plugin.
