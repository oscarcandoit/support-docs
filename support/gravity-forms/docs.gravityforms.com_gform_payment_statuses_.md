## Description

The “gform\_payment\_statuses” filter in Gravity Forms allows custom payment statuses to be defined. Refer to [Gravity Forms Entry Payment Statuses](https://docs.gravityforms.com/gravity-forms-entry-payment-statuses/) article for more information about payment statuses.

## Usage

|     |     |
| --- | --- |
| 1 | `add_filter(``'gform_payment_statuses'``,``'your_function_name'``, 10, 1);` |

## Parameters

- **$payment\_statuses** [array](https://docs.gravityforms.com/array/)

An array of entry payment statuses with the entry value as the key (15 char max) to the text for display.

## Placement

This code can be used in the [functions.php](https://developer.wordpress.org/themes/basics/theme-functions/) file of the active theme, a custom functions plugin, a custom add-on, or with a code snippets plugin.

See also the PHP section in this article: [Where Do I Put This Code?](https://docs.gravityforms.com/where-do-i-put-this-code/)

## Since

This filter was added in Gravity Forms 2.4.

This filter is located in GFCommon::get\_entry\_payment\_statuses() in _gravityforms/common.php_.
