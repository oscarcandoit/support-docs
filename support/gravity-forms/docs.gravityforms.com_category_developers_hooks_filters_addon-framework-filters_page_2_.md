# Add-On Framework Filters

## Articles

[**gform\_allow\_async\_feed\_reprocessing** \\
The gform\_allow\_async\_feed\_reprocessing filter enables the reprocessing feeds that have already been processed via background tasks.](https://docs.gravityforms.com/gform_allow_async_feed_reprocessing/ "gform_allow_async_feed_reprocessing") [**gform\_entry\_meta\_pre\_render\_feed\_settings** \\
Allows population of the JavaScript entry\_meta variable on the feed configuration page.](https://docs.gravityforms.com/gform_entry_meta_pre_render_feed_settings/ "gform_entry_meta_pre_render_feed_settings") [**gform\_feed\_settings\_before\_fields** \\
The gform\_feed\_settings\_before\_fields filter allows developers to change the content that is displayed before the feed settings](https://docs.gravityforms.com/gform_feed_settings_before_fields/ "gform_feed_settings_before_fields") [**gform\_filter\_ui** \\
Allows modifying the existing filter layout or styling.](https://docs.gravityforms.com/gform_filter_ui/ "gform_filter_ui") [**gform\_filters\_pre\_results** \\
Use this filter to modify the filters used in the drop down list of fields in the admin results page.](https://docs.gravityforms.com/gform_filters_pre_results/ "gform_filters_pre_results") [**gform\_form\_list\_count** \\
This filter allows the form list count array to be modified. This is useful when filtering the form count list.](https://docs.gravityforms.com/gform_form_list_count/ "gform_form_list_count") [**gform\_form\_list\_forms** \\
This filter allows the forms displayed on the form listing page to be filtered.](https://docs.gravityforms.com/gform_form_list_forms/ "gform_form_list_forms") [**gform\_form\_pre\_results** \\
Use this filter to modify the Form object prior to calculating the results in the results admin page.](https://docs.gravityforms.com/gform_form_pre_results/ "gform_form_pre_results") [**gform\_gf\_field\_create** \\
Use this filter to modify or replace the GF\_Field object after it has been created.](https://docs.gravityforms.com/gform_gf_field_create/ "gform_gf_field_create") [**gform\_is\_delayed\_pre\_process\_feed** \\
This filter can be used to delay feed processing for Add-On Framework based add-ons.](https://docs.gravityforms.com/gform_is_delayed_pre_process_feed/ "gform_is_delayed_pre_process_feed") [Previous](https://docs.gravityforms.com/category/developers/hooks/filters/addon-framework-filters/) [1](https://docs.gravityforms.com/category/developers/hooks/filters/addon-framework-filters/) 2 [3](https://docs.gravityforms.com/category/developers/hooks/filters/addon-framework-filters/page/3/) [Next](https://docs.gravityforms.com/category/developers/hooks/filters/addon-framework-filters/page/3/)
