## Description

`gform_submission_files_pre_save_field_value` allows filtering (e.g., renaming) of the submission files before they are saved to the form uploads folder and entry.

## Usage

|     |     |
| --- | --- |
| 1 | `add_filter(``'gform_submission_files_pre_save_field_value'``,``'my_function'``, 10, 4 );` |

## Parameters

| **Parameter** | **Type** | **Description** |
| --- | --- | --- |
| **$files** | [array](https://docs.gravityforms.com/array/) | Array of submission files. See [Files Parameters](https://docs.gravityforms.com/gform_submission_files_pre_save_field_value/#h-files-parameters). |
| **$field** | [GF\_Field\_FileUpload](https://docs.gravityforms.com/gf_field_fileupload/) | The File Upload field the files are for. |
| **$entry** | [array](https://docs.gravityforms.com/array/) | The entry currently being saved. Only fields located before the current field are included. |
| **$form** | [array](https://docs.gravityforms.com/array/) | The form currently being processed. |

### Files Parameters

| **Parameter** | **Type** | **Description** |
| --- | --- | --- |
| **new** | [array](https://docs.gravityforms.com/array/) | Array of new files from `$_FILES`. See [New File Parameters](https://docs.gravityforms.com/gform_submission_files_pre_save_field_value/#h-new-file-parameters). |
| **existing** | [array](https://docs.gravityforms.com/array/) | Array of dynamically populated file URLs and/or files that have already been uploaded to the form tmp folder. See [Existing File Parameters](https://docs.gravityforms.com/gform_submission_files_pre_save_field_value/#h-existing-file-parameters). |

#### New File Parameters

| **Parameter** | **Type** | **Description** |
| --- | --- | --- |
| **name** | string | The original name of the file on the client machine. |
| **type** | string | The MIME type of the file provided by the browser. |
| **size** | integer | The size of the file in bytes. |
| **tmp\_name** | string | The temporary filename of the file in which the uploaded file was stored on the server. |
| **error** | integer | The [PHP error code](https://www.php.net/manual/en/filesystem.constants.php#constant.upload-err-cant-write) for the file upload. |
| **full\_path** | string | The full path as submitted by the browser. Only populated with PHP 8.1+. |

#### Existing File Parameters

| **Parameter** | **Type** | **Description** |
| --- | --- | --- |
| **uploaded\_filename** | string | The name of the uploaded file, an existing file, or the name parsed from the populated URL. |
| **temp\_filename** | string\|null | The temporary name of the file. Only present if the file has been saved to the form tmp folder. |
| **url** | string\|null | The file URL. Only present if the file has been dynamically populated on initial form display. |

## Examples

### Rename uploaded files to include the entry and field IDs.

|     |     |
| --- | --- |
| 1<br>2<br>3<br>4<br>5<br>6<br>7<br>8<br>9<br>10<br>11<br>12<br>13<br>14<br>15<br>16<br>17<br>18<br>19<br>20<br>21<br>22<br>23 | `add_filter(``'gform_submission_files_pre_save_field_value'``,``function``(``$files``,``$field``,``$entry``) {`<br>```$entry_id``= absint( rgar(``$entry``,``'id'``) );`<br>```foreach``(``$files``[``'existing'``]``as``&``$existing_file``) {`<br>```$existing_file``[``'uploaded_filename'``] = sprintf(`<br>```'entry-%d-field-%d-%s'``,`<br>```$entry_id``,`<br>```$field``->id,`<br>```$existing_file``[``'uploaded_filename'``]`<br>```);`<br>```}`<br>```foreach``(``$files``[``'new'``]``as``&``$new_file``) {`<br>```$new_file``[``'name'``] = sprintf(`<br>```'entry-%d-field-%d-%s'``,`<br>```$entry_id``,`<br>```$field``->id,`<br>```$new_file``[``'name'``]`<br>```);`<br>```}`<br>```return``$files``;`<br>`}, 10, 3 );` |

## Placement

This code can be used in the [functions.php](https://developer.wordpress.org/themes/basics/theme-functions/) file of the active theme, a custom functions plugin, a custom add-on, or with a code snippets plugin.

See also the PHP section in this article: [Where Do I Put This Code?](https://docs.gravityforms.com/where-do-i-put-this-code/)

## Source Code

The filter is located in `GF_Field_FileUpload::filter_submission_files_pre_save()` in `includes/fields/class-gf-field-fileupload.php`
