# Obtaining Your Campaign Monitor API Key

To begin using the Campaign Monitor Add-On for Gravity Forms, you will first need to add your API key. In this article, we will show you how to retrieve your Campaign Monitor API key.

1. Log into your [Campaign Monitor](https://www.campaignmonitor.com/) account.
2. [![Campaign Monitor Account Settings](https://docs.gravityforms.com/wp-content/uploads/2015/05/Campaign-Monitor-Account-Settings-noted-300x33.png)](https://docs.gravityforms.com/wp-content/uploads/2015/05/Campaign-Monitor-Account-Settings-noted.png) Click on **Account settings** at the top of the page.
3. [![Campaign Monitor API Keys link](https://docs.gravityforms.com/wp-content/uploads/2015/05/Campaign-Monitor-API-Keys-link-249x300.png)](https://docs.gravityforms.com/wp-content/uploads/2015/05/Campaign-Monitor-API-Keys-link.png) Click on the **API Keys** link on the right side of the page to view your API Keys.
