# Monitoring How-Tos

Generated on 25 Nov 2025

DigitalOcean Monitoring is a free, opt-in service that lets you track Droplet resource usage in real time, visualize performance metrics, and receive alerts via email or Slack to proactively manage your infrastructure’s health.

## Getting Started

![](https://docs.digitalocean.com/images/icons/monitoring.svg)

How to Install the DigitalOcean Metrics Agent

Install the DigitalOcean metrics agent to enable extended Droplet monitoring.

![](https://docs.digitalocean.com/images/icons/monitoring.svg)

Install the Metrics Agent via Metrics Agent Repository

Install the DigitalOcean metrics agent manually by configuring the repository directly.

![](https://docs.digitalocean.com/images/icons/monitoring.svg)

How to Uninstall the DigitalOcean Metrics Agent

Uninstall the DigitalOcean metrics agent from your Droplet to stop monitoring and remove related system resources.

![](https://docs.digitalocean.com/images/icons/monitoring.svg)

How to Create, Edit, and Delete Resource Alerts

Create, edit, view, and delete resource alerts to monitor your Droplets and receive notifications when key metrics exceed or fall below thresholds you define.

## Monitoring Management

![](https://docs.digitalocean.com/images/icons/monitoring.svg)

How to Upgrade Your Legacy Metrics Agent

Remove the legacy metrics agent and install the newer Prometheus-based version to restore extended metrics visibility.
