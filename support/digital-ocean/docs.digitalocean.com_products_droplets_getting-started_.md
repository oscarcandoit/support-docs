# Getting Started with Droplets

Validated on 26 Sep 2024 • Last edited on 17 Apr 2025

DigitalOcean Droplets are Linux-based virtual machines (VMs) that run on top of virtualized hardware. Each Droplet you create is a new server you can use, either standalone or as part of a larger, cloud-based infrastructure.

![](https://docs.digitalocean.com/images/icons/droplets.svg)

Droplet Quickstart

Just the essentials to go from zero to working in a few minutes.

![](https://docs.digitalocean.com/images/icons/droplets.svg)

Set up a Production-Ready Droplet

Create a new Droplet with our recommended configuration for improved security, reliability, and monitoring.

![](https://docs.digitalocean.com/images/icons/droplets.svg)

Recommended GPU Setup

Follow our recommended setup for drivers and software on GPU Droplets to use their GPUs.
