# Notebooks Features

Validated on 3 May 2024 • Last edited on 25 Oct 2024

Notebooks are a web-based Jupyter IDE with shared persistent storage for long-term development and inter-notebook collaboration, backed by accelerated compute.

Notebooks provide a fully-managed IDE for exploring data and building machine learning models. Notebooks include a free GPU plan. You can launch a GPU-enabled Jupyter Notebook from your browser, choose a pre-built template or bring your own, and execute Jupyter files and projects in other languages such as Python.

Notebooks are built on Docker containers which allows for fast startup time. You can also create a team, invite collaborators, set permissions, fork public projects, or create your own notebooks.

## Machines Available with Subscription Plans

Paperspace provides a wide variety of [CPU, GPU, and multi-GPU machines](https://docs.digitalocean.com/products/paperspace/pricing/#paperspace-compute). We regularly add new machine types. Some machines are available for free for Notebooks. Types of free machines increase with each level of [Gradient Subscription Plan](https://docs.digitalocean.com/products/paperspace/pricing/#paperspace-gradient).

## Machine Auto-shutdown

Paperspace allows you to [set an auto-shutdown time limit](https://docs.digitalocean.com/products/paperspace/machines/how-to/manage-auto-shutdown/) on all machine types. This is to prevent unnecessary account charges for long running processes. You can set the auto-shutdown timer between 1 hour and 1 week. You can remove the limit but should closely monitor the usage to avoid large utilization charges.

When using free-tier machines, the maximum auto-shutdown timeout is 6 hours.
