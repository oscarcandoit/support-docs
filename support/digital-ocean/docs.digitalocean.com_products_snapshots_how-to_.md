# Snapshots How-Tos

Generated on 25 Nov 2025

Snapshots are on-demand disk images of DigitalOcean Droplets and volumes saved to your account. Use them to create new Droplets and volumes with the same contents.

## Getting Started with Droplet Snapshots

![](https://docs.digitalocean.com/images/icons/snapshots.svg)

How to Create Snapshots of Droplets

Snapshot a Droplet to save the contents of the Droplet’s disk.

![](https://docs.digitalocean.com/images/icons/snapshots.svg)

How to Create or Restore Droplets from Snapshots

Create a new Droplet from a snapshot or restore a Droplet to an older point in time.

![](https://docs.digitalocean.com/images/icons/snapshots.svg)

How to Migrate Droplets Using DigitalOcean Snapshots

Use snapshots to migrate a Droplet from one data center region to another.

## Getting Started with Volume Snapshots

![](https://docs.digitalocean.com/images/icons/snapshots.svg)

How to Create Snapshots of Volumes

Snapshot a volume to save the contents of the volume’s disk.

![](https://docs.digitalocean.com/images/icons/snapshots.svg)

How to Create Volumes from Snapshots

Create a new volume from a snapshot.

## Snapshot Management

![](https://docs.digitalocean.com/images/icons/snapshots.svg)

How to Add Droplet Snapshots to Other Regions

Add a Droplet snapshot to additional regions to create Droplets based on it in those regions.

![](https://docs.digitalocean.com/images/icons/snapshots.svg)

How to Transfer a Droplet Snapshot to a Different Team

Transfer Droplet snapshots to another DigitalOcean team to effectively migrate a Droplet to another owner.

![](https://docs.digitalocean.com/images/icons/snapshots.svg)

How to Delete Snapshots

Delete a snapshot to permanently remove it from your account.
