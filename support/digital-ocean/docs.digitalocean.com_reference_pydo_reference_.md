# DigitalOcean PyDo Python Library Reference

Generated on 25 Nov 2025
from `pydo` version
[`v0.21.0`](https://github.com/digitalocean/pydo/releases/tag/v0.21.0)

PyDo is the official Python client library that allows Python developers to interact with and manage their DigitalOcean resources through a Python abstraction layer on top of the raw [DigitalOcean API HTTP Interface](https://docs.digitalocean.com/reference/api/digitalocean/).

A top priority of this project is to ensure the client abides by the API contract. Therefore, the client itself wraps a generated client based on the DigitalOcean OpenAPI Specification to support all of DigitalOcean’s HTTP APIs.

Content hosted in DigitalOcean’s object storage solution,

![](https://docs.digitalocean.com/images/icons/python.svg)

pydo.snapshots

[Snapshots](https://docs.digitalocean.com/products/snapshots/) are saved
instances of a Droplet or a block storage volume, which is reflected in
the `resource_type` attribute.

![](https://docs.digitalocean.com/images/icons/python.svg)

pydo.spaces\_key

Spaces keys authenticate requests to DigitalOcean Spaces Buckets.

![](https://docs.digitalocean.com/images/icons/python.svg)

pydo.ssh\_keys

Manage SSH keys available on your account.

pydo.tags

A tag is a label that can be applied to a resource (currently Droplets,

better organize or facilitate the lookups and actions on it.

![](https://docs.digitalocean.com/images/icons/python.svg)

pydo.uptime

[DigitalOcean Uptime Checks](https://docs.digitalocean.com/products/uptime/) provide the ability to monitor your endpoints from around the world, and alert you when they’re slow, unavailable, or SSL certificates are expiring.

![](https://docs.digitalocean.com/images/icons/python.svg)

pydo.volume\_actions

Block storage actions are commands that can be given to a DigitalOcean
Block Storage Volume.

![](https://docs.digitalocean.com/images/icons/python.svg)

pydo.volume\_snapshots

between Droplets within a specific region.

pydo.volumes

[DigitalOcean Block Storage Volumes](https://docs.digitalocean.com/products/volumes/)
provide expanded storage capacity for your Droplets and can be moved
between Droplets within a specific region.

![](https://docs.digitalocean.com/images/icons/python.svg)

pydo.vpc\_peerings

[VPC Peerings](https://docs.digitalocean.com/products/networking/vpc/how-to/create-peering/)
join two VPC networks with a secure, private connection.

![](https://docs.digitalocean.com/images/icons/python.svg)

pydo.vpcnatgateways

[VPC NAT Gateways](https://docs.digitalocean.com/products/networking/vpc/how-to/create-nat-gateway/)
allow resources in a private VPC to access the public internet without
exposing them to incoming traffic.

![](https://docs.digitalocean.com/images/icons/python.svg)

allow you to create virtual networks containing resources that can
communicate with each other in full isolation using private IP addresses.
