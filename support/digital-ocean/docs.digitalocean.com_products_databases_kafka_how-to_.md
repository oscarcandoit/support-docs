# Kafka How-Tos

Generated on 25 Nov 2025

Kafka is an open-source distributed event and stream-processing platform built to process demanding real-time data feeds. It is inherently scalable, with high throughput and availability.

## Getting Started

![](https://docs.digitalocean.com/images/icons/kafka.svg)

How to Create Kafka Clusters

Create a Kafka cluster from the DigitalOcean Control Panel.

![](https://docs.digitalocean.com/images/icons/kafka.svg)

How to Connect to Kafka Clusters

Connect to Kafka clusters from the command line or other applications.

![](https://docs.digitalocean.com/images/icons/kafka.svg)

How to Create Kafka Topics

Create a Kafka Topic from the DigitalOcean Control Panel.

![](https://docs.digitalocean.com/images/icons/kafka.svg)

How to Secure Kafka Managed Clusters

Add additional security to a Kafka managed database cluster by restricting incoming connections and using encrypted connections.

![](https://docs.digitalocean.com/images/icons/kafka.svg)

How to Schedule Automatic Software Updates for Kafka Clusters

Customize the maintenance window for automatic software updates to your database cluster.

![](https://docs.digitalocean.com/images/icons/kafka.svg)

How to Tag Kafka Clusters

Organize Kafka databases clusters with tags to group and filter databases or create monitoring alert policies for multiple databases at once.

Access and interpret Kafka database performance metrics.

Access and understand performance metrics for nodes in a database cluster.

How to Forward Logs from a Kafka Cluster

How to Enable Schema Registry on Kafka Clusters
