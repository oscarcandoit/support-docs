# cPanel & WHM® (License)

Generated on 16 Sep 2024
from [the cPanel & WHM® (License) catalog page](https://marketplace.digitalocean.com/add-ons/cpanel-license)

cPanel is the industry-leading web hosting control panel, renowned for its intuitive interface and robust feature set, making it a top choice for hosting providers, resellers, and website owners worldwide. Whether you are a beginner or a seasoned developer, cPanel simplifies website and server management with a user-friendly dashboard that streamlines tasks like file management, database administration, and email setup. With tools like Softaculous, you can launch popular platforms such as WordPress and Joomla with a single click, while advanced email tools and real-time malware scanning enhance productivity and security. Its scalability ensures it works for small personal sites as well as enterprise-level solutions, offering unmatched reliability and control.

Key Features:

- Unlimited Domains & DNS integration
- WP Toolkit with automated staging/cloning included
- Support for NodeJS, Ruby, Docker and LAMP stacks w/ Apache or NGINX
- New Laravel Toolkit and support for .NET on Linux - Security & backups across all levels of a website or application stack
- Subscription Management, Account Management and Reseller Management - cPanel Extensions in the in-app catalog
- Access to Sitejet for a fully integrated DIY website builder
- SocialBee an All-In-One Social Media Management Tool
- SEO Toolkit that guides you towards maximizing your search engine traffic The benefits of cPanel extend beyond ease of use.

It automates complex server tasks such as account activations and suspensions saving time for hosting providers and resellers, while end-users enjoy seamless integration with leading CMS platforms. Security is a priority, with features like malware protection and comprehensive email management keeping your digital assets safe. For developers, cPanel offers API access, command-line options, and compatibility with tools like phpMyAdmin for database management. With over 20 years of industry leadership, cPanel continues to evolve, providing cutting-edge features like Site Quality Monitoring, IPv6 support, and regular updates, all backed by 24/7 expert support and extensive online resources.

cPanel runs on DigitalOcean infrastructure, simplifying the lives of Web Professionals and providing the scalability, security, and performance that your customers depend on.

DigitalOcean droplet required.
