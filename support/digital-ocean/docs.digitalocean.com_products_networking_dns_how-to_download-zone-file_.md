# How to Download DNS Zone Files

Validated on 12 Aug 2025 • Last edited on 2 Sep 2025

Adding a domain you own to your DigitalOcean account lets you manage the domain’s DNS records with the control panel and API. Domains you manage on DigitalOcean integrate with DigitalOcean Load Balancers and Spaces to streamline automatic SSL certificate management.

DNS zone files are plain text files that contain all of the record information for your domain on DigitalOcean. You can download this file to migrate DNS configurations to [other DigitalOcean Teams](https://docs.digitalocean.com/platform/teams/) or DNS providers.

To download a zone file for a domain, log in to the [control panel](https://cloud.digitalocean.com/) and click **Networking** in the main menu, then go to the **Domains** tab.

![Two domains listed, one with its menu open.](https://docs.digitalocean.com/screenshots/dns/download-zone-file.c3e3b6c533bcb139970aa4112d414138953e798f0d6e5618a369bd23866fe27f.png)

Open the **More** menu of the domain whose zone file you want to download, and then click **Download zone**. The zone file downloads to your local machine.
