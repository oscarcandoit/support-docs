# Notebooks Quickstart

Validated on 14 Dec 2023 • Last edited on 18 Dec 2024

Notebooks are a web-based Jupyter IDE with shared persistent storage for long-term development and inter-notebook collaboration, backed by accelerated compute.

## Create a Notebook from a Template

1. Use the **New** button to start creating a notebook.

2. Choose a template that specifies the image and repository the notebook is based on.

3. Choose the machine type that the notebook initially started on.

4. Set an auto-shutdown timeout to automatically shut down the notebook after a period of inactivity.

5. Specify advanced options such as workspace.

6. Click **Start notebook** to spin up your environment.
