# 1-Click App

Last edited on 18 Dec 2024

1-Click Apps streamline the setup process for servers and Kubernetes clusters by automating tasks such as package installation, firewall rule configuration, load balancer provisioning, and software setup.

Droplet 1-Click Apps are pre-built Droplet images that come with preconfigured software, features, and settings. Kubernetes 1-Click Apps are pre-configured applications for Kubernetes with software, features, and configuration details already in place.

`doctl` lets you interact with 1-Click Apps from the command line. For more information, see [`doctl 1-click`](https://docs.digitalocean.com/reference/doctl/reference/1-click/) and [`doctl kubernetes 1-click`](https://docs.digitalocean.com/reference/doctl/reference/kubernetes/1-click/).
