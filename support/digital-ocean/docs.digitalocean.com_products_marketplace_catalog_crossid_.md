# Crossid

Generated on 16 Oct 2023
from [the Crossid catalog page](https://marketplace.digitalocean.com/add-ons/crossid)

## Crossid is an adoptable identity platform.

- Login box using various passwordless authentication factors ( _otp_, _fingerprint_, etc…)
- Log users in via social providers such as _Facebook_ or _Google_.
- Registration.
- Multi-factor authentication.
- Protect APIs using industry-standard protocols such as OAuth2 and OIDC.
- Manage user profiles and access.
