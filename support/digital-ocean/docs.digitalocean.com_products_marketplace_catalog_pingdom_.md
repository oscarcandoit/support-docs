# Pingdom

Generated on 5 Apr 2021
from [the Pingdom catalog page](https://marketplace.digitalocean.com/apps/pingdom)

Web application performance monitoring designed to help you optimize and quickly troubleshoot performance issues.

- Test the availability of web resources every minute from over 100 locations worldwide and be the first to know of an outage
- Get alerted if site flows - such as signup, checkout, and search - are unavailable or broken
- Track page load times and identify performance bottlenecks to optimize the end-user experience
