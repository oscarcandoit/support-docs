# Paperspace API Keys

Validated on 14 Dec 2023 • Last edited on 23 Jan 2025

API keys let you interact with Paperspace through the Core RESTful API, the Core Javascript SDK, and Gradient command-line utility (CLI). API Keys have team-scoped permissions and are associated with your user account.

## Create an API Key

Warning
Keep your API keys secret. They function like passwords. Do not hard code your API keys into applications where they may accidentally be released in version control. Instead, use environmental variables or [secrets](https://docs.digitalocean.com/products/paperspace/accounts-and-teams/use-secrets/). If an API key becomes compromised, delete it.

- Click the profile icon in the top right of the console and then click **Team Settings** from the drop-down.
- On the API keys tab, enter a name and click the Add button.

![](https://docs.digitalocean.com/screenshots/paperspace/security-api-key.1ddd0cf5792ee166808e1b64e8725b78d6dbe3148d45b92fc8a0470bc06c099e.png)

Make sure to copy your new API key now. You won’t be able to see it again after leaving this page.

![](https://docs.digitalocean.com/screenshots/paperspace/security-api-key-2.da1b2d0b3b9e31675409c313979253f0c4a4ca2b75d84c925ce124943aa4d295.png)

## Revoke an API Key

You can revoke your API keys by clicking revoke in the console. If you are a [team admin](https://docs.digitalocean.com/products/paperspace/accounts-and-teams/#team-roles), you can also revoke keys that belong to other team members.

![](https://docs.digitalocean.com/screenshots/paperspace/security-api-key-revoke.819dc3a0430fcb883c05f333e941548956039559e269fd24c5e0e7e654842e74.png)
