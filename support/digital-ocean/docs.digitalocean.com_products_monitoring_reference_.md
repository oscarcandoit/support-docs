# Monitoring Reference

Validated on 22 Apr 2022 • Last edited on 17 Jun 2025

DigitalOcean Monitoring is a free, opt-in service that lets you track Droplet resource usage in real time, visualize performance metrics, and receive alerts via email or Slack to proactively manage your infrastructure’s health.

The [DigitalOcean API](https://docs.digitalocean.com/reference/) lets you manage resources programmatically with standard HTTP requests. All actions available in the control panel are also available through the API.

[`doctl`](https://github.com/digitalocean/doctl) is the command-line interface for the DigitalOcean API. It supports most of the same actions available in the API and DigitalOcean Control Panel.

![](https://docs.digitalocean.com/images/icons/default.svg)

DigitalOcean Monitoring API Reference

Use the DigitalOcean API to programmatically retrieve metrics and configure metrics alert policies.

![](https://docs.digitalocean.com/images/icons/monitoring.svg)

doctl compute volume

The sub-commands of doctl monitoring manages monitoring on your account from the command line.
