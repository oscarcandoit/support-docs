# Supported OpenSearch Plugins

Validated on 1 Jul 2024 • Last edited on 9 Jul 2024

OpenSearch is an open-source search and analytics suite which serves as a centralized location to manage logs forwarded from other resources, such as databases and Droplets.

DigitalOcean OpenSearch managed databases supports the following plugins. This list is current as of the **Validated** date on this page.

You can request support for an extension on [ideas.digitalocean.com](https://ideas.digitalocean.com/) by either [voting for an existing request](https://ideas.digitalocean.com/ideas/search?project=DBAAS&query=extension) or [adding a new idea](https://ideas.digitalocean.com/ideas/new) for managed databases.

- [ICU Analysis](https://www.elastic.co/guide/en/elasticsearch/plugins/current/analysis-icu.html)
- [Phonetic Analysis](https://www.elastic.co/guide/en/elasticsearch/plugins/current/analysis-phonetic.html)
- [Kuromoji (Japanese Analysis)](https://www.elastic.co/guide/en/elasticsearch/plugins/current/analysis-kuromoji.html)
- [Mapper Size](https://www.elastic.co/guide/en/elasticsearch/plugins/current/mapper-size.html)
- [OpenSearch SQL](https://opensearch.org/docs/latest/search-plugins/sql/index/)
- [OpenSearch Dashboards Alerting](https://opensearch.org/docs/latest/monitoring-plugins/alerting/index/)
- [Anomaly Detection](https://opensearch.org/docs/monitoring-plugins/ad/index/)
- [Asynchronous Search](https://opensearch.org/docs/search-plugins/async/index/)
- [Index Management](https://opensearch.org/docs/im-plugin/index/)
- [k-NN](https://opensearch.org/docs/search-plugins/knn/index/)
- [Notebooks](https://opensearch.org/docs/dashboards/notebooks/)
- [OpenSearch Dashboard Reports](https://github.com/opensearch-project/dashboards-reports)
- [Scheduler for Dashboard Reports](https://github.com/opensearch-project/dashboards-reports)
- [OpenSearch Dashboards Gantt Charts](https://opensearch.org/docs/latest/dashboards/gantt/)
- [OpenSearch Dashboards Trace Analytics](https://opensearch.org/docs/latest/monitoring-plugins/trace/index/)
- [ML Commons](https://opensearch.org/docs/latest/ml-commons-plugin/index/)
- [OpenSearch Observability](https://opensearch.org/docs/latest/observing-your-data/index/)
- [OpenSearch Security](https://opensearch.org/docs/latest/security/index/)
- [OpenSearch Security Analytics](https://opensearch.org/docs/latest/security-analytics/index/)
- [OpenSearch Neural Speech](https://opensearch.org/docs/latest/search-plugins/neural-search/)
- [OpenSearch Notification](https://opensearch.org/docs/latest/observing-your-data/notifications/index/)
