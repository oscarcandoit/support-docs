# Storage

Validated on 30 Apr 2024 • Last edited on 20 Oct 2025

Store and access any amount of data reliably in the cloud, with S3-compatible Spaces Object Storage, network-based Volumes block storage, or NFS-based Network File Storage.

![](https://docs.digitalocean.com/images/icons/volumes.svg)

Volumes Block Storage

Volumes are network-based block devices that provide additional data storage for Droplets.

![](https://docs.digitalocean.com/images/icons/spaces.svg)

Spaces Object Storage

Spaces Object Storage is an S3-compatible service for storing and serving large amounts of data.

![](https://docs.digitalocean.com/images/icons/nfs.svg)

Network File Storage

Network File Storage is a fully managed, POSIX-compliant file storage solution built for demanding workloads.

## Latest Updates

### 21 November 2025

- [Network File Storage](https://docs.digitalocean.com/products/nfs/) is now available in the `NYC2` datacenter.


### 20 October 2025

- [Network File Storage](https://docs.digitalocean.com/products/nfs/) is now in general availability in the `ATL1` datacenter. Network File Storage provides a fully managed, POSIX-compliant file storage solution based on the Network File System protocol.


### 10 June 2025

- [Spaces access logs](https://docs.digitalocean.com/products/spaces/how-to/access-logs/) are now in general availability.


For more information, see [the full release notes](https://docs.digitalocean.com/release-notes/).
