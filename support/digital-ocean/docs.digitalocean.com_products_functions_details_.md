# Functions Details

Generated on 25 Nov 2025

Functions are blocks of code that run on demand without the need to manage any infrastructure. Develop on your local machine, test your code from the command line (using `doctl`), then deploy to a production namespace or App Platform — no servers required.

![](https://docs.digitalocean.com/images/icons/functions.svg)

Functions Features

Functions is a function as a service (FaaS) offering that lets you run your local serverless code in the cloud using Node.js, Python, Go, or PHP.

![](https://docs.digitalocean.com/images/icons/functions.svg)
