# Getting Started with App Platform

Validated on 9 Aug 2023 • Last edited on 17 Apr 2025

App Platform is a Platform-as-a-Service (PaaS) offering that allows developers to publish code directly to DigitalOcean servers without worrying about the underlying infrastructure.

![](https://docs.digitalocean.com/images/icons/app-platform.svg)

App Platform Quickstart

Get your source code live on App Platform in a few minutes.

![](https://docs.digitalocean.com/images/icons/app-platform.svg)

App Platform Sample Apps

Sample applications to get started on DigitalOcean App Platform.

![](https://docs.digitalocean.com/images/icons/app-platform.svg)

Deploy Sample Apps to App Platform

Fork and launch sample applications for several popular programming languages.

![](https://docs.digitalocean.com/images/icons/app-platform.svg)

Deploy the Etherpad Collaborative Web Editor to App Platform

Learn how to deploy the official Etherpad Docker image to DigitalOcean’s App Platform and connect it to a managed PostgreSQL database.

![](https://docs.digitalocean.com/images/icons/app-platform.svg)

Deploy a Sample RSS Reader to App Platform

Build a multi-component sample RSS reader app and then deploy the app to App Platform.
