# DigitalOcean Marketplace Droplet 1-Click Apps

Validated on 25 Oct 2024 • Last edited on 6 Jun 2025

Droplet 1-Click Apps are pre-built Droplet images that come with preconfigured software, features, and settings.

These apps minimize setup requirements and provide a environment for testing software or deploying applications.

To create a Droplet using a 1-Click App, click the **Create Droplet** button on any Marketplace 1-Click App page, or use the image IDs or slugs for the 1-Click App in API requests and CLI commands to create Droplets.
