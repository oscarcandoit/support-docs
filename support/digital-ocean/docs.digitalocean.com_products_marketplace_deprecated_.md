# One-Click Applications Deprecated Before October 2018

Validated on 14 Sep 2018 • Last edited on 1 Mar 2021

As of September 2018, all deprecation notices for One-Click applications will be included in the [release notes](https://docs.digitalocean.com/release-notes/). Prior to this, the following One-Click applications were deprecated:

- **Apache Cassandra**. See instead the [Apache Cassandra homepage](https://cassandra.apache.org/).
- **Drone**. See instead the [Drone homepage](https://drone.io/).
- **Drupal**. See instead the [Drupal homepage](https://www.drupal.org/) and [our community tutorials on Drupal](https://www.digitalocean.com/community/tags/drupal?type=tutorials).
- **ELK**. See instead the [Elastic Stack homepage](https://www.elastic.co/elk-stack) and [our community tutorials on Elasticsearch](https://www.digitalocean.com/community/tags/elasticsearch?type=tutorials).
- **Machine Learning**. You can follow [our community tutorials on machine learning](https://www.digitalocean.com/community/tags/machine-learning?type=tutorials).
- **Magento**. See instead the [Magento homepage](https://magento.com/).
- **MEAN**. See instead [our community tutorials on MEAN](https://www.digitalocean.com/community/tags/mean?type=tutorials).
- **MediaWiki**. See instead the [MediaWiki homepage](https://www.mediawiki.org/wiki/MediaWiki).
- **Mumble**. See instead the [Mumble homepage](https://www.mumble.com/).
- **.Net Core and Powershell**. See instead the [PowerShell homepage](https://docs.microsoft.com/en-us/powershell/) and the [.NET core homepage](https://docs.microsoft.com/en-us/dotnet/core/).
- **Redis**. See instead the [Redis homepage](https://redis.io/) and [our community tutorials on Redis](https://www.digitalocean.com/community/tags/redis?type=tutorials).
- **Redmine**. See instead the [Redmine homepage](https://www.redmine.org/) .

Share your thoughts and feedback with us at [ideas.digitalocean.com](https://ideas.digitalocean.com/).
