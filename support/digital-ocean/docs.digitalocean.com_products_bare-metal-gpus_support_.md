# DigitalOcean Gradient™ AI Bare Metal GPUs Support

Generated on 25 Nov 2025

DigitalOcean Gradient™ AI Bare Metal GPUs are dedicated, single-tenant servers with 8 GPUs of various models that can operate standalone or in multi-node clusters.

![](https://docs.digitalocean.com/images/icons/bare-metal-gpus.svg)

Contact Support for DigitalOcean Gradient™ AI Bare Metal GPUs

Get help with bare metal GPUs by submitting a support ticket.

![](https://docs.digitalocean.com/images/icons/support.svg)

Can I make BIOS-level changes to bare metal GPUs?

You can submit a support ticket to request a change in the BIOS.

![](https://docs.digitalocean.com/images/icons/support.svg)

Why is a speed test for my bare metal GPU showing slower network speeds?

Public speed test servers may not accurately reflect network performance in data center environments. We recommend using tools like `iperf` for end-to-end tests.
