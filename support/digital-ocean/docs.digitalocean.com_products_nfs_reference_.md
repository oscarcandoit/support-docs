# Network File Storage Reference

Validated on 13 Oct 2025 • Last edited on 20 Oct 2025

Network File Storage is a fully managed, POSIX-compliant file storage solution built for demanding workloads like AI/ML pipelines, containerized applications, and DigitalOcean Kubernetes (DOKS) clusters. It provides scalable, high-throughput shared storage that simplifies storage management for distributed applications.

## Additional Tools for Network File Storage

### The DigitalOcean API

The [DigitalOcean API](https://docs.digitalocean.com/reference/) lets you manage resources programmatically with standard HTTP requests. All actions available in the control panel are also available through the API.

To create, delete, and manage Network File Storage shares via the DigitalOcean API, see the [NFS section of our API reference](https://docs.digitalocean.com/reference/api/digitalocean/#tag/NFS).
