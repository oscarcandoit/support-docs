# Obtaining Your MailerLite API Key

To use the MailerLite Add-On, you will need to first obtain the MailerLite API key. In this article, we will show you exactly how to create a new API key.

1. Log into your [MailerLite](https://accounts.mailerlite.com/dashboard) account.
2. Click on **Integrations** in the left sidebar.![MailerLite Integrations](https://docs.gravityforms.com/wp-content/uploads/2024/04/Markup-2024-04-29-at-09.32.48.png)
3. Under the MailerLite API section, click on **Use** next to API.![MailerLite API](https://docs.gravityforms.com/wp-content/uploads/2024/04/Markup-2024-04-29-at-09.38.32.png)
4. Next to API Tokens, click **Generate new token**. ![API Token](https://docs.gravityforms.com/wp-content/uploads/2024/04/Markup-2024-04-29-at-09.40.40.png)
5. Enter a **token name**, **check the box** to accept the terms, and click **Create token**.![Generate API Key](https://docs.gravityforms.com/wp-content/uploads/2024/04/Markup-2024-04-29-at-09.43.43.png)
6. Click the **Copy** button to copy your new token. ![](https://docs.gravityforms.com/wp-content/uploads/2024/04/Markup-2024-04-29-at-09.45.05.png)

Now, you can paste this key into your Gravity Forms MailerLite Add-On settings. Refer to the [Setting up the MailerLite Add-On](https://docs.gravityforms.com/setting-up-the-mailerlite-add-on/) documentation for further details.

If you need further help to get your API key, we recommend you contact [MailerLite support](https://www.mailerlite.com/support).
