## Description

The gform\_field\_map\_choices filter can be used to override the choices available in the field map drop down.

## Usage

```
add_filter( 'gform_field_map_choices', 'your_function_name', 10, 4 );
```

## Parameters

- **$fields** [array](https://docs.gravityforms.com/array)
The value and label properties for each choice.

- **$form\_id** [integer](https://docs.gravityforms.com/integer)

The ID of the form currently being configured.

- **$field\_type** [null](https://docs.gravityforms.com/null) \| [array](https://docs.gravityforms.com/array)

Null or the field types to be included in the drop down.

- **$exclude\_field\_types** [null](https://docs.gravityforms.com/null) \| [array](https://docs.gravityforms.com/array) \| [string](https://docs.gravityforms.com/string)

Null or the field type(s) to be excluded from the drop down.


## Examples

### 1\. Add new choice

```
add_filter( 'gform_field_map_choices', function( $fields, $form_id, $field_type, $exclude_field_types ) {
	$fields[] = array( 'label' => 'The new choice', 'value' => 'new_choice' );

	return $fields;
}, 10, 4 );
```

## Placement

This code should be placed in the [functions.php](http://codex.wordpress.org/Theme_Development#Functions_File) file of your active theme.

## Source Code

This filter is located in _\\Gravity\_Forms\\Gravity\_Forms\\Settings\\Fields\\Generic\_Map::get\_value\_choices()_ in _includes/settings/fields/class-generic-map.php_.

## Since

This filter was added in Gravity Forms 2.5.
