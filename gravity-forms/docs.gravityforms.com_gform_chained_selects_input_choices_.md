## Description

The gform\_chained\_selects\_input\_choices filter is used to modify the choices that will appear in each drop down of a Chained Select field.

**IMPORTANT**: This filter will not work properly when the page where your form is embedded is cached. This is not a Gravity Forms limitation but a consequence of using caching.

## Usage

Filter all chained selects for all forms.

|     |     |
| --- | --- |
| 1 | `add_filter(``'gform_chained_selects_input_choices'``,``'your_function_name'``);` |

Filter all chained selects for a specific form.

|     |     |
| --- | --- |
| 1 | `add_filter(``'gform_chained_selects_input_choices_6'``,``'your_function_name'``);` |

Filter chained selects for a specific form and specific field.

|     |     |
| --- | --- |
| 1 | `add_filter(``'gform_chained_selects_input_choices_6_2'``,``'your_function_name'``);` |

Filter a specific chained select for a specific form and field.

|     |     |
| --- | --- |
| 1 | `add_filter(``'gform_chained_selects_input_choices_6_2_1'``,``'your_function_name'``);` |

## Parameters

- **$input\_choices** [array](https://docs.gravityforms.com/array)

An array of [Choices](https://docs.gravityforms.com/gform_predefined_choices/) that will be used to populate the current drop down.
- **$form\_id** [integer](https://docs.gravityforms.com/integer)

The form ID for the current form.
- **$field** [Field Object](https://docs.gravityforms.com/field-object)

The current field object.
- **$input\_id** [float](https://docs.gravityforms.com/float)

The input ID of the current input (i.e. drop down) being processed. Example: 3.1.
- **$full\_chain\_value** [array](https://docs.gravityforms.com/array)

An array of values representing the selected value for each input in the chain of selects. Example:



```
array(
      '3.1' => 'Toyota',
      '3.2' => 'Matrix',
      '3.3' => 2013
);
```

- **$value** [string](https://docs.gravityforms.com/string)

The value of the current chained select.
- **$index** [int](https://docs.gravityforms.com/int) – Added in version 1.1.

The index of the specific chained select.
