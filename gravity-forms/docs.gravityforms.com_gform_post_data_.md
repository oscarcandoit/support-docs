## Description

This filter is executed right before the post is created, allowing post data to be manipulated before the post is created.

**Note**: This filter only applies to forms that have Post Fields.

## Usage

|     |     |
| --- | --- |
| 1 | `add_filter(``'gform_post_data'``,``'your_function_name'``, 10, 3 );` |

## Parameters

- **$post\_data** [array](https://docs.gravityforms.com/array)
The array containing the post data to be filtered. This array is in the format used by the WP function _[wp\_insert\_post](http://codex.wordpress.org/Function_Reference/wp_insert_post)($post\_data)_

- **$form** [Form Object](https://docs.gravityforms.com/form-object)

The form currently being processed.

- **$entry** [Entry Object](https://docs.gravityforms.com/entry-object)

The entry currently being processed.
