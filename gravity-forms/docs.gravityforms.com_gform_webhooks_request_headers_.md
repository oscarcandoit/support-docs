## Description

The “gform\_webhooks\_request\_headers” filter allows the webhook HTTP request headers to be modified.

## Usage

The following would apply to all forms:

|     |     |
| --- | --- |
| 1 | `add_filter(``'gform_webhooks_request_headers'``'your_function_name'``, 10, 4 );` |

To target a specific form, append the form id to the hook name. (format: gform\_webhooks\_request\_headers\_FORMID)

|     |     |
| --- | --- |
| 1 | `add_filter(``'gform_webhooks_request_headers_14'``'your_function_name'``, 10, 4 );` |

## Parameters

- **$request\_headers** [array](https://docs.gravityforms.com/array)
HTTP request headers.

- **$feed** [Feed Object](https://docs.gravityforms.com/feed-object)

The feed object.

- **$entry** [Entry Object](https://docs.gravityforms.com/entry-object)

The current entry.

- **$form** [Form Object](https://docs.gravityforms.com/form-object)

The form object.

### 4\. Set the authorization header \#3

This example shows how you can add the Basic Authorization header to authenticate a request.

|     |     |
| --- | --- |
| 1<br>2<br>3<br>4<br>5 | `add_filter(``'gform_webhooks_request_headers'``,``function``(``$request_headers``) {`<br>```$request_headers``[``'Authorization'``] =``'Basic '``.``base64_encode``( USERNAME .``':'``. PASSWORD );`<br>```return``$request_headers``;`<br>`}, 10, 2 );` |

## Placement

This code should be placed in the [functions.php](http://codex.wordpress.org/Theme_Development#Functions_File) file of your active theme.
