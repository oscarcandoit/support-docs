## All Categories

[**Standard Fields** \\
Explore the Standard Fields available in the Form Editor, such as the Drop Down and Radio Button fields.](https://docs.gravityforms.com/category/user-guides/standard-fields/) [**Advanced Fields** \\
Advanced Fields offer more complex field types that can provide specific functionality, such as email, address or password fields, polls, surveys, and more.](https://docs.gravityforms.com/category/user-guides/advanced-fields/) [**Pricing Fields** \\
Discover the different Pricing Fields, such as Product, Option, Total, and Shipping.](https://docs.gravityforms.com/category/user-guides/pricing-fields/) [**Post Fields** \\
Learn about the different Post Fields available, such as Post Body, Post Title, and several others.](https://docs.gravityforms.com/category/user-guides/post-fields/) [**Field Settings** \\
User Guides related to Field Properties, such as how to use calculations and conditional logic.](https://docs.gravityforms.com/category/user-guides/field-settings-guide/) [**Entries** \\
Learn how Gravity Forms shows form submissions, which we call "Entries". These are what your form was built for!](https://docs.gravityforms.com/category/user-guides/form-entries/) [**Conditional Logic** \\
Conditional logic allows you to build interactive forms that can show or hide fields based on user selections. Additionally, conditional logic is available to control other elements like buttons, confirmations, notifications and feeds.](https://docs.gravityforms.com/category/user-guides/conditional-logic/) [**Merge Tags** \\
Articles explaining the different Merge Tags available, such as {admin\_email} and {all\_fields}.](https://docs.gravityforms.com/category/user-guides/merge-tags-getting-started/) [**Confirmations** \\
A "confirmation" is the response shown immediately after a successful form submission. They can display a specific message on your page ("Thank you for your inquiry. We respond within 24 hours."), or redirect the the submitter to another page or URL.](https://docs.gravityforms.com/category/user-guides/confirmations/) [**Notifications** \\
Notifications are a way to send information after a form submission.](https://docs.gravityforms.com/category/user-guides/notifications/) [**Design and Layout** \\
Need to style your forms to match your existing WordPress theme? Gravity Forms provides form themes and plenty of CSS selectors to beautify your forms.](https://docs.gravityforms.com/category/user-guides/design-and-layout/) [**Settings Reference** \\
Reference articles for the different settings available in Gravity Forms.](https://docs.gravityforms.com/category/user-guides/settings-reference/)

## Articles

[**Input Mask** \\
Input masks provide a visual guide allowing users to more easily enter data in a specific format such as dates and phone numbers.](https://docs.gravityforms.com/input-mask/ "Input Mask") [**Hidden** \\
The Hidden field can be used to store information that should be not visible to the user but can be processed and stored with the user submission.](https://docs.gravityforms.com/hidden/ "Hidden") [**File Upload** \\
The File Upload field allows users to upload a file with their form submission. You can limit which filetypes a user may upload.](https://docs.gravityforms.com/file-upload/ "File Upload") [**Page Break** \\
The Page Break field allows the creation of multi-page forms. Adding a Page Break field will automatically enable the multi-page functionality for your form.](https://docs.gravityforms.com/page-break/ "Page Break") [**Section Break** \\
The Section Break field allows you to add a content separator to your form to help organize groups of fields. This is a visual element rather than an actual data collecting form field.](https://docs.gravityforms.com/section-break/ "Section Break") [**HTML** \\
The HTML field allows you to place a block of free form HTML anywhere in your form. This is useful for adding images or additional content.](https://docs.gravityforms.com/html/ "HTML") [**Radio Buttons** \\
The Radio Buttons field allows users to select one option from a list. Only one option from the available choices can be selected.](https://docs.gravityforms.com/radio-buttons/ "Radio Buttons") [**Number** \\
The Number field allows a user to enter a number. The field than automatically validates the form data to insure that a number was entered.](https://docs.gravityforms.com/number/ "Number") [**Paragraph Text** \\
The Paragraph Text field allows a user to enter text in a textarea form field.](https://docs.gravityforms.com/paragraph-text/ "Paragraph Text") [**Allow Field to be Populated Dynamically** \\
Checking this option will enable data to be passed to the form and pre-populate this field dynamically.
