## Description

The gform\_entry\_detail\_meta\_boxes filter can be used to add custom meta boxes to the entry detail page.

## Usage

|     |     |
| --- | --- |
| 1 | `add_filter(``'gform_entry_detail_meta_boxes'``,``'your_function_name'``, 10, 3 );` |

## Parameters

- **$meta\_boxes** [array](https://docs.gravityforms.com/array)
The properties for the meta boxes. The default array is defined like so:





|     |     |
| --- | --- |
| 1<br>2<br>3<br>4<br>5<br>6<br>7<br>8<br>9<br>10<br>11<br>12 | `$meta_boxes``=``array``(`<br>```'submitdiv'``=>``array``(`<br>```'title'``=> esc_html__(``'Entry'``,``'gravityforms'``),`<br>```'callback'``=>``array``(``'GFEntryDetail'``,``'meta_box_entry_info'``),`<br>```'context'``=>``'side'``,`<br>```),`<br>```'notifications'``=>``array``(`<br>```'title'``=> esc_html__(``'Notifications'``,``'gravityforms'``),`<br>```'callback'``=>``array``(``'GFEntryDetail'``,``'meta_box_notifications'``),`<br>```'context'``=>``'side'``,`<br>```),`<br>`);` |





If the user has the gravityforms\_view\_entry\_notes capability then it will also contain the following:





|     |     |
| --- | --- |
| 1<br>2<br>3<br>4<br>5 | `$meta_boxes``[``'notes'``] =``array``(`<br>```'title'``=> esc_html__(``'Notes'``,``'gravityforms'``),`<br>```'callback'``=>``array``(``'GFEntryDetail'``,``'meta_box_notes'``),`<br>```'context'``=>``'normal'``,`<br>`);` |





If the entry has been processed by a payment add-on and the payment\_status property is not empty then it will also contain the following:





|     |     |
| --- | --- |
| 1<br>2<br>3<br>4<br>5<br>6 | `$meta_boxes``[``'payment'``] =``array``(`<br>```'title'``=>``$entry``[``'transaction_type'``] == 2 ? esc_html__(``'Subscription Details'``,``'gravityforms'``) : esc_html__(``'Payment Details'``,``'gravityforms'``),`<br>```'callback'``=>``array``(``'GFEntryDetail'``,``'meta_box_payment_details'``),`<br>```'context'``=>``'side'``,`<br>```'callback_args'``=>``array``(``$entry``,``$form``),`<br>`);` |

- **$entry** [Entry Object](https://docs.gravityforms.com/entry-object)

The entry currently being viewed/edited.

- **$form** [Form Object](https://docs.gravityforms.com/form-object)

The form object used to process the current entry.
