## Introduction

Entry meta data is custom data that’s stored and retrieved along with the entry object. For example, entry meta data may contain the results of a calculation made at the time of the entry submission, or a unique id for a subscriber returned by a mailing list service.

Any add-on which extends [GFAddon](https://docs.gravityforms.com/gfaddon), [GFFeedAddOn](https://docs.gravityforms.com/gffeedaddon), or [GFPaymentAddOn](https://docs.gravityforms.com/gfpaymentaddon) can add custom entry meta.

## get\_entry\_meta()

To add entry meta, override the get\_entry\_meta() function and return an associative array with the properties for each meta key your add-on is including.

### Properties

- **label** [string](https://docs.gravityforms.com/string)
The label for the entry meta item.

- **is\_numeric** [boolean](https://docs.gravityforms.com/boolean)

Used for sorting.

- **is\_default\_column** [boolean](https://docs.gravityforms.com/boolean)

Default columns appear in the entry list by default. Otherwise the user has to edit the columns and select the entry meta from the list.

- **update\_entry\_meta\_callback** [string](https://docs.gravityforms.com/string)

The function that should be called when updating this entry meta value.

- **filter** [array](https://docs.gravityforms.com/array)

An array containing the configuration for the filter used on the results pages, the entry list search, and export entries page.


The array should contain one element: operators. e.g. ‘operators’ => array( ‘is’, ‘isnot’, ‘>’, ‘<‘ )

## Accessing the Entry Meta Value

If you have access to the [Entry Object](https://docs.gravityforms.com/entry-object) then you could access the meta value like so:

|     |     |
| --- | --- |
| 1 | `$contact_id``= rgar(``$entry``,``'simpleaddon_contact_id'``);` |

If you don’t have access to the [Entry Object](https://docs.gravityforms.com/entry-object) but you do have the entry id then you can access the value by using [gform\_get\_meta()](https://docs.gravityforms.com/gform_get_meta) like so:

|     |     |
| --- | --- |
| 1 | `$contact_id``= gform_get_meta(``$entry_id``,``'simpleaddon_contact_id'``);` |

## Deleting the Entry Meta Value

The [gform\_delete\_meta()](https://docs.gravityforms.com/gform_delete_meta) function can be used to delete the meta value from the entry like so:

|     |     |
| --- | --- |
| 1 | `gform_delete_meta(``$entry_id``,``'simpleaddon_contact_id'``);` |
