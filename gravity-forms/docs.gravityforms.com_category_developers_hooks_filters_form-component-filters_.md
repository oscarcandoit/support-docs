# Form Component Filters

## Articles

[**gform\_ajax\_spinner\_url** \\
This filter can be used to change the default AJAX spinner image.](https://docs.gravityforms.com/gform_ajax_spinner_url/ "gform_ajax_spinner_url") [**gform\_block\_form\_forms** \\
The gform\_block\_form\_forms filter allows the list of available forms displayed in the Form block to be overridden.](https://docs.gravityforms.com/gform_block_form_forms/ "gform_block_form_forms") [**gform\_cdata\_open** \\
These filters provide the ability to validate your forms with XHTML doctypes by allowing you to wrap the javascript output with the form in CDATA blocks.](https://docs.gravityforms.com/gform_cdata_open/ "gform_cdata_open") [**gform\_disable\_custom\_field\_names\_query** \\
Allows the postmeta query which retrieves the custom field names (meta keys) to be disabled.](https://docs.gravityforms.com/gform_disable_custom_field_names_query/ "gform_disable_custom_field_names_query") [**gform\_display\_add\_form\_button** \\
Use this filter to allow the “Add Form” button to be displayed on other pages.](https://docs.gravityforms.com/gform_display_add_form_button/ "gform_display_add_form_button") [**gform\_enable\_legacy\_markup** \\
The gform\_enable\_legacy\_markup filter can be used to enable the legacy markup for forms created with Gravity Forms 2.5 and greater.](https://docs.gravityforms.com/gform_enable_legacy_markup/ "gform_enable_legacy_markup") [**gform\_form\_after\_open** \\
This filter allows markup to be added directly after the opening form wrapper.](https://docs.gravityforms.com/gform_form_after_open/ "gform_form_after_open") [**gform\_form\_not\_found\_message** \\
This filer allows the default form message, “Oops! We could not locate your form.”, to be changed.](https://docs.gravityforms.com/gform_form_not_found_message/ "gform_form_not_found_message") [**gform\_input\_mask\_script** \\
Use this filter to change the initialization script for the input mask script. Can be used to specify different initialization parameters.](https://docs.gravityforms.com/gform_input_mask_script/ "gform_input_mask_script") [**gform\_is\_form\_editor** \\
Allows pages to be marked as Form Editor pages or not.](https://docs.gravityforms.com/gform_is_form_editor/ "gform_is_form_editor") 1 [2](https://docs.gravityforms.com/category/developers/hooks/filters/form-component-filters/page/2/) [Next](https://docs.gravityforms.com/category/developers/hooks/filters/form-component-filters/page/2/)
