[**Form Data** \\
Form Data ($form\_data) is an associative array containing all the payment related properties for the entry being processed by the Authorize.Net add-on.](https://docs.gravityforms.com/form-data/ "Form Data") [**Freshbooks Feed Meta** \\
The Feed Meta object for the Freshbooks Add-On is an associative array containing the properties which determine how the add-on should process the form submission.](https://docs.gravityforms.com/freshbooks-feed-meta/ "Freshbooks Feed Meta") [**gaddon\_no\_output\_field\_properties** \\
This filter allows you to allow or deny specific properties to be added to the field HTML.](https://docs.gravityforms.com/gaddon_no_output_field_properties/ "gaddon_no_output_field_properties") [**GetResponse Feed Meta** \\
The Feed Meta object for the GetResponse Add-On is an associative array containing the properties which determine how the add-on should process the form submission.](https://docs.gravityforms.com/getresponse-feed-meta/ "GetResponse Feed Meta") [**GF\_User\_Registration** \\
GF\_User\_Registration is the class which houses the main functionality of the Gravity Forms User Registration Add-on; it extends the GFFeedAddOn class which is part of the add-on framework.](https://docs.gravityforms.com/gf_user_registration/ "GF_User_Registration") [**GFCoupons** \\
GFCoupons is the class which houses the main functionality of the Gravity Forms Coupons Add-on; it extends the GFFeedAddOn class which is part of the add-on framework.](https://docs.gravityforms.com/gfcoupons/ "GFCoupons") [**gform JavaScript Object** \\
