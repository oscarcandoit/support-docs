## Articles

[**gform\_entry\_field\_value** \\
Use this filter to change the field’s value before getting displayed on the Entry detail page.](https://docs.gravityforms.com/gform_entry_field_value/ "gform_entry_field_value") [**gform\_entry\_id\_pre\_save\_lead** \\
Allows entry id to be changed before submission is saved. Useful to update an existing entry instead of creating a new one.](https://docs.gravityforms.com/gform_entry_id_pre_save_lead/ "gform_entry_id_pre_save_lead") [**gform\_entry\_ids\_automatic\_deletion** \\
Allows modification of the array of entry IDs before automatic deletion.](https://docs.gravityforms.com/gform_entry_ids_automatic_deletion/ "gform_entry_ids_automatic_deletion") [**gform\_entry\_info** \\
Use this hook to add custom entry information to the Info area on the Entry detail page.](https://docs.gravityforms.com/gform_entry_info/ "gform_entry_info") [**gform\_entry\_is\_spam** \\
The gform\_entry\_is\_spam filter is used to mark entries as spam.](https://docs.gravityforms.com/gform_entry_is_spam/ "gform_entry_is_spam") [**gform\_entry\_list\_action** \\
This action fires after the default entry list actions have been processed.](https://docs.gravityforms.com/gform_entry_list_action/ "gform_entry_list_action") [**gform\_entry\_list\_action\_helpscout** \\
This hook uses Gravity Forms gform\_entry\_list\_action to run the action. See that hook for more details.](https://docs.gravityforms.com/gform_entry_list_action_helpscout/ "gform_entry_list_action_helpscout") [**gform\_entry\_list\_bulk\_actions** \\
This filter allows the drop down of available bulk actions on the entries list to be modified.](https://docs.gravityforms.com/gform_entry_list_bulk_actions/ "gform_entry_list_bulk_actions") [**gform\_entry\_list\_column\_input\_label\_only** \\
This filter can be used to override the default behavior of only including the input label in the entry list column header.](https://docs.gravityforms.com/gform_entry_list_column_input_label_only/ "gform_entry_list_column_input_label_only") [**gform\_entry\_list\_columns** \\
