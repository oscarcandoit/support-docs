## Description

This filter can be used to modify the TO number before the SMS is sent to Twilio.

This hook has been deprecated since Twilio version 2.4. Please use [gform\_twilio\_message](https://docs.gravityforms.com/gform_twilio_message) instead.

## Usage

The filter which would run for all Twilio feeds can be used like so:

|     |     |
| --- | --- |
| 1 | `add_filter(``'gform_twilio_set_to_phone_number'``,``'your_function_name'``, 10, 3 );` |

## Parameters

- **$to** [string](https://docs.gravityforms.com/string) The number the SMS will be sent TO. Formatted with a ‘+’ and country code e.g. +17571234567 (E.164 format).
- **$entry** [Entry Object](https://docs.gravityforms.com/entry-object)
The Entry which is currently being processed.

- **$feed\_id** [integer](https://docs.gravityforms.com/integer)
The ID of the [Feed Object](https://docs.gravityforms.com/feed-object) which is currently being processed. Available since v2.1.
