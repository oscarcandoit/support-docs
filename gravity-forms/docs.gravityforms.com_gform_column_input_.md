# gform\_column\_input

- [Description](https://docs.gravityforms.com/gform_column_input/#h-description)
- [Usage](https://docs.gravityforms.com/gform_column_input/#h-usage)
- [Examples](https://docs.gravityforms.com/gform_column_input/#h-examples)
- [Placement](https://docs.gravityforms.com/gform_column_input/#h-placement)
- [Source Code](https://docs.gravityforms.com/gform_column_input/#h-source-code)

## Description

This filter can be used to specify a different input type for a list field column. Currently supported field types are “select” (Drop Down) and “text” (Text Field)

## Usage

|     |
| --- |
| `add_filter(``'gform_column_input'``,``'set_column_input'``, 10, 5 );` |

You can also target a specific column by adding the form id, field id and column number after the hook name. (format: gform\_column\_input\_FORMID\_FIELDID\_COLUMN)

|     |
| --- |
| `//This filter declaration targets the first column of field whose id is 2 in form whose id is 6`<br>`add_filter(``'gform_column_input_6_2_1'``,``'set_column_input'``, 10, 5 );` |

## Parameters

- **$input\_info** [array](https://docs.gravityforms.com/array)
The input info array to be filtered, in the following format:


|     |
| --- |
| `array``(`<br>```'type'``=>``'select'``,`<br>```'choices'``=>``'First Choice,Second Choice'``,`<br>`);`<br>`// OR, if values need to be specified for each choice, the following format can also be used:`<br>`array``(`<br>```'type'``=>``'select'``,`<br>```'choices'``=>``array``(`<br>```array``(``'text'``=>``'First Choice'``,``'value'``=>``'First'``),`<br>```array``(``'text'``=>``'Second Choice'``,``'value'``=>``'Second'``),`<br>```),`<br>`);` |

- **$field** [Field Object](https://docs.gravityforms.com/field-object)
Current field.

- **$column** [string](https://docs.gravityforms.com/string)
Current column name.

- **$value** [string](https://docs.gravityforms.com/string)
Currently entered/selected value for the column’s input.

- **$form\_id** [integer](https://docs.gravityforms.com/integer)
ID of current form.


## Examples

This example changes column 1 of list field 2 in a form with id 187 to a drop down.

|     |
| --- |
| `add_filter(``'gform_column_input_187_2_1'``,``'set_column'``, 10, 5 );`<br>`function``set_column(``$input_info``,``$field``,``$column``,``$value``,``$form_id``) {`<br>```return``array``(``'type'``=>``'select'``,``'choices'``=>``'First Choice,Second Choice'``);`<br>`}` |

## Placement

This code should be placed in the [functions.php](http://codex.wordpress.org/Theme_Development#Functions_File) file of your active theme

## Source Code

This filter is located in GF\_Field\_List::get\_list\_input() in _includes/fields/class-gf-field-list.php_
