# Import/Export Filters

## Articles

[**gform\_entries\_field\_header\_pre\_export** \\
The gform\_entries\_field\_header\_pre\_export filter can be used to override the field header in the entries export.](https://docs.gravityforms.com/gform_entries_field_header_pre_export/ "gform_entries_field_header_pre_export") [**gform\_export\_entries\_forms** \\
The “gform\_export\_entries\_forms” filter in Gravity Forms allows the forms displayed on the Export Entries page to be filtered.](https://docs.gravityforms.com/gform_export_entries_forms/ "gform_export_entries_forms") [**gform\_export\_field\_value** \\
Use the gform\_export\_field\_value filter to override the field value before it is included in the CSV export.](https://docs.gravityforms.com/gform_export_field_value/ "gform_export_field_value") [**gform\_export\_fields** \\
Use the gform\_export\_fields filter to add custom columns to the entry export.](https://docs.gravityforms.com/gform_export_fields/ "gform_export_fields") [**gform\_export\_form** \\
Allows modification of the form meta before export.](https://docs.gravityforms.com/gform_export_form/ "gform_export_form") [**gform\_export\_forms\_forms** \\
The “gform\_export\_forms\_forms” filter in Gravity Forms allows the forms displayed on the Export Forms page to be filtered.](https://docs.gravityforms.com/gform_export_forms_forms/ "gform_export_forms_forms") [**gform\_export\_line** \\
This filter allows the filtering of the current line being exported.](https://docs.gravityforms.com/gform_export_line/ "gform_export_line") [**gform\_export\_lines** \\
The gform\_export\_lines filter can be used to filter the entry export lines before they are saved to the temporary file.](https://docs.gravityforms.com/gform_export_lines/ "gform_export_lines") [**gform\_export\_max\_execution\_time** \\
Allows the maximum execution time for exporting entries to be changed.](https://docs.gravityforms.com/gform_export_max_execution_time/ "gform_export_max_execution_time") [**gform\_export\_menu** \\
Use the gform\_export\_menu filter to add new or modify default menu items which will appear in the Import/Export section menu.](https://docs.gravityforms.com/gform_export_menu/ "gform_export_menu") 1 [2](https://docs.gravityforms.com/category/developers/hooks/filters/importexport-filters/page/2/) [Next](https://docs.gravityforms.com/category/developers/hooks/filters/importexport-filters/page/2/)
