## Description

Filters the input types to use when checking if a field is a product field, allowing support for custom product fields.

## Usage

|     |     |
| --- | --- |
| 1 | `add_filter(``'gform_product_field_types'``,``'your_function_name'``);` |

## Parameters

- **$product\_fields** [array](https://docs.gravityforms.com/array)
An array of the product field types.

The default product types are:





|     |     |
| --- | --- |
| 1<br>2<br>3<br>4<br>5<br>6<br>7<br>8<br>9<br>10<br>11<br>12 | `array``(`<br>`'option'``,`<br>`'quantity'``,`<br>`'product'``,`<br>`'total'``,`<br>`'shipping'``,`<br>`'calculation'``,`<br>`'price'``,`<br>`'hiddenproduct'``,`<br>`'singleproduct'``,`<br>`'singleshipping'`<br>`)` |


## Examples

|     |     |
| --- | --- |
| 1<br>2<br>3<br>4<br>5<br>6<br>7<br>8 | `add_filter(``'gform_product_field_types'``,``'set_product_types'``);`<br>`function``set_product_types(``$product_types``){`<br>```if``( ! in_array(``'my_custom_product'``,``$product_types``)){`<br>```$product_types``[] =``'my_custom_product'``;`<br>```}`<br>```return``$product_types``;`<br>`}` |

## Placement

This code should be placed in the [functions.php](http://codex.wordpress.org/Theme_Development#Functions_File) file of your active theme.

## Since

This filter was added in Gravity Forms 1.9.14.

## Source Code

This filter is located in GFCommon::is\_product\_field() in _common.php_.
