This hook was removed in version 2.5. [gform\_post\_update\_form\_meta](https://docs.gravityforms.com/gform_post_update_form_meta/) can be used instead.

## Description

Fires after the notification is successfully saved.

## Usage

|     |
| --- |
| `add_action(``'gform_post_notification_save'``,``'your_function_name'``, 10,3 );` |

## Parameters

- **$notification** [Notifications Object](https://docs.gravityforms.com/notifications-object)


The notifications object.

- **$form** [Form Object](https://docs.gravityforms.com/form-object)


The current form.

- **$is\_new\_notification** [bool](https://docs.gravityforms.com/bool)


True if this is a new notification. False otherwise.
