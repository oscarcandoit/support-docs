# Constants

## Articles

[**GF\_DEBUG** \\
Allows the contents of the AJAX iframe to be viewed for AJAX forms.](https://docs.gravityforms.com/gf_debug/ "GF_DEBUG") [**GF\_ENABLE\_NOTIFICATION\_EMAIL\_HEADER** \\
This constant enables the X-Gravity-Forms-Source email header for notification emails.](https://docs.gravityforms.com/gf_enable_notification_email_header/ "GF_ENABLE_NOTIFICATION_EMAIL_HEADER") [**GF\_IMPORT\_FILE** \\
Allows a Form JSON file from which a set of default forms can be pre-generated when a new site is created on your Multi-Site install to be specified.](https://docs.gravityforms.com/gf_import_file/ "GF_IMPORT_FILE") [**GF\_LICENSE\_KEY** \\
Automatically pre-populates your Gravity Forms license key.](https://docs.gravityforms.com/gf_license_key/ "GF_LICENSE_KEY") [**GF\_LOGGING\_DISABLE\_NOTICE** \\
Allows the non-dismissible security notice which displays on every WordPress admin page when logging is enabled to be turned off.](https://docs.gravityforms.com/gf_logging_disable_notice/ "GF_LOGGING_DISABLE_NOTICE") [**GF\_MIN\_WP\_VERSION** \\
Specifies the minimum WordPress version required by Gravity Forms.](https://docs.gravityforms.com/gf_min_wp_version/ "GF_MIN_WP_VERSION") [**GF\_MIN\_WP\_VERSION\_SUPPORT\_TERMS** \\
Specifies the minimum version of WordPress that will be officially supported by Gravity Forms.](https://docs.gravityforms.com/gf_min_wp_version_support_terms/ "GF_MIN_WP_VERSION_SUPPORT_TERMS") [**GF\_RECAPTCHA\_PRIVATE\_KEY** \\
Automatically populates the reCAPTCHA Private Key on activation.](https://docs.gravityforms.com/gf_recaptcha_private_key/ "GF_RECAPTCHA_PRIVATE_KEY") [**GF\_SUPPORTED\_WP\_VERSION** \\
Indicates if the minimum supported version of WordPress has been met.](https://docs.gravityforms.com/gf_supported_wp_version/ "GF_SUPPORTED_WP_VERSION") [**GF\_THEME\_IMPORT\_FILE** \\
Allows you to specify a Form JSON file from which a set of default forms can be pre-generated when a theme containing this option is activated.](https://docs.gravityforms.com/gf_theme_import_file/ "GF_THEME_IMPORT_FILE") 1 [2](https://docs.gravityforms.com/category/developers/php-api/constants/page/2/) [Next](https://docs.gravityforms.com/category/developers/php-api/constants/page/2/)
