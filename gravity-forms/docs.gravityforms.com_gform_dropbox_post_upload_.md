## Description

This hook is fired after a Dropbox feed has been processed and files have been uploaded.

## Usage

The following would apply to all forms.

|     |
| --- |
| `add_action(``'gform_dropbox_post_upload'``,``'your_function_name'``, 10, 3 );` |

To limit the scope of your function to a specific form, append the form id to the end of the hook name. (format: gform\_dropbox\_post\_upload\_FORMID)

|     |
| --- |
| `add_action(``'gform_dropbox_post_upload_6'``,``'your_function_name'``, 10, 3 );` |

## Parameters

- **$feed** [Feed Object](https://docs.gravityforms.com/feed-object)


The current feed being processed.

- **$entry** [Entry Object](https://docs.gravityforms.com/entry-object) The entry that was submitted.

- **$form** [Form Object](https://docs.gravityforms.com/form-object) The current form to be filtered.
