# Obtaining Your Mad Mimi API Key

GoDaddy has announced the [closure of the Mad Mimi service](https://madmimi.com/). Once the service is discontinued at the end of August 2024, the Gravity Forms Mad Mimi Add-On will no longer be available for download.

To use Mad Mimi with Gravity forms, you will need to first obtain your API key. Without this, Gravity Forms will not be able to appropriately communicate with Mad Mimi. In this article, we will show you how to do so.

1. Log into your [Mad Mimi](https://madmimi.com/) account.
2. Click **Account** from the top navigation bar.
3. Under **Settings & Billing** on the right side, click on the **API** tab. Your **Secret API Key** will be listed below.

[![Mad Mimi API Key page](https://docs.gravityforms.com/wp-content/uploads/2015/08/Mad-Mimi-API-Key-page-noted-298x300.png)](https://docs.gravityforms.com/wp-content/uploads/2015/08/Mad-Mimi-API-Key-page-noted.png)
