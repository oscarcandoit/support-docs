# gform\_chainedselects\_import\_file

- [Example](https://docs.gravityforms.com/gform_chainedselects_import_file/#h-example)
- [Placement](https://docs.gravityforms.com/gform_chainedselects_import_file/#h-placement)

## Description

This filter allows an import file for the Chained Selects to be provided programmatically. This import file will override any previously uploaded file via the field settings.

## Usage

The following would apply to all forms:

|     |     |
| --- | --- |
| 1 | `add_filter(``'gform_chainedselects_import_file'``,``'your_function_name'``, 10, 3 );` |

To target a specific form, append the form id to the hook name. (format: gform\_chainedselects\_import\_file\_FORMID)

|     |     |
| --- | --- |
| 1 | `add_filter(``'gform_chainedselects_import_file_14'``,``'your_function_name'``, 10, 3 );` |

To target a specific field id for a form, append the form id and field id to the hook name. (format: gform\_chainedselects\_import\_file\_FORMID\_FIELDID)

|     |     |
| --- | --- |
| 1 | `add_filter(``'gform_chainedselects_import_file_14_2'``,``'your_function_name'``, 10, 3 );` |

## Parameters

- **$import\_file** [array](https://docs.gravityforms.com/array)
An array of details for the file from which choices will be imported:

  - **$url** [string](https://docs.gravityforms.com/string) The URL of the file to be imported.
  - **$expiration** [int](https://docs.gravityforms.com/int) The number of seconds until the import file will be re-imported.
- **$form** [Form Object](https://docs.gravityforms.com/form-object)
The form object.

- **$field** [Field Object](https://docs.gravityforms.com/field-object)

The field object.

## Placement

This code should be placed in the [functions.php](http://codex.wordpress.org/Theme_Development#Functions_File) file of your active theme.

## Source Code

This filter is located in GF\_Chained\_Field\_Select::maybe\_import\_from\_filter() in _gravityformschainedselects/includes/class-gf-field-chainedselect.php_.
