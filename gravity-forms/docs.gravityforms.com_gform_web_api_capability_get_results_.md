- [Description](https://docs.gravityforms.com/gform_web_api_capability_get_results/#h-description)

## Description

Filters the capability required to get form results via the web API.

## Usage

|     |     |
| --- | --- |
| 1 | `add_filter(``'gform_web_api_capability_get_results'``,``function``(``$capability``) {``return``'my_custom_capability'``; } );` |

## Parameters

- **$capability** [string](https://docs.gravityforms.com/string)


The capability required. Defaults to ‘gravityforms\_view\_entries’.
