# Setting up a Quiz Field

- [Pre-Requisites](https://docs.gravityforms.com/setup-the-quiz-add-on/#h-pre-requisites)
- [Field Level Quiz Settings](https://docs.gravityforms.com/setup-the-quiz-add-on/#h-field-level-quiz-settings)
- [Form Level Quiz Settings](https://docs.gravityforms.com/setup-the-quiz-add-on/#h-form-level-quiz-settings)
- [Next Steps](https://docs.gravityforms.com/setup-the-quiz-add-on/#h-next-steps)

## Pre-Requisites

- [Setup the Quiz Add-On](https://docs.gravityforms.com/downloading-and-installing-the-quiz-add-on/).
- A form with the [Quiz field](https://docs.gravityforms.com/quiz-field/) added to it.

## Field Level Quiz Settings

Now that you have a quiz field added to your form, you will need to provide all the necessary settings data to configure it how you wish it to work.

All the Quiz field settings are explained in the [Quiz Field user guide here](https://docs.gravityforms.com/quiz-field/).

[![Quiz General Settings](https://docs.gravityforms.com/wp-content/uploads/2025/06/image-12.png)](https://docs.gravityforms.com/wp-content/uploads/2025/06/image-12.png)[![Quiz Field Choices flyout](https://docs.gravityforms.com/wp-content/uploads/2025/06/image-13-1024x616.png)](https://docs.gravityforms.com/wp-content/uploads/2025/06/image-13.png)

## Form Level Quiz Settings

Refer to this article on [Quiz Form Settings](https://docs.gravityforms.com/configure-quiz-settings/) for more information.

## Next Steps

If you want to add more questions to your form, simply add more Quiz fields and configure them accordingly.

Once your Quiz form is complete, you are ready to setup your quiz grading. Visit [Setting Up Quiz Grading with the Quiz Add-On](https://docs.gravityforms.com/setup-the-quiz-grading/) for instructions.
