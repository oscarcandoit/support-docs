The GFFeedAddOn class provides basic functionality for developers when creating new feed-based add-ons for Gravity Forms. It includes all the features of the GFAddOn class with one exception; the Form Settings are replaced by the Feed Settings UI.](https://docs.gravityforms.com/gffeedaddon/ "GFFeedAddOn") [**GFPaymentAddOn** \\
The GFPaymentAddOn class provides basic functionality for developers when creating new add-ons for Gravity Forms that collect payments.](https://docs.gravityforms.com/gfpaymentaddon/ "GFPaymentAddOn")
