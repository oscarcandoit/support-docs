# Obtaining Your Postmark API Keys

To use the Postmark Add-On, you’ll need to have your Postmark API keys. Postmark won’t know what account is yours without the appropriate API keys. In this article, we’ll show you how to get your Postmark API keys.

1. Log into your [Postmark](https://account.postmarkapp.com/login) account.
2. Click on **Account** at the top of the page.
3. Under Account, click on the **API Tokens** tab.
4. You’ll need both an **Account API token** and **Server API token**. The **Account API token** is listed here, so take note of it.
5. To get your **Server API token**, start typing the name of your Postmark server in the right side search box labeled **Search for server** and click on the server when it appears. Then click the **Get a server API token** button.
6. A new page will load and you will be presented with your Server API token.

[![Postmark API Tokens](https://docs.gravityforms.com/wp-content/uploads/2018/11/Postmark-API-Tokens-300x282.png)](https://docs.gravityforms.com/wp-content/uploads/2018/11/Postmark-API-Tokens.png)

Now that you have your Account API and Server API tokens, you’re ready to continue [setting up the Postmark Add-On](https://docs.gravityforms.com/setting-up-the-postmark-add-on/) or start [sending form notifications using Postmark](https://docs.gravityforms.com/form-notifications-postmark/).
