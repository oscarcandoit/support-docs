## Description

Allows customization of the options used to initialize the autocomplete for the address field.

**Note:** As of Geolocation 1.5.0, this filter is deprecated. Use the [gform\_geolocation\_place\_autocomplete\_element\_place\_details](https://docs.gravityforms.com/?p=43752) filter instead.

## Usage

|     |
| --- |
| `gform.addFilter(``'gform_geolocation_autocomplete_options_pre_init'``,``function``( options, formId, fieldId, address ) {`<br>```return``options;`<br>`} );` |

## Parameters

- **options** [object](https://docs.gravityforms.com/object)

The autocomplete options. The supported options can be found [here](https://developers.google.com/maps/documentation/javascript/reference/places-widget#AutocompleteOptions).
- **formId** [number](https://docs.gravityforms.com/integer)

The ID of the form the address field belongs to.
- **fieldId** [number](https://docs.gravityforms.com/integer)

The ID of the address field autocomplete is to be initialized for.
- **address** [Element](https://developer.mozilla.org/en-US/docs/Web/API/Element)

The Address field element.

## Examples

### Form and field-specific country restriction

Limit the available choices to a specific country (Great Britain) in Form ID 1 and Field ID 2.

| --- | --- |
| 1<br>2<br>3<br>4<br>5<br>6 | `gform.addFilter(``'gform_geolocation_autocomplete_options_pre_init'``,``function``( options, formId, fieldId, address ) {`<br>```if``( formId == 1 && fieldId == 2 ) {`<br>```options.componentRestrictions = { country:``'GB'``};`<br>```}`<br>```return``options;`<br>`} );` |

### Country restriction by field address type

This example adds a data attribute to the field markup containing the country code related to the address type selected in the field settings.

Use [gform\_field\_container](https://docs.gravityforms.com/gform_field_container/) filter to add the data attribute (PHP).
