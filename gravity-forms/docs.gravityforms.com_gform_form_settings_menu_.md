# gform\_form\_settings\_menu

- [Description](https://docs.gravityforms.com/gform_form_settings_menu/#h-description)
- [Usage](https://docs.gravityforms.com/gform_form_settings_menu/#h-usage)
- [Parameters](https://docs.gravityforms.com/gform_form_settings_menu/#h-parameters)
- [Examples](https://docs.gravityforms.com/gform_form_settings_menu/#h-examples)
- [Source Code](https://docs.gravityforms.com/gform_form_settings_menu/#h-source-code)

## Description

Add new or modify default menu items which will appear in the Form Settings section menu.

## Usage

|     |
| --- |
| `add_filter(``'gform_form_settings_menu'``,``'my_custom_function'``);` |

## Parameters

- **$menu\_items** [array](https://docs.gravityforms.com/array)

An array of menu items to be displayed on the Form Settings page menu.

## Examples

This example demonstrates how you can add a custom menu item to the Form Settings page menu. It also demonstrates (with the use of the [gform\_form\_settings\_page\_view](https://docs.gravityforms.com/gform_form_settings_page_view)) how to display content for this page when selected.

|     |
| --- |
| `// add a custom menu item to the Form Settings page menu`<br>`add_filter(``'gform_form_settings_menu'``,``'my_custom_form_settings_menu_item'``);`<br>`function``my_custom_form_settings_menu_item(``$menu_items``) {`<br>``<br>```$menu_items``[] =``array``(`<br>```'name'``=>``'my_custom_form_settings_page'``,`<br>```'label'``=> __(``'My Custom Form Settings Page'``),`<br>```'icon'``=>``'dashicons-flag dashicons'`<br>```);`<br>``<br>```return``$menu_items``;`<br>`}`<br>``<br>`// handle displaying content for our custom menu when selected`<br>`add_action(``'gform_form_settings_page_my_custom_form_settings_page'``,``'my_custom_form_settings_page'``);`<br>`function``my_custom_form_settings_page() {`<br>``<br>```GFFormSettings::page_header();`<br>``<br>```echo``'My Custom Form Settings!'``;`<br>``<br>```GFFormSettings::page_footer();`<br>``<br>`}` |

This filter is located in GFFormSettings::get\_tabs() in _form\_settings.php_.
