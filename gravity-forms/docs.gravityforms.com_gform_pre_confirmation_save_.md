# gform\_pre\_confirmation\_save

- [Description](https://docs.gravityforms.com/gform_pre_confirmation_save/#h-description)
- [Usage](https://docs.gravityforms.com/gform_pre_confirmation_save/#h-usage)
- [Parameters](https://docs.gravityforms.com/gform_pre_confirmation_save/#h-parameters)
- [Examples](https://docs.gravityforms.com/gform_pre_confirmation_save/#h-examples)
- [Source Code](https://docs.gravityforms.com/gform_pre_confirmation_save/#h-source-code)

## Description

Modify the confirmation object before it is saved to the database. This is particularly useful when saving custom confirmation settings to the confirmation object.

## Usage

Apply to all forms:

|     |     |
| --- | --- |
| 1 | `add_filter(``'gform_pre_confirmation_save'``,``'my_custom_function'``, 10, 2 );` |

Apply to a specific form ID:

|     |     |
| --- | --- |
| 1 | `add_filter(``'gform_pre_confirmation_save_5'``,``'my_custom_function'``, 10, 2 );` |

## Parameters

- **$confirmation** [Confirmation Object](https://docs.gravityforms.com/confirmation-object)
The confirmation object about to be saved.

- **$form** [Form Object](https://docs.gravityforms.com/form-object)

The current form object to which the confirmation being saved belongs.


## Examples

This example demonstrates how you can add the value entered via our [gform\_confirmation\_ui\_settings](https://docs.gravityforms.com/gform_confirmation_ui_settings) to the confirmation object before it is saved to the database. Use with the [gform\_confirmation\_ui\_settings](https://docs.gravityforms.com/gform_pre_confirmation_save/) hook to display your custom settings UI.

If your UI settings have a “name” attribute, they will be submitted along with the rest of the default confirmation settings. We can then retrieve our custom value from the $\_POST using the Gravity Forms helper function **rgpost()**.

|     |     |
| --- | --- |
| 1<br>2<br>3<br>4<br>5 | `add_filter(``'gform_pre_confirmation_save'``,``'my_custom_confirmation_save'``, 10, 2 );`<br>`function``my_custom_confirmation_save(``$confirmation``,``$form``) {`<br>```$confirmation``[``'my_custom_setting'``] = rgpost(``'my_custom_setting'``);`<br>```return``$confirmation``;`<br>`}` |
