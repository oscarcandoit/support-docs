# Payment Filters

## Articles

[**gform\_{$short\_slug}\_is\_valid\_payment\_amount** \\
This filter allows custom logic to be used to determine if a payment add-on which extends GFPaymentAddOn should process the submission for the given amount.](https://docs.gravityforms.com/gform_short_slug_is_valid_payment_amount/ "gform_{$short_slug}_is_valid_payment_amount") [**gform\_creditcard\_types** \\
Use this filter to change the default list of supported credit card types.](https://docs.gravityforms.com/gform_creditcard_types/ "gform_creditcard_types") [**gform\_currencies** \\
The gform\_currencies filter can be used to update existing currencies and/or create new currencies.](https://docs.gravityforms.com/gform_currencies/ "gform_currencies") [**gform\_currency** \\
Use this filter to override the currency configured on the Forms > Settings page. It will affect all forms.](https://docs.gravityforms.com/gform_currency/ "gform_currency") [**gform\_currency\_disabled** \\
Use this filter to enable/disable the currency drop down on the settings page.](https://docs.gravityforms.com/gform_currency_disabled/ "gform_currency_disabled") [**gform\_payment\_methods** \\
Use this filter to add alternative payment methods to the Credit Card field.](https://docs.gravityforms.com/gform_payment_methods/ "gform_payment_methods") [**gform\_payment\_statuses** \\
Allows custom payment statuses to be defined.](https://docs.gravityforms.com/gform_payment_statuses/ "gform_payment_statuses")
