# Standard Fields

## Articles

[**Checkboxes** \\
The Checkboxes field allows you to present one or many checkboxes which allow multiple selections.](https://docs.gravityforms.com/checkboxes/ "Checkboxes") [**Drop Down** \\
The Drop Down field allows users to select one option from a list.](https://docs.gravityforms.com/drop-down/ "Drop Down") [**Hidden** \\
The Hidden field can be used to store information that should be not visible to the user but can be processed and stored with the user submission.](https://docs.gravityforms.com/hidden/ "Hidden") [**HTML** \\
The HTML field allows you to place a block of free form HTML anywhere in your form. This is useful for adding images or additional content.](https://docs.gravityforms.com/html/ "HTML") [**Image Choice Field** \\
The Gravity Forms Image Choice field allows you to display images as choices for your single or multi-select field.](https://docs.gravityforms.com/image-choice-field/ "Image Choice Field") [**Multiple Choice Field** \\
The Gravity Forms Multiple Choice field offers both checkbox and radio button form field functionality with a host of customization settings.](https://docs.gravityforms.com/multiple-choice-field/ "Multiple Choice Field") [**Number** \\
The Number field allows a user to enter a number. The field than automatically validates the form data to insure that a number was entered.](https://docs.gravityforms.com/number/ "Number") [**Page Break** \\
The Page Break field allows the creation of multi-page forms. Adding a Page Break field will automatically enable the multi-page functionality for your form.](https://docs.gravityforms.com/page-break/ "Page Break") [**Paragraph Text** \\
The Paragraph Text field allows a user to enter text in a textarea form field.](https://docs.gravityforms.com/paragraph-text/ "Paragraph Text") [**Radio Buttons** \\
The Radio Buttons field allows users to select one option from a list. Only one option from the available choices can be selected.](https://docs.gravityforms.com/radio-buttons/ "Radio Buttons") 1 [2](https://docs.gravityforms.com/category/user-guides/standard-fields/page/2/) [Next](https://docs.gravityforms.com/category/user-guides/standard-fields/page/2/)
