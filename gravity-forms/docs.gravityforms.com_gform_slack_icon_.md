## Description

This filter is used to dynamically change the icon being displayed with the message sent to Slack. The icon image size should be 48×48.

|     |     |
| --- | --- |
| 1 | `apply_filters(``'gform_slack_icon'``,``$this``->get_base_url() .``'/images/icon.png'``,``$feed``,``$entry``,``$form``)` |

## Usage

|     |     |
| --- | --- |
| 1 | `add_filter(``'gform_slack_icon'``,``'change_slack_icon'``, 10, 4 );` |

## Parameters

- **$icon\_url** [string](https://docs.gravityforms.com/string)
The current icon being used for the Slack message.

- **$feed** [Feed Object](https://docs.gravityforms.com/feed-object)

The current feed object.

- **$entry** [Entry Object](https://docs.gravityforms.com/entry-object)

The current entry object.

- **$form** [Form Object](https://docs.gravityforms.com/form-object)

The current form object.


## Examples

This example changes the icon URL if the form ID is 4.

|     |     |
| --- | --- |
| 1<br>2<br>3<br>4<br>5<br>6<br>7<br>8<br>9 | `add_filter(``'gform_slack_icon'``,``'change_slack_icon'``, 10, 4 );`<br>`function``change_slack_icon(``$icon_url``,``$feed``,``$entry``,``$form``) {`<br>```if``(``$form``[``'id'``] == 4 ) {`<br>```$icon_url``=``'http://placehold.it/48'``;`<br>```}`<br>```return``$icon_url``;`<br>`}` |

## Placement

This code should be placed in the [functions.php](http://codex.wordpress.org/Theme_Development#Functions_File) file of your active theme.

## Since

This filter was added in Slack 1.4.
