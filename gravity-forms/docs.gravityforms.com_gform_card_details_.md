## Description

This filter is executed when creating the Stripe Card or Square field and can be used to modify the “Card Details” label.

## Usage

|     |     |
| --- | --- |
| 1 | `add_filter(``'gform_card_details'``,``'change_details'``, 10, 2 );` |

## Parameters

- **$label** [string](https://docs.gravityforms.com/string)
The label to be filtered.

- **$form\_id** [integer](https://docs.gravityforms.com/integer)

The current form’s id.


## Examples

This example changes the default Card Details label:

|     |     |
| --- | --- |
| 1<br>2<br>3<br>4 | `add_filter(``'gform_card_details'``,``'change_details'``, 10, 2 );`<br>`function``change_details(``$label``,``$form_id``) {`<br>```return``"Credit Card Specifics"``;`<br>`}` |

## Placement

This code should be placed in the [functions.php](http://codex.wordpress.org/Theme_Development#Functions_File) file of your active theme.

## Since

This filter was added in Stripe 2.6 and Square 1.0.

## Source Code

**Stripe add-on:** This filter is located in GF\_Field\_Stripe\_CreditCard::get\_form\_editor\_inline\_script\_on\_page\_render() in _gravityformsstripe/includes/class-gf-field-stripe-creditcard.php_

**Square add-on:** This filter is located in GF\_Field\_Square\_CreditCard::get\_form\_editor\_inline\_script\_on\_page\_render() in _gravityformssquare/includes/class-gf-field-square-creditcard.php_
