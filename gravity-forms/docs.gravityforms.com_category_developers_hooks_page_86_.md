Description.](https://docs.gravityforms.com/gform_trim_input_value/ "gform_trim_input_value") [**gform\_turnstile\_enable\_no\_conflict** \\
Use this filter to enable Turnstile’s No-Conflict mode, which dequeues other CAPTCHA scripts to avoid conflicts.](https://docs.gravityforms.com/gform_turnstile_enable_no_conflict/ "gform_turnstile_enable_no_conflict") [**gform\_twilio\_message** \\
This filter is used to modify the SMS message arguments before it is sent.](https://docs.gravityforms.com/gform_twilio_message/ "gform_twilio_message") [**gform\_twilio\_set\_to\_phone\_number** \\
This hook has been deprecated since Twilio version 2.4. Please use gform\_twilio\_message instead.](https://docs.gravityforms.com/gform_twilio_set_to_phone_number/ "gform_twilio_set_to_phone_number") [**gform\_uninstalling** \\
