# Save Button Field Type

- [Introduction](https://docs.gravityforms.com/save-button/#h-introduction)
- [Example](https://docs.gravityforms.com/save-button/#h-example)

## Introduction

The **save** type field, part of the [Settings API](https://docs.gravityforms.com/settings-api), renders a submit button type of input.

## Example

The following example renders a section with a text input and a save button with custom error and success messages.

|     |     |
| --- | --- |
| 1<br>2<br>3<br>4<br>5<br>6<br>7<br>8<br>9<br>10<br>11<br>12<br>13<br>14<br>15<br>16<br>17<br>18 | `array``(`<br>```'title'``=>``'Save Button Messaging'``,`<br>```'fields'``=>``array``(`<br>```array``(`<br>```'type'``=>``'text'``,`<br>```'name'``=>``'mytextfield'``,`<br>```'label'``=> esc_html__(``'This is my text field'``,``'sometextdomain'``),`<br>```),`<br>```array``(`<br>```'type'``=>``'save'``,`<br>```'messages'``=>``array``(`<br>```'error'``=> esc_html__(``'Settings could not be updated.'``,``'sometextdomain'``),`<br>```'success'``=> esc_html__(``'Success! Settings have been updated.'``,``'sometextdomain'``),`<br>```),`<br>```'value'``=>``'My Save Button Text'`<br>```),`<br>```)`<br>`)` |
