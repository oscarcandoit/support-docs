## Description

The `gform_form_theme_slug` filter can be used to filter the form theme.

## Usage

Applies to all forms:

|     |
| --- |
| `add_filter(``'gform_form_theme_slug'``,``'your_function_name'``, 10, 2 );` |

## Parameters

- **$slug** [string](https://docs.gravityforms.com/string/)

The current theme slug for the form.
- **$form [Form Object](https://docs.gravityforms.com/form-object/)**

The form object being processed.

## Examples

### 1\. Force theme when form embedded in specific post type

This example shows how you can set a specific theme to be always used when a form is embedded in a specific post type.

|     |     |
| --- | --- |
| 1<br>2<br>3<br>4<br>5<br>6<br>7<br>8<br>9 | `add_filter(``'gform_form_theme_slug'``,``function``(``$slug``,``$form``) {`<br>```if``( get_post_type() !==``'regional_group'``) {`<br>```return``$slug``;`<br>```}`<br>```return``'gravity-theme'``;`<br>`}, 10, 2 );` |

### 2\. Force Orbital theme for form preview

This example shows how you can force the Orbital theme to be used for the form preview.

|     |     |
| --- | --- |
| 1<br>2<br>3<br>4<br>5<br>6<br>7<br>8<br>9<br>10 | `add_filter(``'gform_form_theme_slug'``,``function``(``$slug``,``$form``) {`<br>```// Force Orbital theme for form preview.`<br>```if``( GFCommon::is_preview() ) {`<br>```return``'orbital'``;`<br>```}`<br>```return``$slug``;`<br>`}, 10, 2 );` |

## Placement

This code can be used in the [functions.php](https://developer.wordpress.org/themes/basics/theme-functions/) file of the active theme, a custom functions plugin, a custom add-on, or with a code snippets plugin.

See also the PHP section in this article: [Where Do I Put This Code?](https://docs.gravityforms.com/where-do-i-put-this-code/)

This filter was added in Gravity Forms 2.7
