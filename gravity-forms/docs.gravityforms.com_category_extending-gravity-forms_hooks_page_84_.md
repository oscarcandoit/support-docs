## Articles

[**gform\_stripe\_url\_port** \\
This filter sets the Stripe URL port if it’s not 80.](https://docs.gravityforms.com/gform_stripe_url_port/ "gform_stripe_url_port") [**gform\_stripe\_webhook** \\
This filter can be used to add support for performing actions on additional Stripe events.](https://docs.gravityforms.com/gform_stripe_webhook/ "gform_stripe_webhook") [**gform\_stripe\_webhook\_signing\_secret** \\
This filter allows the webhook signing secret for the specified API mode to be modified.](https://docs.gravityforms.com/gform_stripe_webhook_signing_secret/ "gform_stripe_webhook_signing_secret") [**gform\_submission\_data\_pre\_process\_payment** \\
This filter can be used to modify the submission data before it’s used during feed processing by a payment add-on.](https://docs.gravityforms.com/gform_submission_data_pre_process_payment/ "gform_submission_data_pre_process_payment") [**gform\_submission\_files\_pre\_save\_field\_value** \\
gform\_submission\_files\_pre\_save\_field\_value allows filtering (e.g., renaming) of the submission files before they are saved to the form uploads folder and entry.](https://docs.gravityforms.com/gform_submission_files_pre_save_field_value/ "gform_submission_files_pre_save_field_value") [**gform\_submission\_values\_pre\_save** \\
Allows the modification of submitted values before the incomplete submission is saved.](https://docs.gravityforms.com/gform_submission_values_pre_save/ "gform_submission_values_pre_save") [**gform\_submit\_button** \\
This filter is executed when the form is displayed and can be used to completely change the form button tag.](https://docs.gravityforms.com/gform_submit_button/ "gform_submit_button") [**gform\_subscription\_canceled** \\
This action hook can be used to perform custom actions when a subscription has been canceled.](https://docs.gravityforms.com/gform_subscription_canceled/ "gform_subscription_canceled") [**gform\_subscription\_details\_trial\_description** \\
The gform\_subscription\_details\_trial\_description filter allows customization of the trial description text in the subscription details on the entry detail page.](https://docs.gravityforms.com/gform_subscription_details_trial_description/ "gform_subscription_details_trial_description") [**gform\_suppress\_confirmation\_redirect** \\
