- [Since](https://docs.gravityforms.com/gform_field_filter_from_post/#h-since)

## Description

Enables the filter settings for the form fields retrieved from $\_POST to be modified.

## Usage

The following would apply to all forms:

|     |     |
| --- | --- |
| 1 | `add_filter(``'gform_field_filter_from_post'``,``'your_function_name'``, 10, 3 );` |

## Parameters

- **$field\_filter** [array](https://docs.gravityforms.com/array)

The form field, entry properties, and entry meta filter settings.
- **$form** [Form Object](https://docs.gravityforms.com/array)

The form object.
- **$field** [Field Object](https://docs.gravityforms.com/field-object)

The field object.

## Placement

This code can be used in the [functions.php](https://developer.wordpress.org/themes/basics/theme-functions/) file of the active theme, a custom functions plugin, a custom add-on, or with a code snippets plugin.

See also the PHP section in this article: [Where Do I Put This Code?](https://docs.gravityforms.com/where-do-i-put-this-code/)

## Source Code

This hook is located in _common.php_.
