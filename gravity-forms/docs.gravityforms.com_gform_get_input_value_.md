## Description

Use this filter to change the field’s value after being retrieved from the database. Use in conjunction with [gform\_save\_field\_value](https://docs.gravityforms.com/gform_save_field_value) to perform low level transformations, such as encrypting/decrypting a field.

## Usage

The base filter which would run for all forms and all fields would be used like so:

|     |     |
| --- | --- |
| 1 | `add_filter(``'gform_get_input_value'``,``'your_function_name'``, 10, 4 );` |

To target a specific form append the form id to the hook name. (format: gform\_get\_input\_value\_FORMID)

|     |     |
| --- | --- |
| 1 | `add_filter(``'gform_get_input_value_10'``,``'your_function_name'``, 10, 4 );` |

To target a specific field append both the form id and the field id to the hook name. (format: gform\_get\_input\_value\_FORMID\_FIELDID)

|     |     |
| --- | --- |
| 1 | `add_filter(``'gform_get_input_value_10_3'``,``'your_function_name'``, 10, 4 );` |

## Parameters

- **$value** [string](https://docs.gravityforms.com/string)
Field value to be filtered.

- **$entry** [Entry Object](https://docs.gravityforms.com/entry-object)

Current entry object.

- **$field** [Field Object](https://docs.gravityforms.com/field-object)

Current Field object.

- **$input\_id** [float](https://docs.gravityforms.com/float)

For multi-input fields (i.e. name, address, checkboxes), this parameter will hold the input ID. For single input fields, this parameter will be blank.

## Placement

This code should be placed in the [functions.php](http://codex.wordpress.org/Theme_Development#Functions_File) file of your active theme.

## Since

The base filter was added in Gravity Forms 1.5.2.1. The form and field specific versions were added in Gravity Forms 1.9.13.6
