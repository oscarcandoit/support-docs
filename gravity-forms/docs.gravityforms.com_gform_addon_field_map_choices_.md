## Description

The gform\_addon\_field\_map\_choices filter can be used to override the choices available in the field map drop down.

This filter is deprecated since version 2.5. Use the [gform\_field\_map\_choices](https://docs.gravityforms.com/gform_field_map_choices) filter instead.

## Usage

**Generic**: applies to all add-ons and all forms.

```
add_filter( 'gform_addon_field_map_choices', 'your_function_name', 10, 3 );
```

**Addon-specific**: applies to a specific add-on for all forms. Requires a minimum PHP version of 5.3.

```
add_filter( 'gform_{ADDON_SLUG}_field_map_choices', 'your_function_name', 10, 3 );
```

See the [Gravity Forms Add-On Slugs](https://docs.gravityforms.com/gravity-forms-add-on-slugs) article for a list of possible slugs.

## Parameters

- **$fields** [array](https://docs.gravityforms.com/array) The value and label properties for each choice.
- **$form\_id** [integer](https://docs.gravityforms.com/integer)
The ID of the form currently being configured.

- **$field\_type** [null](https://docs.gravityforms.com/null) \| [array](https://docs.gravityforms.com/array)
Null or the field types to be included in the drop down.

- **$exclude\_field\_types** [null](https://docs.gravityforms.com/null) \| [array](https://docs.gravityforms.com/array) \| [string](https://docs.gravityforms.com/string)
Null or the field type(s) to be excluded from the drop down.


## Examples

### 1\. Add new choice to all add-ons

```
add_filter( 'gform_addon_field_map_choices', function( $fields, $form_id, $field_type, $exclude_field_types ) {
    $fields[] = array( 'label' =&amp;amp;amp;amp;gt; 'The new choice', 'value' =&amp;amp;amp;amp;gt; 'new_choice' );

    return $fields;
}, 10, 4 );
```

## Placement

This code should be placed in the [functions.php](http://codex.wordpress.org/Theme_Development#Functions_File) file of your active theme.

## Source Code

This filter is located in _GFAddOn::get\_field\_map\_choices()_ in _/includes/addon/class-gf-addon.php_.

## Since

This filter was added in Gravity Forms 2.0.7.11.
