## Description

This hook runs when a transaction is completed successfully for the PayPal Standard Add-on and can be used to fire actions dependent on a successful PayPal transaction.

## Usage

|     |     |
| --- | --- |
| 1 | `add_action(``'gform_paypal_fulfillment'``,``'your_function_name'``, 10, 4 );` |

## Parameters

- **$entry** [Entry Object](https://docs.gravityforms.com/entry-object)
The entry used to generate the PayPal transaction.

- **$feed** [Feed Object](https://docs.gravityforms.com/feed-object)

The PayPal Feed configuration data used to generate the order.

- **$transaction\_id** [string](https://docs.gravityforms.com/string)

The transaction ID returned by PayPal.

- **$amount** [float](https://docs.gravityforms.com/float)

The amount of the transaction returned by PayPal.

This code should be placed in the [functions.php](http://codex.wordpress.org/Theme_Development#Functions_File) file of your active theme.
