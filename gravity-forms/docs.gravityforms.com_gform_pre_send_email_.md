## Description

Use this filter to modify the email before a notification has been sent. You may also use this to prevent an email from being sent.

## Usage

|     |
| --- |
| `add_filter(``'gform_pre_send_email'``,``'your_function_name'``, 10, 4 );` |

## Parameters

- **$email** [array](https://docs.gravityforms.com/array)


The email to be sent.


|     |
| --- |
| `Array`<br>`(`<br>```[to] => contactus@example.com`<br>```[subject] => New submission from A`<br>```[message] => This is the email body.`<br>```[headers] => Array`<br>```(`<br>```[From] => From:``"test@example.com"``<test@example.com>`<br>```[Content-type] => Content-type: text/html; charset=UTF-8`<br>```)`<br>``<br>```[attachments] => Array`<br>```(`<br>```)`<br>``<br>```[abort_email] => FALSE`<br>`)` |

- **$message\_format** [string](https://docs.gravityforms.com/string)


The message format, html or text.

- **$notification** [array](https://docs.gravityforms.com/array)


An array of properties which make up a notification object. See [Notifications Object](https://docs.gravityforms.com/notifications-object) for possible properties. Available from 1.9.15.6.

- **$entry** [array](https://docs.gravityforms.com/array)


The current [Entry Object](https://docs.gravityforms.com/entry-object). Available from 2.2.3.8.

### Abort emails

|     |     |
| --- | --- |

### Update email meta

This example updates the to address, subject, and message.

|     |     |
| --- | --- |
| 1<br>2<br>3<br>4<br>5<br>6<br>7 | `add_filter(``'gform_pre_send_email'``,``'before_email'``);`<br>`function``before_email(``$email``) {`<br>```$email``[``'to'``] =``'test@somedomain.com'``;`<br>```$email``[``'message'``] =``'This is my new message.'``;`<br>```$email``[``'subject'``] =``'This is a new subject.'``;`<br>```return``$email``;`<br>`}` |

### Add custom source header

The following shows how you can add a custom header to the email just before it is sent. In this case, the header would identify which form, entry, and notification the email originates from.

|     |     |
| --- | --- |
| 1<br>2<br>3<br>4<br>5 | `add_filter(``'gform_pre_send_email'``,``function``(``$email``,``$message_format``,``$notification``,``$entry``) {`<br>```$email``[``'headers'``][``'X-gf-source'``] = sprintf(``'X-gf-source: %d|%d|%s'``, rgar(``$entry``,``'form_id'``), rgar(``$entry``,``'id'``), rgar(``$notification``,``'id'``) );`<br>``<br>```return``$email``;`<br>`}, 10, 4 );` |

|     |     |
