# Merge Tags

## Articles

[**{entry} Merge Tag** \\
Can be used in areas such as notifications and confirmations to display entry properties after the entry has been saved.](https://docs.gravityforms.com/entry-merge-tag/ "{entry} Merge Tag") [**{form\_id} Merge Tag** \\
Displays the ID of the submitted form.](https://docs.gravityforms.com/form_id-merge-tag/ "{form_id} Merge Tag") [**{form\_title} Merge Tag** \\
Displays the title of the submitted form. Can be used in areas such as notifications and confirmations to access form properties.](https://docs.gravityforms.com/form_title-merge-tag/ "{form_title} Merge Tag") [**{ip} Merge Tag** \\
Displays the form submitter’s IP address.](https://docs.gravityforms.com/ip-merge-tag/ "{ip} Merge Tag") [**{payment\_action} Merge Tag** \\
The {payment\_action} merge tag can be used in notifications assigned to events added by payment add-ons such as Authorize.net, PayPal Standard, PayPal Payments Pro, and Stripe to output details of the event which triggered the notification.](https://docs.gravityforms.com/payment_action-merge-tag/ "{payment_action} Merge Tag") [**{post\_edit\_url} Merge Tag** \\
Displays the edit URL for the post which was created during submission.](https://docs.gravityforms.com/post_edit_url-merge-tag/ "{post_edit_url} Merge Tag") [**{post\_id} Merge Tag** \\
Displays the ID of the post which was created during submission.](https://docs.gravityforms.com/post-id-merge-tag/ "{post_id} Merge Tag") [**{pricing\_fields} Merge Tag** \\
Displays the product/order table in a confirmation or notification.](https://docs.gravityforms.com/pricing_fields-merge-tag/ "{pricing_fields} Merge Tag") [**{quiz\_grade} Merge Tag** \\
Checking this adds a Select All option to the top of the checkboxes list which allows the user to select all of the available list items.](https://docs.gravityforms.com/quiz_grade-merge-tag/ "{quiz_grade} Merge Tag") [**{quiz\_passfail} Merge Tag** \\
