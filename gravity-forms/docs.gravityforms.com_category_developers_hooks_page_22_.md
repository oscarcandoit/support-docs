## Articles

[**gform\_disable\_form\_settings\_sanitization** \\
Use this filter to disable the automatic sanitization of form settings. Gravity Forms will skip sanitizing form data such as titles, descriptions, CSS classes, and other form properties when enabled.](https://docs.gravityforms.com/gform_disable_form_settings_sanitization/ "gform_disable_form_settings_sanitization") [**gform\_disable\_form\_theme\_css** \\
The gform\_disable\_form\_theme\_css filter allows the default form theme used by forms created with Gravity Forms 2.5 and greater to be disabled.](https://docs.gravityforms.com/gform_disable_form_theme_css/ "gform_disable_form_theme_css") [**gform\_disable\_installation\_status** \\
This filter is used to disable the display of the Installation Status section on the Forms > Settings page.](https://docs.gravityforms.com/gform_disable_installation_status/ "gform_disable_installation_status") [**gform\_disable\_notification** \\
Use this filter to disable admin and user notification emails.](https://docs.gravityforms.com/gform_disable_notification/ "gform_disable_notification") [**gform\_disable\_post\_creation** \\
Use this filter to disable post creation when submitting a Gravity Form.](https://docs.gravityforms.com/gform_disable_post_creation/ "gform_disable_post_creation") [**gform\_disable\_print\_form\_scripts** \\
This filter can be used to prevent scripts and stylesheets being printed.](https://docs.gravityforms.com/gform_disable_print_form_scripts/ "gform_disable_print_form_scripts") [**gform\_disable\_registration** \\
This filter allows add-ons to prevent the User Registration add-on from registering/updating a user.](https://docs.gravityforms.com/gform_disable_registration/ "gform_disable_registration") [**gform\_disable\_resend\_notification** \\
The gform\_disable\_resend\_notification filter allows developers to prevent a notification email from being resent.](https://docs.gravityforms.com/gform_disable_resend_notification/ "gform_disable_resend_notification") [**gform\_disable\_view\_counter** \\
