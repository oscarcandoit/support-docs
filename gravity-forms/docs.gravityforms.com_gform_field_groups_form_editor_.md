## Description

The “gform\_field\_groups\_form\_editor” filter in Gravity Forms is used to modify the field groups in the Form Editor before fields are added, allowing control over what displays.

## Usage

|     |     |
| --- | --- |
| 1 | `add_filter(``'gform_field_groups_form_editor'``,``'your_function_name'``, 10, 1 );` |

## Parameters

- **$field\_groups** [array](https://docs.gravityforms.com/array) The field groups, including group name, label and fields.

## Example

This example changes the label for Standard Fields to Testing.

|     |     |
| --- | --- |
| 1<br>2<br>3<br>4<br>5<br>6<br>7<br>8<br>9 | `add_filter(``'gform_field_groups_form_editor'``,``'change_title'``, 10, 1 );`<br>`function``change_title(``$field_groups``){`<br>`foreach``(``$field_groups``as``&amp;``$group``){`<br>`if``(``$group``[``'name'``] ==``'standard_fields'``){`<br>`$group``[``'label'``] =``'Testing'``;`<br>`}`<br>`}`<br>`return``$field_groups``;`<br>`}` |

## Placement

Your code snippet should be placed in the [functions.php](https://codex.wordpress.org/Theme_Development#Functions_File) file of your active theme.

## Source Code

This filter is located in GFFormDetail::get\_field\_groups() in _gravityforms/form\_detail.php_.
