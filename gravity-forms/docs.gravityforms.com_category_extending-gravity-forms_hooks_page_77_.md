[**gform\_search\_criteria\_export\_entries** \\

Allows for override of SSL replacement.](https://docs.gravityforms.com/gform_secure_file_download_is_https/ "gform_secure_file_download_is_https") [**gform\_secure\_file\_download\_location** \\
By default the real location of the uploaded file will be hidden. Return FALSE to display the real location. Not recommended – use with caution!](https://docs.gravityforms.com/gform_secure_file_download_location/ "gform_secure_file_download_location") [**gform\_secure\_file\_download\_url** \\
Allows for manual filtering of the download URL to handle conditions such as unusual domain mapping and others.](https://docs.gravityforms.com/gform_secure_file_download_url/ "gform_secure_file_download_url") [**gform\_send\_email\_failed** \\
Triggered when an email from Gravity Forms fails to send.](https://docs.gravityforms.com/gform_send_email_failed/ "gform_send_email_failed") [**gform\_sendgrid\_email** \\
Allows the email being sent by SendGrid to be modified.](https://docs.gravityforms.com/gform_sendgrid_email/ "gform_sendgrid_email") [**gform\_sendgrid\_send\_email\_failed** \\
