[**gform\_user\_select\_query\_params** \\
The gform\_user\_select\_query\_params allows to filter the parameters for the query for the initial list of users in the user select dropdown when the Settings API renders a field of type user\_select.](https://docs.gravityforms.com/gform_user_select_query_params/ "gform_user_select_query_params") [**gform\_user\_updated** \\
This action is used to trigger an event once a user has been updated through the User Registration Add-On.](https://docs.gravityforms.com/gform_user_updated/ "gform_user_updated") [**gform\_username** \\
This filter is used to dynamically alter or generate a username during the registration process.](https://docs.gravityforms.com/gform_username/ "gform_username") [**gform\_userregistration\_associate\_entry\_with\_user** \\
Used to determine if the entry should be associated with newly created user.](https://docs.gravityforms.com/gform_userregistration_associate_entry_with_user/ "gform_userregistration_associate_entry_with_user") [**gform\_userregistration\_delete\_signup** \\
This action allows custom actions to be performed when pending activations are deleted.](https://docs.gravityforms.com/gform_userregistration_delete_signup/ "gform_userregistration_delete_signup") [**gform\_userregistration\_feed\_settings\_fields** \\
This filter can be used to modify the setting fields that appear on the feed page.](https://docs.gravityforms.com/gform_userregistration_feed_settings_fields/ "gform_userregistration_feed_settings_fields") [**gform\_userregistration\_login\_form** \\
This filter allows the modification of the object for the login form used in the Gravity Forms Login Widget.](https://docs.gravityforms.com/gform_user_registration_login_form/ "gform_userregistration_login_form") [**gform\_validate\_required\_file\_exists** \\
