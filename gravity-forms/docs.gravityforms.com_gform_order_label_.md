## Description

The “gform\_order\_label” filter in Gravity Forms is executed when displaying the “Order” heading on the entry detail page or when sending the order information via email.

## Usage

Applies to all forms

|     |     |
| --- | --- |
| 1 | `add_filter(``'gform_order_label'``,``'change_order_label'``, 10, 2 );` |

Applies to a specific form. In this case, form Id 5

|     |     |
| --- | --- |
| 1 | `add_filter(``'gform_order_label_5'``,``'change_order_label'``, 10, 2 );` |

## Parameters

- **$label** [string](https://docs.gravityforms.com/string)


The label to be filtered.

- **$form\_id** [integer](https://docs.gravityforms.com/integer)



The current form’s id.


## Examples

This example changes the default Order label to Estimate;

|     |     |
| --- | --- |
| 1<br>2<br>3<br>4 | `add_filter(``'gform_order_label'``,``'change_order_label'``, 10, 2 );`<br>`function``change_order_label(``$label``,``$form_id``) {`<br>`return``'Estimate'``;`<br>`}` |

## Source Code

This filter is located in the following methods:

- GFCommon::get\_submitted\_pricing\_fields() in _common.php_
- GFEntryDetail::lead\_detail\_grid() in _entry\_detail.php_
