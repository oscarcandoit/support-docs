## Description

The gform\_dropbox\_should\_upload\_file filter can be used to determine if a file for a fileupload or dropbox type field is uploaded to Dropbox.

## Usage

The following would apply to all forms:

|     |     |
| --- | --- |
| 1 | `add_filter(``'gform_dropbox_should_upload_file'``,``'your_function_name'``, 10, 7 );` |

To target a specific form append the form id to the hook name. (format: gform\_dropbox\_should\_upload\_file\_FORMID)

|     |     |
| --- | --- |
| 1 | `add_filter(``'gform_dropbox_should_upload_file_2'``,``'your_function_name'``, 10, 7 );` |

To target a specific field append both the form id and the field id to the hook name. (format: gform\_dropbox\_should\_upload\_file\_FORMID\_FIELDID)

|     |     |
| --- | --- |
| 1 | `add_filter(``'gform_dropbox_should_upload_file_2_3'``,``'your_function_name'``, 10, 7 );` |

## Parameters

- **$should\_upload\_file** [boolean](https://docs.gravityforms.com/boolean/)

Indicates if the file should be uploaded to Dropbox.
- **$url** [string](https://docs.gravityforms.com/string/)

The URL of the file from the entry.
- **$existing** [false](https://docs.gravityforms.com/boolean/) \| [array](https://docs.gravityforms.com/array/)

False or an array of files previously uploaded to Dropbox for the current field and entry.
- **$field** [Field Object](https://docs.gravityforms.com/field-object/)

The current fileupload or dropbox field object
- **$form** [Form Object](https://docs.gravityforms.com/form-object/)

The form containing the current field.
- **$entry** [Entry Object](https://docs.gravityforms.com/entry-object/)

The entry currently being processed.
- **$feed** [Feed Object](https://docs.gravityforms.com/feed-object/)

The feed currently being processed.
