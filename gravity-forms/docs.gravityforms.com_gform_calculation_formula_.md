## Description

This filter can be used to dynamically modify the formula of a number field calculation or calculated product field.

## Usage

The gform\_calculation\_formula filter has both a [JavaScript version](https://docs.gravityforms.com/gform_calculation_formula/#javascript-version) and a [PHP version](https://docs.gravityforms.com/gform_calculation_formula/#php-version). **Both versions should be used.**

The JavaScript version only overrides the calculation formula on the front-end.

|     |     |
| --- | --- |
| 1<br>2<br>3<br>4<br>5 | `gform.addFilter(``'gform_calculation_formula'``,``function``( formula, formulaField, formId, calcObj ) {`<br>`// do stuff`<br>`return``formula;`<br>`} );` |

The PHP version overrides the calculation formula when the calculation is rerun during submission using the field values saved in the entry.

|     |     |
| --- | --- |
| 1<br>2<br>3<br>4<br>5 | `add_filter(``'gform_calculation_formula'``,``function``(``$formula``,``$field``,``$form``,``$entry``) {`<br>`// do stuff`<br>`return``$formula``;`<br>`}, 10, 4 );` |

## JavaScript Version

### Parameters

- **formula** [string](https://docs.gravityforms.com/string)
The calculation formula.

- **formulaField** [Javascript Object](https://docs.gravityforms.com/javascript-object)

The current calculation field object. e.g.


|     |     |
| --- | --- |
| 1 | `{``"field_id"``:3,``"formula"``:``"{:1}+{:2}"``,``"rounding"``:``""``}` |

**formId** [integer](https://docs.gravityforms.com/integer)

The ID of the form in use.

**calcObj** [Javascript Object](https://docs.gravityforms.com/javascript-object)

The calculation object.

### JS Example

|     |     |
| --- | --- |
| 1<br>2<br>3<br>4<br>5<br>6 | `gform.addFilter(``'gform_calculation_formula'``,``function``( formula, formulaField, formId, calcObj ) {`<br>`if``( formId ==``'10'``&& formulaField.field_id ==``'3'``) {`<br>`formula +=``'+5'``;`<br>`}`<br>`return``formula;`<br>`} );` |

### Placement

Your code snippet can be placed in a HTML field on your form or in a theme custom JavaScript file.

### Source Code

This filter is located in _js/gravityforms.js_

## PHP Version

### Parameters

- **$formula** [string](https://docs.gravityforms.com/string)
The calculation formula.

- **$field** [Field Object](https://docs.gravityforms.com/field-object)

The calculation field currently being processed.

- **$form** [Form Object](https://docs.gravityforms.com/form-object)

The form currently being processed.

- **$entry** [Entry Object](https://docs.gravityforms.com/entry-object)

The entry currently being processed.


### PHP Example

|     |     |
| --- | --- |
| 1<br>2<br>3<br>4<br>5<br>6<br>7<br>8 | `add_filter(``'gform_calculation_formula'``,``'change_formula'``, 10, 4 );`<br>`function``change_formula(``$formula``,``$field``,``$form``,``$entry``) {`<br>`if``(``$form``[``'id'``] == 10 &&``$field``->id == 3 ) {`<br>`$formula``.=``'+5'``;`<br>`}`<br>`return``$formula``;`<br>`}` |

### Placement

This code should be placed in the **functions.php** file of your active theme.

### Source Code

This filter is located in GFCommon::calculate() in _common.php_
