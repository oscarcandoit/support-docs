## Description

This filter can be used to modify the lead arguments before they are sent to Zoho CRM.

## Usage

The following would apply to all feeds:

|     |     |
| --- | --- |
| 1 | `add_filter(``'gform_zohocrm_lead'``,``'your_function_name'``, 10, 4 );` |

To target feeds for a specific form append the form id to the hook name. (format: gform\_zohocrm\_lead\_FORMID)

|     |     |
| --- | --- |
| 1 | `add_filter(``'gform_zohocrm_lead_4'``,``'your_function_name'``, 10, 4 );` |

## Parameters

- **$lead** [array](https://docs.gravityforms.com/array)
The lead arguments are an associative array.


|     |     |
| --- | --- |
| 1<br>2<br>3<br>4<br>5<br>6<br>7<br>8<br>9<br>10<br>11<br>12<br>13 | `array``(`<br>`'Email Opt Out'``=>``'false'``,`<br>`'Description'``=>``'some text'``,`<br>`'Lead Source'``=>``'Advertisement'``,`<br>`'Lead Status'``=>``'Not Contacted'``,`<br>`'Rating'``=>``'Acquired'``,`<br>`'SMOWNERID'``=>``'The-Zoho-CRM-User-ID-Here'``,`<br>`'options'``=>``array``(`<br>`'duplicateCheck'``=>``'1'``,`<br>`'isApproval'``=>``'false'``,`<br>`'wfTrigger'``=>``'false'`<br>`),`<br>`)` |

**$feed** [Feed Object](https://docs.gravityforms.com/feed-object)
The feed currently being processed.

**$entry** [Entry Object](https://docs.gravityforms.com/entry-object)
The entry currently being processed.

**$form** [Form Object](https://docs.gravityforms.com/form-object)
The form currently being processed.

## Examples

### 1\. Change Email Opt Out

This example shows how you can change the ‘Email Opt Out’ setting based on a field value in the [Entry Object](https://docs.gravityforms.com/entry-object).

|     |     |
| --- | --- |
| 1<br>2<br>3<br>4<br>5<br>6<br>7<br>8 | `add_filter(``'gform_zohocrm_lead_4'``,``'change_lead_argument'``, 10, 4 );`<br>`function``change_lead_argument(``$lead``,``$feed``,``$entry``,``$form``) {`<br>`if``( rgars(``$feed``,``'meta/feedName'``) ==``'Zoho CRM Feed 2'``&& rgar(``$entry``,``'5'``) ==``'No'``) {`<br>`$lead``[``'Email Opt Out'``] =``'true'``;`<br>`}`<br>`return``$lead``;`<br>`}` |

### 2\. Set Owner based on country

|     |     |
| --- | --- |
| 1<br>2<br>3<br>4<br>5<br>6<br>7<br>8<br>9<br>10<br>11<br>12<br>13<br>14<br>15<br>16<br>17<br>18<br>19 | `add_filter(``'gform_zohocrm_lead_9'``,``'change_lead_argument'``, 10, 4 );`<br>`function``change_lead_argument(``$lead``,``$feed``,``$entry``,``$form``) {`<br>`if``( rgars(``$feed``,``'meta/feedName'``) ==``'Zoho Sales Inquiry'``) {`<br>`// get the selected country from field 4`<br>`$country``= rgar(``$entry``,``'4'``);`<br>`// define an array containing the countries and the zoho crm id to be used for that country`<br>`$owners``=``array``(`<br>`'Afghanistan'``=>``'The-Zoho-CRM-User-ID-Here'``,`<br>`'Albania'``=>``'The-Zoho-CRM-User-ID-Here'``,`<br>`);`<br>`// replace the feed configured owner with the id by using the $country as the key to the value in the $owners array`<br>`$lead``[``'SMOWNERID'``] = rgar(``$owners``,``$country``);`<br>`}`<br>`return``$lead``;`<br>`}` |

### 3\. Set the Layout

This example shows how you can define the Layout the lead should use for a specific feed of form 4.

|     |     |
| --- | --- |
| 1<br>2<br>3<br>4<br>5<br>6<br>7 | `add_filter(``'gform_zohocrm_lead_4'``,``function``(``$lead``,``$feed``,``$entry``,``$form``) {`<br>`if``( rgars(``$feed``,``'meta/feedName'``) ==``'Zoho CRM Feed 2'``) {`<br>`$lead``[``'Layout'``] =``array``(``'name'``=>``'the layout name here'``,``'id'``=>``'the layout id here'``);`<br>`}`<br>`return``$lead``;`<br>`}, 10, 4 );` |

### 4\. Preserve Lead Source on Update

This example shows how you can preserve any existing Lead Source in Zoho for all feeds on form 33.

|     |     |
| --- | --- |
| 1<br>2<br>3<br>4<br>5 | `add_filter(``'gform_zohocrm_lead_33'``,``function``(``$lead``,``$feed``,``$entry``,``$form``) {`<br>`unset(``$lead``[``'Lead_Source'``] );`<br>`return``$lead``;`<br>`}, 10, 4 );` |

### 5\. Add the $gclid parameter

This example shows how to get the value from a [Hidden field](https://docs.gravityforms.com/hidden/) of a form, where [dynamic population](https://docs.gravityforms.com/using-dynamic-population/) has been enabled and **gclid** used as the Parameter Name, and pass it to Zoho CRM as value for $gclid

|     |     |
| --- | --- |
| 1<br>2<br>3<br>4<br>5<br>6<br>7<br>8<br>9<br>10<br>11<br>12 | `add_filter(``'gform_zohocrm_lead'``,``function``(``$lead``,``$feed``,``$entry``,``$form``) {`<br>```gf_zohocrm()->log_debug(``'Adding $glid parameter...'``);`<br>```foreach``(``$form``[``'fields'``]``as``&``$field``) {`<br>```// Your hidden field must use gclid (case sensitive) as the Parameter Name for dynamic population.`<br>```if``(``'gclid'``===``$field``->inputName ) {`<br>```$lead``[``'$gclid'``] = rgar(``$entry``,``$field``->id );`<br>```gf_zohocrm()->log_debug(``'$glid set to: '``.``$lead``[``'$gclid'``] );`<br>```}`<br>```}`<br>```return``$lead``;`<br>`}, 10, 4 );` |

### 6\. Use a field value for Lead Description

By default the Lead Description can be set manually in your [feed settings](https://docs.gravityforms.com/creating-a-feed-for-the-zoho-crm-add-on/#h-creating-a-new-lead). The following example shows how it can be set using a value from a field in your form where you have set **zohocrm\_lead\_description** as the field **Admin Field Label**.

|     |     |
| --- | --- |
| 1<br>2<br>3<br>4<br>5<br>6<br>7<br>8<br>9<br>10<br>11<br>12 | `add_filter(``'gform_zohocrm_lead'``,``function``(``$lead``,``$feed``,``$entry``,``$form``) {`<br>```gf_zohocrm()->log_debug(``'Adding custom Lead Description from a field...'``);`<br>``<br>```foreach``(``$form``[``'fields'``]``as``&``$field``) {`<br>```// Your field must use zohocrm_lead_description (case sensitive) as the Admin Field Label.`<br>```if``(``'zohocrm_lead_description'``===``$field``->adminLabel ) {`<br>```$lead``[``'Description'``] = rgar(``$entry``,``$field``->id );`<br>```gf_zohocrm()->log_debug(``'Description set to: '``.``$lead``[``'Description'``] );`<br>```}`<br>```}`<br>```return``$lead``;`<br>`}, 10, 4 );` |
