# List of Settings API Field Types

The following field types are available when creating settings using the Settings API. The link takes you to user guide documentation for that field type.

| Field Type | Description |
| --- | --- |
| [Card](https://docs.gravityforms.com/card-field/) | Creates radio inputs in a stylized card. |
| [Checkbox](https://docs.gravityforms.com/checkbox-field/) | Creates a checkbox field. |
| [Checkbox and Select](https://docs.gravityforms.com/checkbox_and_select) | Creates a complex field where a hidden drop-down field is displayed when the checkbox is checked. |
| [Custom Field](https://docs.gravityforms.com/custom-field-types) | Learn how to create custom field types. |
| [Dynamic Field Map](https://docs.gravityforms.com/dynamic_field_map-field) | Creates a set of fields that allow the user to map their form fields to custom keys (fields) entered by the user. |
| [Field Map](https://docs.gravityforms.com/field_map-field) | Creates a set of fields that allow the user to map their form fields to a custom field selected from a predefined list. |
| [Field Select](https://docs.gravityforms.com/field_select-field/) | Creates a drop down field populated with fields from the form. |
| [Generic Map](https://docs.gravityforms.com/generic_map-field/) | Creates a set of fields that allow the user to map custom keys to custom values, based on submission or other data. |
| [Hidden](https://docs.gravityforms.com/hidden-field) | Creates a hidden field. |
| [Post Select](https://docs.gravityforms.com/post-select-field-type/) | Creates a searchable drop down field populated with posts from specific post type. |
| [Radio Button](https://docs.gravityforms.com/radio-field) | Creates a radio button field. |
| [Save Button](https://docs.gravityforms.com/save-button) | Creates a save button. |
| [Select](https://docs.gravityforms.com/select-field) | Creates a select/drop-down field. |
| [Select Custom](https://docs.gravityforms.com/select_custom-field) | Creates a select/drop-down field which also allows users to enter a custom value. |
| [Simple Condition](https://docs.gravityforms.com/field-types/the-simple_condition-helper/) | Creates a complex set of fields allowing users to specify a condition (i.e. Country is Brazil). |
| [Text](https://docs.gravityforms.com/text-field) | Creates a text field. |
| [Textarea](https://docs.gravityforms.com/textarea-field) | Creates a textarea field. |
| [User Select](https://docs.gravityforms.com/?p=42829&preview=true) | Creates a select/drop-down field. |
