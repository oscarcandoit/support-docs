# Merge Tags Reference

Gravity Forms uses **merge tags** to dynamically insert submitted field values and other relevant information into notifications, confirmations, post content templates, and more. This article offers a quick reference guide and links to detailed documentation for each tag.

**Note**: Not all merge tags are supported in every context where merge tags are accepted. For instance, some merge tags can’t be used as default values. Be sure to review the individual documentation pages for usage guidelines and limitations.

| Merge Tag | Description |
| --- | --- |
| **Form Related** | – [form id](https://docs.gravityforms.com/form-id-merge-tag/): The ID of the form.<br>– [form title](https://docs.gravityforms.com/form-title-merge-tag/): The title of the form.<br>– [embed url](https://docs.gravityforms.com/embed-url-merge-tag/): The URL where the form is embedded. |
| **Field Related** | – [Field merge tag format & modifiers](https://docs.gravityforms.com/merge-tags/#field-merge-tags)<br>– [all\_fields](https://docs.gravityforms.com/all-fields-merge-tag/): Outputs all submitted fields.<br>– [Custom field](https://docs.gravityforms.com/custom-field-merge-tag/): Reference custom meta fields.<br>– [Standard Fields](https://docs.gravityforms.com/category/fields/standard-fields/)<br>– [Advanced Fields](https://docs.gravityforms.com/category/fields/advanced-fields/)<br>– [Pricing Fields](https://docs.gravityforms.com/category/fields/pricing-fields/)<br>– [Post Fields](https://docs.gravityforms.com/category/fields/post-fields/)<br>– [Coupon field](https://docs.gravityforms.com/coupon-merge-tag/)<br>– [Dropbox Upload field](https://docs.gravityforms.com/dropbox-upload-merge-tag/) |
| **Entry Related** | – [entry meta](https://docs.gravityforms.com/entry-meta-merge-tag/): Access entry metadata.<br>– [entry url](https://docs.gravityforms.com/entry-url-merge-tag/): Direct link to the entry.<br>– [submitter’s IP](https://docs.gravityforms.com/ip-merge-tag/): User’s IP address.<br>– [user agent](https://docs.gravityforms.com/user-agent-merge-tag/): Browser info.<br>– [entry creator](https://docs.gravityforms.com/entry-creator-merge-tag/): User who submitted entry.<br>– [entry date tags](https://docs.gravityforms.com/entry-date-merge-tags/): Date/time entry submitted.<br>– `{entry}` |
| **User Properties** | – [user property tags](https://docs.gravityforms.com/user-merge-tag/): Display user data like name or email. |
| **Post Field Related** | – [post id](https://docs.gravityforms.com/post-id-merge-tag/): ID of post created by form.<br>– [embed post](https://docs.gravityforms.com/embed_post-merge-tag/): Post embed merge tag.<br>– Post edit URL, post images, and custom fields also supported. |
| **Other Merge Tags** | – `{today}`: Outputs the current date/time.<br>– `{pricing_fields}`: Displays pricing table (products, totals, etc.).<br>– `{referer}`: Referring page’s URL.<br>– `{admin_email}`: Email from WP general settings.<br>– `{helpscout:[prop]}`: Help Scout conversation property. |
