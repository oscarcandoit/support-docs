## Description

Triggered after the payment details are displayed within a Gravity Forms entry.

## Usage

|     |     |
| --- | --- |
| 1 | `add_action(``'gform_payment_details'``,``'my_function'``, 10, 2 );` |

## Parameters

- **$form\_id** [integer](https://docs.gravityforms.com/integer)
The current form ID.

- **$entry** [array](https://docs.gravityforms.com/array)

The current entry.
