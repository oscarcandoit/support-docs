## Description

Use this filter to add custom form actions which display below the form title in the Form List view.

## Usage

|     |     |
| --- | --- |
| 1 | `add_filter(``'gform_form_actions'``,``'form_actions'``, 10, 2 );` |

## Parameters

- **$actions** [array](https://docs.gravityforms.com/array)
An associative array containing all of the default form actions (ie Edit, Preview, Notifications, etc.).





|     |     |
| --- | --- |
| 1<br>2<br>3<br>4<br>5<br>6<br>7<br>8 | `array``(`<br>```'edit'``=>``'<a href="" ... >Edit</a>'``,`<br>```'preview'``=>``'<a href="" ... >Preview</a>'``,`<br>```'entries'``=>``'<a href="" ... >Entries</a>'``,`<br>```'notifications'``=>``'<a href="" ... >Notifications</a>'``,`<br>```'duplicate'``=>``'<a href="" ... >Duplicate</a>'``,`<br>```'delete'``=>``'<a href="" ... >Delete</a>'`<br>`);` |

- $form\_id [integer](https://docs.gravityforms.com/integer)

The ID of the field from which the entry value was submitted.
