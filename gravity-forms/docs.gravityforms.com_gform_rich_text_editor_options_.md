## Description

The “gform\_rich\_text\_editor\_options” filter in Gravity Forms allows the rich text editor options to be modified. A specific FORMID or FORMID + FIELDID may also be used with the hook.

## Usage

Applies to all forms:

|     |     |
| --- | --- |
| 1 | `add_filter(``'gform_rich_text_editor_options'``,``'my_function'``, 10, 4 );` |

Targets a specific form:

|     |     |
| --- | --- |
| 1 | `add_filter(``'gform_rich_text_editor_options_3'``,``'my_function'``, 10, 4 );` |

Targets a specific field for a specific form:

|     |     |
| --- | --- |
| 1 | `add_filter(``'gform_rich_text_editor_options_3_6'``,``'my_function'``, 10, 4 );` |

## Parameters

- **$editor\_settings** [array](https://docs.gravityforms.com/array)
Array of buttons to include within the TinyMCE editor inside Gravity Forms.

Defaults to the following:





|     |     |
| --- | --- |
| 1<br>2<br>3<br>4<br>5<br>6<br>7<br>8<br>9<br>10 | `array``(`<br>```'textarea_name'``=>``'input_'``.``$id``,`<br>```'wpautop'``=> true,`<br>```'editor_class'``=>``$class``,`<br>```'textarea_rows'``=> 10,`<br>```'tabindex'``=>``$tabindex``,`<br>```'media_buttons'``=> false,`<br>```'quicktags'``=> false,`<br>```'tinymce'``=>``array``(``'init_instance_callback'``=>``'function...'``),`<br>`);` |

- **$field\_object** [object](https://docs.gravityforms.com/object)

The full field object returned.

- **$form** [array](https://docs.gravityforms.com/array)

The form that the filter is being run on.

- **$entry** [array](https://docs.gravityforms.com/array)

The entry, if available.


## Examples

### 1\. Set a custom height value

|     |     |
| --- | --- |
| 1<br>2<br>3<br>4<br>5 | `function``my_function(``$editor_settings``,``$field_object``,``$form``,``$entry``) {`<br>```$editor_settings``[``'editor_height'``] = 400;`<br>```return``$editor_settings``;`<br>`}`<br>`add_filter(``'gform_rich_text_editor_options'``,``'my_function'``, 10, 4 );` |

### 2\. Enable the Add Media button

This will add the Add Media button before the toolbar just like WP does in Post/Page editor. The user must be logged in to see the button; this is a WordPress requirement not a Gravity Forms limitation.

|     |     |
| --- | --- |
| 1<br>2<br>3<br>4<br>5 | `function``show_media_button(``$editor_settings``,``$field_object``,``$form``,``$entry``) {`<br>```$editor_settings``[``'media_buttons'``] = true;`<br>```return``$editor_settings``;`<br>`}`<br>`add_filter(``'gform_rich_text_editor_options'``,``'show_media_button'``, 10, 4 );` |
