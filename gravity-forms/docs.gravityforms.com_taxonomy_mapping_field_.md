## Description

Allows users to modify which field is used when looking up terms via `get_term_by()`. Defaults to `name`, but can be any of: ‘slug’, ‘name’, ‘term\_id’ (or ‘id’, ‘ID’), or ‘term\_taxonomy\_id’

## Usage

The following would apply to all forms:

|     |     |
| --- | --- |
| 1 | `add_filter(``'gform_advancedpostcreation_taxonomy_mapping_field'``,``'your_function_name'``, 10, 3 );` |

## Parameters

- **$field** [string](https://docs.gravityforms.com/array/)

Field used when looking up terms via `get_term_by()`.
- **$taxonomy** [string](https://docs.gravityforms.com/array/)

Taxonomy currently being mapped _to_, ie `category`
- **$values** [array](https://docs.gravityforms.com/array/)

Array of the values _to be mapped_ to the current Taxonomy. These represent the submitted field values.


## Examples

### Assign Taxonomy based on Term ID

Use the `term_id` instead of `name` to assign the taxonomy.

|     |     |
| --- | --- |
| 1<br>2<br>3<br>4 | `// Use term_id instead of name to assign the taxonomy`<br>`add_filter(``'gform_advancedpostcreation_taxonomy_mapping_field'``,``function``(``$field``,``$taxonomy``) {`<br>```return``'term_id'``;``// can also use 'id' or 'ID'`<br>`}, 10, 2 );` |

### Assign Taxonomy based on Slug

Use the `slug` instead of `name` to assign the taxonomy.

|     |     |
| --- | --- |
| 1<br>2<br>3<br>4 | `// Use slug instead of name for assigning the taxonomy`<br>`add_filter(``'gform_advancedpostcreation_taxonomy_mapping_field'``,``function``(``$field``,``$taxonomy``) {`<br>```return``'slug'``;`<br>`}, 10, 2 );` |

The filter also passes through the `$taxonomy` being checked against, so you could conditionally choose which field to use for mapping based on that.

|     |     |
| --- | --- |
| 1<br>2<br>3<br>4<br>5<br>6<br>7 | `add_filter(``'gform_advancedpostcreation_taxonomy_mapping_field'``,``function``(``$field``,``$taxonomy``) {`<br>```if``(``$taxonomy``!==``'category'``) {`<br>```return``$field``;`<br>```}`<br>``<br>```return``'term_id'``;`<br>`}, 10, 2 );` |

## Placement

This code can be used in the [functions.php](https://developer.wordpress.org/themes/basics/theme-functions/) file of the active theme, a custom functions plugin, a custom add-on, or with a code snippets plugin.

See also the PHP section in this article: [Where Do I Put This Code?](https://docs.gravityforms.com/where-do-i-put-this-code/)

## Since

This filter was added in the Gravity Forms Advanced Post Creation add-on version 1.3.

## Source Code

This filter is located in **GF\_AdvancedPost\_Creation::get\_taxonomy\_mapping\_term\_ids** in _class-gf-advancedpostcreation.php_.
