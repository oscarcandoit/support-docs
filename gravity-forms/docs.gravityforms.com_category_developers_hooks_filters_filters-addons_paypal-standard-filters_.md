# PayPal Standard Filters

## Articles

[**gform\_paypal\_action\_fields** \\
This hook is used add individual options to the PayPal feed.](https://docs.gravityforms.com/gform_paypal_action_fields/ "gform_paypal_action_fields") [**gform\_paypal\_add\_option\_group** \\
This hook is used to add groups of options to the PayPal feed.](https://docs.gravityforms.com/gform_paypal_add_option_group/ "gform_paypal_add_option_group") [**gform\_paypal\_business\_email** \\
Allows changes to be made to the PayPal business email while validating the IPN from PayPal.](https://docs.gravityforms.com/gform_paypal_business_email/ "gform_paypal_business_email") [**gform\_paypal\_config\_validation** \\
Used to validate custom settings/meta specified using the gform\_paypal\_action\_fields or gform\_paypal\_add\_option\_group action hooks.](https://docs.gravityforms.com/gform_paypal_config_validation/ "gform_paypal_config_validation") [**gform\_paypal\_feed\_settings\_fields** \\
This filter allows adding custom settings fields to PayPal Standard.](https://docs.gravityforms.com/gform_paypal_feed_settings_fields/ "gform_paypal_feed_settings_fields") [**gform\_paypal\_field\_value** \\
This filter can be used to modify a value before it is sent to PayPal using the PayPal Standard Add-On.](https://docs.gravityforms.com/gform_paypal_field_value/ "gform_paypal_field_value") [**gform\_paypal\_fulfillment** \\
This hook runs when a transaction is completed successfully for the PayPal Standard Add-on and can be used to fire actions dependent on a successful PayPal transaction.](https://docs.gravityforms.com/gform_paypal_fulfillment/ "gform_paypal_fulfillment") [**gform\_paypal\_get\_payment\_feed** \\
This filter allows the filtering of return configuration feeds.](https://docs.gravityforms.com/gform_paypal_get_payment_feed/ "gform_paypal_get_payment_feed") [**gform\_paypal\_hash\_matches** \\
This filter allows additional validation of the hash to be performed.](https://docs.gravityforms.com/gform_paypal_hash_matches/ "gform_paypal_hash_matches") [**gform\_paypal\_invoice** \\
This filter allows the invoice number, sent to PayPal for the PayPal Standard Add-on, to be changed.](https://docs.gravityforms.com/gform_paypal_invoice/ "gform_paypal_invoice") 1 [2](https://docs.gravityforms.com/category/developers/hooks/filters/filters-addons/paypal-standard-filters/page/2/) [3](https://docs.gravityforms.com/category/developers/hooks/filters/filters-addons/paypal-standard-filters/page/3/) [Next](https://docs.gravityforms.com/category/developers/hooks/filters/filters-addons/paypal-standard-filters/page/2/)
