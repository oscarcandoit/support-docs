[**gform\_cleanup\_target\_dir** \\
The gform\_cleanup\_target\_dir filter is used to bypass functionality that relies on open\_dir(), enabling support for the multi-file upload field on servers that don’t support open\_dir(), such as WordPress VIP.](https://docs.gravityforms.com/gform_cleanup_target_dir/ "gform_cleanup_target_dir") [**gform\_column\_input** \\
This filter can be used to specify a different input type for a list field column. Currently supported field types are "select" (Drop Down) and "text" (Text Field).](https://docs.gravityforms.com/gform_column_input/ "gform_column_input") [**gform\_column\_input\_content** \\
