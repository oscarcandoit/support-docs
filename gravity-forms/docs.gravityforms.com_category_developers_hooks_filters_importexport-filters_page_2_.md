# Import/Export Filters

## Articles

[**gform\_export\_separator** \\
Use the gform\_export\_separator filter to change the column separator character for the entry export file.](https://docs.gravityforms.com/gform_export_separator/ "gform_export_separator") [**gform\_field\_filter\_from\_post** \\
Enables the filter settings for the form fields retrieved from $\_POST to be modified.](https://docs.gravityforms.com/gform_field_filter_from_post/ "gform_field_filter_from_post") [**gform\_filters\_get\_users** \\
Use this filter to control the list of users available in the filters on the entry list, export entries conditional logic and results pages.](https://docs.gravityforms.com/gform_filters_get_users/ "gform_filters_get_users") [**gform\_form\_export\_filename** \\
Allows the form export filename to be changed.](https://docs.gravityforms.com/gform_form_export_filename/ "gform_form_export_filename") [**gform\_import\_form\_xml\_options** \\
Use the gform\_import\_form\_xml\_options filter to modify the settings used to import a form from an XML export file.](https://docs.gravityforms.com/gform_import_form_xml_options/ "gform_import_form_xml_options") [**gform\_include\_bom\_export\_entries** \\
Use the gform\_include\_bom\_export\_entries filter to exclude the BOM character from the beginning of entry export files.](https://docs.gravityforms.com/gform_include_bom_export_entries/ "gform_include_bom_export_entries") [**gform\_leads\_before\_export** \\
The gform\_leads\_before\_export filter allows entries to be changed before the entry export is executed.](https://docs.gravityforms.com/gform_leads_before_export/ "gform_leads_before_export") [**gform\_search\_criteria\_export\_entries** \\
Use this filter to allow entries of different statuses (i.e. trash, abuse) to be exported.](https://docs.gravityforms.com/gform_search_criteria_export_entries/ "gform_search_criteria_export_entries") [Previous](https://docs.gravityforms.com/category/developers/hooks/filters/importexport-filters/) [1](https://docs.gravityforms.com/category/developers/hooks/filters/importexport-filters/) 2
