## Description

Use this action hook to perform actions right before a form is deleted.

## Usage

|     |     |
| --- | --- |
| 1 | `add_action(``'gform_before_delete_form'``,``'do_cleanup'``);` |

## Parameters

| Parameter | Type | Description |
| --- | --- | --- |
| **$form\_id** | [integer](https://docs.gravityforms.com/integer) | ID of current form. |

## Examples

This example adds a log file entry when a form is deleted.

Form ID: {$form_id}. Entries: {$entry_count}\n"``);`<br>```fclose(``$f``);`<br>`}` |

## Placement

This code can be used in the [functions.php](https://developer.wordpress.org/themes/basics/theme-functions/) file of the active theme, a custom functions plugin, a custom add-on, or with a code snippets plugin.

See also the PHP section in this article: [Where Do I Put This Code?](https://docs.gravityforms.com/where-do-i-put-this-code/)
