## Description

This filter allows you to override the default choice markup used when rendering radio button, checkbox and drop down type fields.

## Usage

To apply the filter to all radio, checkbox and drop down type fields on all forms you can do this:

|     |
| --- |
| `add_filter(``'gform_field_choice_markup_pre_render'``,``'your_function_name'``, 10, 4 );` |

To target a specific form append the form id to the hook name. (format: gform\_field\_choice\_markup\_pre\_render\_FORMID)

|     |
| --- |
| `add_filter(``'gform_field_choice_markup_pre_render_10'``,``'your_function_name'``, 10, 4 );` |

To target a specific field on a specific form append the form id and the field id to the hook name. (format: gform\_field\_choice\_markup\_pre\_render\_FORMID\_FIELDID)

|     |
| --- |
| `add_filter(``'gform_field_choice_markup_pre_render_10_5'``,``'your_function_name'``, 10, 4 );` |

## Parameters

- **$choice\_markup** [string](https://docs.gravityforms.com/string)

The string containing the choice markup to be filtered.

**Checkbox**

|     |
| --- |
| `<``li``class``=``'gchoice_10_5_1'``>`<br>```<``input``name``=``'input_5.1'``type``=``'checkbox'``value``=``'Item 1|10'``id``=``'choice_10_5_1'``tabindex``=``'1'``/>`<br>```<``label``for``=``'choice_10_5_1'``id``=``'label_10_5_1'``>First Choice</``label``>`<br>`</``li``>` |

**Radio**

|     |
| --- |
| `<``li``class``=``'gchoice_10_5_0'``>`<br>```<``input``name``=``'input_5'``type``=``'radio'``value``=``'Item 1|10'``id``=``'choice_10_5_0'``tabindex``=``'1'``/>`<br>```<``label``for``=``'choice_10_5_0'``id``=``'label_10_5_0'``>First Choice</``label``>`<br>`</``li``>` |

**Select**

|     |
| --- |
| `<``option``value``=``'Item 1'``selected``=``'selected'``>First Choice</``option``>` |

- **$choice** [array](https://docs.gravityforms.com/array)

An associative array containing the choice properties.

|     |
| --- |
| `array``(`<br>```'text'``=>``'First Choice'``,`<br>```'value'``=>``'Item 1'``,`<br>```'isSelected'``=> true,`<br>```'price'``=>``'10'``,`<br>`)` |

- **$field** [Field Object](https://docs.gravityforms.com/field-object)

The field currently being processed.
- **$value** [string](https://docs.gravityforms.com/string)

The value to be selected if the field is being populated.

## Examples

### 1\. Add Custom Attribute

This example demonstrates how to add a custom attribute to the choice markup.

|     |
| --- |
| `add_filter(``'gform_field_choice_markup_pre_render_10'``,``function``(``$choice_markup``,``$choice``,``$field``,``$value``) {`<br>```if``(``$field``->get_input_type() ==``'radio'``&& rgar(``$choice``,``'text'``) ==``'Second Choice'``) {`<br>```return``str_replace``(``"type='radio'"``,``"type='radio' disabled"``,``$choice_markup``);`<br>```}`<br>``<br>```return``$choice_markup``;`<br>`}, 10, 4 );` |

### 2\. Include Price for Product Fields

This example shows how you can update the option markup to include the choice price for the drop down type Product field.

|     |
| --- |
| `add_filter(``'gform_field_choice_markup_pre_render'``,``function``(``$choice_markup``,``$choice``,``$field``,``$value``) {`<br>```if``(``$field``->type ==``'product'``) {`<br>```$new_string``= sprintf(``'>%s - %s<'``,``$choice``[``'text'``], GFCommon::to_money(``$choice``[``'price'``] ) );`<br>```return``str_replace``(``">{$choice['text']}<"``,``$new_string``,``$choice_markup``);`<br>```}`<br>``<br>```return``$choice_markup``;`<br>`}, 10, 4 );` |

### 3\. Remove choice based on date

This example shows how you can remove a choice from a specific field if the specified date has passed.

|     |
| --- |
| `add_filter(``'gform_field_choice_markup_pre_render_10_5'``,``function``(``$choice_markup``,``$choice``) {`<br>```if``( rgar(``$choice``,``'value'``)  ==``'test'``&& time() >=``strtotime``(``'2019-10-03'``) ) {`<br>```return``''``;`<br>```}`<br>``<br>```return``$choice_markup``;`<br>`}, 10, 2 );` |

### 4\. Add CSS class to Multiselect choice

This example shows how you can add a CSS class to a Multiselect field choice labeled as “Second Choice”.

|     |
| --- |
| `// Change 543 and 2 to your form id and field id respectively.`<br>`add_filter(``'gform_field_choice_markup_pre_render_543_2'``,``function``(``$choice_markup``,``$choice``,``$field``,``$value``) {`<br>```// Change Second Choice to the label of your choice.`<br>```if``(``$field``->get_input_type() ==``'multiselect'``&& rgar(``$choice``,``'text'``) ==``'Second Choice'``) {`<br>```// Change my_class to the CSS class name you want to use.`<br>```return``str_replace``(``"value="``,``"class='my_class' value="``,``$choice_markup``);`<br>```}`<br>``<br>```return``$choice_markup``;`<br>`}, 10, 4 );` |

### 5\. Add inline list headings

This example shows how you can replace a choice with an inline heading, by prepending your option label with ###.

## Placement

This code should be placed in the [functions.php](http://codex.wordpress.org/Theme_Development#Functions_File) file of your active theme.

## Source Code

This filter is located in the following methods:

- GFCommon::get\_select\_choices() in _common.php_ – since 1.9.11.10
- GF\_Field\_Checkbox::get\_checkbox\_choices() in _includes/fields/class-gf-field-checkbox.php_ – since 1.9.6
- GF\_Field\_Radio::get\_radio\_choices() in _includes/fields/class-gf-field-radio.php_ – since 1.9.6
