## Description

The “gform\_signature\_show\_in\_all\_fields” filter in the Gravity Forms Signature Add-On allows the inclusion of the signature image in the [{all\_fields}](https://docs.gravityforms.com/gform_signature_show_in_all_fields/fields-merge-tag) merge tag output to be toggled on/off. The default is **true**.

## Usage

|     |     |
| --- | --- |
| 1 | `add_filter(``'gform_signature_show_in_all_fields'``,``'your_function_name'``, 10, 4 );` |

## Parameters

- **$include\_signature** [bool](https://docs.gravityforms.com/bool)

The current value that indicates whether the signature image should be included. The default is true.
- **$field** [Field Object](https://docs.gravityforms.com/field-object)

The Field Object.
- **$options** [string](https://docs.gravityforms.com/string)

The merge tag modifiers. e.g. “value,nohidden” would be the modifiers for `{all_fields:value,nohidden}`.
- **$value**

The value of the field currently being processed.

## Examples

The example below turns off the inclusion of the signature image in the {all\_fields} merge tag.

|     |     |
| --- | --- |
| 1 | `add_filter(``'gform_signature_show_in_all_fields'``,``'__return_false'``);` |

## Placement

This code can be used in the [functions.php](https://developer.wordpress.org/themes/basics/theme-functions/) file of the active theme, a custom functions plugin, a custom add-on, or with a code snippets plugin.

See also the PHP section in this article: [Where Do I Put This Code?](https://docs.gravityforms.com/where-do-i-put-this-code/)

## Since

This filter was added in version 2.3.

## Source Code

This filter is located in GFSignature::merge\_tag\_filter() in _gravityformssignature/class-gf-signature.php_.
