## Articles

[**gform\_default\_choice\_alignment** \\
Allows the default choice alignment to be set for fields that support choice alignment.](https://docs.gravityforms.com/gform_default_choice_alignment/ "gform_default_choice_alignment") [**gform\_disable\_address\_map\_link** \\
This filter is used to disable the inclusion of the “Map It” link when displaying the Address field value.](https://docs.gravityforms.com/gform_disable_address_map_link/ "gform_disable_address_map_link") [**gform\_dropdown\_no\_results\_text** \\
Use this filter to change the default “No Results” placeholder text on Drop Down fields.](https://docs.gravityforms.com/gform_dropdown_no_results_text/ "gform_dropdown_no_results_text") [**gform\_email** \\
This filter is executed when creating the email field and can be used to modify the “Email” label.](https://docs.gravityforms.com/gform_email/ "gform_email") [**gform\_email\_confirm** \\
This filter is executed when creating the email field and can be used to modify the “Confirm Email” label.](https://docs.gravityforms.com/gform_email_confirm/ "gform_email_confirm") [**gform\_email\_fields\_notification\_admin** \\
Use the filter to add/remove fields from the list of email fields that get displayed on the Notification edit page when configuring the “Send To Field”.](https://docs.gravityforms.com/gform_email_fields_notification_admin/ "gform_email_fields_notification_admin") [**gform\_enable\_credit\_card\_field** \\
Use this filter to enable/disable the built-in credit card field.](https://docs.gravityforms.com/gform_enable_credit_card_field/ "gform_enable_credit_card_field") [**gform\_enable\_password\_field** \\
Use this filter to enable the password field. Useful when developing add-ons that require a password field.](https://docs.gravityforms.com/gform_enable_password_field/ "gform_enable_password_field") [**gform\_encrypt\_password** \\
