## Description

The `gform_googleanalytics_submission_event_name` filter can be used to filter the name of the submission event to be sent to Google Analytics. This filter only applies to the Google Analytics or Measurement Protocol (not Tag Manager) connection types.

**Note:** For more information about restrictions on resetting the event name, refer to [\[GA4\] Event naming rules – Analytics Help](https://support.google.com/analytics/answer/13316687?hl=en).

## Usage

Applies to all forms:

|     |
| --- |
| `add_action(``'gform_googleanalytics_submission_event_name'``,``'your_function_name'``, 10, 5 );` |

## Parameters

| Parameter | Type | Description |
| --- | --- | --- |
| **$event\_name** | [string](https://docs.gravityforms.com/string) | The event name sent to Google Analytics. Default is `gforms_submission`. |
| **$mode** | [string](https://docs.gravityforms.com/string) | Google Analytics connection mode. Options: `ga` for Analytics, `gmp` for Measurement Protocol. |
| **$feed** | [Feed Object](https://docs.gravityforms.com/object) | The feed currently being processed. |
| **$entry** | [Entry Object](https://docs.gravityforms.com/object) | The entry currently being processed. |
| **$form** | [Form Object](https://docs.gravityforms.com/object) | The form currently being processed. |

## Examples

### 1\. Append form ID to event name

|     |     |
| --- | --- |
| 1<br>2<br>3<br>4<br>5 | `add_filter(``'gform_googleanalytics_submission_event_name'``,``function``(``$event_name``,``$mode``,``$feed``,``$entry``,``$form``) {`<br>```return``"{$event_name}_{$form['id']}"``;`<br>`}, 10, 5 );` |

## Placement

This code can be used in the [functions.php](https://developer.wordpress.org/themes/basics/theme-functions/) file of the active theme, a custom functions plugin, a custom add-on, or with a code snippets plugin.

See also the PHP section in this article: [Where Do I Put This Code?](https://docs.gravityforms.com/where-do-i-put-this-code/)

## Since

This filter hook was added in Google Analytics v2.0.0
