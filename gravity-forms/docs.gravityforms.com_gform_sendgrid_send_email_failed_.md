# gform\_sendgrid\_send\_email\_failed

- [Description](https://docs.gravityforms.com/gform_sendgrid_send_email_failed/#h-description)
- [Usage](https://docs.gravityforms.com/gform_sendgrid_send_email_failed/#h-usage)
- [Parameters](https://docs.gravityforms.com/gform_sendgrid_send_email_failed/#h-parameters)
- [Since](https://docs.gravityforms.com/gform_sendgrid_send_email_failed/#h-since)
- [Source Code](https://docs.gravityforms.com/gform_sendgrid_send_email_failed/#h-source-code)

## Description

Triggered when an email from Gravity Forms fails to be passed to SendGrid.

## Usage

|     |
| --- |
| `add_action(``'gform_sendgrid_send_email_failed'``,``'my_function'``, 10, 6 );` |

## Parameters

- **$error\_message** [string](https://docs.gravityforms.com/string)




The error message.

- **$sendgrid\_email** [array](https://docs.gravityforms.com/array)




The SendGrid email arguments.

- **$email** [array](https://docs.gravityforms.com/array)




The original email details.

- **$message\_format** [array](https://docs.gravityforms.com/array)




The message format, html or text.

- **$notification** [array](https://docs.gravityforms.com/array)




The Notification object.

- **$entry** [array](https://docs.gravityforms.com/array)




The current Entry object.


## Since

This action hook was added in SendGrid version 1.4.

## Source Code

This action hook is located in GF\_SendGrid::maybe\_send\_email() in _gravityformssendgrid/class-gf-sendgrid.php_.
