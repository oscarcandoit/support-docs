## Introduction

The following functions are located in the [GFAddon](https://docs.gravityforms.com/gfaddon) class; they are also available for use by add-ons which extend the [GFFeedAddOn](https://docs.gravityforms.com/gffeedaddon) or [GFPaymentAddOn](https://docs.gravityforms.com/gfpaymentaddon) classes and can be used to retrieve the mapped form field values during feed processing in the [process\_feed()](https://docs.gravityforms.com/gffeedaddon/#processing-feeds) method.

## get\_mapped\_field\_value()

|     |     |
| --- | --- |
| 1 | `protected``function``get_mapped_field_value(``$setting_name``,``$form``,``$entry``,``$settings``= false ) {}` |

### Parameters

- **$setting\_name** [string](https://docs.gravityforms.com/string)
The setting name; the key to the setting value in the [Feed Object](https://docs.gravityforms.com/feed-object) meta array.

- **$form** [Form Object](https://docs.gravityforms.com/form-object)

The form currently being processed.

- **$entry** [Entry Object](https://docs.gravityforms.com/entry-object)

The entry currently being processed.

- **$settings** [array](https://docs.gravityforms.com/array)

The [Feed Object](https://docs.gravityforms.com/feed-object) meta.


### Returns

( [string](https://docs.gravityforms.com/string)) The value of the mapped field.

### Uses

- includes/addon/class-gf-addon.php: [get\_field\_value()](https://docs.gravityforms.com/mapped-field-values-during-feed-processing/#get_field_value)

### Usage Example

|     |     |
| --- | --- |
| 1 | `$value``=``$this``-&gt;get_mapped_field_value(``$meta_key``,``$form``,``$entry``,``$feed``[``'meta'``] );` |

## get\_{$input\_type}\_field\_value()

To perform custom formatting of values for a specific field or input type you can define a custom function to be used when the [get\_field\_value()](https://docs.gravityforms.com/mapped-field-values-during-feed-processing/#get_field_value) processes that field type. Just replace {$input\_type} in the function name with the field type you want to format e.g.

|     |     |
| --- | --- |
| 1<br>2<br>3<br>4<br>5<br>6<br>7<br>8 | `public``function``get_phone_field_value(``$entry``,``$field_id``,``$field``) {`<br>`$field_value``= rgar(``$entry``,``$field_id``);`<br>`if``( !``empty``(``$field_value``) &amp;&amp;``$field``-&gt;phoneFormat ==``'standard'``&amp;&amp; preg_match(``'/^D?(d{3})D?D?(d{3})D?(d{4})$/'``,``$field_value``,``$matches``) ) {`<br>`$field_value``= sprintf(``'%s-%s-%s'``,``$matches``[1],``$matches``[2],``$matches``[3] );`<br>`}`<br>`return``$field_value``;`<br>`}` |
