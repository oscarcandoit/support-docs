# gform\_ajax\_spinner\_url

- [Description](https://docs.gravityforms.com/gform_ajax_spinner_url/#h-description)
- [Usage](https://docs.gravityforms.com/gform_ajax_spinner_url/#h-usage)
- [Examples](https://docs.gravityforms.com/gform_ajax_spinner_url/#h-examples)
- [Change the spinner image to a custom image file.](https://docs.gravityforms.com/gform_ajax_spinner_url/#h-change-the-spinner-image-to-a-custom-image-file)
- [Placement](https://docs.gravityforms.com/gform_ajax_spinner_url/#h-placement)
- [Source Code](https://docs.gravityforms.com/gform_ajax_spinner_url/#h-source-code)

## Description

This filter can be used to change the default AJAX spinner image

## Usage

|     |     |
| --- | --- |
| 1 | `add_filter(``'gform_ajax_spinner_url'``,``'custom_spinner_image'``, 10, 2 );` |

You can also specify this per form by adding the form id after the hook name.

|     |     |
| --- | --- |
| 1 | `add_filter(``'gform_ajax_spinner_url_6'``,``'custom_spinner_image'``, 10, 2 );` |

## Parameters

- **$image\_src** [string](https://docs.gravityforms.com/string)
The spinner image URL to be filtered

- **$form** [Form Object](https://docs.gravityforms.com/form-object)

Current form.


## Examples

### Change the spinner image to a custom image file.

|     |     |
| --- | --- |
| 1<br>2<br>3<br>4 | `add_filter(``'gform_ajax_spinner_url'``,``'spinner_url'``, 10, 2 );`<br>`function``spinner_url(``$image_src``,``$form``) {`<br>`return``'http://www.somedomain.com/spinner.png'``;`<br>`}` |

## Placement

This code should be placed in the [functions.php](http://codex.wordpress.org/Theme_Development#Functions_File) file of your active theme.

## Source Code

This filter is located in GFFormDisplay::get\_form() in _form\_display.php_
