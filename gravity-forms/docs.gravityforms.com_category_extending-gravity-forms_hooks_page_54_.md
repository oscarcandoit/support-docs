## Articles

[**gform\_notification\_events** \\
Use this filter to add a new event to the drop down of available triggers for notifications. Additional code is necessary to handle the event since this hook only adds it to the Event drop down on the notifications admin page.](https://docs.gravityforms.com/gform_notification_events/ "gform_notification_events") [**gform\_notification\_format** \\
This filter has been deprecated since Gravity Forms version 1.7. Use the filter gform\_notication instead.](https://docs.gravityforms.com/gform_notification_format/ "gform_notification_format") [**gform\_notification\_note** \\
Use this filter to modify the note(s) being added to the entry detail page that contain the sending result of notifications for the entry.](https://docs.gravityforms.com/gform_notification_note/ "gform_notification_note") [**gform\_notification\_services** \\
Description.](https://docs.gravityforms.com/gform_notification_services/ "gform_notification_services") [**gform\_notification\_settings\_fields** \\
The gform\_notification\_settings\_fields filter is used to customize the available settings on the Notification configuration page.](https://docs.gravityforms.com/gform_notification_settings_fields/ "gform_notification_settings_fields") [**gform\_notification\_ui\_settings** \\
This filter has been deprecated. Use the gform\_notification\_settings\_fields filter instead.](https://docs.gravityforms.com/gform_notification_ui_settings/ "gform_notification_ui_settings") [**gform\_notification\_validation** \\
This filter was removed in Gravity Forms v2.5 and is no longer supported. From Gravity Forms 2.5, notification settings and their validation callbacks are now defined via the gform\_notification\_settings\_fields filter.](https://docs.gravityforms.com/gform_notification_validation/ "gform_notification_validation") [**gform\_order\_label** \\
The "gform\_order\_label" filter in Gravity Forms is executed when displaying the "Order" heading on the entry detail page or when sending the order information via email.](https://docs.gravityforms.com/gform_order_label/ "gform_order_label") [**gform\_order\_summary** \\
