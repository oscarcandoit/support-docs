## Articles

[**gform\_delete\_entry\_link** \\
The gform\_delete\_entry\_link filter can be used to modify the HTML for the “Delete Permanently” link that appears in the entry list table when viewing entries in the trash or spam filters.](https://docs.gravityforms.com/gform_delete_entry_link/ "gform_delete_entry_link") [**gform\_delete\_lead** \\
This hook has been deprecated and gform\_delete\_entry should be used instead.](https://docs.gravityforms.com/gform_delete_lead/ "gform_delete_lead") [**gform\_disable\_address\_map\_link** \\
This filter is used to disable the inclusion of the “Map It” link when displaying the Address field value.](https://docs.gravityforms.com/gform_disable_address_map_link/ "gform_disable_address_map_link") [**gform\_disable\_admin\_notification** \\
This hook has been deprecated. Please use gform\_disable\_notification instead.](https://docs.gravityforms.com/gform_disable_admin_notification/ "gform_disable_admin_notification") [**gform\_disable\_ajax\_save** \\
The filter gform\_disable\_ajax\_save can be used to turn off AJAX saving in the form editor.](https://docs.gravityforms.com/gform_disable_ajax_save/ "gform_disable_ajax_save") [**gform\_disable\_auto\_update** \\
The "gform\_disable\_auto\_update" filter allows automatic updates to be turned off in Gravity Forms.](https://docs.gravityforms.com/gform_disable_auto_update/ "gform_disable_auto_update") [**gform\_disable\_css** \\
The filter gform\_disable\_css can be used to disable all CSS files from being loaded on the Front End.](https://docs.gravityforms.com/gform_disable_css/ "gform_disable_css") [**gform\_disable\_custom\_field\_names\_query** \\
Allows the postmeta query which retrieves the custom field names (meta keys) to be disabled.](https://docs.gravityforms.com/gform_disable_custom_field_names_query/ "gform_disable_custom_field_names_query") [**gform\_disable\_dom\_parser** \\
