# Mailchimp Filters

## Articles

[**gform\_mailchimp\_allow\_resubscription** \\
Use this filter to modify whether a user that currently has a status of unsubscribed on your list is resubscribed.](https://docs.gravityforms.com/gform_mailchimp_allow_resubscription/ "gform_mailchimp_allow_resubscription") [**gform\_mailchimp\_field\_value** \\
Use this filter to modify a value before it is sent to the Mailchimp API.](https://docs.gravityforms.com/gform_mailchimp_field_value/ "gform_mailchimp_field_value") [**gform\_mailchimp\_keep\_existing\_groups** \\
Use this filter to modify whether a user that is already subscribed to your list has their groups replaced when submitting the form a second time.](https://docs.gravityforms.com/gform_mailchimp_keep_existing_groups/ "gform_mailchimp_keep_existing_groups") [**gform\_mailchimp\_lists\_params** \\
This filter allows the request parameters to be modified when getting MailChimp lists.](https://docs.gravityforms.com/gform_mailchimp_lists_params/ "gform_mailchimp_lists_params") [**gform\_mailchimp\_override\_empty\_fields** \\
This filter can be used to prevent empty form fields erasing values already stored in the mapped Mailchimp MMERGE fields when updating an existing subscriber.](https://docs.gravityforms.com/gform_mailchimp_override_empty_fields/ "gform_mailchimp_override_empty_fields") [**gform\_mailchimp\_request\_args** \\
This filter allows the request arguments to be modified before sent to Mailchimp.](https://docs.gravityforms.com/gform_mailchimp_request_args/ "gform_mailchimp_request_args") [**gform\_mailchimp\_subscription** \\
This filter modifies the subscription object before it is executed.](https://docs.gravityforms.com/gform_mailchimp_subscription/ "gform_mailchimp_subscription") [**http\_request\_timeout** \\
https://developer.wordpress.org/reference/hooks/http\_request\_timeout/.](https://docs.gravityforms.com/http_request_timeout/ "http_request_timeout") [**Mailchimp Feed Meta** \\
The Feed Meta object for the Mailchimp Add-On is an associative array containing the properties which determine how the add-on should process the form submission.](https://docs.gravityforms.com/mailchimp-feed-meta/ "Mailchimp Feed Meta")
