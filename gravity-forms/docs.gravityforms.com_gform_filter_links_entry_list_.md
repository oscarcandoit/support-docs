## Description

The “gform\_filter\_links\_entry\_list” filter in Gravity Forms allows the list of filters at the top of the Entry List page to be modified.

## Usage

|     |     |
| --- | --- |
| 1 | `add_filter(``'gform_filter_links_entry_list'``,``'your_function_name'``, 10, 3 );` |

Parameters

| **Parameter** | **Type** | **Description** |
| --- | --- | --- |
| **$filter\_links** | array | The filter links. |
| **$form** | array | The form the filter links are being prepared for. |
| **$include\_counts** | bool | Indicates if the database query to get the counts was performed. |
| **$counts** | array | The number of entries that match the filters when $include\_counts is true. |

## Examples

### Basic usage.

This example will simply add the ‘My Custom Link’ to the Entries list page. It will not filter anything because it has a blank array for ‘field\_filters’. So you would need to provide your own filter to make it work.

|     |     |
| --- | --- |
| 1<br>2<br>3<br>4<br>5<br>6<br>7<br>8<br>9 | `add_filter(``'gform_filter_links_entry_list'``,``'change_links'``, 10, 4 );`<br>`function``change_links(``$filter_links``,``$form``,``$include_counts``,``$counts``){`<br>```$filter_links``[] =``array``(`<br>```'id'``=>``'mycustom'``,`<br>```'field_filters'``=>``array``(),``// You need to provide a filter array here to make this link do something.`<br>```'label'``=>``'My Custom Link'``,`<br>```);`<br>```return``$filter_links``;`<br>`}` |

### Filter entries by field value.

The following example will add a new ‘Value test for Field id 1’ link that will reduce the entries list to only entries where field ID 1 has **test** as value.

| --- | --- |
| 1<br>2<br>3<br>4<br>5<br>6<br>7<br>8<br>9<br>10<br>11<br>12<br>13<br>14<br>15<br>16<br>17 | `// Filter Entries List to show only entries with value 'test' for field id 1.`<br>`add_filter(``'gform_filter_links_entry_list'``,``'filter_list_by_field_value'``, 10, 4 );`<br>`function``filter_list_by_field_value(``$filter_links``,``$form``,``$include_counts``,``$counts``){`<br>```$filter_links``[] =``array``(`<br>```'id'``=>``'filterbyfield'``,`<br>```'field_filters'``=>`<br>```array``(`<br>```0 =>`<br>```array``(`<br>```'key'``=> 1,``// ID of the field to check.`<br>```'value'``=>``'test'``,``// Only entries with this value for the above field id.`<br>```),`<br>```),`<br>```'label'``=>``'Value test for Field id 1'``,`<br>```);`<br>```return``$filter_links``;`<br>`}` |

### Add a filter for “Not Starred” entries.

The following example will add a new ‘Not Starred’ link that will filter the entries list to show only entries where the is\_starred property is set to false (entries with the gray star icon).

## Placement

This code can be used in the [functions.php](https://developer.wordpress.org/themes/basics/theme-functions/) file of the active theme, a custom functions plugin, a custom add-on, or with a code snippets plugin.

See also the PHP section in this article: [Where Do I Put This Code?](https://docs.gravityforms.com/where-do-i-put-this-code/)

## Since

This filter was added in Gravity Forms version 1.9.15 and in 2.9.16 the $counts parameter was added.
