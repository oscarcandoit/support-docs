## Pre-Requisites

This field is only available in the Form Editor when the [Coupons Add-On](https://www.gravityforms.com/add-ons/coupons/) is active.

## Summary

The **Coupon** field allows using coupons created with the [Coupons Add-On](https://www.gravityforms.com/add-ons/coupons/). Coupons can be used to offer discounts on products and services.

**Note:** Only one Coupon field can be added per form.

## Common Settings

This field uses only common field settings for the Advanced settings. For a description of each of the common field settings, refer to [this article](https://docs.gravityforms.com/common-field-settings/).
