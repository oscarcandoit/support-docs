# Theme Framework

[**Theme Layers APIs** \\
Gravity Forms provides two APIs for implementing Theme Layers. These APIs allow developers to define form styling without creating PHP classes, offering a flexible way to customize form appearance within WordPress.](https://docs.gravityforms.com/category/developers/theme-framework/theme-layers-apis/)

A Fluent API simplifies complex operations by chaining method calls in a clear, step-by-step sequence. The JSON API allows developers to define Gravity Forms styles using theme.json.
