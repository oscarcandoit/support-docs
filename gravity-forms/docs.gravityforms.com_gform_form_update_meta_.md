## Description

Allows modifying the form meta (fields, form settings, notifications, and confirmations) before it is saved to the database.

## Usage

The following would apply to all forms:

|     |
| --- |
| `add_filter(``'gform_form_update_meta'``,``'my_import_form'``, 10, 3 );` |

To target a specific form, append the form ID to the hook name. (format: gform\_form\_update\_meta\_FORMID):

|     |
| --- |
| `add_filter(``'gform_form_update_meta_1'``,``'my_import_form'``, 10, 3 );` |

## Parameters

- **$form\_meta** [string](https://docs.gravityforms.com/string)
The meta being saved for the current form. The form fields and settings, or notifications, or confirmations.

- **$form\_id** [int](https://docs.gravityforms.com/int)
The current form ID.

- **$meta\_name** [string](https://docs.gravityforms.com/string)
The type of meta being saved. Possible values: `display_meta` (the fields and form settings), `notifications`, or `confirmations`.
