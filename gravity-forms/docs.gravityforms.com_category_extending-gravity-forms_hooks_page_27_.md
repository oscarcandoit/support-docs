The gform\_entries\_filter\_count\_queries filter allows the queries used to get the counts for the entries list filter links to be overridden.](https://docs.gravityforms.com/gform_entries_filter_count_queries/ "gform_entries_filter_count_queries") [**gform\_entries\_first\_column** \\
