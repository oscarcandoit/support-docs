# Merge Tags

## Articles

[**{quiz\_percent} Merge Tag** \\
Displays the percentage score the user achieved. This is affected by weighted scoring.](https://docs.gravityforms.com/quiz_percent-merge-tag/ "{quiz_percent} Merge Tag") [**{quiz\_score} Merge Tag** \\
Displays the total score for the quiz or for a specific quiz question. The score is affected by weighted scoring.](https://docs.gravityforms.com/quiz_score-merge-tag/ "{quiz_score} Merge Tag") [**{quiz} Merge Tag** \\
Displays the specific quiz question along with your answer and the correct answer.](https://docs.gravityforms.com/quiz-merge-tag/ "{quiz} Merge Tag") [**{referer} Merge Tag** \\
Displays the address of the page which referred the user agent to the current page.](https://docs.gravityforms.com/referer-merge-tag/ "{referer} Merge Tag") [**{save\_email\_input} Merge Tag** \\
Using the save\_email\_input merge tag will insert a form into the “Save and Continue” confirmation allowing the user to enter an email address the resume link should be sent to.](https://docs.gravityforms.com/save_email_input/ "{save_email_input} Merge Tag") [**{score} Merge Tag** \\
Displays the score for a specific survey question. Scoring must be enabled on the survey field for it to be tallied.](https://docs.gravityforms.com/score-merge-tag/ "{score} Merge Tag") [**{source\_page\_number} Merge Tag** \\
When using the Google Analytics Add-On, this merge tag returns the number of the page from which the user has navigated.](https://docs.gravityforms.com/source_page_number-merge-tag/ "{source_page_number} Merge Tag") [**{square\_receipt\_url} Merge Tag** \\
The {square\_receipt\_url} merge tag outputs the URL to the receipt hosted by Square for the transaction associated with the current entry.](https://docs.gravityforms.com/square_receipt_url-merge-tag/ "{square_receipt_url} Merge Tag") [**{survey\_total\_score} Merge Tag** \\
Displays the total score for the survey. Scoring must be enabled on the survey field for it to be tallied.](https://docs.gravityforms.com/survey_total_score-merge-tag/ "{survey_total_score} Merge Tag") [**{today} Merge Tags** \\
The {today} merge tags can be used in field default value settings, notifications, confirmations, and with some add-ons to output the date and/or time the merge tag is processed.](https://docs.gravityforms.com/today-merge-tags/ "{today} Merge Tags") [Previous](https://docs.gravityforms.com/category/user-guides/merge-tags-getting-started/page/2/) [1](https://docs.gravityforms.com/category/user-guides/merge-tags-getting-started/) [2](https://docs.gravityforms.com/category/user-guides/merge-tags-getting-started/page/2/) 3 [4](https://docs.gravityforms.com/category/user-guides/merge-tags-getting-started/page/4/) [Next](https://docs.gravityforms.com/category/user-guides/merge-tags-getting-started/page/4/)
