- [Source Code](https://docs.gravityforms.com/gform_email/#h-source-code)

## Description

This filter is executed when creating the email field and can be used to modify the “Email” label.

## Usage

Applies to all forms.

|     |     |
| --- | --- |
| 1 | `add_filter(``'gform_email'``,``'change_email'``, 10, 2 );` |

Applies to a specific form. In this case, form id 5.

|     |     |
| --- | --- |
| 1 | `add_filter(``'gform_email_5'``,``'change_email'``, 10, 2 );` |

## Parameters

- **$label** [string](https://docs.gravityforms.com/string)
The label to be filtered.

- **$form\_id** [integer](https://docs.gravityforms.com/integer)

The current form’s id.


## Examples

This example changes the default email label:

|     |     |
| --- | --- |
| 1<br>2<br>3<br>4 | `add_filter(``'gform_email'``,``'change_email'``, 10, 2 );`<br>`function``change_email(``$label``,``$form_id``) {`<br>`return``'Your Email'``;`<br>`}` |

## Placement

This code should be placed in the [functions.php](http://codex.wordpress.org/Theme_Development#Functions_File) file of your active theme.

## Source Code

This filter is located in GF\_Field\_Email::get\_field\_input() in _includes/fields/class-gf-field-email.php_.
