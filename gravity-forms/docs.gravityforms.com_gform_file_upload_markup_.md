## Description

The “gform\_file\_upload\_markup” filter can be used to modify the HTML for the multi-file upload “preview”.

## Usage

The gform\_file\_upload\_markup filter has both a [JavaScript version](https://docs.gravityforms.com/gform_file_upload_markup/#javascript-version) and a [PHP version](https://docs.gravityforms.com/gform_file_upload_markup/#php-version). **Both versions should be used.**

The JavaScript version modifies the HTML the first time the uploaded file preview displays.

|     |     |
| --- | --- |
| 1<br>2<br>3<br>4<br>5 | `gform.addFilter(``'gform_file_upload_markup'``,``function``( html, file, up, strings, imagesUrl, response ) {`<br>```// do stuff`<br>```return``html;`<br>`} );` |

The PHP version changes the uploaded file preview HTML on a multi-page form when you click Previous/Next after a file has been uploaded.

|     |     |
| --- | --- |
| 1<br>2<br>3<br>4<br>5 | `add_filter(``'gform_file_upload_markup'``,``function``(``$file_upload_markup``,``$file_info``,``$form_id``,``$field_id``) {`<br>```// do stuff`<br>```return``$result``;`<br>`}, 10, 4 );` |

- **html** [string](https://docs.gravityforms.com/string)

- **file** [Javascript Object](https://docs.gravityforms.com/javascript-object)

The URL to the images folder.

- **response** [Javascript Object](https://docs.gravityforms.com/javascript-object)

The response from GFAsyncUpload. Since version 2.4.22.3.
