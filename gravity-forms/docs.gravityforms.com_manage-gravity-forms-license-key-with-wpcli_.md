# Managing the Gravity Forms License Key with WP-CLI

- [Pre-Requisites](https://docs.gravityforms.com/manage-gravity-forms-license-key-with-wpcli/#h-pre-requisites)
- [Updating the License Key](https://docs.gravityforms.com/manage-gravity-forms-license-key-with-wpcli/#h-updating-the-license-key)
- [Examples](https://docs.gravityforms.com/manage-gravity-forms-license-key-with-wpcli/#update-examples)
- [Parameters](https://docs.gravityforms.com/manage-gravity-forms-license-key-with-wpcli/#install-parameters)
- [Deleting the License Key](https://docs.gravityforms.com/manage-gravity-forms-license-key-with-wpcli/#deleting-the-license-key)
- [Examples](https://docs.gravityforms.com/manage-gravity-forms-license-key-with-wpcli/#delete-examples)

Using the Gravity Forms CLI Add-On and the following commands you can update and delete the Gravity Forms license key from the command line.

## Pre-Requisites

- [Download](https://docs.gravityforms.com/downloading-a-gravity-forms-add-on) and [install](https://docs.gravityforms.com/installing-a-gravity-forms-add-on/) [Gravity Forms CLI](https://www.gravityforms.com/my-account/licenses/)

## Updating the License Key

`wp gf license update`

Updates the Gravity Forms license key.

### Examples

|     |
| --- |
| `# Updating the plugin to use the specified license key.`<br>`$ wp gf license update abc123`<br>`License key updated` |

### Parameters

| Argument | Description |
| --- | --- |
| `[<license-key>]` | A valid Gravity Forms license key. |

## Deleting the License Key

`wp gf license delete`

Deletes the Gravity Forms license key.

### Examples

|     |
| --- |
| `# Deleting license key from the plugin.`<br>`$ wp gf license delete`<br>`License key deleted` |
