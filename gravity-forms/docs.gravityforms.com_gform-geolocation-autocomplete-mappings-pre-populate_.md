## Description

Allows the mapped autocomplete values to be customized before the values are added to the inputs.

## Usage

|     |
| --- |
| `gform.addFilter(``'gform_geolocation_autocomplete_mappings_pre_populate'``,``function``( mappings, components, place, formId, fieldId ) {`<br>```return``mappings;`<br>`} );` |

## Parameters

- **mappings** [object](https://docs.gravityforms.com/object)

A collection of address field elements and values to be populated. **Only the values are writable.**

The keys to the object are: `line1`, `line2`, `city`, `state`, `zip`, `country`, `latitude`, and `longitude`. Each key is assigned an object that contains the following:

  - **element** [object](https://docs.gravityforms.com/object)

    A [JavaScript element](https://developer.mozilla.org/en-US/docs/Web/API/Element) containing the input or select to be populated.
  - **value** [string](https://docs.gravityforms.com/string/)

    The value to be populated into the input or select.
- **components** [object](https://docs.gravityforms.com/object)

The components for the selected place. Details about how the component objects are defined can be found [here](https://developers.google.com/maps/documentation/javascript/reference/geocoder#GeocoderAddressComponent). Each component can be accessed using the type as the key, e.g. `components.street_number.long_name`.
- **place** [object](https://docs.gravityforms.com/object)

Data about the place selected by the user. Details about what the place object can contain are [here](https://developers.google.com/maps/documentation/javascript/reference/places-service#PlaceResult).
- **formId** [number](https://docs.gravityforms.com/integer/)

The ID of the form the address field belongs to.
- **fieldId** [number](https://docs.gravityforms.com/integer)

The ID of the address field being populated.

## Examples

### Change line 1 value

The following shows how the entire address of the selected place can be populated into the line 1 input.

|     |     |
| --- | --- |
| 1<br>2<br>3<br>4<br>5<br>6<br>7<br>8 | `gform.addFilter(``'gform_geolocation_autocomplete_mappings_pre_populate'``,``function``( mappings, components, place, formId, fieldId ) {`<br>```const tempElement = document.createElement(``'div'``);`<br>```tempElement.innerHTML = place.adr_address;`<br>```mappings.line1.value = tempElement.innerText || tempElement.textContent;`<br>```tempElement.remove();`<br>```return``mappings;`<br>`} );` |

### Change the line 2 value

The following shows how you can add a custom value to the line 2 input

|     |     |
| --- | --- |
| 1<br>2<br>3<br>4<br>5 | `gform.addFilter(``'gform_geolocation_autocomplete_mappings_pre_populate'``,``function``( mappings, components, place, formId, fieldId ) {`<br>```mappings.line2.value =``'myVal'``;`<br>```return``mappings;`<br>`} );` |

### Change the state value

The following shows how you can populate the state input with the short name.

|     |     |
| --- | --- |
| 1<br>2<br>3<br>4<br>5 | `gform.addFilter(``'gform_geolocation_autocomplete_mappings_pre_populate'``,``function``( mappings, components, place, formId, fieldId ) {`<br>```mappings.state.value = components.administrative_area_level_1.short_name;`<br>``<br>```return``mappings;`<br>`} );` |
