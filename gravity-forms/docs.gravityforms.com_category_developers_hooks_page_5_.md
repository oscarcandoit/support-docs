## Articles

[**gform\_activecampaign\_tags** \\
This filter can be used to modify the tags which will be added to the contact.](https://docs.gravityforms.com/gform_activecampaign_tags/ "gform_activecampaign_tags") [**gform\_add\_field\_buttons** \\
This filter can be used to add/edit/remove the "add field" buttons from the form editor's floating toolbox.](https://docs.gravityforms.com/gform_add_field_buttons/ "gform_add_field_buttons") [**gform\_addnote\_button** \\
This filter allows modification of the HTML of the “Add Note” button for Entry Notes on the Entry Detail page.](https://docs.gravityforms.com/gform_addnote_button/ "gform_addnote_button") [**gform\_addon\_app\_PAGE\_TAB** \\
This action fires when a page and tab is accessed within Gravity Forms. It is typically used to add content to custom settings tabs.](https://docs.gravityforms.com/gform_addon_app_page_tab/ "gform_addon_app_PAGE_TAB") [**gform\_addon\_app\_settings\_menu\_{$SHORT\_SLUG}** \\
This filter controls the addition or removal of tabs within the Gravity Forms settings menu.](https://docs.gravityforms.com/gform_addon_app_settings_menu_slug/ "gform_addon_app_settings_menu_{$SHORT_SLUG}") [**gform\_addon\_feed\_settings\_fields** \\
This filter can be used to modify the feed settings fields before they are rendered on the Feed Settings edit view.](https://docs.gravityforms.com/gform_addon_feed_settings_fields/ "gform_addon_feed_settings_fields") [**gform\_addon\_field\_map\_choices** \\
This filter is deprecated since version 2.5. Use the gform\_field\_map\_choices filter instead.](https://docs.gravityforms.com/gform_addon_field_map_choices/ "gform_addon_field_map_choices") [**gform\_addon\_field\_value** \\
This filter can be used to modify a value before it is sent to a third-party by one of the Add-On Framework based add-ons.](https://docs.gravityforms.com/gform_addon_field_value/ "gform_addon_field_value") [**gform\_addon\_form\_settings\_fields** \\
Allows code to modify the settings fields displayed on a given form settings page.](https://docs.gravityforms.com/gform_addon_form_settings_fields/ "gform_addon_form_settings_fields") [**gform\_addon\_navigation** \\
