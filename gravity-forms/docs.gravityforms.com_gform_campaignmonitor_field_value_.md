## Description

This filter can be used to modify a value before it is sent to Campaign Monitor.

## Usage

The base filter which would run for all forms and all fields would be used like so:

|     |     |
| --- | --- |
| 1 | `add_filter(``'gform_campaignmonitor_field_value'``,``'your_function_name'``, 10, 4 );` |

To target a specific form append the form id to the hook name. (format: gform\_campaignmonitor\_field\_value\_FORMID)

|     |     |
| --- | --- |
| 1 | `add_filter(``'gform_campaignmonitor_field_value_10'``,``'your_function_name'``, 10, 4 );` |

To target a specific field append both the form id and the field id to the hook name. (format: gform\_campaignmonitor\_field\_value\_FORMID\_FIELDID)

|     |     |
| --- | --- |
| 1 | `add_filter(``'gform_campaignmonitor_field_value_10_3'``,``'your_function_name'``, 10, 4 );` |

## Parameters

| Parameter | Type | Description |
| --- | --- | --- |
| **$value** | [string](https://docs.gravityforms.com/string) | The value to be modified. |
| **$form\_id** | [integer](https://docs.gravityforms.com/integer) | The ID of the form being processed. |
| **$field\_id** | [string](https://docs.gravityforms.com/string) | The ID of the field being processed. |
| **$entry** | [Entry Object](https://docs.gravityforms.com/entry-object) | The entry currently being processed. |

## Examples

### 1\. Use GF\_Field::get\_value\_export

This example shows how you can have versions of the Campaign Monitor add-on older than 3.4 use GF\_Field::get\_value\_export() to format the field value for export. **Requires a minimum of Gravity Forms 1.9.12.4.**

|     |     |
| --- | --- |
| 1<br>2<br>3<br>4<br>5<br>6<br>7<br>8<br>9<br>10<br>11 | `add_filter(``'gform_campaignmonitor_field_value'``,``'format_entry_value'``, 10, 4 );`<br>`function``format_entry_value(``$value``,``$form_id``,``$field_id``,``$entry``) {`<br>```$form``= GFAPI::get_form(``$form_id``);`<br>```$field``= GFFormsModel::get_field(``$form``,``$field_id``);`<br>```if``(``is_object``(``$field``) ) {`<br>```$value``=``$field``->get_value_export(``$entry``,``$field_id``, true );`<br>```}`<br>```return``$value``;`<br>`}` |

### 2\. Change Signature Value

This example shows how you can send the url of the signature image.

|     |     |
| --- | --- |
| 1<br>2<br>3<br>4<br>5<br>6<br>7<br>8<br>9<br>10<br>11 | `add_filter(``'gform_campaignmonitor_field_value'``,``'change_signature_value'``, 10, 4 );`<br>`function``change_signature_value(``$value``,``$form_id``,``$field_id``,``$entry``) {`<br>```$form``= GFAPI::get_form(``$form_id``);`<br>```$field``= GFFormsModel::get_field(``$form``,``$field_id``);`<br>```if``(``is_object``(``$field``) &&``$field``->get_input_type() ==``'signature'``) {`<br>```$value``= RGFormsModel::get_upload_url_root() .``'signatures/'``.``$value``;`<br>```}`<br>```return``$value``;`<br>`}` |
