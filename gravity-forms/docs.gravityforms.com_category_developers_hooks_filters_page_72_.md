## Articles

[**gform\_webhooks\_request\_method** \\
This filter allows the webhook HTTP request method to be modified.](https://docs.gravityforms.com/gform_webhooks_request_method/ "gform_webhooks_request_method") [**gform\_webhooks\_request\_url** \\
This filter allows the webhook HTTP request url to be modified.](https://docs.gravityforms.com/gform_webhooks_request_url/ "gform_webhooks_request_url") [**gform\_zap\_before\_save** \\
This filter allows changes to be made to the Zapier feed setup before it is saved to the database.](https://docs.gravityforms.com/gform_zap_before_save/ "gform_zap_before_save") [**gform\_zapier\_actions** \\
This filter allows the modification of the feed actions allowed (delete and edit).](https://docs.gravityforms.com/gform_zapier_actions/ "gform_zapier_actions") [**gform\_zapier\_feed\_conditional\_logic** \\
This filter allows the feed conditional logic rule to be overridden during submission, allowing multiple rules to be defined.](https://docs.gravityforms.com/gform_zapier_feed_conditional_logic/ "gform_zapier_feed_conditional_logic") [**gform\_zapier\_field\_value** \\
This filter can be used to modify a value before it is sent to Zapier.](https://docs.gravityforms.com/gform_zapier_field_value/ "gform_zapier_field_value") [**gform\_zapier\_products** \\
This filter allows the products array to be modified before it is sent to Zapier.](https://docs.gravityforms.com/gform_zapier_products/ "gform_zapier_products") [**gform\_zapier\_request\_body** \\
