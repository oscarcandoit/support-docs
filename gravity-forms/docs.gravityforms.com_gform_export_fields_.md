## Description

Use this filter to add custom columns to the entry export. The new fields show in the field selection list. Use this filter in conjunction with the filter [gform\_export\_field\_value](https://docs.gravityforms.com/gform_export_field_value).

## Usage

|     |     |
| --- | --- |
| 1 | `add_filter(``'gform_export_fields'``,``'your_function_name'``);` |

## Parameters

- **$form** [Form Object](https://docs.gravityforms.com/form-object)

The current form.
