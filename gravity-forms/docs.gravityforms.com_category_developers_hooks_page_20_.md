[**gform\_dashboard\_title** \\
This filter allows the title of the forms statistics section displayed on the Dashboard to be changed.](https://docs.gravityforms.com/gform_dashboard_title/ "gform_dashboard_title") [**gform\_date\_max\_year** \\
Use this filter to specify the maximum year displayed in the date field’s year drop down and the HTML5 max attribute for the date field’s year input.](https://docs.gravityforms.com/gform_date_max_year/ "gform_date_max_year") [**gform\_date\_min\_year** \\
Use this filter to specify the minimum year displayed in the date field’s year drop down and the HTML5 min attribute for the date field’s year input.](https://docs.gravityforms.com/gform_date_min_year/ "gform_date_min_year") [**gform\_datepicker\_options\_pre\_init** \\
This filter can be used to modify the options used to initialize a fields datepicker.](https://docs.gravityforms.com/gform_datepicker_options_pre_init/ "gform_datepicker_options_pre_init") [**gform\_default\_address\_type** \\
This filter can be used to override the default address type for new fields.](https://docs.gravityforms.com/gform_default_address_type/ "gform_default_address_type") [**gform\_default\_choice\_alignment** \\
Allows the default choice alignment to be set for fields that support choice alignment.](https://docs.gravityforms.com/gform_default_choice_alignment/ "gform_default_choice_alignment") [**gform\_default\_notification** \\
Use this filter to disable the default admin notification automatically generated for new forms.](https://docs.gravityforms.com/gform_default_notification/ "gform_default_notification") [**gform\_default\_styles** \\
