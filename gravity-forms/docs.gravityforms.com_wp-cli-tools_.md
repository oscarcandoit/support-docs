This article includes a few extra commands which are available with the CLI Add-On for Gravity Forms.

## Pre-Requisites

- [Download](https://docs.gravityforms.com/downloading-a-gravity-forms-add-on) and [install](https://docs.gravityforms.com/installing-a-gravity-forms-add-on/) [Gravity Forms CLI](https://www.gravityforms.com/my-account/licenses/)

## Clearing Transients

`wp gf tool clear_transients`

`wp gf tool clear-transients`

Clears the Gravity Forms transients.

### Examples

|     |
| --- |
| `# Successful clearing of the transients.`<br>`$ wp gf tool``clear``-transients`<br>`Success: Gravity Forms transients cleared successfully.` |

|     |
| --- |
| `# Failed clearing of the transients.`<br>`$ wp gf tool clear_transients`<br>`Error: There was a problem clearing the Gravity Forms transients.` |

## Emptying Trash

`wp gf tool empty_trash`

`wp gf tool empty-trash`

Delete the trashed entries.

### Examples

|     |
| --- |
| `# Successful deletion of trashed entries for all forms.`<br>`$ wp gf tool empty-trash`<br>`Success: Trash emptied successfully``for``all forms.` |

|     |
| --- |
| `# Successful deletion of trashed entries for a specific form.`<br>`$ wp gf tool empty-trash 2`<br>`Success: Trash emptied successfully``for``form ID 2` |

### Parameters

| Argument | Description |
| --- | --- |
| `[<form-id>]` | The ID of the form to delete the entries from. Default: all forms. |

## Verifying Checksums

`wp gf tool verify_checksums`

`wp gf tool verify-checksums`

Verify the Gravity Forms plugin files against the checksums.

### Examples

|     |
| --- |
| `# Successful verification of the plugin checksums.`<br>`$ wp gf tool verify-checksums`<br>`Success: Gravity Forms``install``verifies against checksums.` |

|     |
| --- |
| `# Failed checksum verification.`<br>`$ wp gf tool verify-checksums`<br>`Warning: File doesn't exist: css``/admin``.min.css`<br>`Error: Gravity Forms``install``doesn't verify against checksums.` |

### Parameters

| Argument | Description |
| --- | --- |
| `[--version=<value>]` | The version number the checksums should be validated against. |

## Getting the System Report

`wp gf tool system_report`

`wp gf tool system-report`

`wp gf tool status`

Outputs the [system report](https://docs.gravityforms.com/checking-environment-details/) from the Forms > System Status page.

### Examples

|     |
| --- |
| `# Successfully using the "system-report" command to output the system report.`<br>`$ wp gf tool system-report`<br>`Success:`<br>`### Gravity Forms ###`<br>`...` |

|     |
| --- |
| `# Using the "status" alias with an older Gravity Forms version.`<br>`$ wp gf tool status`<br>`Error: The system report is only available with Gravity Forms 2.2 or greater.` |
