# gform\_entries\_filter\_count\_queries

- [Usage](https://docs.gravityforms.com/gform_entries_filter_count_queries/#h-usage)
- [Parameters](https://docs.gravityforms.com/gform_entries_filter_count_queries/#h-parameters)
- [Examples](https://docs.gravityforms.com/gform_entries_filter_count_queries/#h-examples)
- [Add Custom Entry Count for Toxic Status.](https://docs.gravityforms.com/gform_entries_filter_count_queries/#h-add-custom-entry-count-for-toxic-status)
- [Placement](https://docs.gravityforms.com/gform_entries_filter_count_queries/#h-placement)
- [Since](https://docs.gravityforms.com/gform_entries_filter_count_queries/#h-since)
- [Source Code](https://docs.gravityforms.com/gform_entries_filter_count_queries/#h-source-nbsp-code)

The `gform_entries_filter_count_queries` filter allows the queries used to get the counts for the entries list filter links to be overridden.

## Usage

|     |     |
| --- | --- |
| 1 | `add_filter(``'gform_entries_filter_count_queries'``,``'your_function_name'``, 10, 2 );` |

## Parameters

| Parameter | Type | Description |
| --- | --- | --- |
| **$parameter** | array | The filter count queries. |
| **$form\_id** | int | The ID of the form for which the queries are being prepared. |

The default value of `$queries`:

|     |     |
| --- | --- |
| 1<br>2<br>3<br>4<br>5<br>6<br>7 | `$queries``=``array``(`<br>```"COUNT(DISTINCT CASE WHEN l.status='active' THEN l.id END) as total"``,`<br>```"COUNT(DISTINCT CASE WHEN l.is_read=0 AND l.status='active' THEN l.id END) as unread"``,`<br>```"COUNT(DISTINCT CASE WHEN l.is_starred=1 AND l.status='active' THEN l.id END) as starred"``,`<br>```"COUNT(DISTINCT CASE WHEN l.status='spam' THEN l.id END) as spam"``,`<br>```"COUNT(DISTINCT CASE WHEN l.status='trash' THEN l.id END) as trash"``,`<br>`);` |

## Examples

### **Add Custom Entry Count for Toxic Status.**

|     |     |
| --- | --- |
| 1<br>2<br>3<br>4<br>5 | `add_filter(``'gform_entries_filter_count_queries'``,``function``(``$queries``) {`<br>```$queries``[] =``"COUNT(DISTINCT CASE WHEN l.status='toxic' THEN l.id END) as toxic"``;`<br>```return``$queries``;`<br>`} );` |

## Placement

This code can be used in the [functions.php](https://developer.wordpress.org/themes/basics/theme-functions/) file of the active theme, a custom functions plugin, a custom add-on, or with a code snippets plugin.

See also the PHP section in this article: [Where Do I Put This Code?](https://docs.gravityforms.com/where-do-i-put-this-code/)

## Since

This filter was added in Gravity Forms 2.9.16

## Source Code

This filter is located in  `GFFormsModel::get_form_counts()` in `forms_model.php`
