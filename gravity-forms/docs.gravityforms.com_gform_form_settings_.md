# gform\_form\_settings

- [Description](https://docs.gravityforms.com/gform_form_settings/#h-description)
- [Usage](https://docs.gravityforms.com/gform_form_settings/#h-usage)
- [Parameters](https://docs.gravityforms.com/gform_form_settings/#h-parameters)
- [Examples](https://docs.gravityforms.com/gform_form_settings/#h-examples)
- [Source Code](https://docs.gravityforms.com/gform_form_settings/#h-source-code)

This filter was deprecated in Gravity Forms v2.5 and will be removed in v3.0. Since Gravity Forms 2.5, form settings are defined via the [gform\_form\_settings\_fields](https://docs.gravityforms.com/gform_form_settings_fields/) filter.

## Description

Add new or modify existing form settings that display on the Form Settings screen.

## Usage

Apply to all forms:

|     |     |
| --- | --- |
| 1 | `add_filter(``'gform_form_settings'``,``'my_custom_function'``, 10, 2 );` |

## Parameters

- **$settings** [array](https://docs.gravityforms.com/array/)

An array of settings for the Form Settings UI. Each setting contains the HTML which will be displayed for that setting. The array keys are the group titles so could be translated.
