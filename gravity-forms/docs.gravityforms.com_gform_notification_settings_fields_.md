## Description

The gform\_notification\_settings\_fields filter is used to customize the available settings on the Notification configuration page.

## Usage

The filter which runs for all would be used like so:

|     |     |
| --- | --- |
| 1 | `add_filter(``'gform_notification_settings_fields'``,``'your_function_name'``, 10, 3 );` |

You can also target a specific form by adding the form id after the hook name.

|     |     |
| --- | --- |
| 1 | `add_filter(``'gform_notification_settings_fields_6'``,``'your_function_name'``, 10, 3 );` |

## Parameters

- **$fields** [array](https://docs.gravityforms.com/array/)

The Form Settings fields. See the [Settings API](https://docs.gravityforms.com/category/developers/php-api/add-on-framework/settings-api/) for details about how the fields are defined.

- **$notification** [Notification Object](https://docs.gravityforms.com/notifications-object/)

The meta for the form notification being viewed/edited.

- **$form** [Form Object](https://docs.gravityforms.com/form-object/)

The current form.


## Examples

### Add a new field

This example would add a hidden field to the notification configuration page.

|     |     |
| --- | --- |
| 1<br>2<br>3<br>4<br>5 | `add_filter(``'gform_notification_settings_fields'``,``function``(``$fields``,``$notification``,``$form``) {`<br>```$fields``[0][``'fields'``][] =``array``(``'type'``=>``'hidden'``,``'name'``=>``'my_custom_hidden_field'``);`<br>``<br>```return``$fields``;`<br>`}, 10, 3 );` |

### Replace From Email default value when adding a new notification

This example would apply only to new notifications added from the Notifications list page. The default Admin Notification settings can’t be filtered.

|     |     |
| --- | --- |
| 1<br>2<br>3<br>4<br>5<br>6<br>7<br>8<br>9<br>10<br>11 | `add_filter(``'gform_notification_settings_fields'``,``function``(``$fields``,``$notification``,``$form``) {`<br>```foreach``(``$fields``[0][``'fields'``]``as``&``$field``) {`<br>```if``( rgar(``$field``,``'name'``) !==``'from'``) {`<br>```continue``;`<br>```}`<br>```$field``[``'default_value'``] =``str_replace``(``'{admin_email}'``,``'user@example.com'``,``$field``[``'default_value'``] );`<br>```}`<br>```return``$fields``;`<br>`}, 10, 3 );` |

## Placement

This code should be placed in the [functions.php](http://codex.wordpress.org/Theme_Development#Functions_File) file of your active theme or a custom functions plugin.

## Since

This filter was added in Gravity Forms v2.5.

This filter is located in GFNotification::settings\_fields() in _notification.php_.
