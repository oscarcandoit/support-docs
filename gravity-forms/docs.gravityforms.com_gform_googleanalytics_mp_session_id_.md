## Description

The **gform\_googleanalytics\_mp\_session\_id** filter allows developers to override the session ID used by the Google Analytics 4 Measurement Protocol integration.

## Usage

|     |     |
| --- | --- |
| 1 | `add_filter( 'gform_googleanalytics_mp_session_id', 'your_function_name', 10, 3 );` |

## Parameters

| Parameter | Type | Description |
| --- | --- | --- |
| **$session\_id** | [string](https://docs.gravityforms.com/string/) \|null | Parsed session ID from the cookie, or `null` if not found. |
| **$cookie\_name** | [string](https://docs.gravityforms.com/string/) | Name of the Google Analytics cookie being used. |
| **`$`measurement\_id** | `string` | The GA4 Measurement ID associated with the current tracking configuration. |
