# gform\_secure\_file\_download\_url

- [Description](https://docs.gravityforms.com/gform_secure_file_download_url/#h-description)

- [Placement](https://docs.gravityforms.com/gform_secure_file_download_url/#h-placement)
- [Since](https://docs.gravityforms.com/gform_secure_file_download_url/#h-since)
- [Source Code](https://docs.gravityforms.com/gform_secure_file_download_url/#h-source-code)

## Description

Allows for manual filtering of the download URL to handle conditions such as unusual domain mapping and others.

## Usage

The base filter which would run for all forms would be used like so:

|     |     |
| --- | --- |
| 1 | `add_filter(``'gform_secure_file_download_url'``,``'your_function_name'``, 10, 2 );` |

## Parameters

- **$download\_url** [string](https://docs.gravityforms.com/string)
The URL from which to download the file.

- **$field** [GF\_Field\_Fileupload](https://docs.gravityforms.com/gf_field_fileupload)

The field object for further context.


## Placement

This code should be placed in the [functions.php](http://codex.wordpress.org/Theme_Development#Functions_File) file of your active theme.

## Since

This filter was added in Gravity Forms 2.1.1.1.

## Source Code

This filter is located in GF\_Field\_FileUpload::get\_download\_url() in _includes/fields/class-gf-field-fileupload.php_.

- URL


This field is for validation purposes and should be left unchanged.
