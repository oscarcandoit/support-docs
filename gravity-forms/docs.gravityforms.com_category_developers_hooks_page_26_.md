## Articles

[**gform\_enable\_credit\_card\_field** \\
Use this filter to enable/disable the built-in credit card field.](https://docs.gravityforms.com/gform_enable_credit_card_field/ "gform_enable_credit_card_field") [**gform\_enable\_download\_nocache\_headers** \\
The gform\_enable\_download\_nocache\_headers filter allows developers to enable or disable the no-cache headers for file downloads.](https://docs.gravityforms.com/gform_enable_download_nocache_headers/ "gform_enable_download_nocache_headers") [**gform\_enable\_entry\_info\_payment\_details** \\
The “gform\_enable\_entry\_info\_payment\_details” action enables the payment details panel for the current entry. This hook has been deprecated because the payment details are now visible on the entry if the payment status is set.](https://docs.gravityforms.com/gform_enable_entry_info_payment_details/ "gform_enable_entry_info_payment_details") [**gform\_enable\_field\_label\_visibility\_settings** \\
As of Gravity Forms 2.4, this functionality is part of the Form Editor and the hook has been deprecated.](https://docs.gravityforms.com/gform_enable_field_label_visibility_settings/ "gform_enable_field_label_visibility_settings") [**gform\_enable\_legacy\_markup** \\
The gform\_enable\_legacy\_markup filter can be used to enable the legacy markup for forms created with Gravity Forms 2.5 and greater.](https://docs.gravityforms.com/gform_enable_legacy_markup/ "gform_enable_legacy_markup") [**gform\_enable\_password\_field** \\
Use this filter to enable the password field. Useful when developing add-ons that require a password field.](https://docs.gravityforms.com/gform_enable_password_field/ "gform_enable_password_field") [**gform\_enable\_shortcode\_notification\_message** \\
Allows the disabling of the notification message defined in the shortcode.](https://docs.gravityforms.com/gform_enable_shortcode_notification_message/ "gform_enable_shortcode_notification_message") [**gform\_encrypt\_password** \\
