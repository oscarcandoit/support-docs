- [Description](https://docs.gravityforms.com/gform_site_created/#h-description)

- [Source Code](https://docs.gravityforms.com/gform_site_created/#h-source-code)

## Description

Fires after the site has been created. Only applicable to Network installs with the “Create Site” option enabled.

## Usage

Applies to all forms.

|     |     |
| --- | --- |
| 1 | `add_action(``'gform_site_created'``,``'your_function_name'``, 10, 5 );` |

## Parameters

- **$site\_id** [integer](https://docs.gravityforms.com/integer)
The ID of the created site.

- **$user\_id** [string](https://docs.gravityforms.com/string)

The ID of the registered user.

- **$entry** [Entry Object](https://docs.gravityforms.com/entry-object)

The entry object from which the site was created.

- **$feed** [Feed Object](https://docs.gravityforms.com/feed-object)

The Feed which is currently being processed.

- **$password** [string](https://docs.gravityforms.com/string)

The password associated with the user; either submitted by the user or sent via email from WordPress.


## Examples

Here is an example where we use this hook to create a new welcome post on the newly created site.

|     |     |
| --- | --- |
| 1<br>2<br>3<br>4<br>5<br>6<br>7<br>8<br>9<br>10<br>11<br>12<br>13<br>14 | `add_action(``'gform_site_created'``,``'create_welcome_post'``, 10, 5 );`<br>`function``create_welcome_post(``$site_id``,``$user_id``,``$entry``,``$feed``,``$password``) {`<br>```$user``= get_userdata(``$user_id``);`<br>```$post``=``array``();`<br>```$post``[``'post_title'``] =``'Welcome to your new site!'``;`<br>```$post``[``'post_content'``] =``"Hi {$user->first_name}, nWe just wanted to personally welcome you to your new site!"``;`<br>```switch_to_blog(``$site_id``);`<br>```wp_insert_post(``$post``);`<br>```restore_current_blog();`<br>`}` |

## Placement

This code should be placed in the [functions.php](http://codex.wordpress.org/Theme_Development#Functions_File) file of your active theme.
