# gform.utils

- [Introduction](https://docs.gravityforms.com/gform-utils/#h-introduction)
- [Overview](https://docs.gravityforms.com/gform-utils/#h-overview)

## Introduction

The `gform.utils` object in Gravity Forms provides a collection of utility functions and modules to assist developers in performing common tasks such as DOM manipulation, data processing, browser compatibility, etc.

**Note**: You can access comprehensive documentation for gform.utils at: [https://docs.js.gravity.com/js-utils/](https://docs.js.gravity.com/js-utils/)
